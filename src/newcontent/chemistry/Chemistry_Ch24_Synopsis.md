# IGCSE Chemistry Chapter 24: Alkanes

## Chapter Synopsis – Quick Revision Summary
**Pearson Edexcel IGCSE Chemistry**

## Overview

Alkanes are the simplest **homologous series** of hydrocarbons. They are **saturated** (single bonds only) with the general formula **CₙH₂ₙ₊₂**. Alkanes are important fuels derived from crude oil and undergo combustion reactions to release energy.

---

## Key Points at a Glance

### 1. General Formula

```
CₙH₂ₙ₊₂
```

- **n** = number of carbon atoms
- Each carbon forms **4 bonds**
- Each hydrogen forms **1 bond**

### 2. First Five Alkanes (MUST KNOW)

| Name | Formula | B.P. | State at RTP | Main Use |
|------|---------|------|--------------|----------|
| **Methane** | CH₄ | -162°C | Gas | Natural gas |
| **Ethane** | C₂H₆ | -89°C | Gas | Cracking feedstock |
| **Propane** | C₃H₈ | -42°C | Gas | LPG |
| **Butane** | C₄H₁₀ | -0.5°C | Gas | LPG, lighter fuel |
| **Pentane** | C₅H₁₂ | 36°C | Liquid | Petrol component |

### 3. Naming Pattern

| Carbon Atoms | Prefix | Name |
|--------------|--------|------|
| 1 | Meth- | Methane |
| 2 | Eth- | Ethane |
| 3 | Prop- | Propane |
| 4 | But- | Butane |
| 5 | Pent- | Pentane |

### 4. Properties Trends

As chain length **increases**:

| Property | Trend | Reason |
|----------|-------|--------|
| Boiling Point | ↑ Increases | Stronger van der Waals forces |
| Viscosity | ↑ Increases | Chains tangle more |
| Volatility | ↓ Decreases | Harder to vaporise |
| Flammability | ↓ Decreases | Less easily ignited |

### 5. Combustion

**Complete Combustion (excess O₂):**
```
Alkane + O₂ → CO₂ + H₂O  [Blue flame]
```

**Incomplete Combustion (limited O₂):**
```
Alkane + O₂ → CO + H₂O  [Yellow flame, toxic CO]
Alkane + O₂ → C + H₂O   [Sooty flame]
```

### 6. Key Combustion Equations

| Alkane | Complete Combustion Equation |
|--------|------------------------------|
| Methane | CH₄ + 2O₂ → CO₂ + 2H₂O |
| Ethane | 2C₂H₆ + 7O₂ → 4CO₂ + 6H₂O |
| Propane | C₃H₈ + 5O₂ → 3CO₂ + 4H₂O |
| Butane | 2C₄H₁₀ + 13O₂ → 8CO₂ + 10H₂O |

### 7. Uses Summary

| Alkane | Primary Uses |
|--------|--------------|
| Methane | Natural gas for heating/cooking |
| Propane | LPG for portable heating, camping |
| Butane | LPG, cigarette lighters, aerosol propellants |
| C₅-C₁₀ | Petrol for cars |
| C₁₀-C₁₆ | Kerosene for jet fuel |
| C₁₅-C₂₀ | Diesel for vehicles |

## Essential Definitions

| Term | Definition |
|------|------------|
| **Alkane** | Saturated hydrocarbon with formula CₙH₂ₙ₊₂ |
| **Saturated** | Contains only single bonds |
| **Homologous series** | Family with same general formula, similar properties |
| **Isomer** | Same formula, different structure |

---

## Key Facts

- Alkanes are **insoluble** in water
- Alkanes are **unreactive** (except combustion, halogenation with UV)
- **Pentane onwards** = liquids at room temperature
- Isomerism starts from **butane (C₄H₁₀)**
- CO from incomplete combustion is **toxic** (binds to haemoglobin)

---

## Exam Alert – Common Mistakes

| ❌ Wrong | ✅ Correct |
|---------|-----------|
| General formula is CₙH₂ₙ | CₙH₂ₙ₊₂ |
| Alkanes have double bonds | Only **single bonds** (saturated) |
| Higher BP = stronger bonds | Stronger **intermolecular forces** |
| Pentane is a gas at RTP | Pentane is a **liquid** at RTP |
| Halogenation needs no conditions | Requires **UV light** |
