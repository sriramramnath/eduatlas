## 1. Introduction to Alkanes

### 1.1 Definition

**Alkanes** are a **homologous series** of saturated hydrocarbons.

**Key characteristics:**
- Contain **only carbon and hydrogen atoms** (hydrocarbons)
- Have **only single covalent bonds** (saturated)
- Follow the **general formula CₙH₂ₙ₊₂**

### 1.2 What is a Homologous Series?

A **homologous series** is a family of organic compounds that:
- Have the **same general formula**
- Show a **gradual change in physical properties** (e.g., boiling point)
- Have **similar chemical properties**
- Differ by a **CH₂ group** between consecutive members

## 2. General Formula and Structure

### 2.1 General Formula

The general formula for alkanes is:

## **CₙH₂ₙ₊₂**

Where **n** = number of carbon atoms

### 2.2 Working Out Molecular Formulae

| n (carbon atoms) | Formula = CₙH₂ₙ₊₂ | Molecular Formula |
|------------------|---------------------|-------------------|
| 1 | C₁H₂(1)+₂ | CH₄ |
| 2 | C₂H₂(2)+₂ | C₂H₆ |
| 3 | C₃H₂(3)+₂ | C₈H₈ |
| 4 | C₄H₂(4)+₂ | C₄H₁₀ |
| 5 | C₅H₂(5)+₂ | C₅H₁₂ |
| 6 | C₆H₂(6)+₂ | C₆H₁₄ |
| 7 | C₇H₂(7)+₂ | C₇H₁₆ |
| 8 | C₈H₂(8)+₂ | C₈H₁₈ |

### 2.3 Structure of Alkanes

**Key structural features:**
- Each carbon atom forms **four single bonds**
- Each hydrogen atom forms **one single bond**
- Carbon atoms bond to other carbon atoms in a **chain**
- All bonds are **single covalent bonds** (sigma bonds)

**Displayed Formulae:**

**Methane (CH₄):**
```
       H
       |
   H — C — H
       |
       H
```

**Ethane (C₂H₆):**
```
   H   H
   |   |
 H—C — C—H
   |   |
   H   H
```

**Propane (C₃H₈):**
```
   H   H   H
   |   |   |
 H—C — C — C—H
   |   |   |
   H   H   H
```

**Butane (C₄H₁₀):**
```
   H   H   H   H
   |   |   |   |
 H—C — C — C — C—H
   |   |   |   |
   H   H   H   H
```

## 4. Physical Properties of Alkanes

### 4.1 Trend Summary

| Property | Trend with Increasing Chain Length |
|----------|-------------------------------------|
| **Boiling Point** | Increases |
| **Melting Point** | Increases |
| **Viscosity** | Increases |
| **Volatility** | Decreases |
| **Flammability** | Decreases |
| **Solubility in Water** | Insoluble (all alkanes) |
| **State at Room Temperature** | Gas → Liquid → Solid |

### 4.2 State at Room Temperature (25°C)

| Alkane | Boiling Point | State at 25°C |
|--------|---------------|---------------|
| Methane (CH₄) | -162°C | Gas |
| Ethane (C₂H₆) | -89°C | Gas |
| Propane (C₃H₈) | -42°C | Gas |
| Butane (C₄H₁₀) | -0.5°C | Gas |
| Pentane (C₅H₁₂) | 36°C | Liquid |
| Hexane (C₆H₁₄) | 69°C | Liquid |
| ...up to C₁₇ | | Liquid |
| C₁₈ and above | | Solid |

### 4.3 Boiling Point Trend Explained

**Why does boiling point increase with chain length?**

1. Longer alkane molecules have **more electrons**
2. This creates **stronger van der Waals forces** (intermolecular forces)
3. **More energy** is required to overcome these forces
4. Therefore, **higher boiling point**

**Important:** The increase in boiling point is due to **intermolecular forces**, NOT stronger covalent bonds.

### 4.4 Solubility in Water

- Alkanes are **insoluble in water**
- They are **non-polar molecules** (no charge separation)
- Water is a **polar solvent**
- "Like dissolves like" – non-polar alkanes do not dissolve in polar water
- Alkanes are soluble in **non-polar solvents** (like hexane, other alkanes)

### 4.5 Density

- Alkanes are **less dense than water**
- They **float on water**
- This is important for oil spill management

## 5. Combustion of Alkanes

### 5.1 Why Alkanes Burn

Alkanes are **good fuels** because:
- They release **large amounts of energy** when burned
- They are **readily available** from crude oil
- Short-chain alkanes are **easy to ignite**

### 5.2 Complete Combustion

**Conditions:** **Excess/plentiful oxygen**

**General Equation:**
```
Alkane + Oxygen → Carbon Dioxide + Water + Energy
```

**Balanced Equations:**

**Methane:**
```
CH₄ + 2O₂ → CO₂ + 2H₂O
```

**Ethane:**
```
2C₂H₆ + 7O₂ → 4CO₂ + 6H₂O
```

**Propane:**
```
C₃H₈ + 5O₂ → 3CO₂ + 4H₂O
```

**Butane:**
```
2C₄H₁₀ + 13O₂ → 8CO₂ + 10H₂O
```

**Characteristics of Complete Combustion:**
- **Blue flame**
- **Clean burning** (no smoke or soot)
- **Maximum energy released**
- Products are **CO₂ and H₂O only**

### 5.3 Balancing Combustion Equations

**Step-by-step method:**

For alkane CₓHᵧ:

1. Balance carbon: CₓHᵧ + ... O₂ → **x** CO₂ + ...
2. Balance hydrogen: CₓHᵧ + ... O₂ → x CO₂ + **(y/2)** H₂O
3. Count oxygen atoms needed: O needed = 2x + (y/2)
4. Balance oxygen: If O needed is odd, multiply all products by 2

**Example: Balance C₃H₈ + O₂ → CO₂ + H₂O**

1. C₃H₈ + ... O₂ → **3** CO₂ + ...
2. C₃H₈ + ... O₂ → 3 CO₂ + **4** H₂O
3. O atoms needed = (3 × 2) + 4 = 10
4. C₃H₈ + **5** O₂ → 3 CO₂ + 4 H₂O ✓

### 5.4 Incomplete Combustion

**Conditions:** **Limited/insufficient oxygen**

**Possible Products:**
- Carbon monoxide (CO) and water
- Carbon (soot/C) and water
- Both CO and C

**Equations:**

**Carbon monoxide production:**
```
2CH₄ + 3O₂ → 2CO + 4H₂O
```

**Carbon (soot) production:**
```
CH₄ + O₂ → C + 2H₂O
```

**Characteristics of Incomplete Combustion:**
- **Yellow/orange flame**
- **Sooty flame** (black smoke)
- **Less energy released**
- **Carbon monoxide is toxic**

### 5.5 Dangers of Incomplete Combustion

**Carbon Monoxide (CO):**
- Colourless and odourless – "silent killer"
- Binds to haemoglobin 200× more strongly than oxygen
- Prevents oxygen transport in blood
- Causes headaches, dizziness, unconsciousness, death
- Produced by faulty gas appliances, car engines

**Prevention:**
- Ensure adequate ventilation
- Regular servicing of gas appliances
- Install carbon monoxide detectors

## 7. Reactivity of Alkanes

### 7.1 General Reactivity

Alkanes are **relatively unreactive** under normal conditions because:
- They have **strong C-C and C-H single bonds**
- Bonds are **non-polar** (no charge separation)
- No areas of high electron density to attract reactants

### 7.2 Main Reactions

Despite being unreactive, alkanes undergo:
1. **Combustion** – reaction with oxygen
2. **Halogenation** – reaction with halogens (UV light required)
3. **Cracking** – breaking into smaller molecules

### 7.3 Reaction with Halogens (Substitution)

Alkanes react with halogens (Cl₂, Br₂) in the presence of **UV light**:

**Example – Methane with Chlorine:**
```
CH₄ + Cl₂ → CH₃Cl + HCl
    (UV light)
```

**Products:**
- Chloromethane (CH₃Cl)
- Hydrogen chloride (HCl)

This is a **substitution reaction** – one H atom is replaced by a Cl atom.

Further substitution can occur:
```
CH₃Cl + Cl₂ → CH₂Cl₂ + HCl
CH₂Cl₂ + Cl₂ → CHCl₃ + HCl
CHCl₃ + Cl₂ → CCl₄ + HCl
```

## 8. Environmental Impact

### 8.1 Fossil Fuel Concerns

Burning alkanes (fossil fuels) produces **carbon dioxide**:
- A **greenhouse gas**
- Contributes to **global warming** and **climate change**

### 8.2 Carbon Monoxide Pollution

- Produced by **incomplete combustion**
- Particularly from **vehicle engines**
- Dangerous to human health

### 8.3 Sustainable Alternatives

- **Renewable energy sources** (solar, wind, hydroelectric)
- **Biofuels** (biodiesel, bioethanol)
- **Hydrogen fuel cells**
- **Electric vehicles**

## 9. Isomerism in Alkanes

### 9.1 What are Isomers?

**Isomers** are compounds with the **same molecular formula** but **different structural arrangements**.

### 9.2 Structural Isomers

Isomerism starts from **butane (C₄H₁₀)** onwards.

**Butane (C₄H₁₀) has two isomers:**

1. **Butane** (straight chain):
```
CH₃—CH₂—CH₂—CH₃
```

2. **2-methylpropane** (branched):
```
    CH₃
     |
CH₃—CH—CH₃
```

**Pentane (C₅H₁₂) has three isomers:**

1. **Pentane** (straight chain)
2. **2-methylbutane** (branched)
3. **2,2-dimethylpropane** (highly branched)

## Common Exam Mistakes

### Mistake 1: Wrong General Formula
❌ **Wrong:** "CₙH₂ₙ"
✅ **Correct:** "CₙH₂ₙ₊₂" (alkanes have TWO extra hydrogens)

### Mistake 2: Confusing Alkanes and Alkenes
❌ **Wrong:** "Alkanes have double bonds"
✅ **Correct:** "Alkanes have ONLY single bonds (saturated)"

### Mistake 3: Incorrect Balancing of Combustion Equations
❌ **Wrong:** "CH₄ + O₂ → CO₂ + H₂O" (unbalanced)
✅ **Correct:** "CH₄ + 2O₂ → CO₂ + 2H₂O" (balanced)

### Mistake 4: Explaining Boiling Point Trend
❌ **Wrong:** "Higher boiling point because bonds are stronger"
✅ **Correct:** "Higher boiling point because there are stronger intermolecular forces (van der Waals forces)"

### Mistake 5: Naming Errors
❌ **Wrong:** "Methane = C₂H₆"
✅ **Correct:** "Methane = CH₄; Ethane = C₂H₆"

### Mistake 6: State at Room Temperature
❌ **Wrong:** "Pentane is a gas at room temperature"
✅ **Correct:** "Pentane (b.p. 36°C) is a liquid at room temperature (25°C)"

### Mistake 7: Incomplete Combustion Products
❌ **Wrong:** "Incomplete combustion always produces CO"
✅ **Correct:** "Incomplete combustion can produce CO and/or C (soot)"

### Mistake 8: Forgetting UV Light for Halogenation
❌ **Wrong:** "Alkanes react easily with chlorine"
✅ **Correct:** "Alkanes react with chlorine ONLY in the presence of UV light"

## Key Definitions Summary

| Term | Definition |
|------|------------|
| **Alkane** | A saturated hydrocarbon with single bonds only, general formula CₙH₂ₙ₊₂ |
| **Homologous series** | A family of compounds with the same general formula and similar chemical properties |
| **Saturated** | Containing only single bonds (cannot add more atoms) |
| **Hydrocarbon** | A compound containing only carbon and hydrogen |
| **Complete combustion** | Burning with excess oxygen, producing CO₂ and H₂O |
| **Incomplete combustion** | Burning with limited oxygen, producing CO and/or C |
| **Isomers** | Compounds with the same molecular formula but different structures |
| **Substitution reaction** | A reaction where one atom replaces another |
