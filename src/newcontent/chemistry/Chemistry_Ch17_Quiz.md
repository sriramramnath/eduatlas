## Part A: Multiple Choice Answers

| Question | Answer | Explanation |
|----------|--------|-------------|
| **Q1** | **A** | Metal + acid → salt + hydrogen. Zn + 2HCl → ZnCl₂ + H₂ |
| **Q2** | **B** | Sulfuric acid produces sulfates. CuO + H₂SO₄ → CuSO₄ + H₂O |
| **Q3** | **C** | Acid + carbonate → salt + water + carbon dioxide |
| **Q4** | **C** | Barium sulfate (BaSO₄) is an insoluble sulfate. All sodium, potassium, and ammonium salts are soluble. |
| **Q5** | **B** | H⁺(aq) + OH⁻(aq) → H₂O(l) is the ionic equation for neutralisation |
| **Q6** | **B** | Copper is below hydrogen in reactivity series, so use acid + insoluble base method |
| **Q7** | **B** | Effervescence (CO₂ gas) and solid dissolves (CaCO₃ + 2HCl → CaCl₂ + H₂O + CO₂) |
| **Q8** | **C** | Silver chloride (AgCl) is an insoluble chloride. Ag⁺ + Cl⁻ → AgCl(s) |
| **Q9** | **B** | Iron forms Fe²⁺ ions with acids (iron below hydrogen but not very reactive). Iron(II) chloride. |
| **Q10** | **B** | Excess base ensures all acid is used up, giving a neutral salt solution |
| **Q11** | **C** | BaCl₂ + Na₂SO₄ → BaSO₄(s) + 2NaCl. BaSO₄ is insoluble |
| **Q12** | **B** | Solubility of most solids decreases with cooling, causing crystallisation |

## Part B: True/False Answers

| Question | Answer | Correct Statement (if FALSE) |
|----------|--------|------------------------------|
| **Q1** | **FALSE** | Most chlorides are soluble, but **silver chloride (AgCl) and lead(II) chloride (PbCl₂)** are insoluble. |
| **Q2** | **FALSE** | Only metals **above hydrogen** in the reactivity series react with acids to produce hydrogen. Copper, silver, and gold do not. |
| **Q3** | **FALSE** | Copper does NOT react with dilute hydrochloric acid as it is **below hydrogen** in the reactivity series. |
| **Q4** | **TRUE** | Salts from nitric acid are named as nitrates (e.g., sodium nitrate). |
| **Q5** | **TRUE** | This is the definition of a precipitation reaction. |
| **Q6** | **FALSE** | Crystallisation involves concentrating the solution and allowing it to cool. Evaporating to dryness gives a powder, not crystals. |
| **Q7** | **FALSE** | **Sodium carbonate is soluble** - all sodium compounds are soluble. |
| **Q8** | **TRUE** | This is the correct ionic equation for acid + carbonate reactions. |
| **Q9** | **TRUE** | Filtration removes unreacted solid base before crystallisation. |
| **Q10** | **FALSE** | Barium sulfate is used because it is **insoluble** and will not be absorbed by the body. |
| **Q11** | **TRUE** | Iron can form Fe²⁺ (iron(II)) or Fe³⁺ (iron(III)) ions, giving different salts. |
| **Q12** | **TRUE** | Salts from ethanoic acid are named as ethanoates (e.g., sodium ethanoate). |

## Part C: Short Answer Mark Schemes

### Q1. Balanced Equations (3 marks)

**(a) Magnesium oxide with hydrochloric acid (1 mark):**
```
MgO + 2HCl → MgCl₂ + H₂O
```
*(Award mark for correct products and balanced equation)*

**(b) Sodium carbonate with sulfuric acid (1 mark):**
```
Na₂CO₃ + H₂SO₄ → Na₂SO₄ + H₂O + CO₂
```
*(All three products required)*

**(c) Zinc with nitric acid (1 mark):**
```
Zn + 2HNO₃ → Zn(NO₃)₂ + H₂
```
*(Note: zinc nitrate formula must be correct)*

---

### Q2. Preparing Copper(II) Sulfate (3 marks)

**Balanced equation (1 mark):**
```
CuO(s) + H₂SO₄(aq) → CuSO₄(aq) + H₂O(l)
```

**Method points (2 marks total):**

*Award 1 mark for each of the following (max 2 marks):*
- Add copper(II) oxide to warm sulfuric acid **in excess** until no more reacts
- Filter to remove excess copper(II) oxide
- Heat the filtrate to concentrate / evaporate some water
- Allow to cool and crystallise
- Filter crystals, wash with distilled water and dry

*Minimum expected: mention excess base, filtration, and crystallisation*

---

### Q3. Precipitation (3 marks)

**(a) Definition of precipitate (1 mark):**
- An **insoluble solid** formed when two solutions react together
- OR: A solid that forms from mixing two solutions

**(b) Ionic equation (1 mark):**
```
Ag⁺(aq) + Cl⁻(aq) → AgCl(s)
```
*(Must show correct state symbols and no spectator ions)*

**(c) Observation (1 mark):**
- A **white precipitate** forms
- OR: White solid appears

---

### Q4. Solubility Rules (3 marks)

**(a) One insoluble sulfate (1 mark):**
- **Barium sulfate / BaSO₄** OR **Lead(II) sulfate / PbSO₄** OR **Calcium sulfate / CaSO₄** (slightly soluble)
*(Accept any correct insoluble sulfate)*

**(b) One soluble hydroxide (1 mark):**
- **Sodium hydroxide / NaOH** OR **Potassium hydroxide / KOH** OR **Ammonium hydroxide / NH₄OH**
*(All Na, K, NH₄ compounds are soluble)*

**(c) Explanation (1 mark):**
- Sodium compounds are **always soluble** / all sodium salts are soluble
- Calcium carbonate is a **carbonate of a metal other than Na, K, or NH₄** / most carbonates are insoluble
*(Need both parts for full mark)*
