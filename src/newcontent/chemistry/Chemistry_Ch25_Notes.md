## 25.1 Introduction to Alkenes

### Definition
**Alkenes** are a homologous series of **unsaturated hydrocarbons** that contain at least one carbon-carbon double bond (C=C). They are more reactive than alkanes due to the presence of this double bond.

### Key Characteristics
- **Unsaturated**: Contain a C=C double bond (not all carbon atoms have the maximum number of hydrogen atoms)
- **General Formula**: CnH2n (where n is the number of carbon atoms, n ≥ 2)
- **Functional Group**: Carbon-carbon double bond (C=C)
- **Homologous Series**: Each member differs by CH2 from the next

---

## 25.2 General Formula and Structure of Alkenes

### General Formula
**CnH2n**

| Number of Carbons (n) | Molecular Formula | Name |
|-----------------------|-------------------|------|
| 2 | C2H4 | Ethene |
| 3 | C3H6 | Propene |
| 4 | C4H8 | Butene |
| 5 | C5H10 | Pentene |
| 6 | C6H12 | Hexene |

### The Carbon-Carbon Double Bond

#### Structure of the C=C Bond
- The double bond consists of **two pairs of shared electrons** between two carbon atoms
- Each carbon in the double bond is bonded to **two other atoms** (not four like in alkanes)
- The double bond is **reactive** because:
  1. It contains a high electron density
  2. The bond is weaker than a C-C single bond
  3. It can "open up" to allow other atoms to add on

#### Electron Dot and Cross Diagram
```
      H   H
       \ /
    H - C = C - H     (Ethene)
       / \
      H   H
```

#### Displayed Formula Rules
- Each carbon in the C=C double bond has **two other bonds**
- These other bonds are to hydrogen atoms or other carbon atoms
- The double bond is represented by **two lines** (=)

### Why n ≥ 2?
The smallest alkene must have **at least 2 carbon atoms** to form a C=C double bond. There is no "methene" because a single carbon cannot form a double bond.

## 25.3 Naming Alkenes

### Naming Rules (IUPAC System)

1. **Count the carbon atoms** in the longest chain containing the C=C bond
2. **Use the appropriate prefix** (same as alkanes):
   - 2 carbons: **eth-**
   - 3 carbons: **prop-**
   - 4 carbons: **but-**
   - 5 carbons: **pent-**
   - 6 carbons: **hex-**

3. **Add the suffix "-ene"** to indicate an alkene

4. **For alkenes with 4+ carbons**: Add a number to show the position of the double bond
   - Number from the end that gives the **lowest number** for the double bond

### The First Three Alkenes

#### Ethene (C2H4)
```
    H   H
     \ /
      C = C
     / \
    H   H
```
- Simplest alkene
- No positional isomerism possible
- Important industrial chemical

#### Propene (C3H6)
```
    H   H   H
     \ /    |
      C = C - C - H
     / \        |
    H   H       H
```
- Double bond must be between C1 and C2
- No positional isomerism possible

#### Butene (C4H8)
Two positional isomers:

**But-1-ene** (double bond between C1 and C2):
```
    H   H   H   H
     \ /    |   |
      C = C - C - C - H
     / \        |   |
    H   H       H   H
```

**But-2-ene** (double bond between C2 and C3):
```
    H   H   H   H
    |    \ /    |
H - C - C = C - C - H
    |       \ /
    H       H   H
```

### Important Naming Notes
- The number in the name indicates the **first carbon of the double bond**
- Always number from the end giving the **lowest number** for the double bond
- For example: But-2-ene, not but-3-ene (numbering from the other end would give 3)

## 25.4 Addition Reactions of Alkenes

### What is an Addition Reaction?

**Addition reaction**: A reaction where two molecules combine to form one larger molecule. The C=C double bond "opens up" to allow atoms to add on.

**Key Point**: Alkenes undergo addition reactions because the double bond is reactive. The product is always **saturated**.

### Types of Addition Reactions

---

#### 1. Addition of Hydrogen (Hydrogenation)

**General Equation:**
```
CnH2n + H2 → CnH2n+2
Alkene + Hydrogen → Alkane
```

**Specific Example:**
```
    H   H                     H   H
     \ /                      |   |
      C = C      +  H2  →     C - C
     / \                      |   |
    H   H                     H   H
   Ethene                     Ethane
```

**Conditions:**
- **Catalyst**: Nickel (Ni) or Platinum (Pt)
- **Temperature**: 150°C (with nickel catalyst)
- **Pressure**: Often carried out at elevated pressure

**Industrial Application:**
- **Hardening of vegetable oils** to make margarine
- Unsaturated vegetable oils (containing C=C bonds) are hydrogenated to produce semi-solid fats
- This increases the melting point, making the fat solid at room temperature

**Word Equation:**
Vegetable oil + Hydrogen → Margarine (nickel catalyst, 150°C)

---

#### 2. Addition of Water/Steam (Hydration)

**General Equation:**
```
CnH2n + H2O → CnH2n+1OH
Alkene + Steam → Alcohol
```

**Specific Example:**
```
    H   H                     H   H
     \ /                      |   |
      C = C      +  H2O  →    C - C - OH
     / \                      |   |
    H   H                     H   H
   Ethene                     Ethanol
```

**Conditions:**
- **Catalyst**: Phosphoric acid (H3PO4) on silica support
- **Temperature**: 300°C
- **Pressure**: 60-70 atmospheres

**Important Notes:**
- This is an **industrial method** for producing ethanol
- The product is an **alcohol** (hydroxyl group -OH added)
- With propene, the OH group mainly adds to the **middle carbon** (Markovnikov's rule - but this is not required for IGCSE)

**Word Equation:**
Ethene + Steam ⇌ Ethanol (phosphoric acid catalyst, 300°C, 60-70 atm)

---

#### 3. Addition of Halogens (Halogenation)

**General Equation:**
```
CnH2n + X2 → CnH2nX2
Alkene + Halogen → Dihaloalkane
```

**Specific Example (with Bromine):**
```
    H   H                     H   H
     \ /                      |   |
      C = C      +  Br2  →    C - C
     / \                      |   |
    H   H                    Br   Br
   Ethene                    1,2-dibromoethane
```

**Conditions:**
- Room temperature
- No catalyst required
- Can use bromine (Br2) or chlorine (Cl2)

**Observation with Bromine Water:**
- **Bromine water** is orange/brown in color
- When added to an alkene, it becomes **colorless**
- This is the **test for unsaturation** (see section 25.5)

**Reaction with Chlorine:**
```
Ethene + Cl2 → 1,2-dichloroethane
```

## 25.5 Testing for Unsaturation (Bromine Water Test)

### The Bromine Water Test

This is the standard test to distinguish between **saturated** and **unsaturated** hydrocarbons.

### How to Perform the Test

1. Add **bromine water** (orange/brown solution) to the hydrocarbon
2. **Shake** the mixture
3. Observe the color change

### Results and Interpretation

| Hydrocarbon Type | Observation | Explanation |
|------------------|-------------|-------------|
| **Alkene (unsaturated)** | Orange/brown → Colorless | Bromine adds across the C=C double bond |
| **Alkane (saturated)** | Stays orange/brown | No reaction; no double bond to react |

### Chemical Equation
```
C2H4 + Br2 → C2H4Br2
Ethene + Bromine → 1,2-Dibromoethane
(Orange)          (Colorless)
```

### Why Does the Color Change?
- Bromine water contains Br2 molecules (orange/brown)
- When Br2 reacts with the alkene, it becomes incorporated into the product
- The product (dibromoalkane) is **colorless**
- Therefore, the solution loses its orange/brown color

### Common Exam Mistakes

❌ **Wrong**: "The solution turns clear"
✅ **Correct**: "The solution turns from orange/brown to colorless"

❌ **Wrong**: "The bromine decolorizes because it reacts with hydrogen"
✅ **Correct**: "The bromine decolorizes because it adds across the C=C double bond"

## 25.6 Uses of Alkenes

### Industrial Importance

Alkenes are extremely important in the chemical industry as they are used to make many useful products.

### 1. Ethene (C2H4)

| Use | Product | Notes |
|-----|---------|-------|
| Making ethanol | Ethanol (C2H5OH) | Hydration reaction with steam |
| Making poly(ethene) | Polythene plastic | Polymerization reaction |
| Making ethylene glycol | Antifreeze | Used in car radiators |
| Making PVC precursors | Chloroethene | For making PVC plastic |

### 2. Propene (C3H6)

| Use | Product | Notes |
|-----|---------|-------|
| Making poly(propene) | Polypropylene plastic | Used for ropes, crates, car parts |
| Making propanone | Acetone | Solvent |
| Making propylene glycol | Antifreeze, food additive | Safe for food use |

### 3. Making Polymers (Polymerization)

Alkenes undergo **addition polymerization** to form plastics:

```
n Ethene → Poly(ethene)
n C2H4 → (-CH2-CH2-)n
```

**Common Polymers from Alkenes:**
- Poly(ethene) from ethene - plastic bags, bottles
- Poly(propene) from propene - ropes, crates
- Poly(chloroethene)/PVC from chloroethene - pipes, window frames

### 4. Other Uses

- **Starting materials** for making alcohols, aldehydes, and other organic compounds
- **Ripening fruit** - ethene is used commercially to ripen fruit
- **Manufacture of detergents** and other chemicals

## 25.7 Comparing Alkenes and Alkanes

| Property | Alkenes | Alkanes |
|----------|---------|---------|
| **General Formula** | CnH2n | CnH2n+2 |
| **Type** | Unsaturated | Saturated |
| **Bond Type** | Contains C=C double bond | Only C-C single bonds |
| **Reactivity** | More reactive | Less reactive |
| **Main Reaction Type** | Addition reactions | Substitution reactions |
| **Bromine Water Test** | Decolorizes (orange → colorless) | No change (stays orange) |
| **Combustion** | Burns with smoky flame | Burns with clean flame |

## 25.8 Key Definitions Summary

| Term | Definition |
|------|------------|
| **Alkene** | An unsaturated hydrocarbon containing a carbon-carbon double bond |
| **Unsaturated** | A compound containing a carbon-carbon double bond (can add more atoms) |
| **Saturated** | A compound containing only single bonds (cannot add more atoms) |
| **Addition Reaction** | A reaction where two molecules combine to form one larger molecule |
| **Hydrogenation** | Addition of hydrogen to an unsaturated compound |
| **Hydration** | Addition of water to a compound |
| **Halogenation** | Addition of a halogen to a compound |
| **Functional Group** | The part of a molecule responsible for its characteristic reactions |

## 25.9 Common Exam Mistakes

### 1. Formula Errors
❌ Writing CnH2n+2 instead of CnH2n for alkenes
✅ Remember: Alkenes have **2 fewer** hydrogens than alkanes

### 2. Naming Errors
❌ Writing "but-3-ene" instead of "but-1-ene"
✅ Always number from the end giving the **lowest number** for the double bond

### 3. Bromine Water Test
❌ Writing "turns clear" or "becomes transparent"
✅ Write "**turns from orange/brown to colorless**"

### 4. Addition Reaction Products
❌ Forgetting that addition reactions produce **one product only**
✅ The C=C bond opens up; two molecules become one

### 5. Conditions
❌ Forgetting to state the catalyst for hydration
✅ **Phosphoric acid catalyst**, 300°C, 60-70 atm

### 6. Hydrogenation
❌ Confusing hydrogenation conditions
✅ **Nickel catalyst**, 150°C

### 7. Drawing Structures
❌ Drawing carbon atoms with 5 bonds around the C=C
✅ Each carbon in the double bond has **two other bonds** (total 4 bonds per carbon)

## Key Takeaways

✓ Alkenes have the general formula **CnH2n** and contain a **C=C double bond**

✓ Alkenes are **unsaturated** and undergo **addition reactions**

✓ Addition reactions include: hydrogenation (H2), hydration (H2O), and halogenation (Br2, Cl2)

✓ The **bromine water test** distinguishes alkenes from alkanes (orange → colorless)

✓ Alkenes are important for making **plastics, alcohols, and other chemicals**

✓ Always remember the correct **conditions** for each reaction
