## рҹ“Ң Essential Definitions

| Term | Definition |
|------|------------|
| **Acid** | Proton donor - releases HвҒә ions in water |
| **Base** | Proton acceptor - accepts HвҒә ions |
| **Alkali** | Soluble base that produces OHвҒ» ions in solution |

- **pH < 7**: Acidic

## вҡЎ Strong vs Weak Acids

| Property | Strong Acid | Weak Acid |
|----------|-------------|-----------|
| **Dissociation** | Complete (вҶ’) | Partial (вҮҢ) |
| **Examples** | HCl, HвӮӮSOвӮ„, HNOвӮғ | CHвӮғCOOH, citric acid |

**Key distinction:** 
- **Strong/Weak** = degree of dissociation

## рҹ§Ә Titration Essentials

### The Setup
```
      Burette (acid of known conc.)
           вҶ“

  в”Ӯ  Conical flask  в”Ӯ  + indicator
  в”Ӯ  (alkali of     в”Ӯ
  в”Ӯ  unknown conc.) в”Ӯ
  в””в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”ҳ
       White tile
```

### Key Steps
1. Rinse burette with solution to be used
2. Fill burette, record initial reading
3. Pipette known volume of unknown into flask
4. Add indicator (2-3 drops)
5. Add from burette slowly, swirling
6. Add dropwise near endpoint
7. Record volume at colour change
8. Repeat for concordant results (Вұ0.1 cmВі)

## рҹ“җ Titration Calculation Formula

**Method 1: Ratio method**
```
CвӮҒVвӮҒ    CвӮӮVвӮӮ
в”Җв”Җв”Җв”Җ = в”Җв”Җв”Җв”Җ
 nвӮҒ      nвӮӮ
```
Where n = moles ratio from equation

**Method 2: Step-by-step**
1. Moles of known = C Г— V(dmВі)
2. Use ratio to find moles of unknown
3. Concentration = moles Г· V(dmВі)

**Remember:** Convert cmВі вҶ’ dmВі (Г·1000)

## рҹ“қ Worked Example

**Problem:** 25.0 cmВі NaOH needs 20.0 cmВі of 0.150 mol/dmВі HCl

**Equation:** NaOH + HCl вҶ’ NaCl + HвӮӮO (1:1 ratio)

**Solution:**
```
1. Moles HCl = 0.150 Г— (20.0/1000) = 0.00300 mol
2. Moles NaOH = 0.00300 mol (1:1 ratio)
3. [NaOH] = 0.00300 Г· (25.0/1000) = 0.120 mol/dmВі
```

## вҡ пёҸ Common Exam Pitfalls

1. **Alkali вү  Base** - All alkalis are bases, not vice versa
2. **Don't forget units** - Convert cmВі to dmВі in calculations
3. **Use the ratio** - Write balanced equation, use stoichiometry
4. **Strong вү  Concentrated** - Different concepts!
5. **Indicator choice** - Learn which indicator for which titration
6. **Endpoint precision** - Dropwise addition, white tile, concordant results

## рҹ“Ӣ Essential Equations

1. **Neutralisation:** HвҒә(aq) + OHвҒ»(aq) вҶ’ HвӮӮO(l)

2. **Strong acid:** HCl вҶ’ HвҒә + ClвҒ» (complete)

3. **Weak acid:** CHвӮғCOOH вҮҢ HвҒә + CHвӮғCOOвҒ» (partial)
