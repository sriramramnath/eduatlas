### Question 1 (3 marks)
**Define "addition polymerisation" and state two essential features of the monomers.**

**Mark Scheme:**
- Definition: A reaction in which unsaturated monomers join together without loss of any atoms to form a polymer (1 mark)
- Feature 1: Must contain C=C double bond (1 mark)
- Feature 2: Can join together when double bond opens (1 mark)

### Question 6 (3 marks)
**Difference between addition and condensation polymerisation**

**Mark Scheme:**
- Addition: Monomers join without loss of any atoms / no by-products (1 mark)
- Condensation: Monomers join with loss of small molecule (H₂O or HCl) (1 mark)
- Example addition: Poly(ethene), poly(propene), PVC, PTFE (0.5 marks)
- Example condensation: Polyester, Terylene/PET (0.5 marks)

## Exam Tips for Chapter 29

### Drawing Polymer Structures
1. Always show the repeating unit in square brackets
2. Add subscript 'n' outside the brackets
3. Remember the double bond OPENS - no C=C in polymer
4. Include all side groups correctly

### Key Equations to Remember
```
Ethene → Poly(ethene):     n CH₂=CH₂ → -[-CH₂-CH₂-]-n-
Propene → Poly(propene):   n CH₂=CHCH₃ → -[-CH₂-CH(CH₃)-]-n-
Chloroethene → PVC:        n CH₂=CHCl → -[-CH₂-CHCl-]-n-
```
