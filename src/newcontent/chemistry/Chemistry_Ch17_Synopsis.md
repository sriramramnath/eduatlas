## ًں”¬ Required Practical: Making Copper(II) Sulfate

### The Method

```
Step 1: REACTION
CuO + Hâ‚‚SOâ‚„ â†’ CuSOâ‚„ + Hâ‚‚O
(Warm acid, add excess CuO)

Step 2: FILTRATION
Remove unreacted CuO

Step 3: CRYSTALLISATION
Concentrate, cool, crystallise

Step 4: COLLECTION
Filter, wash, dry crystals
```

### Key Points

| Step | Why? |
|------|------|
| Warm the acid | Faster reaction |
| Add EXCESS base | Ensures all acid used up |
| Filter | Remove unreacted base |
| Don't evaporate to dryness | Gives crystals, not powder |
| Cool slowly | Larger, purer crystals |
| Wash with cold water | Remove impurities |

##  Common Exam Pitfalls

1. **Wrong salt name** - Check which acid was used
2. **Missing Roman numerals** - Iron(II), Iron(III), Copper(II)
3. **Wrong method** - Use acid+base for unreactive metals
4. **Evaporating to dryness** - Gives powder, not crystals
5. **Forgetting excess base** - Need excess to use all acid
6. **Spectator ions in ionic equations** - Omit them!
7. **State symbols** - Include (s), (l), (g), (aq)
