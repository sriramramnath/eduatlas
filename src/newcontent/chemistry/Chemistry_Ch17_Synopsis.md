## 📌 Essential Definitions

| Term | Definition |
|------|------------|
| **Salt** | Ionic compound where H⁺ in acid is replaced by metal ion |
| **Neutralisation** | Acid + Base → Salt + Water |
| **Precipitate** | Insoluble solid formed from mixing solutions |
| **Precipitation** | Reaction forming an insoluble solid |
| **Crystallisation** | Forming pure crystals from solution |

---

## 🧪 Reactions of Acids Summary

| Reaction Type | General Equation | Observations |
|---------------|------------------|--------------|
| **Acid + Metal** | Acid + Metal → Salt + H₂ | Effervescence, metal dissolves |
| **Acid + Base** | Acid + Base → Salt + H₂O | Solid dissolves, no gas |
| **Acid + Carbonate** | Acid + Carbonate → Salt + H₂O + CO₂ | Effervescence, solid dissolves |

## 📋 Salt Naming Rules

| Acid Used | Salt Ending | Example |
|-----------|-------------|---------|
| Hydrochloric acid (HCl) | **-chloride** | Sodium chloride |
| Sulfuric acid (H₂SO₄) | **-sulfate** | Copper(II) sulfate |
| Nitric acid (HNO₃) | **-nitrate** | Calcium nitrate |
| Ethanoic acid (CH₃COOH) | **-ethanoate** | Sodium ethanoate |

**First part = Metal name (with Roman numeral if needed)**

## 🔧 Methods for Making Salts

### Making SOLUBLE Salts

| Method | Used For | Example |
|--------|----------|---------|
| **Acid + Metal** | Reactive metals (Mg, Zn, Fe) | Mg + 2HCl → MgCl₂ + H₂ |
| **Acid + Insoluble Base** | Unreactive metals (Cu, Pb) | CuO + H₂SO₄ → CuSO₄ + H₂O |
| **Acid + Alkali (Titration)** | Soluble reactants (Na, K, NH₄⁺) | NaOH + HCl → NaCl + H₂O |

### Making INSOLUBLE Salts

**Method: Precipitation**
- Mix two soluble salt solutions
- Insoluble salt precipitates out
- Filter, wash, dry

```
BaCl₂(aq) + Na₂SO₄(aq) → BaSO₄(s) + 2NaCl(aq)
Ba²⁺(aq) + SO₄²⁻(aq) → BaSO₄(s)
```

## 📊 Solubility Rules (Essential!)

| Compound | Rule |
|----------|------|
| **All sodium compounds** | ✓ Soluble |
| **All potassium compounds** | ✓ Soluble |
| **All ammonium compounds** | ✓ Soluble |
| **All nitrates** | ✓ Soluble |
| **Most chlorides** | ✓ Soluble (except AgCl, PbCl₂) |
| **Most sulfates** | ✓ Soluble (except BaSO₄, PbSO₄, CaSO₄) |
| **Most carbonates** | ✗ Insoluble (except Na, K, NH₄⁺) |
| **Most hydroxides** | ✗ Insoluble (except Na, K, NH₄⁺) |

## 🔬 Required Practical: Making Copper(II) Sulfate

### The Method

```
Step 1: REACTION
CuO + H₂SO₄ → CuSO₄ + H₂O
(Warm acid, add excess CuO)

Step 2: FILTRATION
Remove unreacted CuO

Step 3: CRYSTALLISATION
Concentrate, cool, crystallise

Step 4: COLLECTION
Filter, wash, dry crystals
```

### Key Points

| Step | Why? |
|------|------|
| Warm the acid | Faster reaction |
| Add EXCESS base | Ensures all acid used up |
| Filter | Remove unreacted base |
| Don't evaporate to dryness | Gives crystals, not powder |
| Cool slowly | Larger, purer crystals |
| Wash with cold water | Remove impurities |

## ⚡ Key Equations to Learn

### Acid + Metal
```
Mg + 2HCl → MgCl₂ + H₂
Zn + H₂SO₄ → ZnSO₄ + H₂
Fe + 2HCl → FeCl₂ + H₂
```

### Acid + Metal Oxide
```
CuO + H₂SO₄ → CuSO₄ + H₂O
MgO + 2HNO₃ → Mg(NO₃)₂ + H₂O
```

### Acid + Metal Hydroxide
```
NaOH + HCl → NaCl + H₂O
Ca(OH)₂ + 2HNO₃ → Ca(NO₃)₂ + 2H₂O
```

### Acid + Carbonate
```
CaCO₃ + 2HCl → CaCl₂ + H₂O + CO₂
Na₂CO₃ + H₂SO₄ → Na₂SO₄ + H₂O + CO₂
```

### Ionic Equations
```
Neutralisation:    H⁺ + OH⁻ → H₂O
Carbonate reaction: CO₃²⁻ + 2H⁺ → H₂O + CO₂
Precipitation:     Ba²⁺ + SO₄²⁻ → BaSO₄
                   Ag⁺ + Cl⁻ → AgCl
```

## ⚠️ Common Exam Pitfalls

1. **Wrong salt name** - Check which acid was used
2. **Missing Roman numerals** - Iron(II), Iron(III), Copper(II)
3. **Wrong method** - Use acid+base for unreactive metals
4. **Evaporating to dryness** - Gives powder, not crystals
5. **Forgetting excess base** - Need excess to use all acid
6. **Spectator ions in ionic equations** - Omit them!
7. **State symbols** - Include (s), (l), (g), (aq)

## 🎯 Quick Method Selection Guide

**Question:** How do I make salt X?

| Is the salt SOLUBLE? | Method |
|---------------------|--------|
| **YES** - and metal is reactive (Mg, Zn, Fe) | Acid + Metal |
| **YES** - and metal is unreactive (Cu, Pb) | Acid + Insoluble Base |
| **YES** - and contains Na⁺, K⁺, NH₄⁺ | Titration (Acid + Alkali) |
| **NO** (insoluble) | Precipitation |

## 💡 Memory Tips

- **"NAK" = **N**itrates, **A**mmonium, **K** (potassium) and **N**a (sodium) - all soluble
- **Chlorides** - remember **Ag** and **Pb** are exceptions
- **Sulfates** - remember **Ba** is insoluble (barium meal uses BaSO₄)
- **Carbonates** - most are insoluble (hence limestone doesn't dissolve easily)
- **Precipitation = "Meeting of ions"** - when two solutions meet, insoluble product forms
