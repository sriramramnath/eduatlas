## 📌 Essential Definitions

| Term | Definition |
|------|------------|
| **Flame test** | Identifies metal ions by their characteristic flame colours |
| **Precipitate** | Insoluble solid formed when two solutions react |
| **Limewater** | Calcium hydroxide solution (tests for CO₂) |
| **Effervescence** | Bubbling due to gas being released |
| **Amphoteric** | Reacts with both acids and bases (e.g., Al(OH)₃, Zn(OH)₂) |

## 🧪 Gas Tests Summary

| Gas | Test | Positive Result | Key Point |
|-----|------|-----------------|-----------|
| **Oxygen (O₂)** | Glowing splint | **Relights** | Not a lighted splint! |
| **Hydrogen (H₂)** | Lighted splint | **"Pop" sound** | Small explosion |
| **Carbon dioxide (CO₂)** | Limewater | **Turns milky** | CaCO₃ forms |
| **Ammonia (NH₃)** | Damp red litmus | **Turns blue** | Must be DAMP |
| **Chlorine (Cl₂)** | Damp blue litmus | **Red → White** (bleaches) | Acidic then bleaches |

## 🔥 Flame Tests

### Method
1. Clean wire loop in HCl, heat in flame (no colour)
2. Dip in sample, hold in flame
3. Observe colour

### Colours to Memorise

| Metal Ion | Flame Colour | Memory Tip |
|-----------|--------------|------------|
| **Lithium (Li⁺)** | Crimson red | "Li" = Light crimson |
| **Sodium (Na⁺)** | Yellow | Street lamps are sodium (yellow) |
| **Potassium (K⁺)** | Lilac | "K" = Kool lilac |
| **Calcium (Ca²⁺)** | Orange-red / Brick red | Like a red brick |
| **Copper (Cu²⁺)** | Blue-green | Copper pipes are blue-green when oxidised |

### Important Notes
- **Sodium masks potassium** - view through blue cobalt glass
- Learn EXACT colour words: crimson, lilac, orange-red, blue-green

## 🧬 Cation Tests with NaOH

### Summary Table

| Cation | Ppt Colour | Soluble in excess NaOH? |
|--------|------------|------------------------|
| **Cu²⁺** | Blue | ❌ No |
| **Fe²⁺** | Green | ❌ No |
| **Fe³⁺** | Brown | ❌ No |
| **Al³⁺** | White | ✅ Yes |
| **Zn²⁺** | White | ✅ Yes |
| **Ca²⁺** | White | ❌ No |

### Key Equations
```
Cu²⁺ + 2OH⁻ → Cu(OH)₂(s) [blue]
Fe²⁺ + 2OH⁻ → Fe(OH)₂(s) [green]
Fe³⁺ + 3OH⁻ → Fe(OH)₃(s) [brown]
Al³⁺ + 3OH⁻ → Al(OH)₃(s) [white, dissolves in excess]
Zn²⁺ + 2OH⁻ → Zn(OH)₂(s) [white, dissolves in excess]
Ca²⁺ + 2OH⁻ → Ca(OH)₂(s) [white, insoluble in excess]
```

## 🔍 Distinguishing White Precipitates (Al³⁺, Zn²⁺, Ca²⁺)

| Test | Al³⁺ | Zn²⁺ | Ca²⁺ |
|------|------|------|------|
| **Add NaOH** | White ppt | White ppt | White ppt |
| **Excess NaOH** | ✅ Dissolves | ✅ Dissolves | ❌ Insoluble |
| **Excess NH₃** | ❌ Insoluble | ✅ Dissolves | N/A |

**Key:** Use ammonia to distinguish Al³⁺ from Zn²⁺

## 🧪 Anion Tests

### Test for Carbonate (CO₃²⁻)
```
Test:  Add dilute HCl
Result: Effervescence → CO₂ produced
Confirm: Gas turns limewater milky

CO₃²⁻ + 2H⁺ → H₂O + CO₂
```

### Test for Sulfate (SO₄²⁻)
```
Test:  Add HCl (removes carbonate), then BaCl₂
Result: White precipitate

Ba²⁺ + SO₄²⁻ → BaSO₄(s)
```

### Test for Halides (Cl⁻, Br⁻, I⁻)
```
Test:  Add HNO₃ (removes carbonate/hydroxide), then AgNO₃
```

| Halide | Precipitate | Colour |
|--------|-------------|--------|
| **Cl⁻** | AgCl | White |
| **Br⁻** | AgBr | Cream |
| **I⁻** | AgI | Yellow |

## 📊 Instrumental Analysis: Flame Emission Spectroscopy

### Principle
1. Sample sprayed into flame
2. Metal atoms excited by heat
3. Electrons emit light returning to ground state
4. Wavelength identifies element
5. Intensity measures concentration

### Advantages
| Advantage | Description |
|-----------|-------------|
| **Rapid** | Results in minutes |
| **Accurate** | Objective measurements |
| **Sensitive** | Detects trace amounts |
| **Quantitative** | Measures concentration |
| **Multi-element** | Analyses several metals at once |

### Flame Test vs FES
| Feature | Flame Test | FES |
|---------|------------|-----|
| Speed | Very fast | Fast |
| Accuracy | Subjective | Objective |
| Sensitivity | Low | High |
| Quantitative | No | Yes |
| Cost | Low | High |

##  Common Exam Pitfalls

1. **O₂ vs H₂ test** - O₂ uses GLOWING splint; H₂ uses LIGHTED splint
2. **Flame colours** - Learn exact terms: crimson, lilac, orange-red, blue-green
3. **"Excess NaOH"** - Only Al³⁺ and Zn²⁺ dissolve; Ca²⁺ doesn't
4. **Add acid first** - HCl before BaCl₂; HNO₃ before AgNO₃
5. **Silver halide colours** - White → Cream → Yellow (Cl → Br → I)
6. **Damp litmus for Cl₂** - Must be damp, not dry
7. **Al³⁺ vs Zn²⁺** - Use NH₃ test to distinguish
8. **State symbols** - Include (s), (aq), (g) in equations

## 🎯 Quick Decision Guide

### Identifying a Gas
1. Colourless, no smell → Test for O₂, H₂, CO₂
2. Pungent smell → Test for NH₃ (red litmus)
3. Choking smell, green-yellow → Cl₂ (blue litmus)

### Identifying a Cation in Solution
1. Note solution colour (blue = Cu²⁺)
2. Add NaOH dropwise → note precipitate colour
3. Add excess NaOH → does precipitate dissolve?
4. If white ppt dissolves, use NH₃ to distinguish Al³⁺ from Zn²⁺

### Identifying an Anion
1. Add HCl → fizzing? = CO₃²⁻
2. Add HCl then BaCl₂ → white ppt? = SO₄²⁻
3. Add HNO₃ then AgNO₃ → ppt colour identifies halide

##  Memory Tips

### Gas Tests
- **O₂ = Oxygen brings Old embers back to life** (relights glowing splint)
- **H₂ = Hydrogen goes Pop with a lighted splint**
- **CO₂ = Cloudy** (limewater turns milky)
- **NH₃ = Neutralises red litmus** (turns blue)
- **Cl₂ = Chlorine Changes** blue to red to white

### Flame Colours
- **Li** = **Li**ght crimson
- **Na** = street l**amp**s (yellow)
- **K** = **K**ool lilac
- **Ca** = **Ca**rrot orange or brick
- **Cu** = **Cu**te blue-green

### Silver Halides (get darker down group)
- **Cl** = Clear white
- **Br** = Beige/cream
- **I** = Intense yellow

## 📝 Key Equations Summary

### Gas Test Equations
```
O₂: No equation (combustion)
H₂: 2H₂ + O₂ → 2H₂O (explosion)
CO₂: Ca(OH)₂ + CO₂ → CaCO₃(s) + H₂O
NH₃: NH₃ + H₂O → NH₄⁺ + OH⁻ (alkaline)
Cl₂: Cl₂ + H₂O → HCl + HOCl (acid + bleach)
```

### Precipitation Equations
```
Cu²⁺ + 2OH⁻ → Cu(OH)₂(s) blue
Fe³⁺ + 3OH⁻ → Fe(OH)₃(s) brown
Ba²⁺ + SO₄²⁻ → BaSO₄(s) white
Ag⁺ + Cl⁻ → AgCl(s) white
```
