## 6.1 Balanced Chemical Equations

**Key Principles:**
- Law of Conservation of Mass: atoms cannot be created or destroyed
- Balance using coefficients (numbers in front of formulae)
- NEVER change subscripts to balance

**Method:**
1. Write correct formulae
2. Count atoms on each side
3. Add coefficients to balance
4. Check all atoms balanced

---

## 6.2 State Symbols

| Symbol | Meaning |
|--------|---------|
| (s) | Solid |
| (l) | Liquid (pure) |
| (g) | Gas |
| (aq) | Aqueous (dissolved in water) |

**Remember:**
- Water as reactant/product = (l)
- Solutions = (aq)
- Precipitates = (s)

---

## 6.3 Ionic Equations

**Definition:** Shows only species that actually react

**Spectator ions:** Ions that appear unchanged on both sides

**Method:**
1. Write balanced equation with state symbols
2. Split aqueous ionic compounds into ions
3. Cancel spectator ions
4. Write simplified ionic equation

**Key Ionic Equations:**
```
H⁺(aq) + OH⁻(aq) → H₂O(l)        [Neutralisation]
2H⁺(aq) + CO₃²⁻(aq) → H₂O(l) + CO₂(g)   [Acid + Carbonate]
Ag⁺(aq) + Cl⁻(aq) → AgCl(s)      [Precipitation]
```

---

## 6.4 Calculating Reacting Masses

**Key Formula:**
```
n = m/Mr
```
- n = moles (mol)
- m = mass (g)
- Mr = molar mass (g/mol)

**Steps:**
1. Calculate moles of given substance
2. Use mole ratio from balanced equation
3. Calculate mass of required substance

**Limiting Reactant:** The reactant used up first; determines product amount

---

## 6.5 Percentage Yield

**Formula:**
```
% Yield = (actual yield ÷ theoretical yield) × 100
```

**Reasons for yield < 100%:**
- Incomplete reactions
- Side reactions
- Loss during transfer
- Reversible reactions
- Impure reactants

---

## 6.6 Atom Economy

**Formula:**
```
Atom economy = (Mr of desired product ÷ Sum of Mr of all products) × 100
```

**Importance:**
- Measures efficiency of atom usage
- High atom economy = sustainable, less waste
- Independent of experimental technique

## Common Exam Errors

❌ Changing subscripts to balance equations
❌ Confusing (l) and (aq) state symbols
❌ Splitting solids/liquids/gases into ions
❌ Forgetting mole ratios in calculations
❌ Using reactants instead of products for atom economy
❌ Mixing up percentage yield and atom economy
