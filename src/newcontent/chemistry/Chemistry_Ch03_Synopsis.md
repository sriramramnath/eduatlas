# Chapter 3: Atomic Structure - Concise Synopsis

## Quick Revision Guide for Pearson Edexcel IGCSE Chemistry

---

## 3.1 Atoms, Elements & Compounds

| Term | Definition |
|------|------------|
| **Element** | Substance containing only one type of atom |
| **Atom** | Smallest part of an element that still has properties of that element |
| **Compound** | Two or more elements chemically combined in fixed proportions |
| **Molecule** | Two or more atoms chemically bonded together |

**Key Point:** Compounds can only be separated by chemical reactions, not physical means.

---

## 3.2 Subatomic Particles

| Particle | Relative Mass | Relative Charge | Location |
|----------|---------------|-----------------|----------|
| **Proton** | 1 | +1 | Nucleus |
| **Neutron** | 1 | 0 | Nucleus |
| **Electron** | 1/1836 (negligible) | -1 | Electron shells |

**Remember:** 
- Nucleus = protons + neutrons (contains most of the mass)
- Electrons have negligible mass but occupy most of the volume

---

## 3.3 Atomic & Mass Numbers

| Term | Definition | Formula |
|------|------------|---------|
| **Atomic Number (Z)** | Number of protons | Z = protons |
| **Mass Number (A)** | Protons + neutrons | A = protons + neutrons |
| **Neutrons** | Calculated | N = A - Z |

### Atomic Notation: $\large _Z^A X$

**Example:** $_{11}^{23}Na$
- Protons = 11
- Electrons = 11 (neutral atom)
- Neutrons = 23 - 11 = 12

---

## 3.4 Isotopes

**Definition:** Atoms of the **same element** with the **same number of protons** but **different numbers of neutrons**.

| Property | Same or Different? |
|----------|-------------------|
| Atomic number | Same |
| Mass number | Different |
| Chemical properties | Same (same electrons) |
| Physical properties | Different (different mass) |

**Examples:**
- Carbon isotopes: $^{12}C$, $^{13}C$, $^{14}C$
- Chlorine isotopes: $^{35}Cl$ (75%), $^{37}Cl$ (25%)

---

## 3.5 Electron Configuration (2,8,8 Rule)

**Maximum electrons per shell:**
- 1st shell: **2** electrons
- 2nd shell: **8** electrons  
- 3rd shell: **8** electrons (for elements 1-20)

### First 20 Elements:

| Period 1 | Period 2 | Period 3 | Period 4 |
|----------|----------|----------|----------|
| H: 1 | Li: 2,1 | Na: 2,8,1 | K: 2,8,8,1 |
| He: 2 | Be: 2,2 | Mg: 2,8,2 | Ca: 2,8,8,2 |
| | B: 2,3 | Al: 2,8,3 | |
| | C: 2,4 | Si: 2,8,4 | |
| | N: 2,5 | P: 2,8,5 | |
| | O: 2,6 | S: 2,8,6 | |
| | F: 2,7 | Cl: 2,8,7 | |
| | Ne: 2,8 | Ar: 2,8,8 | |

**Pattern:**
- **Group number** = outer shell electrons
- **Period number** = number of shells

## 3.6 Relative Atomic Mass (Aᵣ)

**Definition:** Weighted average mass of atoms relative to 1/12th of carbon-12.

### Formula:
$$A_r = \frac{(m_1 \times a_1) + (m_2 \times a_2)}{100}$$

**Where:** m = mass number, a = % abundance

### Worked Example:
Chlorine: $^{35}Cl$ (75%), $^{37}Cl$ (25%)

$$A_r = \frac{(35 \times 75) + (37 \times 25)}{100} = \frac{2625 + 925}{100} = 35.5$$

## Quick Memory Aid: "PEN"

**P**rotons = Atomic number  
**E**lectrons = Same as protons (in neutral atom)  
**N**eutrons = Mass number - Atomic number

## Common Mistakes to Avoid

1. ❌ Mass number = protons + electrons → ✅ Mass number = protons + neutrons
2. ❌ K has configuration 2,8,9 → ✅ K has configuration 2,8,8,1
3. ❌ Isotopes have different protons → ✅ Isotopes have different neutrons
4. ❌ Forgetting that Aᵣ is a weighted average
