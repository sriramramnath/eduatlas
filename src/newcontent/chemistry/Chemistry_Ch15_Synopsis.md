## Metal Extraction Methods

| Reactivity | Metals | Extraction Method |
|------------|--------|-------------------|
| Very High | K, Na, Ca, Mg, Al | **Electrolysis** of molten compounds |
| Medium | Zn, Fe, Sn, Pb, Cu | **Reduction with carbon** (smelting) |
| Low | Ag, Au, Pt | Found **native** or simple heating |

---

## Extraction of Iron (Blast Furnace)

### Raw Materials
| Material | Formula | Purpose |
|----------|---------|---------|
| Haematite | Fe₂O₃ | Iron ore |
| Coke | C | Fuel + reducing agent |
| Limestone | CaCO₃ | Removes impurities |
| Hot air | O₂ | Combustion |

### Key Equations
```
C + O₂ → CO₂           (combustion)
CO₂ + C → 2CO          (forms reducing agent)
Fe₂O₃ + 3CO → 2Fe + 3CO₂  (reduction)
CaCO₃ → CaO + CO₂      (limestone decomposes)
CaO + SiO₂ → CaSiO₃    (slag formation)
```

### Products
- **Molten iron** - sinks to bottom
- **Slag (CaSiO₃)** - floats on iron, used in roads/cement

## Extraction of Aluminium

| Aspect | Details |
|--------|---------|
| Ore | Bauxite (Al₂O₃) |
| Method | Electrolysis |
| Electrolyte | Molten Al₂O₃ in **cryolite** |
| Cathode reaction | Al³⁺ + 3e⁻ → Al |
| Anode reaction | 2O²⁻ → O₂ + 4e⁻ |
| Temperature | ~950°C (cryolite lowers MP) |

### Why Cryolite?
- Lowers melting point (2050°C → 950°C)
- Saves energy and cost

### Anode Problem
- Oxygen reacts with carbon anodes
- C + O₂ → CO₂
- Anodes must be replaced regularly

## Extraction of Copper

### From High-Grade Ores
- Smelting + electrolytic refining
- Produces 99.99% pure copper

### From Low-Grade Ores

| Method | Process | Key Feature |
|--------|---------|-------------|
| **Phytomining** | Plants absorb Cu → harvest → burn → extract from ash | Uses plants |
| **Bioleaching** | Bacteria produce leachate containing Cu²⁺ | Uses bacteria |

**Extracting Cu from solution:**
```
Cu²⁺ + Fe → Cu + Fe²⁺ (displacement with scrap iron)
```

## Uses of Metals

### Aluminium
| Property | Use |
|----------|-----|
| Low density | Aircraft, drinks cans |
| Corrosion-resistant | Window frames |
| Good conductor | Overhead power cables |

**Why corrosion-resistant?** Forms protective Al₂O₃ layer

### Copper
| Property | Use |
|----------|-----|
| Excellent conductor | Electrical wires |
| Malleable, ductile | Water pipes |
| Corrosion-resistant | Roofing, statues |

### Iron and Steel
| Type | Composition | Properties | Uses |
|------|-------------|------------|------|
| Mild steel | Fe + <1% C | Malleable, strong | Cars, buildings |
| High-carbon steel | Fe + 1-1.5% C | Hard, brittle | Tools, knives |
| Stainless steel | Fe + Cr + Ni | Corrosion-resistant | Cutlery, surgical instruments |

## Alloys

### Definition
An **alloy** = mixture of a metal + other elements

### Why Harder than Pure Metals?
- Different-sized atoms disrupt regular structure
- Layers cannot slide easily
- Result: Harder and stronger

### Common Alloys
| Alloy | Composition | Uses |
|-------|-------------|------|
| Brass | Cu + Zn | Musical instruments, fittings |
| Bronze | Cu + Sn | Statues, medals |
| Stainless steel | Fe + Cr + Ni | Cutlery, surgical tools |
| Solder | Sn + Pb (or Ag) | Joining electronics |
| Amalgam | Hg + Ag, Sn | Dental fillings |

## Corrosion and Rusting

### Rusting
```
Iron + Oxygen + Water → Hydrated iron(III) oxide
4Fe + 3O₂ + 2xH₂O → 2Fe₂O₃·xH₂O
```

**Both O₂ AND H₂O needed!**

### Prevention Methods

| Method | How it Works | Examples |
|--------|--------------|----------|
| **Painting** | Barrier | Cars, bridges |
| **Oiling/Greasing** | Barrier | Tools, chains |
| **Galvanizing** | Zn coating (barrier + sacrificial) | Roofs, bins |
| **Sacrificial anodes** | More reactive metal corrodes instead | Ship hulls, pipes |
| **Stainless steel** | Alloying with Cr, Ni | Cutlery |

### Sacrificial Protection
- Zinc/magnesium attached to iron
- More reactive metal corrodes instead of iron
- Works even if coating is scratched

## Key Equations to Learn

### Blast Furnace
```
Fe₂O₃ + 3CO → 2Fe + 3CO₂
CaCO₃ → CaO + CO₂
CaO + SiO₂ → CaSiO₃
```

### Aluminium Electrolysis
```
Cathode: Al³⁺ + 3e⁻ → Al
Anode: 2O²⁻ → O₂ + 4e⁻
```

### Rusting
```
4Fe + 3O₂ + 2xH₂O → 2Fe₂O₃·xH₂O
```

## Common Exam Errors

| ❌ Wrong | ✓ Correct |
|---------|----------|
| "Carbon reduces Fe₂O₃" | "CO reduces Fe₂O₃" |
| "Al forms at anode" | "Al forms at cathode" |
| "Al₂O₃ electrolysed at 2050°C" | "Al₂O₃ in cryolite at 950°C" |
| "Bacteria in phytomining" | "Plants in phytomining" |
| "Limestone removes carbon" | "Limestone removes silica (acidic impurities)" |
