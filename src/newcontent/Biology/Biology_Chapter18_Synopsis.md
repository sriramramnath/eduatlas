# Chapter 18: Genes and Inheritance - Concise Synopsis

## Quick Revision Guide for Pearson Edexcel IGCSE Biology

---

## 1. Key Terms

| Term | Definition |
|------|------------|
| **Gene** | DNA section coding for protein |
| **Allele** | Different version of same gene |
| **Dominant** | Always expressed (capital letter) |
| **Recessive** | Only expressed if homozygous (lowercase) |
| **Genotype** | Genetic makeup |
| **Phenotype** | Observable characteristics |
| **Homozygous** | Two identical alleles (BB, bb) |
| **Heterozygous** | Two different alleles (Bb) |
| **Carrier** | Heterozygous for recessive allele |

---

## 2. Punnett Squares

### Monohybrid Cross (Bb × Bb)

| | B | b |
|---|---|---|
| **B** | BB | Bb |
| **b** | Bb | bb |

- Genotype ratio: 1 BB : 2 Bb : 1 bb
- Phenotype ratio: 3 dominant : 1 recessive

---

## 3. Types of Crosses

| Cross | Genotype Ratio | Phenotype Ratio |
|-------|----------------|-----------------|
| BB × bb | All Bb | All dominant |
| Bb × bb | 1 Bb : 1 bb | 1 dominant : 1 recessive |
| Bb × Bb | 1 BB : 2 Bb : 1 bb | 3 dominant : 1 recessive |
| Bb × BB | 1 BB : 1 Bb | All dominant |

---

## 4. Codominance

Both alleles expressed equally in heterozygote.

### Blood Groups Example

| Genotype | Phenotype |
|----------|-----------|
| IᴬIᴬ, Iᴬi | Blood group A |
| IᴮIᴮ, Iᴮi | Blood group B |
| IᴬIᴮ | Blood group AB (codominant) |
| ii | Blood group O |

---

## 5. Sex Determination

| Sex | Chromosomes |
|-----|-------------|
| Female | XX |
| Male | XY |

### Sex Cross (XX × XY)

| | X | Y |
|---|---|---|
| **X** | XX | XY |
| **X** | XX | XY |

- Ratio: 1 female : 1 male

---

## 6. Sex-Linked Inheritance

Gene on sex chromosome (usually X)

### X-linked Recessive (e.g., Haemophilia)

| Genotype | Phenotype |
|----------|-----------|
| XᴴXᴴ | Normal female |
| XᴴXʰ | Carrier female |
| XʰXʰ | Affected female |
| XᴴY | Normal male |
| XʰY | Affected male |

### Key Points
- Males cannot be carriers (only one X)
- More males affected
- Fathers don't pass to sons

---

## 7. Pedigree Chart Symbols

- □ = Unaffected male
- ○ = Unaffected female
- ■ = Affected male
- ● = Affected female

### Patterns

| Type | Features |
|------|----------|
| **Autosomal dominant** | Every affected has affected parent; both sexes equally |
| **Autosomal recessive** | Can skip generations; unaffected parents can have affected children |
| **X-linked recessive** | More males affected; fathers don't pass to sons |

---

## 8. Exam Focus Points

### High-Yield Facts
1. Capital = dominant; lowercase = recessive
2. Always show gametes in crosses
3. Give both genotype AND phenotype ratios
4. X-linked: males only have one X
5. Codominance: both alleles expressed

### Common Mistakes
- Wrong letter case
- Not showing gametes
- Forgetting ratios
- Not explaining sex-linked
- Wrong pedigree interpretation
