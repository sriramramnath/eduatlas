# Chapter 16: Chromosomes, Genes and DNA - Concise Synopsis

## Quick Revision Guide for Pearson Edexcel IGCSE Biology

---

## 1. DNA Structure

### Components of a Nucleotide
- **Phosphate group**
- **Deoxyribose sugar**
- **Nitrogenous base** (A, T, C, or G)

### Base Pairing Rules
| Pairs with | Bonds |
|------------|-------|
| A — T | 2 hydrogen bonds |
| C — G | 3 hydrogen bonds |

### Key Features
- **Double helix** structure
- **Antiparallel** strands
- **Polymer** of nucleotides
- Sugar-phosphate **backbone** on outside
- **Bases** on inside

---

## 2. Chromosomes

### Key Facts
- Made of DNA + proteins
- Found in **nucleus**
- Humans have **46** chromosomes (23 pairs)
- Carry **genes** arranged along their length

### Homologous Pairs
- One from each parent
- Same genes in same positions
- May have different **alleles**

### Sex Chromosomes
| Sex | Chromosomes |
|-----|-------------|
| Female | XX |
| Male | XY |

---

## 3. Genes and Alleles

| Term | Definition |
|------|------------|
| **Gene** | Section of DNA coding for a protein |
| **Allele** | Different version of the same gene |
| **Locus** | Position of a gene on chromosome |

### Example
- Gene: Eye colour
- Alleles: Brown (B), Blue (b)
- Genotype: BB, Bb, or bb
- Phenotype: Brown or blue eyes

---

## 4. Protein Synthesis

### Two Stages

| Stage | Location | Process |
|-------|----------|---------|
| **Transcription** | Nucleus | DNA → mRNA |
| **Translation** | Ribosome | mRNA → Protein |

### Transcription Steps
1. DNA unwinds
2. mRNA strand made using DNA template
3. mRNA leaves nucleus

### Translation Steps
1. mRNA attaches to ribosome
2. tRNA brings amino acids
3. Amino acids joined into protein

---

## 5. Key Terms for Inheritance

| Term | Definition |
|------|------------|
| **Dominant** | Always expressed when present |
| **Recessive** | Only expressed when homozygous |
| **Homozygous** | Two identical alleles (BB, bb) |
| **Heterozygous** | Two different alleles (Bb) |
| **Genotype** | Genetic makeup |
| **Phenotype** | Observable characteristics |
| **Carrier** | Heterozygous for recessive allele |

---

## 6. The Genetic Code

- **Triplet code**: 3 bases = 1 amino acid
- **Universal**: Same in all organisms
- **Degenerate**: Multiple codons for some amino acids
- Start codon = AUG
- Stop codons signal end

---

## 7. Exam Focus Points

### High-Yield Facts
1. DNA = polymer of nucleotides
2. A pairs with T; C pairs with G
3. Genes code for proteins
4. Alleles = different versions of same gene
5. Transcription in nucleus; Translation at ribosome

### Common Mistakes
- Confusing gene with allele
- Wrong case letters (B vs b)
- Forgetting chromosomes are in pairs
- Not saying DNA is a polymer
- Mixing up transcription and translation
