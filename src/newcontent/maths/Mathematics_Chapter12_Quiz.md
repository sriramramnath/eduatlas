### Question 1
**Simplify:** $\frac{8x}{12}$

A) $\frac{2x}{3}$

B) $\frac{4x}{6}$

C) $\frac{8x}{12}$

D) $\frac{2}{3}$

**Answer: A**

**Explanation:**
$$\frac{8x}{12} = \frac{4 \cdot 2x}{4 \cdot 3} = \frac{2x}{3}$$
The common factor of 8 and 12 is 4. Divide both numerator and denominator by 4.

### Question 2
**Simplify:** $\frac{x^2 - 9}{x + 3}$

A) $x - 3$

B) $x + 3$

C) $x^2 - 3$

D) $x - 9$

**Answer: A**

**Explanation:**
Factorise numerator using difference of two squares:
$$x^2 - 9 = (x + 3)(x - 3)$$
Then:
$$\frac{(x + 3)(x - 3)}{x + 3} = x - 3$$

---

### Question 16
**Statement:** $\frac{x + 4}{x} = 4$

**Answer: FALSE**

**Correction:** $\frac{x + 4}{x}$ cannot be simplified further. The $x$ in the numerator is not a factor (it's part of an addition), so it cannot cancel with the denominator.

**Explanation:** You can only cancel factors that are multiplied, not terms that are added. $x + 4$ is not the same as $x \times 4$.

---

### Question 17
**Statement:** The equation $x^2 = 25$ has exactly one solution, $x = 5$.

**Answer: FALSE**

**Correction:** $x^2 = 25$ has TWO solutions: $x = 5$ and $x = -5$.

**Explanation:** Both $5^2 = 25$ and $(-5)^2 = 25$. Square equations always have two solutions (if solutions exist).

---

### Question 18
**Statement:** $x^3 \times x^4 = x^7$

**Answer: TRUE**

**Explanation:** Using Law 1 for indices: when multiplying expressions with the same base, add the powers. $x^3 \times x^4 = x^{3+4} = x^7$.

---

### Question 19
**Statement:** When solving $-2x > 8$, the solution is $x > -4$.

**Answer: FALSE**

**Correction:** The solution is $x < -4$.

**Explanation:** When dividing both sides of an inequality by a negative number, you must REVERSE the inequality sign. $-2x > 8$ Ôćĺ $x < -4$.

---

### Question 20
**Statement:** $(x^2)^3 = x^5$

**Answer: FALSE**

**Correction:** $(x^2)^3 = x^6$.

**Explanation:** Using Law 3: when raising a power to a power, MULTIPLY the powers. $(x^2)^3 = x^{2 \times 3} = x^6$.

---

### Question 21
**Statement:** On a number line, $x < 3$ is represented with an open circle at 3.

**Answer: TRUE**

**Explanation:** The inequality $x < 3$ uses $<$ (strict inequality), which means 3 is NOT included in the solution set. Open circles indicate that the endpoint is excluded.

---

### Question 22
**Statement:** $\sqrt{x} = -4$ has solution $x = 16$.

**Answer: FALSE**

**Correction:** $\sqrt{x} = -4$ has NO solution (in real numbers).

**Explanation:** $\sqrt{x}$ is defined as the positive square root, so $\sqrt{x}$ cannot equal a negative number. This equation has no solution. Even though $x = 16$ would give $\sqrt{16} = 4$, this is not $-4$.

---

### Question 23
**Statement:** $\frac{x^2 - 4}{x - 2} = x + 2$

**Answer: TRUE**

**Explanation:** Factorise the numerator as a difference of two squares: $x^2 - 4 = (x+2)(x-2)$. Then $\frac{(x+2)(x-2)}{x-2} = x + 2$.

---

### Question 24
**Statement:** The solution to $2 < x \leq 5$ includes the integers 2, 3, 4, and 5.

**Answer: FALSE**

**Correction:** The integer solutions are 3, 4, and 5. (2 is NOT included because $x > 2$, not $x \geq 2$).

**Explanation:** The inequality $2 < x$ means $x$ is strictly greater than 2, so 2 is not included. $x \leq 5$ means 5 is included.

---

### Question 25
**Statement:** $x^0 = 0$ for all values of $x$.

**Answer: FALSE**

**Correction:** $x^0 = 1$ for all non-zero values of $x$.

**Explanation:** By the index laws, $\frac{x^1}{x^1} = x^{1-1} = x^0$, and $\frac{x}{x} = 1$. Therefore $x^0 = 1$ (when $x \neq 0$).

### Question 38 [6 marks]

| A1 | $x^2 = 5$, so $x = \pm\sqrt{5}$ |

$\boxed{x = \sqrt{5} \text{ or } x = -\sqrt{5}}$

*Note: Neither solution makes any denominator zero, so both are valid.*

### Question 40 [7 marks]
**A rectangle has length $(x + 3)$ cm and width $x$ cm.**

**(a)** Write an expression for the area of the rectangle.

**(b)** The area is 40 cm┬▓. Form an equation and solve for $x$.

**(c)** Hence find the length and width of the rectangle.

**(d)** Verify your answer.

---

**Solution:**

**(a)** Expression for area

| M1 | Area = length ├Ś width |
|----|-----------------------|
| A1 | $\boxed{x(x + 3) \text{ cm}^2 \text{ or } x^2 + 3x \text{ cm}^2}$ |

---

**(b)** Form and solve equation

| M1 | Form equation: $x(x + 3) = 40$ |
|----|--------------------------------|
| M1 | Expand: $x^2 + 3x = 40$ |
| M1 | Rearrange: $x^2 + 3x - 40 = 0$ |
| M1 | Factorise: $(x + 8)(x - 5) = 0$ |
| M1 | Solve: $x = -8$ or $x = 5$ |
| A1 | $x = -8$ rejected (negative length), so $\boxed{x = 5}$ |

---

**(c)** Length and width

| M1 | Width = $x = 5$ cm |
|----|--------------------|
| A1 | Length = $x + 3 = \boxed{8 \text{ cm}}$ |
| A1 | $\boxed{\text{Width: } 5 \text{ cm, Length: } 8 \text{ cm}}$ |

---

**(d)** Verification

| M1 | Area = $5 \times 8 = 40$ cm┬▓ Ôťô |
|----|--------------------------------|
| A1 | Both dimensions are positive Ôťô |

## EXAM FOCUS & STRATEGY

### Frequently Tested Question Types

1. **Simplifying algebraic fractions** (2-4 marks)
   - Factorise first
   - Cancel common factors
   - Watch for difference of two squares

2. **Solving $x^2 = k$** (2-3 marks)
   - Remember TWO solutions
   - Use ┬▒ correctly

3. **Index law problems** (2-4 marks)
   - Know all three laws
   - Handle coefficients separately
   - Apply laws in correct order

4. **Inequalities** (2-4 marks)
   - Check for negative divisors
   - Reverse sign when needed
   - Correct number line representation

### Command Words Guidance

| Command | Action Required |
|---------|-----------------|
| **Simplify** | Use index laws or cancel factors |
| **Solve** | Find all values of the variable |
| **Factorise** | Express as product of factors |
| **Represent** | Draw on number line |
| **Find** | Determine the value(s) |

### How to Gain Full Marks

1. **Show all working** - Every step earns method marks
2. **Check solutions** - Substitute back to verify
3. **Two solutions** - Don't forget negative roots
4. **Factorise first** - Before cancelling fractions
5. **Reverse signs** - When dividing inequalities by negatives

### Common Mistakes to Avoid

| Mistake | How to Avoid |
|---------|--------------|
| One solution for $x^2 = k$ | Always write $\pm$ |
| Cancelling added terms | Only cancel factors |
| Wrong index law | Memorise all three laws |
| Not reversing inequality | Check sign before dividing |
