## 1. SIMPLIFYING ALGEBRAIC EXPRESSIONS

### Key Definitions
- **Expression:** Numbers, variables and operations (no equals sign)
- **Term:** Single number, variable, or product of both
- **Coefficient:** Number multiplied by a variable

### Like Terms
- **Same variable(s)** AND **same power(s)**

| ✅ LIKE TERMS | ❌ NOT LIKE TERMS |
|--------------|-------------------|
| $3x$ and $-5x$ | $3x$ and $3x^2$ |
| $2ab$ and $7ab$ | $2ab$ and $2a$ |
| $4y^2$ and $-y^2$ | $4x$ and $4y$ |
| $5$ and $-3$ (constants) | $x$ and $xy$ |

### Simplifying Rules
- Only **like terms** can be combined
- Add/subtract **coefficients only**
- Keep the **variable part unchanged**

**Example:** $3x + 5x = (3+5)x = 8x$

## 2. EXPRESSIONS WITH BRACKETS

### Common Translations

| Words | Expression |
|-------|------------|
| Sum of $a$ and $b$ | $a + b$ |
| Product of $a$ and $b$ | $ab$ |
| $a$ more than $b$ | $b + a$ |
| $a$ less than $b$ | $b - a$ |
| Twice $x$ | $2x$ |
| $x$ squared | $x^2$ |
| Sum of $x$ and $y$, squared | $(x + y)^2$ |
| 5 times $x$ plus 2 | $5(x + 2)$ or $5x + 2$ |

>  **"5 less than x"** = $x - 5$ (NOT $5 - x$)

### Substitution Rules
- Replace variable with given value
- Use BODMAS for calculation
- **Power brackets:** $(-2)^2 = 4$ vs $-2^2 = -4$

## 3. EXPANDING SINGLE BRACKETS

### The Golden Rule

$$\boxed{a(b + c) = ab + ac}$$

**Multiply EVERY term inside by the term outside.**

### Sign Rules Table

| Outside × Inside | Result |
|------------------|--------|
| $(+) \times (+)$ | $+$ |
| $(+) \times (-)$ | $-$ |
| $(-) \times (+)$ | $-$ |
| $(-) \times (-)$ | $+$ |

### Quick Examples

| Expression | Expanded Form |
|------------|---------------|
| $3(x + 4)$ | $3x + 12$ |
| $5(2x - 3)$ | $10x - 15$ |
| $-2(x + 5)$ | $-2x - 10$ |
| $-3(2x - 4)$ | $-6x + 12$ |
| $x(x + 7)$ | $x^2 + 7x$ |
| $2x(3x - 1)$ | $6x^2 - 2x$ |

### Common Errors to Avoid

| ❌ WRONG | ✅ CORRECT |
|----------|------------|
| $3(x + 4) = 3x + 4$ | $3x + 12$ |
| $-2(x - 3) = -2x - 6$ | $-2x + 6$ |
| $x(x + 3) = x + 3x$ | $x^2 + 3x$ |

## 4. SOLVING EQUATIONS

### One-Step Equations

| Type | Equation | Method | Solution |
|------|----------|--------|----------|
| Addition | $x + a = b$ | Subtract $a$ | $x = b - a$ |
| Subtraction | $x - a = b$ | Add $a$ | $x = b + a$ |
| Multiplication | $ax = b$ | Divide by $a$ | $x = \frac{b}{a}$ |
| Division | $\frac{x}{a} = b$ | Multiply by $a$ | $x = ab$ |

### Two-Step Equations

**Standard Form:** $ax + b = c$

**Method:**
1. Subtract/add constant first
2. Divide/multiply coefficient second

**Example:**
$$3x + 5 = 17$$
$$3x = 12$$ (subtract 5)
$$x = 4$$ (divide by 3)

### Equations with Brackets

**Two methods:**
1. **Expand first:** $3(x + 4) = 21$ → $3x + 12 = 21$
2. **Divide first (if divisible):** $3(x + 4) = 21$ → $x + 4 = 7$

### Unknown on Both Sides

**Method:** Get all $x$ terms on ONE side

$$4x + 3 = 2x + 11$$
$$4x - 2x = 11 - 3$$
$$2x = 8$$
$$x = 4$$

## HIGH-YIELD EXAM TIPS

### Time-Savers
1. **Mental maths:** $3x + 5x = 8x$ (no working needed)
2. **Sign shortcuts:** Count negatives: odd = negative, even = positive
3. **Check answers:** Substitute back into original equation

### Exam Technique
1. **Show ALL working** - every step on a new line
2. **Write clearly** - examiners need to follow your logic
3. **Check units** - if units are given, include in answer
4. **Answer the question** - "Find x" means write "$x = ...$"

### Command Words Quick Guide

| Command | Action |
|---------|--------|
| Simplify | Collect like terms |
| Expand | Remove brackets only |
| Expand and simplify | Remove brackets, then collect |
| Solve | Find the value |
| Factorise | Put into brackets |

## FORMULA SHEET

### Algebraic Manipulation
- **Like terms:** Same variable, same power
- **Expand:** $a(b + c) = ab + ac$
- **Collect:** $(a + b)x + (c + d)x = (a + b + c + d)x$

### Solving Equations
- **One-step:** Use inverse operation
- **Two-step:** Remove constant first, then coefficient
- **Balance:** Do the same to both sides

### Sign Rules
- $(-) \times (-) = +$
- $(-) \times (+) = -$
- $(+) \times (+) = +$
- $(+) \times (-) = -$
