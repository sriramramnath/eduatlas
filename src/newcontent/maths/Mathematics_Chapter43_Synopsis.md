## 📌 ESSENTIAL DEFINITIONS

| Term | Definition |
|------|------------|
| **Tangent Ratio** | $\tan \theta = \frac{\text{Opposite}}{\text{Adjacent}}$ |
| **Opposite** | Side across from angle θ |
| **Adjacent** | Side next to angle θ (not hypotenuse) |
| **Hypotenuse** | Longest side (opposite 90°) |

---

## MASTER FORMULA

$$\boxed{\tan \theta = \frac{O}{A}}$$

---

### Finding Unknowns

| Given | Find | Formula |
|-------|------|---------|
| θ and A | O | $O = A \times \tan \theta$ |
| θ and O | A | $A = \frac{O}{\tan \theta}$ |
| O and A | θ | $\theta = \tan^{-1}\left(\frac{O}{A}\right)$ |

---

## 🎯 WHEN TO USE TANGENT

✅ Use tangent when:
- You have **Opposite** and **Adjacent** (to find angle)
- You have **Angle** and **Adjacent** (to find opposite)
- You have **Angle** and **Opposite** (to find adjacent)

❌ Do NOT use tangent when:
- Hypotenuse is involved → use sin or cos instead

## 📐 ANGLES OF ELEVATION & DEPRESSION

| Type | Description |
|------|-------------|
| **Angle of Elevation** | Looking UP from horizontal |
| **Angle of Depression** | Looking DOWN from horizontal |

> **Key Rule:** Angle of elevation = Angle of depression (alternate angles)

## ⚡ QUICK RULES

### Rule 1: SOHCAHTOA Memory Aid
- **T**OA = **T**angent = **O**pposite over **A**djacent

### Rule 2: Multiplication vs Division
- Finding **OPPOSITE** → **MULTIPLY**: $O = A \times \tan \theta$
- Finding **ADJACENT** → **DIVIDE**: $A = \frac{O}{\tan \theta}$

### Rule 3: Finding Angle
- Calculate ratio first: $\frac{O}{A}$
- Then use inverse: $\tan^{-1}$

##  COMMON EXAM MISTAKES

| Mistake | Correction |
|---------|------------|
| Using hypotenuse with tan | Tan uses O and A only |
| Calculator in radians | Check for D/DEG mode |
| Not labeling sides | Always mark O, A, H |
| Rounding too early | Keep full calculator display |
| Mixing up elevation/depression | Draw horizontal line first |

---

## ✅ EXAM METHOD STEPS

### Finding a Side (2-3 marks)
1. Label sides (O, A, H)
2. Write: $\tan \theta = \frac{O}{A}$
3. Substitute known values
4. Rearrange
5. Calculate and round

### Finding an Angle (2-3 marks)
1. Label sides (O, A, H)
2. Calculate: $\frac{O}{A}$
3. Write: $\theta = \tan^{-1}\left(\frac{O}{A}\right)$
4. Calculate and round
