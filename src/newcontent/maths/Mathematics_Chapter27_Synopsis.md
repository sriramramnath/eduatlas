## 📊 SOLVING QUADRATIC EQUATIONS GRAPHICALLY - AT A GLANCE

### Core Principle
**Solutions of $f(x) = 0$ = x-intercepts of $y = f(x)$**

### Quick Reference Table

| Equation | What to Draw | Solutions At |
|----------|--------------|--------------|
| $ax^2 + bx + c = 0$ | Graph $y = ax^2 + bx + c$ | x-intercepts |
| $ax^2 + bx + c = k$ | Graph $y = ax^2 + bx + c$ + line $y = k$ | Intersections with line |
| $ax^2 + bx + c = mx + d$ | Both graphs | Intersection points |

---

## 🔢 DISCRIMINANT QUICK GUIDE

$$\Delta = b^2 - 4ac$$

| Discriminant | Graph Behavior | Solutions |
|--------------|----------------|-----------|
| $b^2 - 4ac > 0$ | Crosses x-axis twice | **2 distinct solutions** |
| $b^2 - 4ac = 0$ | Touches x-axis (tangent) | **1 repeated solution** |
| $b^2 - 4ac < 0$ | Does not touch x-axis | **No real solutions** |

---

## 📐 SOLVING $ax^2 + bx + c = k$

### Method: Horizontal Line

1. Draw $y = ax^2 + bx + c$ (parabola)
2. Draw $y = k$ (horizontal line)
3. Find intersections → read x-values

### Example
Solve $x^2 - 4x + 3 = 2$:
- Draw parabola $y = x^2 - 4x + 3$
- Draw line $y = 2$
- Intersections give solutions

---

## 🔄 REARRANGING FOR GRAPHICAL SOLUTION

### Common Rearrangement Patterns

| Original | Rearranged | Line to Draw |
|----------|------------|--------------|
| $x^2 = 2x + 3$ | $x^2 - 2x = 3$ | $y = 3$ on $y = x^2 - 2x$ |
| $x^2 + 5 = 2x$ | $x^2 - 2x = -5$ | $y = -5$ on $y = x^2 - 2x$ |
| $x^2 + 3x = 7$ | Already in form | $y = 7$ on $y = x^2 + 3x$ |

---

## ✖️ INTERSECTION METHOD (TWO GRAPHS)

### When to Use
When both graphs are simple to draw

### Method
1. Draw first graph (e.g., $y = x^2$)
2. Draw second graph (e.g., $y = 2x + 3$)
3. Find intersection points
4. Read both x AND y coordinates

## 📈 SOLVING OTHER EQUATIONS

### Cubic Equations
$$ax^3 + bx^2 + cx + d = 0$$

| Feature | Detail |
|---------|--------|
| Solutions | x-intercepts of cubic graph |
| Number of solutions | 1, 2, or 3 real solutions |
| For $f(x) = k$ | Draw horizontal line $y = k$ |

### Reciprocal Equations
$$y = \frac{a}{x}$$

| Equation | Method |
|----------|--------|
| $\frac{a}{x} = k$ | Draw $y = k$ horizontal line |
| $\frac{a}{x} = mx + c$ | Draw both graphs, find intersection |

### Circle Equations
$$(x-a)^2 + (y-b)^2 = r^2$$

| Feature | Detail |
|---------|--------|
| Center | $(a, b)$ |
| Radius | $r$ |
| With line | Find intersection points |

## 🔀 NON-LINEAR SIMULTANEOUS EQUATIONS

### Definition
At least one equation is non-linear (quadratic, cubic, reciprocal, circle)

### Graphical Method

**Step 1:** Draw both graphs on same axes

**Step 2:** Mark intersection points clearly

**Step 3:** Read coordinates $(x, y)$

>  **Essential:** Give BOTH x and y coordinates!

---

## 📋 LINEAR + QUADRATIC SIMULTANEOUS

### Graphical Method
$$\begin{cases} y = ax^2 + bx + c \\ y = mx + d \end{cases}$$

1. Draw parabola
2. Draw line
3. Find intersections → give $(x, y)$ pairs

### Algebraic Method (Substitution)

**Step 1:** Set expressions equal
$$ax^2 + bx + c = mx + d$$

**Step 2:** Rearrange to quadratic
$$ax^2 + (b-m)x + (c-d) = 0$$

**Step 3:** Solve quadratic (factorise or formula)

**Step 4:** Find y-values using linear equation

## 🎯 NUMBER OF SOLUTIONS

### Linear + Quadratic

| Graph Behavior | Solutions |
|----------------|-----------|
| Line cuts parabola twice | **2 solutions** |
| Line touches (tangent) | **1 solution** |
| Line misses parabola | **No solutions** |

### Two Quadratics

| Scenario | Solutions |
|----------|-----------|
| Two parabolas intersect | Up to 4 solutions |
| Same parabola reflected | 2 solutions |

### Circle + Line

| Scenario | Solutions |
|----------|-----------|
| Line cuts circle twice | **2 solutions** |
| Line tangent to circle | **1 solution** |
| Line misses circle | **No solutions** |

## ⚡ QUICK FORMULAS

### Quadratic Formula
$$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$

### Key Substitution Pattern
For $\begin{cases} y = ax^2 + bx + c \\ y = mx + d \end{cases}$

Substitute: $ax^2 + bx + c = mx + d$

Rearrange: $ax^2 + (b-m)x + (c-d) = 0$

##  EXAM QUESTION TYPES

### Type 1: Solve $f(x) = k$
**Command:** "Use your graph to solve..."

**Method:**
1. Draw horizontal line $y = k$
2. Find intersections
3. Read x-values

### Type 2: Simultaneous Equations
**Command:** "Find the coordinates of the points of intersection"

**Method:**
1. Draw both graphs
2. Mark intersections
3. Give $(x, y)$ pairs

### Type 3: Number of Solutions
**Command:** "How many solutions are there?"

**Method:**
- Count intersection points
- Use discriminant if algebraic

### Type 4: Rearranging
**Command:** "Given the graph of $y = ...$, solve..."

**Method:**
1. Rearrange equation to match graph form
2. Draw appropriate line
3. Find intersections

##  COMMON ERRORS

| Error | Correction |
|-------|------------|
| Only giving x | Give **both** x and y coordinates |
| Wrong line drawn | Check equation carefully |
| Poor accuracy | Read to 1 d.p. or 0.5 small square |
| Missing solutions | Check for **all** intersections |
| No construction lines | Draw lines clearly on graph |
| Wrong rearrangement | Verify algebraically |

## 📌 KEY REMEMBERS

### Graphical Solutions
1. ✅ $f(x) = 0$ → x-intercepts
2. ✅ $f(x) = k$ → horizontal line $y = k$
3. ✅ Two graphs → intersections
4. ✅ Always give both coordinates
5. ✅ Check all quadrants/regions

### Simultaneous Equations
1. ✅ Draw both graphs on same axes
2. ✅ Label each graph clearly
3. ✅ Mark intersection points
4. ✅ Solutions are coordinate pairs
5. ✅ Verify by substitution

### Accuracy
1. ✅ Read to 1 decimal place typically
2. ✅ Show construction lines
3. ✅ Use appropriate scale
4. ✅ Check reasonableness

## 🏆 A* TIPS

### Speed Techniques
1. **Know key points:** $(0, c)$ is y-intercept for any graph
2. **Use symmetry:** Parabolas are symmetric about vertex
3. **Factorise quickly:** Check for simple factors before formula

### Accuracy Techniques
1. **Draw construction lines** lightly but clearly
2. **Circle intersection points** on graph
3. **State coordinates** separately after reading from graph

### Verification
1. **Substitute back** into both original equations
2. **Visual check:** Does the solution look reasonable?
3. **Discriminant check:** Know how many solutions to expect
