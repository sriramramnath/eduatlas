## Part A: Multiple Choice Questions (10 marks)

### Question 1
A cuboid has dimensions 3 cm, 4 cm, and 12 cm. What is the length of the space diagonal?

A) 13 cm  
B) 19 cm  
C) $\sqrt{19}$ cm  
D) $\sqrt{169}$ cm

**Answer: A) 13 cm**

**Explanation:**
Using the space diagonal formula:
$$d = \sqrt{l^2 + w^2 + h^2}$$
$$d = \sqrt{3^2 + 4^2 + 12^2} = \sqrt{9 + 16 + 144} = \sqrt{169} = 13 \text{ cm}$$

Note: This is a nice example where the answer simplifies to a whole number.

### Question 2
In a right cone, the radius is 6 cm and the slant height is 10 cm. The vertical height is:

A) 4 cm  
B) 8 cm  
C) $\sqrt{136}$ cm  
D) 16 cm

**Answer: B) 8 cm**

**Explanation:**
Using $l^2 = r^2 + h^2$:
$$h = \sqrt{l^2 - r^2} = \sqrt{10^2 - 6^2} = \sqrt{100 - 36} = \sqrt{64} = 8 \text{ cm}$$

### Question 3
The angle between the space diagonal of a cube and its base is:

A) 30°  
B) 45°  
C) $\tan^{-1}(\sqrt{2})$  
D) 60°

**Answer: C) $\tan^{-1}(\sqrt{2})$**

**Explanation:**
For a cube of side $s$:
- Vertical height = $s$
- Base diagonal = $s\sqrt{2}$

$$\tan\theta = \frac{s}{s\sqrt{2}} = \frac{1}{\sqrt{2}}$$

Wait, let me recalculate. The angle between space diagonal and base:
- Height = $s$ (opposite to angle)
- Base diagonal = $s\sqrt{2}$ (adjacent to angle)

$$\tan\theta = \frac{s}{s\sqrt{2}} = \frac{1}{\sqrt{2}}$$

$$\theta = \tan^{-1}\left(\frac{1}{\sqrt{2}}\right) \approx 35.3°$$

Actually, let me reconsider. The angle we want is between the space diagonal and the base. The space diagonal, the base diagonal, and the height form a right triangle.

For a cube with side $a$:
- Height = $a$ (vertical from one corner)
- Base diagonal = $a\sqrt{2}$

$\tan\theta = \frac{a}{a\sqrt{2}} = \frac{1}{\sqrt{2}}$

$\theta = \tan^{-1}\left(\frac{1}{\sqrt{2}}\right)$

This doesn't match any option directly. Let me check if the question means something different.

If we're looking at the angle between the space diagonal and an edge:
$\cos\theta = \frac{a}{a\sqrt{3}} = \frac{1}{\sqrt{3}}$
$\theta = \cos^{-1}\left(\frac{1}{\sqrt{3}}\right) \approx 54.7°$

The correct answer for the angle with the base is approximately 35.3° or $\tan^{-1}\left(\frac{1}{\sqrt{2}}\right)$.

However, option C says $\tan^{-1}(\sqrt{2})$ which equals approximately 54.7° - this would be the angle with an edge, not the base.

Given the options, **C is closest** as it relates to the correct trigonometric relationship, though the exact value should be $\tan^{-1}\left(\frac{1}{\sqrt{2}}\right)$ for the angle with the base.

**Corrected Answer:** The angle between the space diagonal and the base of a cube is approximately 35.3°.

### Question 4
A square-based pyramid has base edges of 8 cm and vertical height 6 cm. The slant height of a triangular face is:

A) 10 cm  
B) $\sqrt{28}$ cm  
C) $\sqrt{52}$ cm  
D) $\sqrt{100}$ cm

**Answer: A) 10 cm**

**Explanation:**
The slant height forms a right triangle with:
- Vertical height = 6 cm
- Half the base edge = 4 cm

$$l = \sqrt{h^2 + \left(\frac{s}{2}\right)^2} = \sqrt{6^2 + 4^2} = \sqrt{36 + 16} = \sqrt{52} \approx 7.21 \text{ cm}$$

Wait, I need to reconsider. The slant height is the height of the triangular face from the base edge to the apex.

Actually, the distance from the center of the base to the midpoint of an edge is $\frac{s}{2} = 4$ cm.

So: $l = \sqrt{6^2 + 4^2} = \sqrt{52} \approx 7.21$ cm

The answer should be **C) $\sqrt{52}$ cm**

Let me verify: In a square-based pyramid with side $s$ and height $h$, the slant height (height of triangular face):
- From center to midpoint of edge = $\frac{s}{2}$
- Vertical height = $h$
- Slant height = $\sqrt{h^2 + \left(\frac{s}{2}\right)^2}$

With $s = 8$ and $h = 6$:
$l = \sqrt{6^2 + 4^2} = \sqrt{52}$ cm

**Correct Answer: C) $\sqrt{52}$ cm**

### Question 5
To find the angle between a line and a plane, which trigonometric ratio is most commonly used?

A) $\sin\theta = \frac{\text{horizontal}}{\text{slant}}$  
B) $\cos\theta = \frac{\text{vertical}}{\text{slant}}$  
C) $\tan\theta = \frac{\text{vertical}}{\text{horizontal}}$  
D) $\tan\theta = \frac{\text{horizontal}}{\text{vertical}}$

**Answer: C) $\tan\theta = \frac{\text{vertical}}{\text{horizontal}}$**

**Explanation:**
The angle between a line and a plane is found using:
- Vertical height as the opposite side
- Horizontal distance (projection on plane) as the adjacent side

$$\tan\theta = \frac{\text{vertical height}}{\text{horizontal distance}}$$

### Question 6
A cuboid measures 5 cm × 12 cm × 13 cm. The face diagonal on the 5 cm × 12 cm face measures:

A) 13 cm  
B) 17 cm  
C) $\sqrt{17}$ cm  
D) $\sqrt{314}$ cm

**Answer: A) 13 cm**

**Explanation:**
Using Pythagoras for the face diagonal:
$$d = \sqrt{5^2 + 12^2} = \sqrt{25 + 144} = \sqrt{169} = 13 \text{ cm}$$

This is a Pythagorean triple (5, 12, 13).

In 3D trigonometry problems, the FIRST step should always be:

A) Write down the formula

**Answer: B) Draw and label a diagram**

**Explanation:**

### Question 8
The space diagonal of a cuboid makes an angle θ with the base. If the height of the cuboid is equal to the base diagonal, then θ equals:

A) 30°  
B) 45°  
C) 60°  
D) 90°

**Answer: B) 45°**

**Explanation:**
If height = base diagonal:
$$\tan\theta = \frac{\text{height}}{\text{base diagonal}} = \frac{d}{d} = 1$$

$$\theta = \tan^{-1}(1) = 45°$$

When opposite = adjacent in a right triangle, the angle is 45°.

### Question 9
A cone has a base radius of 8 cm and a vertical height of 15 cm. The angle between the slant height and the base is closest to:

A) 28°  
B) 48°  
C) 62°  
D) 72°

**Answer: C) 62°**

**Explanation:**
$$\tan\theta = \frac{\text{vertical height}}{\text{radius}} = \frac{15}{8} = 1.875$$

$$\theta = \tan^{-1}(1.875) \approx 61.9° \approx 62°$$

### Question 10
When calculating angles in 3D problems, your calculator should be in:

A) Radian mode  
B) Degree mode  
C) Gradient mode  
D) Either radian or degree mode

**Answer: B) Degree mode**

**Explanation:**
IGCSE Mathematics always uses degrees for angles. Check your calculator displays "D" or "DEG" before calculating angles. Using radians will give incorrect answers (e.g., $\sin^{-1}(0.5) = 30°$ but in radians = 0.524 rad).

## Part B: True/False Questions (10 marks)

### Question 11
The space diagonal of a cuboid is always longer than any face diagonal.

**Answer: TRUE**

**Correction if False:** N/A

**Explanation:**
The space diagonal formula is $d = \sqrt{l^2 + w^2 + h^2}$, while a face diagonal on the $l \times w$ face is $\sqrt{l^2 + w^2}$. Since we add the positive term $h^2$, the space diagonal is always greater: $\sqrt{l^2 + w^2 + h^2} > \sqrt{l^2 + w^2}$ for $h > 0$.

---

### Question 12
In a right cone, the slant height, radius, and vertical height form a right-angled triangle.

**Answer: TRUE**

**Correction if False:** N/A

**Explanation:**
The slant height is the hypotenuse, with the radius and vertical height as the two shorter sides: $l^2 = r^2 + h^2$. This is because the radius is perpendicular to the height at the center of the base.

---

### Question 13
The angle between a line and a plane is found using the angle between the line and any line in the plane.

**Answer: FALSE**

**Correction:** The angle between a line and a plane is found using the angle between the line and its projection onto the plane (not just any line in the plane).

**Explanation:**
You must use the projection of the line onto the plane, which forms a right angle with the perpendicular from the line to the plane. Any arbitrary line in the plane won't give the correct angle.

---

### Question 14
When finding the angle between a diagonal of a cuboid and the base, the horizontal distance used is a face diagonal.

**Answer: TRUE**

**Correction if False:** N/A

**Explanation:**
The projection of the space diagonal onto the base is indeed the face diagonal. So when finding the angle with the base:
- Opposite = height of cuboid
- Adjacent = face diagonal of base
- $\tan\theta = \frac{\text{height}}{\text{face diagonal}}$

---

### Question 15
In a square-based pyramid, all triangular faces are always identical.

**Answer: FALSE**

**Correction:** In a square-based pyramid, all triangular faces are identical only if the pyramid is "regular" (the apex is directly above the center of the base).

**Explanation:**
If the apex is not centered above the base (an irregular pyramid), the triangular faces will have different shapes and sizes. The faces are identical only in a regular pyramid.

---

### Question 16
Pythagoras' theorem can only be used once in a 3D problem.

**Answer: FALSE**

**Correction:** Pythagoras' theorem may be used multiple times in a 3D problem, typically to find intermediate lengths before applying trigonometry.

**Explanation:**
Multi-stage problems often require using Pythagoras multiple times—for example, first to find a face diagonal, then to find the space diagonal.

---

### Question 17
$\sin\theta$ can never be greater than 1.

**Answer: TRUE**

**Correction if False:** N/A

**Explanation:**
Since $\sin\theta = \frac{\text{opposite}}{\text{hypotenuse}}$ and the opposite side cannot be longer than the hypotenuse in a right-angled triangle, $\sin\theta \leq 1$ always. Similarly, $\cos\theta \leq 1$ always.

---

### Question 18
If $\tan\theta = 1$, then $\theta = 45°$.

**Answer: TRUE**

**Correction if False:** N/A

**Explanation:**
$\tan 45° = 1$ because in a 45-45-90 triangle, the opposite and adjacent sides are equal. Therefore, if $\tan\theta = 1$ and $\theta$ is acute (between 0° and 90°), then $\theta = 45°$.

---

### Question 19
The distance from the center of a cube to any corner equals the space diagonal.

**Answer: FALSE**

**Correction:** The distance from the center of a cube to any corner equals half the space diagonal.

**Explanation:**
The space diagonal goes from one corner to the opposite corner. The center is at the midpoint of this diagonal, so the distance from center to corner is $\frac{\text{space diagonal}}{2} = \frac{s\sqrt{3}}{2}$ for a cube of side $s$.

---

### Question 20
When showing working for 3D trigonometry, drawing the 2D triangle you're working with earns method marks.

**Answer: TRUE**

**Correction if False:** N/A

**Explanation:**
Drawing a clear 2D diagram of the specific right-angled triangle being used demonstrates understanding of the problem and earns method marks (M1), even if subsequent calculations contain errors.

## Part C: Structured Questions

### Question 21 (3 marks)
A cuboid has dimensions 9 cm, 12 cm, and 20 cm.

**(a)** Calculate the length of the diagonal on the 9 cm × 12 cm face. (1 mark)

**(b)** Calculate the length of the space diagonal of the cuboid. (2 marks)

**(a) Face diagonal**

$$d = \sqrt{9^2 + 12^2}$$ **(M1)**

$$= \sqrt{81 + 144} = \sqrt{225} = 15 \text{ cm}$$ **(A1)**

**Answer: 15 cm**

---

**(b) Space diagonal**

$$D = \sqrt{9^2 + 12^2 + 20^2}$$ **(M1)**

$$= \sqrt{81 + 144 + 400}$$

$$= \sqrt{625} = 25 \text{ cm}$$ **(A1)**

**Answer: 25 cm**

**Note:** Alternatively, use $D = \sqrt{15^2 + 20^2} = \sqrt{225 + 400} = \sqrt{625} = 25$ cm

This is another Pythagorean triple extended: (9, 12, 15) is a triple, and (15, 20, 25) is also a triple.

---

**Mark Scheme:**
- (a) M1: Correct Pythagoras formula; A1: Correct answer with units
- (b) M1: Correct formula (three terms or using part (a)); A1: Correct answer with units

### Question 22 (4 marks)
A right circular cone has a base radius of 12 cm and a vertical height of 9 cm.

**(a)** Calculate the slant height of the cone. (2 marks)

**(b)** Find the angle between the slant height and the base. Give your answer to 1 decimal place. (2 marks)

**(a) Slant height**

$$l = \sqrt{r^2 + h^2}$$ **(M1)**

$$= \sqrt{12^2 + 9^2}$$

$$= \sqrt{144 + 81} = \sqrt{225} = 15 \text{ cm}$$ **(A1)**

**Answer: 15 cm**

---

**(b) Angle between slant height and base**

```
         apex
          /|
         / |
    15  /  | 9
       /θ  |
      /____|
    base   12
```

$$\cos\theta = \frac{\text{adjacent}}{\text{hypotenuse}} = \frac{12}{15} = 0.8$$ **(M1)**

$$\theta = \cos^{-1}(0.8) \approx 36.9°$$ **(A1)**

**Alternative using tangent:**
$$\tan\theta = \frac{9}{12} = 0.75$$
$$\theta = \tan^{-1}(0.75) \approx 36.9°$$

**Answer: 36.9° (to 1 d.p.)**

---

**Mark Scheme:**
- (a) M1: Correct Pythagoras; A1: Correct answer
- (b) M1: Correct trig ratio used; A1: Correct answer to 1 d.p.

### Question 23 (5 marks)
A square-based pyramid has a base with sides of length 10 cm and a vertical height of 12 cm.

**(a)** Calculate the horizontal distance from the center of the base to a corner of the base. (2 marks)

**(b)** Find the angle that a slant edge makes with the base. Give your answer to 1 decimal place. (3 marks)

**(a) Distance from center to corner**

The distance from center to corner is half the diagonal of the square base.

$$\text{Diagonal of square} = \sqrt{10^2 + 10^2} = \sqrt{200} = 10\sqrt{2} \text{ cm}$$ **(M1)**

$$\text{Distance from center to corner} = \frac{10\sqrt{2}}{2} = 5\sqrt{2} \text{ cm}$$ **(A1)**

**Answer: $5\sqrt{2}$ cm or approximately 7.07 cm**

---

**(b) Angle that slant edge makes with base**

```
         apex
          /|
         / |
   edge /  | 12
       /θ  |
      /____|
   corner  5√2
```

The triangle formed has:
- Height = 12 cm (opposite to angle)
- Horizontal distance = $5\sqrt{2}$ cm (adjacent to angle)

$$\tan\theta = \frac{12}{5\sqrt{2}}$$ **(M1)**

$$= \frac{12}{5\sqrt{2}} \times \frac{\sqrt{2}}{\sqrt{2}} = \frac{12\sqrt{2}}{10} = \frac{6\sqrt{2}}{5}$$ **(M1)**

$$\theta = \tan^{-1}\left(\frac{6\sqrt{2}}{5}\right) \approx 59.5°$$ **(A1)**

**Answer: 59.5° (to 1 d.p.)**

---

**Mark Scheme:**
- (a) M1: Correct method for diagonal or half-diagonal; A1: Correct answer
- (b) M1: Correct triangle identified; M1: Correct trig ratio; A1: Correct answer to 1 d.p.

### Question 24 (6 marks)
The diagram shows a cuboid ABCDEFGH with AB = 16 cm, BC = 12 cm, and AE = 5 cm.

```
    H _______ G
     /|      /|
   E /_|_____/F|
    | |     | |
    |D|_____|_|C
    |/      |/
    A_______B
```

**(a)** Calculate the length of the face diagonal AC. (2 marks)

**(b)** Calculate the length of the space diagonal AG. (2 marks)

**(c)** Find the angle that AG makes with the base ABCD. Give your answer to 1 decimal place. (2 marks)

**(a) Face diagonal AC**

$$AC = \sqrt{AB^2 + BC^2}$$ **(M1)**

$$= \sqrt{16^2 + 12^2} = \sqrt{256 + 144} = \sqrt{400} = 20 \text{ cm}$$ **(A1)**

**Answer: 20 cm**

---

**(b) Space diagonal AG**

Using AC and AE:

$$AG = \sqrt{AC^2 + AE^2} = \sqrt{20^2 + 5^2}$$ **(M1)**

$$= \sqrt{400 + 25} = \sqrt{425} = 5\sqrt{17} \text{ cm}$$

Or approximately: $AG \approx 20.6$ cm **(A1)**

**Answer: $\sqrt{425}$ cm or $5\sqrt{17}$ cm or 20.6 cm**

---

**(c) Angle that AG makes with base**

```
    G
    |\
    | \
  5 |  \ AG
    | θ \
    A----C
      20
```

$$\tan\theta = \frac{AE}{AC} = \frac{5}{20} = 0.25$$ **(M1)**

$$\theta = \tan^{-1}(0.25) \approx 14.0°$$ **(A1)**

**Answer: 14.0° (to 1 d.p.)**

---

- (b) M1: Correct method; A1: Correct answer (exact or decimal)

### Question 25 (5 marks)
A storage tank is in the shape of a cylinder with a conical top. The cylinder has radius 3 m and height 4 m. The cone has the same radius and height 2 m.

**(a)** Find the slant height of the conical top. (2 marks)

**(b)** Find the angle that the slant height makes with the horizontal base of the cone. (2 marks)

**(c)** A safety cable runs from the top of the cone to the ground at a point 5 m from the base of the cylinder. Assuming the cable is straight, find its length if the ground is level with the base of the cylinder. (1 mark)

**(a) Slant height of cone**

$$l = \sqrt{r^2 + h^2} = \sqrt{3^2 + 2^2}$$ **(M1)**

$$= \sqrt{9 + 4} = \sqrt{13} \text{ m}$$ **(A1)**

**Answer: $\sqrt{13}$ m or approximately 3.61 m**

---

**(b) Angle with horizontal**

The angle between the slant height and the horizontal base of the cone:

$$\tan\theta = \frac{\text{height}}{\text{radius}} = \frac{2}{3}$$ **(M1)**

$$\theta = \tan^{-1}\left(\frac{2}{3}\right) \approx 33.7°$$ **(A1)**

**Answer: 33.7° (to 1 d.p.)**

---

**(c) Length of cable**

The cable runs from:
- Top of cone: height = 4 + 2 = 6 m above ground
- Point on ground: 3 + 5 = 8 m horizontally from the axis of the tank (radius + distance from base)

$$\text{Cable length} = \sqrt{6^2 + 8^2}$$ **(M1)**

$$= \sqrt{36 + 64} = \sqrt{100} = 10 \text{ m}$$ **(A1)**

**Answer: 10 m**

---

**Mark Scheme:**
- (a) M1: Correct Pythagoras; A1: Correct answer
- (b) M1: Correct trig; A1: Correct angle
- (c) M1: Correct dimensions used in Pythagoras; A1: Correct length

### Typical Problem-Solving Patterns

**Pattern 1: Cuboid Angle Problem**
```
Given dimensions → Find face diagonal → Use height and diagonal → Find angle
```

**Pattern 2: Pyramid Problem**
```
Given base and height → Find half-diagonal of base → Use with height → Find slant edge or angle
```

**Pattern 3: Multi-Stage**
```
Stage 1: Pythagoras for intermediate length
Stage 2: Use result in another triangle
Stage 3: Trigonometry for final angle
```

4. Check answer is reasonable

1. Show logical steps
2. State what you're finding at each stage
