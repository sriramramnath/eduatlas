# Chapter 21: Graphs 1 - Concise Synopsis
## Concise Synopsis: Straight Line Graphs

---

# Quick Reference Card

## 1. Gradient Formula

$$\boxed{m = \frac{y_2 - y_1}{x_2 - x_1} = \frac{\text{rise}}{\text{run}}}$$

| Gradient Type | Value | Line Direction |
|---------------|-------|----------------|
| Positive | $m > 0$ | Uphill (↗) |
| Negative | $m < 0$ | Downhill (↘) |
| Zero | $m = 0$ | Horizontal (→) |
| Undefined | — | Vertical (↑) |

---

## 2. Equation of a Straight Line

$$\boxed{y = mx + c}$$

| Symbol | Meaning |
|--------|---------|
| **$m$** | Gradient (steepness) |
| **$c$** | Y-intercept (crosses y-axis at $(0, c)$) |

### Alternative Form (Point-Gradient)
$$y - y_1 = m(x - x_1)$$

---

## 3. Parallel & Perpendicular Lines

| Relationship | Condition |
|--------------|-----------|
| **Parallel** | Same gradient: $m_1 = m_2$ |
| **Perpendicular** | $m_1 \times m_2 = -1$ or $m_2 = -\frac{1}{m_1}$ |

**Example:**
- Line $y = 3x + 2$ has gradient $m = 3$
- Parallel line gradient: $m = 3$
- Perpendicular line gradient: $m = -\frac{1}{3}$

---

## 4. Plotting Straight Lines

### Method: Table of Values
1. Choose $x$ values (typically $-3, -2, -1, 0, 1, 2, 3$)
2. Calculate $y$ for each $x$
3. Plot points accurately
4. Draw straight line with ruler

### Quick Method (Gradient-Intercept)
1. Mark the $y$-intercept $(0, c)$
2. From this point, use gradient: rise ÷ run
3. Plot additional points
4. Draw straight line

---

## 5. Conversion Graphs

- **Gradient = Conversion rate**
- Pass through origin if direct proportion

**Example:** £ to $ graph through $(£10, $13)$
- Gradient: $m = \frac{13}{10} = 1.3$
- Conversion: **£1 = $1.30**
- Equation: $D = 1.3P$

---

## 6. Real-Life Graphs

### Distance-Time Graphs

| Feature | Meaning |
|---------|---------|
| Gradient | Speed |
| Steeper line | Faster speed |
| Flat section | Stationary (stopped) |
| Straight line | Constant speed |

$$\text{Speed} = \frac{\text{Distance traveled}}{\text{Time taken}}$$

### Speed-Time Graphs

| Feature | Meaning |
|---------|---------|
| Gradient | Acceleration |
| Flat section | Constant speed |
| Area under graph | Distance traveled |

---

## 7. Midpoint Formula

$$\boxed{\text{Midpoint} = \left(\frac{x_1 + x_2}{2}, \frac{y_1 + y_2}{2}\right)}$$

**Example:** Midpoint of $(2, 4)$ and $(6, 10)$:
$$\left(\frac{2+6}{2}, \frac{4+10}{2}\right) = (4, 7)$$

---

# Essential Rules

## ✅ Do's
- ✅ Always rearrange to $y = mx + c$ form first
- ✅ Show working when calculating gradients
- ✅ Use ruler for straight lines
- ✅ Check scale on both axes
- ✅ Include units in real-life context questions

## ❌ Don'ts
- ❌ Mix up rise and run in gradient formula
- ❌ Forget negative signs in coordinates
- ❌ Say vertical lines have gradient 0
- ❌ Round too early in calculations
- ❌ Forget units when interpreting graphs

# High-Yield Exam Points

| Topic | Key Points |
|-------|------------|
| **Gradient** | y-difference ÷ x-difference; positive = uphill, negative = downhill |
| **Equation** | Rearrange to $y = mx + c$; $m$ = gradient, $c$ = y-intercept |
| **Parallel** | Same gradient, different y-intercept |
| **Perpendicular** | Gradients multiply to $-1$ (negative reciprocal) |
| **Distance-time** | Gradient = speed; flat = stopped |
| **Conversion** | Gradient = rate; equation from any point |

---

# Common Exam Traps

1. **"Find gradient of $x = 3$"** → Undefined (vertical line)
2. **"Find gradient of $y = 5$"** → Zero (horizontal line)
3. **Equation $2y = 4x + 6$"** → Divide by 2 first: $y = 2x + 3$
4. **"Perpendicular to $y = \frac{1}{3}x$"** → Gradient is $-3$, not $3$
5. **Reading scales** → Check if grid squares = 1, 2, 5, or 10 units
