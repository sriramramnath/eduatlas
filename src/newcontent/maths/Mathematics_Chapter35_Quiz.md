### Question 1

The frequency table shows the number of pets owned by students in a class:

| Number of pets | Frequency |
|----------------|-----------|
| 0              | 6         |
| 1              | 12        |
| 2              | 8         |
| 3              | 4         |

What is the total number of students?

**A** 28  
**B** 30  
**C** 32  
**D** 6  

**Answer: B**

**Explanation:** Total = 6 + 12 + 8 + 4 = 30 students. The sum of all frequencies equals the total number of data items.

### Question 2

Which of the following is an example of **continuous data**?

**A** Number of goals scored in a match  
**B** Shoe size  
**C** Height of students  
**D** Number of siblings  

**Answer: C**

**Explanation:** Height is measured and can take any value within a range (e.g., 165.3 cm). Goals scored, shoe size (discrete categories), and number of siblings are all countable and therefore discrete.

### Question 3

For the class interval 20–29, what is the midpoint?

**A** 24  
**B** 24.5  
**C** 25  
**D** 25.5  

**Answer: B**

**Explanation:** Midpoint = $\frac{20 + 29}{2} = \frac{49}{2} = 24.5$

### Question 4

A grouped frequency table has classes 0–9, 10–19, 20–29. What are the class boundaries for the class 10–19?

**A** 10 and 19  
**B** 10.5 and 19.5  
**C** 9.5 and 19.5  
**D** 9 and 20  

**Answer: C**

**Explanation:** The gap between classes is $10 - 9 = 1$. Half the gap is 0.5. Subtract 0.5 from the lower limit: $10 - 0.5 = 9.5$. Add 0.5 to the upper limit: $19 + 0.5 = 19.5$. Class boundaries are 9.5 and 19.5.

### Question 5

The table shows the masses of parcels:

| Mass (kg) | Frequency |
|-----------|-----------|
| 0–10      | 8         |
| 10–20     | 15        |
| 20–30     | 12        |

What is the **modal class**?

**A** 0–10 kg  
**B** 10–20 kg  
**C** 20–30 kg  
**D** Cannot be determined  

**Answer: B**

**Explanation:** The modal class is the class with the highest frequency. The frequency for 10–20 kg is 15, which is the highest.

### Question 6

Which graph is most appropriate for displaying continuous data?

**A** Bar chart with gaps  
**B** Pictogram  
**C** Histogram without gaps  
**D** Pie chart  

**Answer: C**

**Explanation:** Continuous data should be displayed using a histogram with no gaps between bars, as the data values are continuous along a scale.

### Question 7

The cumulative frequency for the last class in a frequency table is 80. What is the total frequency?

**A** 80  
**B** 40  
**C** Cannot be determined  
**D** Depends on class width  

**Answer: A**

**Explanation:** The cumulative frequency for the last class equals the total frequency (all data items up to and including that class).

### Question 8

Given the grouped data:

| Score | Frequency |
|-------|-----------|
| 0–20  | 10        |
| 21–40 | 20        |
| 41–60 | 15        |
| 61–80 | 5         |

Which class contains the median?

**A** 0–20  
**B** 21–40  
**C** 41–60  
**D** 61–80  

**Answer: B**

**Explanation:** Total frequency $n = 10 + 20 + 15 + 5 = 50$. Median position = $\frac{50}{2} = 25$. Cumulative frequencies: 10, 30, 45, 50. The 25th value falls in the class 21–40 (cumulative frequency reaches 30 in this class).

### Question 9

What is the relative frequency of a class with frequency 12 if the total frequency is 60?

**A** 0.12  
**B** 0.2  
**C** 0.5  
**D** 5  

**Answer: B**

**Explanation:** Relative frequency = $\frac{\text{Frequency}}{\text{Total frequency}} = \frac{12}{60} = 0.2$

### Question 10

Which of the following is **discrete data**?

**A** Time taken to run 100m  
**B** Temperature in a room  
**C** Number of books on a shelf  
**D** Weight of a bag of rice  

**Answer: C**

**Explanation:** Number of books can only take whole number values (0, 1, 2, 3, ...) and can be counted. Time, temperature, and weight are all measured and can take any value (continuous).

### Question 11

For the class interval $30 \leq x < 40$, which value is **included** in this class?

**A** 29.9  
**B** 30  
**C** 40  
**D** 40.1  

**Answer: B**

**Explanation:** The notation $30 \leq x < 40$ means $x$ is greater than or equal to 30, and less than 40. So 30 is included (30 ≤ 30 is true), but 40 is not (40 < 40 is false).

### Question 12

A histogram shows data with class widths 5, 10, and 20. Which statement is correct?

**A** All bars should have the same width  
**B** Frequency density should be used  
**C** The histogram cannot be drawn  
**D** Heights directly represent frequency  

**Answer: B**

**Explanation:** When class widths are unequal, frequency density = $\frac{\text{frequency}}{\text{class width}}$ must be calculated. The area of each bar represents the frequency, not the height.

## SECTION B: TRUE OR FALSE

**Instructions:** State whether each statement is TRUE or FALSE. If FALSE, provide the correct statement.

---

### Question 1

**Statement:** The mean calculated from grouped data is always exact.

**Answer: FALSE**

**Correction:** The mean calculated from grouped data is an **estimate** because we assume all values in a class are at the midpoint.

---

### Question 2

**Statement:** Discrete data can only take whole number values.

**Answer: FALSE**

**Correction:** Discrete data takes **specific, separate values**. These are often whole numbers but can include values like shoe size (3, 3.5, 4, 4.5...). The key is that there are gaps between possible values.

---

### Question 3

**Statement:** The sum of all relative frequencies should equal 1.

**Answer: TRUE**

---

### Question 4

**Statement:** Bar charts and histograms are interchangeable for any type of data.

**Answer: FALSE**

**Correction:** Bar charts are for **discrete data** (with gaps between bars), while histograms are for **continuous data** (without gaps between bars).

---

### Question 5

**Statement:** The modal class is the class with the largest class width.

**Answer: FALSE**

**Correction:** The modal class is the class with the **highest frequency**.

---

### Question 6

**Statement:** Class boundaries are the same as class limits.

**Answer: FALSE**

**Correction:** Class limits are the stated values (e.g., 10–19), while class boundaries are the true mathematical limits (e.g., 9.5–19.5) found by adjusting for the gap between classes.

---

### Question 7

**Statement:** For continuous data, a value of exactly 20 kg could belong to either class 10–20 or class 20–30.

**Answer: TRUE** (This is why we need clear class boundaries or inequality notation)

---

### Question 8

**Statement:** Cumulative frequency is calculated by multiplying frequencies together.

**Answer: FALSE**

**Correction:** Cumulative frequency is calculated by **adding** frequencies progressively (running total).

---

### Question 9

**Statement:** Height, weight, and temperature are all examples of continuous data.

**Answer: TRUE**

---

### Question 10

**Statement:** When finding the median from grouped data, we use the position $\frac{n+1}{2}$.

**Answer: FALSE**

**Correction:** For grouped data, we use position $\frac{n}{2}$ (not $\frac{n+1}{2}$). The position $\frac{n+1}{2}$ is used for ungrouped data when finding the median from a list.

---

### Question 11

**Statement:** The interquartile range is found by subtracting the lower quartile from the upper quartile.

**Answer: TRUE**

---

### Question 12

**Statement:** In a histogram, the height of each bar represents the frequency.

**Answer: FALSE**

**Correction:** In a histogram, the **area** of each bar represents the frequency. When class widths are unequal, the height (frequency density) = $\frac{\text{frequency}}{\text{class width}}$.

### Question 2 [3 marks]

A frequency table shows the time taken to complete a puzzle:

| Time (seconds) | Frequency |
|----------------|-----------|
| 0–20           | 6         |
| 21–40          | 14        |
| 41–60          | 18        |
| 61–80          | 8         |
| 81–100         | 4         |

**(a)** Find the midpoint of each class. [1]

**(b)** Calculate an estimate for the mean time. [2]

**Solution:**

**(a)** Midpoints: 10, 30, 50, 70, 90

**(b)**

| Time (s) | $f$ | Midpoint ($x$) | $fx$ |
|----------|-----|----------------|------|
| 0–20     | 6   | 10             | 60   |
| 21–40    | 14  | 30             | 420  |
| 41–60    | 18  | 50             | 900  |
| 61–80    | 8   | 70             | 560  |
| 81–100   | 4   | 90             | 360  |
| **Total**| **50** |              | **2300** |

$$\bar{x} = \frac{\sum fx}{\sum f} = \frac{2300}{50} = 46 \text{ seconds}$$

**Mark Scheme:**
- (a) A1: All midpoints correct
- (b) M1: Correct method using $\bar{x} = \frac{\sum fx}{\sum f}$ with midpoints
- A1: Correct answer (46 or 46.0 seconds)

### Question 4 [3 marks]

Classify each of the following as discrete or continuous data. Give a reason for each answer.

**(a)** The number of students in a class [1]

### Question 5 [4 marks]

A grouped frequency table shows the masses of parcels:

| Mass (kg) | Frequency |
|-----------|-----------|
| 0–5       | 10        |
| 5–10      | 25        |
| 10–15     | 35        |
| 15–20     | 20        |
| 20–25     | 10        |

**(a)** State the class boundaries for the class 10–15 kg. [2]

**(b)** Calculate the percentage frequency for the class 15–20 kg. [2]

**Solution:**

**(a)** The gap between classes is $5 - 0 = 5$, but actually looking at the classes 0–5, 5–10, etc., there is **no gap** (0–5, then 5–10).

However, using the standard approach for continuous data with class intervals like this:
- Lower boundary: 10 kg
- Upper boundary: 15 kg

Or if treating as overlapping boundaries needing adjustment:
- Gap between upper of one class and lower of next = 0
- So boundaries are at the stated values.

**Class boundaries: 10 kg and 15 kg**

**(b)** Total frequency = $10 + 25 + 35 + 20 + 10 = 100$

Percentage frequency = $\frac{20}{100} \times 100\% = 20\%$

**Mark Scheme:**
- (a) M1: Understanding the concept of class boundaries
- A1: Correct boundaries (10 and 15 kg, or correct reasoning)
- (b) M1: Correct total and/or percentage calculation method
- A1: Correct answer (20%)

### Question 6 [5 marks]

The table shows the distances travelled by 60 cyclists in one week:

| Distance (km) | Frequency |
|---------------|-----------|
| 0–50          | 8         |
| 50–100        | 15        |
| 100–150       | 22        |
| 150–200       | 10        |
| 200–250       | 5         |

**(a)** Calculate an estimate for the mean distance. [3]

**(b)** Write down the modal class. [1]

**(c)** State, with a reason, whether distance is discrete or continuous data. [1]

**Solution:**

**(a)**

| Distance (km) | $f$ | Midpoint ($x$) | $fx$ |
|---------------|-----|----------------|------|
| 0–50          | 8   | 25             | 200  |
| 50–100        | 15  | 75             | 1125 |
| 100–150       | 22  | 125            | 2750 |
| 150–200       | 10  | 175            | 1750 |
| 200–250       | 5   | 225            | 1125 |
| **Total**     | **60** |              | **6950** |

$$\bar{x} = \frac{6950}{60} = 115.83... \approx 115.8 \text{ km (to 1 d.p.)}$$

**(b)** Modal class: **100–150 km** (highest frequency = 22)

**(c)** **Continuous** – distance is measured and can take any value within a range (e.g., 125.7 km is possible).

**Mark Scheme:**
- (a) M1: Correct midpoints calculated
- M1: Correct $\sum fx$ and $\sum f$ values
- A1: Correct mean (accept 115.8, 116, or 115.83)
- (b) B1: Correct modal class
- (c) B1: Correct classification with reason

**(b)** Find the class containing the median. [2]

**Solution:**

### Question 8 [5 marks]

A histogram is to be drawn for the following data:

| Time (minutes) | Frequency |
|----------------|-----------|
| 0–10           | 12        |
| 10–20          | 18        |
| 20–40          | 30        |
| 40–50          | 15        |

**(a)** State which class has an unusual width. [1]

**(b)** Calculate the frequency density for each class. [3]

**(c)** If the frequency density for 0–10 is represented by a bar of height 1.2 units, what would be the height of the bar for the 20–40 class? [1]

**Solution:**

**(a)** The class **20–40** has width 20 (double the others which have width 10).

**(b)** Frequency density = $\frac{\text{Frequency}}{\text{Class Width}}$

| Time (min) | $f$ | Width | Frequency Density |
|------------|-----|-------|-------------------|
| 0–10       | 12  | 10    | $\frac{12}{10} = 1.2$ |
| 10–20      | 18  | 10    | $\frac{18}{10} = 1.8$ |
| 20–40      | 30  | 20    | $\frac{30}{20} = 1.5$ |
| 40–50      | 15  | 10    | $\frac{15}{10} = 1.5$ |

**(c)** The height of the 20–40 bar would be **1.5 units** (the frequency density).

**Mark Scheme:**
- (a) B1: Correct identification of 20–40 class
- (b) M1: Correct formula for frequency density
- M1: At least two correct frequency densities
- A1: All four frequency densities correct
- (c) B1: Correct height (1.5 units)

## Typical Problem-Solving Patterns

### Pattern 1: Estimating Mean

```
Step 1: Draw table with columns: Class, f, x (midpoint), fx
Step 2: Calculate all midpoints
Step 3: Multiply f × x for each row
Step 4: Sum fx column and f column
Step 5: Apply formula: mean = Σfx ÷ Σf
Step 6: Round to appropriate accuracy + include units
```

### Pattern 2: Finding Median Class

```
Step 1: Add cumulative frequency column
Step 2: Calculate total (n)
Step 3: Find median position = n/2
Step 4: Find which class contains this position
Step 5: State the median class
```

### Pattern 3: Data Classification

```
Step 1: Ask "Can it be counted?"
Step 2: Ask "Can it take any value (like 2.5)?"
Step 3: If counted AND specific values only → Discrete
Step 4: If measured AND any value possible → Continuous
```

## Common Mistakes to Avoid

| ❌ Mistake | ✅ Correct Approach |
|-----------|---------------------|
| Using limits instead of midpoints | Always use midpoints for grouped mean |
| Gaps in histogram bars | No gaps for continuous data |
| Wrong median position | Use n/2 for grouped data |
| Missing units | Always include units in final answer |
| Forgetting to check totals | Verify Σf = total data items |
| Confusing class limits with boundaries | Boundaries adjust for gaps |

---

## Final Exam Tips

> ** Tip 1:** Always show your working—marks are given for method even if the answer is wrong.

> ** Tip 2:** For estimated mean, explicitly state "Estimated mean = ..." to show understanding.

> ** Tip 3:** When classifying data, always give a reason (key words: "counted," "measured," "any value").

> ** Tip 4:** Check that the last cumulative frequency equals the total frequency—a quick way to verify your table.

> ** Tip 5:** In histogram questions with unequal class widths, remember: area = frequency, height = frequency density.
