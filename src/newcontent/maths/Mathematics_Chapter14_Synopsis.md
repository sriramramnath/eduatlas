## 1. USING FORMULAE – QUICK RULES

### What is a Formula?
- An equation showing relationship between **2+ variables**
- The **subject** appears alone on one side (usually left)
- Example: In $A = lw$, **$A$ is the subject**

### Creating Formulae from Words
| Words → Formula | Example |
|-----------------|---------|
| "multiply x by 5" | $5x$ |
| "add 3 to y" | $y + 3$ |
| "square n" | $n^2$ |
| "divide a by b" | $\frac{a}{b}$ |

**Example:** "Cost $C$ is £20 deposit plus £3 per hour $h$"
$$C = 20 + 3h$$

## 2. CHANGE OF SUBJECT – QUICK RULES

### The Golden Rule
> **Whatever you do to one side, do to the other**

### Inverse Operations Table
| Operation | Inverse |
|-----------|---------|
| $+a$ | $-a$ |
| $-a$ | $+a$ |
| $\times a$ | $\div a$ |
| $\div a$ | $\times a$ |
| $x^2$ | $\sqrt{x}$ |
| $x^3$ | $\sqrt[3]{x}$ |
| $\sqrt{x}$ | $x^2$ |

### Order of Rearranging (Reverse BIDMAS)
1. **First:** Remove additions/subtractions
2. **Then:** Remove multiplications/divisions
3. **Finally:** Remove powers/roots

### Quick Examples

| Original | Make New Subject | Answer |
|----------|------------------|--------|
| $C = 2\pi r$ | Make $r$ subject | $r = \frac{C}{2\pi}$ |
| $A = \frac{1}{2}bh$ | Make $h$ subject | $h = \frac{2A}{b}$ |
| $v = u + at$ | Make $t$ subject | $t = \frac{v-u}{a}$ |
| $A = \pi r^2$ | Make $r$ subject | $r = \sqrt{\frac{A}{\pi}}$ |
| $V = \frac{4}{3}\pi r^3$ | Make $r$ subject | $r = \sqrt[3]{\frac{3V}{4\pi}}$ |

### Subject Appears Twice? → **FACTORISE FIRST!**

**Example:** Make $r$ the subject of $A = \pi r^2 + 2\pi rh$
$$A = \pi r(r + 2h)$$
$$r = \frac{A}{\pi(r + 2h)}$$

## 3. SUBSTITUTION – QUICK RULES

### Essential Rules
- ✅ **Use brackets** for negative numbers: $(-3)^2 = 9$
- ✅ **Use brackets** for fractions: $\frac{1}{2} \times \frac{3}{4}$
- ✅ **Follow BIDMAS** in calculations
- ❌ **Never** skip steps in exams

### Substitution Examples

| Formula | Values | Calculation | Answer |
|---------|--------|-------------|--------|
| $A = lw$ | $l = 5, w = 3$ | $5 \times 3$ | $15$ |
| $A = \frac{1}{2}bh$ | $b = 10, h = 6$ | $\frac{1}{2} \times 10 \times 6$ | $30$ |
| $v = u + at$ | $u = 2, a = 3, t = 4$ | $2 + 3 \times 4$ | $14$ |
| $y = x^2 - 3x$ | $x = -2$ | $(-2)^2 - 3(-2)$ | $10$ |

### Negative Number Warning!
| Expression | Correct | Wrong |
|------------|---------|-------|
| $(-5)^2$ | $25$ | — |
| $-5^2$ | — | $-25$ |
| $(-3)^3$ | $-27$ | — |

## 4. KEY FORMULAE TABLE

### 2D Shapes
| Shape | Area Formula |
|-------|--------------|
| Rectangle | $A = lw$ |
| Triangle | $A = \frac{1}{2}bh$ |
| Circle | $A = \pi r^2$ |
| Trapezium | $A = \frac{1}{2}(a+b)h$ |

### 3D Shapes
| Shape | Volume | Surface Area |
|-------|--------|--------------|
| Cylinder | $V = \pi r^2 h$ | $A = 2\pi r(r+h)$ |
| Cone | $V = \frac{1}{3}\pi r^2 h$ | $A = \pi r(l+r)$ |
| Sphere | $V = \frac{4}{3}\pi r^3$ | $A = 4\pi r^2$ |

### Circle
- Circumference: $C = 2\pi r$ or $C = \pi d$
- Area: $A = \pi r^2$

### Motion Formulae (SUVAT)
| Formula | Meaning |
|---------|---------|
| $v = u + at$ | Final velocity |
| $s = ut + \frac{1}{2}at^2$ | Displacement |
| $v^2 = u^2 + 2as$ | Velocity squared |
| $s = \frac{(u+v)}{2}t$ | Average velocity |

### Other Important Formulae
| Formula | Use |
|---------|-----|
| $I = \frac{PRT}{100}$ | Simple interest |
| $A = P(1+\frac{r}{100})^n$ | Compound interest |
| $F = \frac{9C}{5} + 32$ | Celsius to Fahrenheit |
| $C = \frac{5(F-32)}{9}$ | Fahrenheit to Celsius |

## 5. COMMON MISTAKES 

| Mistake | Example | Correction |
|---------|---------|------------|
| Wrong sign when moving | $x - 5 = 3$ → $x = 3 - 5$ | $x = 3 + 5 = 8$ |
| Forgetting brackets | $(-3)^2 = -9$ | $(-3)^2 = 9$ |
| Dividing only one term | $\frac{x+y}{2} = \frac{x}{2} + y$ | $\frac{x+y}{2} = \frac{x}{2} + \frac{y}{2}$ |
| Expanding wrongly | $(x+2)^2 = x^2 + 4$ | $(x+2)^2 = x^2 + 4x + 4$ |
| Not factorising | Can't solve $ax + bx = c$ | $x(a+b) = c$, so $x = \frac{c}{a+b}$ |
