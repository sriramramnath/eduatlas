# Topic 1: Comparative Costs

## Key Concepts
- **Best value** = Lowest cost per unit (NOT lowest total price)
- Always compare using the **same units**
- Consider all offers: discounts, multi-buys, buy X get Y free

---

## Essential Formulas

| Formula | Description |
|---------|-------------|
| $\text{Unit Price} = \frac{\text{Total Cost}}{\text{Quantity}}$ | Cost per single unit |
| $\text{Discounted Price} = \text{Original} \times (1 - \frac{\text{discount\%}}{100})$ | After percentage discount |

---

## Quick Rules

✅ **DO:** Calculate unit price for ALL options before deciding

✅ **DO:** Convert to same units first (kg ↔ g, L ↔ mL)

✅ **DO:** For "buy X get Y free": Effective cost = $\frac{X}{X+Y} \times$ original price

✅ **DO:** Calculate total cost for the quantity you actually need

---

## Best Value Comparison Steps
1. Convert to same units
2. Calculate unit price for each option
3. Compare unit prices
4. State which is better value with numerical reason

# Topic 2: Taxation

## Key Terms

| Term | Definition |
|------|------------|
| **Gross Pay** | Total earnings before deductions |
| **Net Pay** | Take-home pay after deductions |
| **Personal Allowance** | Tax-free income threshold |
| **Taxable Income** | Gross Pay minus Personal Allowance |

---

## Pay Formulas

| Formula | Description |
|---------|-------------|
| $\text{Monthly Salary} = \frac{\text{Annual Salary}}{12}$ | Monthly from annual |
| $\text{Weekly Salary} = \frac{\text{Annual Salary}}{52}$ | Weekly from annual |
| $\text{Basic Pay} = \text{Hours} \times \text{Hourly Rate}$ | Standard wages |
| $\text{Net Pay} = \text{Gross Pay} - \text{Deductions}$ | Take-home pay |

---

## Overtime Multipliers

| Type | Multiplier |
|------|------------|
| Time-and-a-half | $1.5 \times$ normal rate |
| Double time | $2 \times$ normal rate |

---

## Income Tax Calculation Steps

```
Step 1: Gross Annual Income
Step 2: Taxable Income = Gross − Personal Allowance
Step 3: Apply tax bands sequentially
Step 4: Sum tax from all bands
```

---

## Tax Band Example Structure

| Band | Income Range | Rate |
|------|--------------|------|
| Personal Allowance | First £12,570 | 0% |
| Basic Rate | Next £37,700 | 20% |
| Higher Rate | Remaining | 40% |

> **Note:** Values vary by year - always use exam question values

---

## Tax Calculation Formula

$$\text{Tax} = \sum(\text{Income in Band} \times \text{Band Rate})$$

---

## Commission Formula

$$\text{Commission} = \frac{\text{Rate}}{100} \times \text{Sales Value}$$

$$\text{Total Earnings} = \text{Basic Salary} + \text{Commission}$$

# Topic 3: Foreign Currency Exchange

## Exchange Rate Interpretation

| Notation | Meaning |
|----------|---------|
| £1 = $1.25 | £1 buys $1.25 dollars |
| €1 = £0.86 | €1 buys £0.86 pounds |

---

## Conversion Formulas

| Direction | Formula |
|-----------|---------|
| Home → Foreign | $\text{Foreign} = \text{Home} \times \text{Rate}$ |
| Foreign → Home | $\text{Home} = \frac{\text{Foreign}}{\text{Rate}}$ |

---

## Quick Conversion Rules

🔹 **Pounds to foreign currency:** MULTIPLY by rate

🔹 **Foreign currency to pounds:** DIVIDE by rate

🔹 **Bank selling you currency:** Use the rate that gives you LESS

🔹 **Bank buying from you:** Use the rate that gives you LESS

---

## Commission Calculation

| Type | Formula |
|------|---------|
| Percentage | $\text{Fee} = \frac{\text{Rate}}{100} \times \text{Amount}$ |
| With minimum fee | $\text{Fee} = \max(\text{Calculated Fee}, \text{Minimum Fee})$ |

---

## Currency Exchange Total Cost

$$\text{Total Cost} = \text{Exchange Amount} + \text{Commission/Fees}$$

$$\text{Dollars Received} = (\text{Pounds} - \text{Commission}) \times \text{Rate}$$

# High-Yield Exam Points

##  Most Common Mistakes

| Mistake | Correction |
|---------|------------|
| Comparing total prices instead of unit prices | Always calculate cost per unit |
| Forgetting to subtract personal allowance | Taxable income ≠ Gross income |
| Applying same tax rate to all income | Use correct bands with different rates |
| Using wrong conversion direction | Pounds × rate = foreign; foreign ÷ rate = pounds |
| Ignoring commission/fees | Add fees to cost or subtract from amount |
| Using bank buy rate when selling | Match rate to who is buying/selling |

---

## Exam Technique Tips

✅ **Show all working** - method marks save you

✅ **Work in consistent units** - convert at the start

✅ **Draw a table for tax bands** - organizes calculations

✅ **Check reasonableness** - net pay should be less than gross

✅ **Round only at the end** - premature rounding loses accuracy

✅ **State units in answers** - £, $, €, kg, etc.

# Quick Reference: Three Key Topics

## 📊 Comparative Costs
```
Unit Price = Cost ÷ Quantity
Lower unit price = Better value
Convert units first if needed
```

## 💰 Taxation
```
Taxable Income = Gross − Personal Allowance
Tax = Sum of (Band Income × Band Rate)
Net Pay = Gross − Tax − NI − Other Deductions
```

## 💱 Currency Exchange
```
£ → Foreign: Multiply
Foreign → £: Divide
Include fees in calculations
Bank always gives worse rate
```
