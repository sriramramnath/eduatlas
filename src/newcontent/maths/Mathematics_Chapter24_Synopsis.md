# 1. Quadratic Graphs Basics

## Standard Form
$$y = ax^2 + bx + c$$

Where:
- $a \neq 0$
- Graph is a **parabola** (U-shaped or ∩-shaped)

## Key Features

| Feature | Description | How to Find |
|---------|-------------|-------------|
| **Shape** | Parabola | Determined by $a$ |
| **y-intercept** | Where curve crosses y-axis | At $(0, c)$ |
| **x-intercepts** | Where curve crosses x-axis | Solve $ax^2 + bx + c = 0$ |
| **Vertex** | Turning point | $x = -\frac{b}{2a}$ |
| **Axis of symmetry** | Line through vertex | $x = -\frac{b}{2a}$ |

---

# 2. Shape of Parabolas

## Direction of Opening

| Value of $a$ | Direction | Vertex Type |
|--------------|-----------|-------------|
| $a > 0$ | Opens **upwards** (U) | **Minimum** |
| $a < 0$ | Opens **downwards** (∩) | **Maximum** |

## Width of Parabola

| Value of $\|a\|$ | Effect |
|------------------|--------|
| Large ($> 1$) | **Narrower** parabola |
| Small ($0 < \|a\| < 1$) | **Wider** parabola |
| $\|a\| = 1$ | Standard width |

---

# 3. Key Formulas

## Vertex Coordinates

> **x-coordinate:**
> $$x = -\frac{b}{2a}$$

> **y-coordinate:** Substitute $x$ value into the equation

## Axis of Symmetry
$$x = -\frac{b}{2a}$$

## Quadratic Formula (for x-intercepts)
$$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$

## Discriminant
$$\Delta = b^2 - 4ac$$

| Discriminant | x-intercepts | Graph Position |
|--------------|--------------|----------------|
| $\Delta > 0$ | Two distinct | Crosses x-axis twice |
| $\Delta = 0$ | One (repeated) | Touches x-axis |
| $\Delta < 0$ | None | Does not cross x-axis |

---

# 4. Drawing Quadratic Graphs

## Method: Table of Values

1. **Choose x-values** (typically 5-7 points)
2. **Substitute** each x-value into equation
3. **Calculate** corresponding y-values
4. **Plot** all points on grid
5. **Join** with smooth curve

## Quick Example: $y = x^2 - 4x + 3$

| $x$ | 0 | 1 | 2 | 3 | 4 |
|-----|-----|-----|-----|-----|-----|
| $y$ | 3 | 0 | -1 | 0 | 3 |

- y-intercept: $(0, 3)$
- x-intercepts: $(1, 0)$ and $(3, 0)$
- Vertex: $(2, -1)$

---

# 5. Solving Equations Graphically

## Type 1: $ax^2 + bx + c = 0$

**Method:**
1. Draw $y = ax^2 + bx + c$
2. Find where curve crosses **x-axis**
3. Read off x-values

## Type 2: $ax^2 + bx + c = k$

**Method:**
1. Draw $y = ax^2 + bx + c$
2. Draw line $y = k$
3. Find **intersection points**
4. Read off x-values

## Type 3: Comparing Two Graphs

**Example:** Solve $x^2 = 2x + 3$
1. Draw $y = x^2$ and $y = 2x + 3$
2. Find **intersection points**
3. Read off x-values

---

# 6. Maximum and Minimum Values

## Finding Maximum/Minimum

| Method | Steps |
|--------|-------|
| **Graphical** | Read y-coordinate of vertex |
| **Algebraic** | Calculate $x = -\frac{b}{2a}$, then substitute |

## Quick Rule
- If $a > 0$: **Minimum** value at vertex
- If $a < 0$: **Maximum** value at vertex

---

# 7. Effects of Parameters

## Effect of $a$ in $y = ax^2 + bx + c$

| Change | Effect on Graph |
|--------|-----------------|
| Increase $\|a\|$ | Narrower |
| Decrease $\|a\|$ | Wider |
| $a$ positive | Opens up |
| $a$ negative | Opens down |

## Effect of $c$

| Value of $c$ | Effect |
|--------------|--------|
| $c > 0$ | Shift **up** |
| $c < 0$ | Shift **down** |
| $c$ | y-intercept |

---

# 8. Quick Reference: Key Points

## For $y = x^2$
- Standard parabola
- Vertex at origin $(0, 0)$
- Opens upwards
- Minimum at $y = 0$

## For $y = -x^2$
- Same shape as $y = x^2$
- Opens **downwards**
- Maximum at $y = 0$

## For $y = ax^2$
- Width depends on $\|a\|$
- Direction depends on sign of $a$
- Vertex at $(0, 0)$

## For $y = ax^2 + c$
- Shifted up/down by $c$
- y-intercept = $c$

# 11. Common Mistakes to Avoid

| ❌ Mistake | ✅ Correction |
|------------|---------------|
| $(-x)^2 = -x^2$ | $(-x)^2 = x^2$ |
| Joining points with straight lines | Use smooth curve |
| Confusing x and y intercepts | x-intercept: where $y = 0$; y-intercept: where $x = 0$ |
| Wrong direction of parabola | Check sign of $a$ |
| Reading wrong values from graph | Check scale carefully |
| Not extending curve | Draw beyond plotted points |

# 12. Quick Problem-Solving Steps

## Drawing $y = ax^2 + bx + c$:
1. Create table with 5-7 x-values
2. Calculate each y-value carefully
3. Plot points on grid
4. Join with smooth curve

## Solving $ax^2 + bx + c = 0$:
1. Find where curve crosses x-axis
2. Read x-values

## Solving $ax^2 + bx + c = k$:
1. Draw horizontal line at $y = k$
2. Find intersections with curve
3. Read x-values

## Finding Vertex:
1. Calculate $x = -\frac{b}{2a}$
2. Substitute into equation for y
3. State coordinates $(x, y)$

# 13. Worked Quick Examples

## Example 1: Find Vertex
**$y = x^2 - 6x + 5$**

$$x = -\frac{-6}{2(1)} = 3$$

$$y = 3^2 - 6(3) + 5 = 9 - 18 + 5 = -4$$

**Vertex: $(3, -4)$** (minimum since $a > 0$)

---

## Example 2: Find Intercepts
**$y = x^2 - 5x + 6$**

**y-intercept:** $(0, 6)$

**x-intercepts:** 
$$x^2 - 5x + 6 = 0$$
$$(x - 2)(x - 3) = 0$$
$$x = 2 \text{ or } x = 3$$

**x-intercepts: $(2, 0)$ and $(3, 0)$**

---

## Example 3: Describe Shape
**$y = -2x^2 + 4x - 1$**

- $a = -2 < 0$ → Opens **downwards** (∩-shape)
- $\|a\| = 2 > 1$ → **Narrower** than $y = x^2$
- Vertex is a **maximum** point
- y-intercept at $(0, -1)$

---

## Example 4: Discriminant Check
**$y = x^2 + 2x + 5$**

$$\Delta = b^2 - 4ac = 2^2 - 4(1)(5) = 4 - 20 = -16$$

Since $\Delta < 0$: **No x-intercepts** (doesn't cross x-axis)

** Key Takeaways:**
- Parabola shape depends on coefficient $a$
- Vertex formula: $x = -\frac{b}{2a}$
- y-intercept always at $(0, c)$
- Graphical solutions: read x-values where curve meets axis or line
- Use smooth curves when drawing
