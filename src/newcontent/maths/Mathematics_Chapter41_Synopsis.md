# 📐 ESSENTIAL FORMULA TABLE

## Angle Sums

| Shape | Formula |
|-------|---------|
| Triangle | $180°$ |
| Quadrilateral | $360°$ |
| $n$-sided polygon | $(n-2) \times 180°$ |
| Exterior angles (any polygon) | $360°$ |

---

## Polygon Formulas

| Quantity | Formula |
|----------|---------|
| Interior angle sum | $(n-2) \times 180°$ |
| Each interior angle (regular) | $\frac{(n-2) \times 180°}{n}$ |
| Each exterior angle (regular) | $\frac{360°}{n}$ |
| Number of sides from exterior angle | $n = \frac{360°}{\text{exterior angle}}$ |

---

## Similar Shapes

| Relationship | Formula |
|--------------|---------|
| Scale factor | $k = \frac{\text{image length}}{\text{object length}}$ |
| Area ratio | $\frac{A_1}{A_2} = \left(\frac{L_1}{L_2}\right)^2 = k^2$ |

# QUICK RULES

## Triangle Rules

- ✅ Angles sum to **180°**
- ✅ Exterior angle = sum of two interior opposite angles
- ✅ Isosceles: two equal sides → two equal angles
- ✅ Equilateral: all sides equal → all angles = **60°**
- ✅ Vertically opposite angles are equal

---

## Quadrilateral Rules

- ✅ Angles sum to **360°**
- ✅ Parallelogram: opposite angles equal, adjacent angles supplementary
- ✅ Kite: one pair of equal opposite angles (between unequal sides)
- ✅ Rectangle: all angles = **90°**
- ✅ Square: all sides equal, all angles = **90°**

---

## Similarity Conditions

**Triangles are similar if:**

| Condition | Meaning |
|-----------|---------|
| **AAA** | Three pairs of equal angles |
| **SSS** | Three pairs of proportional sides |
| **SAS** | Two proportional sides + equal included angle |

>  **TIP:** For AAA, proving two angles equal is sufficient (third follows automatically)

# 🏗️ CONSTRUCTIONS CHECKLIST

## Required Constructions

| Construction | Key Steps |
|--------------|-----------|
| **Perpendicular bisector** | Arcs from both ends → connect intersections |
| **Angle bisector** | Arc from vertex → arcs from intersections → connect |
| **60° angle** | Arc from point → arc with same radius → equilateral triangle |
| **Triangle (SSS)** | Base line + arcs from both ends with given radii |
| **Triangle (SAS)** | Base line + angle + side length |
| **Triangle (ASA)** | Base line + two angles |

>  **EXAM TIP:** Always leave construction arcs visible - they earn marks!

## Quick Angle Rules Summary

| Rule | When to Use |
|------|-------------|
| Straight line = 180° | Angles on a line |
| Point = 360° | Angles around a point |
| Triangle = 180° | Interior angles of triangle |
| Exterior angle theorem | Exterior + two opposite interior |
| Vertically opposite | Intersecting lines |
| Parallel lines | Alternate, corresponding, co-interior |

---

## Common Mistakes to Avoid

| ❌ Mistake | ✅ Correct Approach |
|-----------|---------------------|
| Wrong scale factor for area | Area ratio = (length ratio)² |
| Missing construction arcs | Leave all arcs visible |
| No reasons given | State the theorem/rule used |
| Forgetting units | Include ° for angles |
| Using $360°/n$ for interior | Interior = $\frac{(n-2) \times 180°}{n}$ |

# 📐 KEY DERIVATIONS

## Polygon Interior Angle Sum

$$\text{Sum} = (n-2) \times 180°$$

**Why?** A polygon with $n$ sides can be split into $(n-2)$ triangles.

---

## Scale Factor for Area

$$\frac{A_1}{A_2} = k^2$$

**Why?** Area is 2-dimensional, so length factor applies twice.

## Quick Calculation Shortcuts

For **regular polygons**:
- Hexagon exterior = $360° ÷ 6 = 60°$
- Interior = $180° -$ exterior
- Number of sides = $360° ÷$ exterior angle
