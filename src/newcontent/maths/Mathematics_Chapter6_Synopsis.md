# 📋 Essential Formulas at a Glance

## Direct Proportion

| Statement | Formula | Key Feature |
|-----------|---------|-------------|
| $y \propto x$ | $y = kx$ | Straight line through origin |
| Constant ratio | $\frac{y}{x} = k$ | Same value for all pairs |
| Gradient | $k$ | Equals constant of proportionality |

## Inverse Proportion

| Statement | Formula | Key Feature |
|-----------|---------|-------------|
| $y \propto \frac{1}{x}$ | $y = \frac{k}{x}$ | Curve (hyperbola) |
| Constant product | $xy = k$ | Same value for all pairs |
| Graph shape | Hyperbola | Never touches axes |

## Index Laws Summary

| Type | Rule | Example |
|------|------|---------|
| **Negative** | $a^{-n} = \frac{1}{a^n}$ | $2^{-3} = \frac{1}{8}$ |
| **Fractional** | $a^{\frac{1}{n}} = \sqrt[n]{a}$ | $8^{\frac{1}{3}} = 2$ |
| **Combined** | $a^{-\frac{m}{n}} = \frac{1}{(\sqrt[n]{a})^m}$ | $8^{-\frac{2}{3}} = \frac{1}{4}$ |
| **Zero** | $a^0 = 1$ | $5^0 = 1$ |

# 📊 Quick Rules

## Proportion Rules

### Direct Proportion
- ✅ $y = kx$
- ✅ Graph: straight line through $(0,0)$
- ✅ As $x$ doubles, $y$ doubles
- ✅ $\frac{y_1}{x_1} = \frac{y_2}{x_2}$

### Inverse Proportion
- ✅ $y = \frac{k}{x}$ or $xy = k$
- ✅ Graph: curve (hyperbola)
- ✅ As $x$ doubles, $y$ halves
- ✅ $x_1 \times y_1 = x_2 \times y_2$

---

## Index Rules

### Evaluating $a^{\frac{m}{n}}$
```
Step 1: Take the nth root (denominator)
Step 2: Raise to power m (numerator)
```
**Memory aid:** Denominator = Down (root), Numerator = Up (power)

### Evaluating Negative Indices
```
Step 1: Reciprocate (flip)
Step 2: Change sign of index to positive
Step 3: Evaluate
```
**Memory aid:** Negative means "flip the fraction"

### Fraction with Negative Index
$$\left(\frac{a}{b}\right)^{-n} = \left(\frac{b}{a}\right)^n$$
**Memory aid:** "Flip the fraction, keep the power positive"

#  Common Exam Traps

| Trap | Correct Approach |
|------|------------------|
| $y \propto x$ written as $y = x$ | Must include $k$: $y = kx$ |
| $y \propto \frac{1}{x}$ written as $y = \frac{1}{x}$ | Must include $k$: $y = \frac{k}{x}$ |
| $a^{-n} = -a^n$ | $a^{-n} = \frac{1}{a^n}$ (reciprocal, NOT negative) |
| Evaluating $a^{\frac{m}{n}}$ in wrong order | Root first (denominator), then power (numerator) |
| $\left(\frac{a}{b}\right)^{-1} = -\frac{a}{b}$ | $\left(\frac{a}{b}\right)^{-1} = \frac{b}{a}$ (flip the fraction) |
| Drawing straight line for inverse proportion | Inverse proportion gives a CURVE |
| Forgetting to find $k$ | ALWAYS find $k$ first |

| Term | Meaning |
|------|---------|
| **Directly proportional** | $y = kx$ (straight line through origin) |
| **Inversely proportional** | $y = \frac{k}{x}$ (hyperbola) |
| **Constant of proportionality** | $k$ in the equation |
| **Reciprocal** | $\frac{1}{n}$ is the reciprocal of $n$ |
| **Index/Power** | The exponent in $a^n$ |
| **Root** | The inverse operation of a power |

# 📐 Special Cases

## $y \propto x^2$ (Direct proportion with square)

$$y = kx^2$$

- Not a straight line (parabola)
- Still passes through origin
- $\frac{y}{x^2} = k$ is constant

## $y \propto \frac{1}{x^2}$ (Inverse proportion with square)

$$y = \frac{k}{x^2}$$

- Curve (hyperbola-type)
- $x^2 y = k$ is constant
- Steeper than simple inverse proportion
