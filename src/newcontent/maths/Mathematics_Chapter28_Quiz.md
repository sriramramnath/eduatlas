### Question 1
The gradient of a curve at a specific point is equal to:

A) The gradient of the normal at that point  
B) The gradient of the tangent at that point  
C) The average gradient between two points  
D) The gradient of the secant line  

**Answer: B**

**Explanation:** By definition, the gradient of a curve at any point equals the gradient of the tangent drawn at that point. The tangent touches the curve at exactly one point and has the same direction as the curve at that point.

---

### Question 2
The graph of $y = f(x)$ is transformed to $y = f(x + 3)$. This transformation is:

A) Translation 3 units right  
B) Translation 3 units left  
C) Translation 3 units up  
D) Horizontal stretch with scale factor 3  

**Answer: B**

**Explanation:** When we have $f(x + a)$, the graph shifts **left** by $a$ units (opposite direction to the sign). This is because for the same output, we need an input that is $a$ less. The transformation vector is $\begin{pmatrix} -3 \\ 0 \end{pmatrix}$.

---

### Question 3
The graph of $y = x^2$ passes through $(2, 4)$. After the transformation $y = -x^2$, this point becomes:

A) $(2, 4)$  
B) $(-2, 4)$  
C) $(2, -4)$  
D) $(-2, -4)$  

**Answer: C**

**Explanation:** The transformation $y = -f(x)$ reflects the graph in the x-axis. This negates the y-coordinate while keeping the x-coordinate the same. So $(2, 4) \rightarrow (2, -4)$.

### Question 4
The transformation $y = 3f(x)$ represents:

A) Horizontal stretch with scale factor 3  
B) Horizontal stretch with scale factor $\frac{1}{3}$  
C) Vertical stretch with scale factor 3  
D) Vertical stretch with scale factor $\frac{1}{3}$  

**Answer: C**

**Explanation:** When the coefficient is outside the function (multiplying $f(x)$), it causes a **vertical stretch** with scale factor equal to that coefficient. Each y-coordinate is multiplied by 3.

### Question 5
A distance-time graph shows a curve. The gradient of the tangent at a point on this curve represents:

A) Total distance traveled  
B) Average speed  
C) Instantaneous speed  
D) Acceleration  

**Answer: C**

**Explanation:** On a distance-time graph, the gradient represents speed. The gradient of the tangent at a specific point gives the instantaneous speed at that moment in time.

### Question 6
The graph of $y = \sin(x)$ has period 360°. What is the period of $y = \sin(2x)$?

A) 720°  
B) 360°  
C) 180°  
D) 90°  

**Answer: C**

**Explanation:** The transformation $y = f(ax)$ causes a horizontal stretch with scale factor $\frac{1}{a}$. The period becomes $\frac{360°}{2} = 180°$. The function completes one cycle twice as fast.

### Question 7
The point $(4, 2)$ lies on $y = f(x)$. What are the coordinates of this point after the transformation $y = f(x - 1)$?

A) $(3, 2)$  
B) $(5, 2)$  
C) $(4, 3)$  
D) $(4, 1)$  

**Answer: B**

**Explanation:** The transformation $f(x - 1)$ shifts the graph **right** by 1 unit. The x-coordinate increases by 1: $4 + 1 = 5$. The y-coordinate stays the same. New point: $(5, 2)$.

### Question 8
Which transformation maps $y = f(x)$ onto $y = f(-x)$?

A) Reflection in the x-axis  
B) Reflection in the y-axis  
C) 90° rotation about the origin  
D) Translation in the negative x-direction  

**Answer: B**

**Explanation:** The transformation $y = f(-x)$ negates the x-coordinate while keeping y the same, which is a reflection in the y-axis (the vertical axis).

---

### Question 9
The curve $y = x^3 - 2x$ has a turning point at $x = 1$. What is the gradient at this point?

A) 1  
B) -1  
C) 0  
D) Cannot be determined  

**Answer: C**

**Explanation:** At any turning point (maximum or minimum), the tangent is horizontal, so the gradient is **zero**. This is a key property of turning points on curves.

---

### Question 10
The graph of $y = f(x)$ passes through the origin. After which transformation will the graph still pass through the origin?

A) $y = f(x) + 3$  
B) $y = f(x - 2)$  
C) $y = 2f(x)$  
D) $y = f(x) + 1$  

**Answer: C**

**Explanation:** The origin $(0, 0)$ has $y = 0$. After transformation $y = 2f(x)$, when $x = 0$: $y = 2f(0) = 2 \times 0 = 0$. The point remains $(0, 0)$. For vertical translations (options A and D), the y-coordinate changes, and for horizontal shifts (option B), the x-coordinate changes.

---

### Question 11
The transformation $y = f(2x)$ causes the graph to become:

A) Twice as wide  
B) Half as wide  
C) Twice as tall  
D) Half as tall  

**Answer: B**

**Explanation:** The transformation $y = f(ax)$ causes a horizontal stretch with scale factor $\frac{1}{a}$. With $a = 2$, the scale factor is $\frac{1}{2}$, making the graph **half as wide** (narrower).

---

### Question 12
A point on a curve has a tangent with gradient $-3$. This means the curve at this point is:

A) Increasing rapidly  
B) Decreasing rapidly  
C) At a maximum point  
D) At a minimum point  

**Answer: B**

**Explanation:** A negative gradient means the curve is decreasing (going downhill from left to right). The magnitude $|-3| = 3$ indicates it's decreasing relatively steeply ("rapidly").

# Section B: True/False Statements

**For each statement, state whether it is TRUE or FALSE. If FALSE, provide the correct statement.**

---

### Question 1
**Statement:** The gradient of a curve at a point equals the gradient of the normal to the curve at that point.

**Answer: FALSE**

**Correction:** The gradient of a curve at a point equals the gradient of the **tangent** to the curve at that point. (The normal is perpendicular to the tangent.)

---

### Question 2
**Statement:** The transformation $y = f(x + 5)$ shifts the graph 5 units to the right.

**Answer: FALSE**

**Correction:** The transformation $y = f(x + 5)$ shifts the graph 5 units to the **left**. (Horizontal shifts are in the opposite direction to the sign inside the function.)

---

### Question 3
**Statement:** The transformation $y = -f(x)$ reflects the graph in the x-axis.

**Answer: TRUE**

**Explanation:** This transformation negates all y-values, which is equivalent to reflecting the graph across the horizontal x-axis.

---

### Question 4
**Statement:** Points on the y-axis are invariant (unchanged) under a reflection in the y-axis.

**Answer: TRUE**

**Explanation:** Points on the y-axis have $x = 0$. After reflection in the y-axis, $x$ becomes $-x = -0 = 0$, so the point stays in the same position.

---

### Question 5
**Statement:** The transformation $y = f(3x)$ represents a horizontal stretch with scale factor 3.

**Answer: FALSE**

**Correction:** The transformation $y = f(3x)$ represents a horizontal stretch with scale factor **$\frac{1}{3}$**. (The scale factor for horizontal stretches is the reciprocal of the coefficient.)

---

### Question 6
**Statement:** When drawing a tangent to find the gradient of a curve, the tangent should pass through two points on the curve.

**Answer: FALSE**

**Correction:** A tangent should touch the curve at **exactly one point**. If it passes through multiple points on the curve, it is a secant line, not a tangent.

---

### Question 7
**Statement:** The gradient at a turning point of a curve is always zero.

**Answer: TRUE**

**Explanation:** At turning points (local maxima and minima), the tangent is horizontal, giving a gradient of zero. This is a fundamental property of turning points.

---

### Question 8
**Statement:** The transformation $y = 2f(x) + 3$ shifts the graph 3 units up and stretches it vertically by scale factor 2.

**Answer: TRUE**

**Explanation:** This combined transformation first applies a vertical stretch (multiply y by 2) and then a vertical translation (add 3 to y).

---

### Question 9
**Statement:** For a horizontal stretch, points on the x-axis move to new positions.

**Answer: FALSE**

**Correction:** Points on the x-axis (where $y = 0$) are **invariant** under a horizontal stretch. They remain in the same position because only x-coordinates change, but $y = 0$ stays as $y = 0$.

---

### Question 10
**Statement:** The transformation $y = f(-x)$ is equivalent to reflecting the graph in the x-axis.

**Answer: FALSE**

**Correction:** The transformation $y = f(-x)$ reflects the graph in the **y-axis**. (The minus sign affects x, so it reflects across the vertical axis.)

---

### Question 11
**Statement:** The rate of change shown on a distance-time graph can be found by calculating the gradient.

**Answer: TRUE**

**Explanation:** On a distance-time graph, gradient = $\frac{\text{change in distance}}{\text{change in time}}$ = speed, which is the rate of change of distance with respect to time.

---

### Question 12
**Statement:** A vertical stretch with scale factor 0.5 makes the graph taller.

**Answer: FALSE**

**Correction:** A vertical stretch with scale factor 0.5 makes the graph **shorter**. Scale factors less than 1 compress the graph toward the x-axis.

# Exam Focus & Strategy

## Frequently Tested Question Types

### 1. Gradient of Curve Questions
- **Drawing tangents** at specified points
- **Calculating gradients** from tangents
- **Interpreting gradients** in real-world contexts (speed, rate of change)
- **Identifying zero gradients** at turning points

### 2. Transformation Description Questions
- **Identifying transformation type** from function notation
- **Describing transformations fully** with direction and magnitude
- **Using correct terminology**: translation, reflection, stretch, scale factor

### 3. Coordinate Transformation Questions
- **Finding new coordinates** after transformations
- **Tracking key points** (maxima, minima, intercepts)

### 4. Combined Transformation Questions
- **Order of transformations**
- **Multiple operations on the same graph**

---

## Typical Problem-Solving Patterns

### Pattern 1: Tangent Drawing
```
1. Locate the point on the curve
2. Draw tangent carefully (one point of contact)
3. Extend the tangent
4. Read two well-spaced points
5. Calculate gradient
```

### Pattern 2: Transformation Description
```
1. Look INSIDE the function for x-transformations
   - f(x + a): horizontal shift (opposite direction)
   - f(ax): horizontal stretch (reciprocal scale factor)
2. Look OUTSIDE the function for y-transformations
   - a·f(x): vertical stretch (same scale factor)
   - f(x) + a: vertical shift (same direction)
   - -f(x): reflection in x-axis
3. Combine in correct order
```

### Pattern 3: Finding Transformed Coordinates
```
1. Write down original coordinates
2. Apply x-transformations to x-coordinate
3. Apply y-transformations to y-coordinate
4. Write final coordinates
```

## How to Gain Full Marks

### For Gradient Questions:
- [✓] Draw accurate tangent
- [✓] Use points ON tangent (not curve)
- [✓] Show calculation
- [✓] Include units if context requires
- [✓] Give appropriate accuracy

### For Transformation Questions:
- [✓] State **TYPE** (translation/reflection/stretch)
- [✓] State **DIRECTION** clearly
- [✓] Give **MAGNITUDE** (units or SF)
- [✓] Use vector notation if asked

### Example Full-Marks Answer:
**Q:** Describe the transformation mapping $y = f(x)$ to $y = f(x-2)+3$

**A:** Translation 2 units right and 3 units up (or: Translation by vector $\begin{pmatrix} 2 \\ 3 \end{pmatrix}$)

---

## Key Exam Tips

1. **Check direction for horizontal shifts** - it's opposite to the sign!

2. **Use reciprocal for horizontal stretch scale factor**

3. **Always include units** for real-world gradient problems

4. **Draw tangents carefully** - accuracy is crucial

5. **Label all key points** on sketch graphs

6. **Show all working** - marks are given for method

7. **Use correct vocabulary**: "translation" not "shift", "scale factor" not "times"
