# Topic 1: Rational and Irrational Numbers

## Key Definitions

| Type | Definition | Examples |
|------|------------|----------|
| **Rational** | Can be written as $\frac{p}{q}$ ($p, q$ integers, $q \neq 0$) | $\frac{1}{2}, 3, -4, 0.25, 0.\overline{3}$ |
| **Irrational** | Cannot be written as a fraction | $\sqrt{2}, \pi, e, \sqrt{3}$ |

---

## Quick Identification Rules

| Number Type | Rational or Irrational? |
|-------------|-------------------------|
| Terminating decimal | ✓ Rational |
| Recurring decimal | ✓ Rational |
| Integers | ✓ Rational |
| $\sqrt{n}$ where $n$ is a perfect square | ✓ Rational |
| $\sqrt{n}$ where $n$ is NOT a perfect square | ✗ Irrational |
| $\pi$ and multiples of $\pi$ | ✗ Irrational |

---

## Recurring Decimal to Fraction Method

```
For 0.aaa... (1 digit repeats):
  Let x = 0.aaa...
  10x = a.aaa...
  10x - x = a → x = a/9

For 0.abab... (2 digits repeat):
  Let x = 0.abab...
  100x = ab.abab...
  100x - x = ab → x = ab/99
```

---

## Key Properties

| Operation | Result |
|-----------|--------|
| Rational + Rational | Rational |
| Rational + Irrational | **Irrational** |
| Rational × Irrational (non-zero) | **Irrational** |
| $\sqrt{a} \times \sqrt{a}$ | $a$ (Rational) |
| $\sqrt{a} \times \sqrt{b}$ | $\sqrt{ab}$ |

# Topic 2: Surds

## What is a Surd?
An **irrational root** that cannot be simplified to a rational number.

| Surds ✓ | Not Surds ✗ |
|---------|-------------|
| $\sqrt{2}$ | $\sqrt{4} = 2$ |
| $\sqrt{3}$ | $\sqrt{9} = 3$ |
| $\sqrt{5}$ | $\sqrt{16} = 4$ |
| $\sqrt[3]{2}$ | $\sqrt[3]{8} = 2$ |

---

## Essential Rules

| Rule | Formula |
|------|---------|
| Product | $\sqrt{a} \times \sqrt{b} = \sqrt{ab}$ |
| Quotient | $\frac{\sqrt{a}}{\sqrt{b}} = \sqrt{\frac{a}{b}}$ |
| Square | $(\sqrt{a})^2 = a$ |
| With coefficient | $k\sqrt{a} \times m\sqrt{b} = km\sqrt{ab}$ |

# Simplifying Surds

## Method
```
Step 1: Find largest perfect square factor
Step 2: Split: √(ab) = √a × √b
Step 3: Simplify the square root
Step 4: Write as k√n
```

# Adding and Subtracting Surds

## Key Rule
**Only "like surds" can be combined!**

| Expression | Result |
|------------|--------|
| $3\sqrt{2} + 5\sqrt{2}$ | $8\sqrt{2}$ |
| $4\sqrt{3} - \sqrt{3}$ | $3\sqrt{3}$ |
| $2\sqrt{5} + 3\sqrt{7}$ | Cannot combine |

## Strategy
```
1. Simplify ALL surds first
2. Then combine like surds
3. Different surds stay separate
```

**Example:**
$$\sqrt{8} + \sqrt{18} = 2\sqrt{2} + 3\sqrt{2} = 5\sqrt{2}$$

# Rationalising the Denominator

## Type 1: Simple Denominator $\frac{a}{\sqrt{b}}$

**Method:** Multiply by $\frac{\sqrt{b}}{\sqrt{b}}$

$$\frac{a}{\sqrt{b}} = \frac{a\sqrt{b}}{b}$$

**Examples:**
- $\frac{6}{\sqrt{3}} = \frac{6\sqrt{3}}{3} = 2\sqrt{3}$
- $\frac{10}{\sqrt{5}} = \frac{10\sqrt{5}}{5} = 2\sqrt{5}$

---

## Type 2: Binomial Denominator $\frac{a}{p + q\sqrt{r}}$

**Method:** Multiply by the **conjugate**

| Expression | Conjugate |
|------------|-----------|
| $a + b\sqrt{c}$ | $a - b\sqrt{c}$ |
| $a - b\sqrt{c}$ | $a + b\sqrt{c}$ |

**Key Formula:**
$$(a + b\sqrt{c})(a - b\sqrt{c}) = a^2 - b^2c$$

**Examples:**
- $\frac{5}{2+\sqrt{3}} \times \frac{2-\sqrt{3}}{2-\sqrt{3}} = \frac{5(2-\sqrt{3})}{1} = 10 - 5\sqrt{3}$
- $\frac{3}{4-\sqrt{7}} \times \frac{4+\sqrt{7}}{4+\sqrt{7}} = \frac{12+3\sqrt{7}}{9} = \frac{4+\sqrt{7}}{3}$

# High-Yield Exam Points

##  Most Common Mistakes

| Mistake | Correction |
|---------|------------|
| $\sqrt{a} + \sqrt{b} = \sqrt{a+b}$ | WRONG! Cannot combine |
| $2\sqrt{3} + 3\sqrt{5} = 5\sqrt{8}$ | WRONG! Different surds |
| $\sqrt{2}^2 = \sqrt{4}$ | WRONG! $(\sqrt{2})^2 = 2$ |
| Forgetting to simplify | $\frac{6\sqrt{3}}{3} = 2\sqrt{3}$ |
| Wrong conjugate sign | $(a+b\sqrt{c})$ needs $(a-b\sqrt{c})$ |

---

## Exam Technique Checklist

✅ Simplify all surds before combining

✅ Always rationalise denominators

✅ Leave answers in exact surd form (unless asked for decimal)

✅ Show all working when rationalising

✅ Check if different surds simplify to the same surd

✅ Multiply BOTH numerator and denominator when rationalising

## Question Type Summary

| Type | Key Steps |
|------|-----------|
| Simplify surd | Find largest square factor, split, simplify |
| Add/subtract surds | Simplify first, combine like surds only |
| Multiply surds | Multiply coefficients, multiply surds, simplify |
| Rationalise $\frac{a}{\sqrt{b}}$ | Multiply by $\frac{\sqrt{b}}{\sqrt{b}}$ |
| Rationalise $\frac{a}{p+q\sqrt{r}}$ | Multiply by conjugate |
| Geometric problem | Use Pythagoras, leave in exact form |
