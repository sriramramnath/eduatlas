### Question 1

What is $0.\dot{3}$ expressed as a fraction?

A) $\frac{3}{10}$

B) $\frac{1}{3}$

C) $\frac{3}{100}$

D) $\frac{33}{100}$

**Answer: B) $\frac{1}{3}$**

**Explanation:**

Let $x = 0.\dot{3} = 0.333...$

Multiply by 10:
$$10x = 3.\dot{3} = 3.333...$$

Subtract:
$$10x - x = 3$$
$$9x = 3$$
$$x = \frac{3}{9} = \frac{1}{3}$$

Alternatively, using the formula: $0.\dot{3} = \frac{3}{9} = \frac{1}{3}$

### Question 4

Which of these fractions will give a terminating decimal?

A) $\frac{1}{6}$

B) $\frac{1}{7}$

C) $\frac{1}{8}$

D) $\frac{1}{9}$

**Answer: C) $\frac{1}{8}$**

**Explanation:**

A fraction terminates if the denominator (in lowest terms) has only prime factors 2 and/or 5.

| Option | Denominator | Prime Factors | Terminating? |
|--------|-------------|---------------|--------------|
| $\frac{1}{6}$ | 6 | 2 × 3 | No (has 3) |
| $\frac{1}{7}$ | 7 | 7 | No (has 7) |
| $\frac{1}{8}$ | 8 | 2³ | **Yes** (only 2) |
| $\frac{1}{9}$ | 9 | 3² | No (has 3) |

$\frac{1}{8} = 0.125$ (terminates)

Let me check with $a = b = 9$:
$0.\dot{9}\dot{9} = \frac{99}{99} = 1$
$\frac{a}{11} = \frac{9}{11} \neq 1$

So this doesn't work.

Let me check $a = b$:
$0.\dot{a}\dot{a} = \frac{11a}{99} = \frac{a}{9}$

We want $\frac{a}{9} = \frac{a}{11}$... this is only true if $a = 0$.

So $a = 0, b = 0$: $0.\dot{0}\dot{0} = 0 = \frac{0}{11}$ ✓

But the problem says different digits...

**The problem may have an error, or I need to interpret differently.**

**Revised Answer:**

The equation $\frac{10a + b}{99} = \frac{a}{11}$ gives $b = -a$.

Since $a, b$ are positive digits and must be different, **no solution exists** with different digits.

However, if we allow $a = b$: the only solution is $a = b = 0$, giving $0.\dot{0}\dot{0} = 0$.

**Correction to the problem's intended solution:**

The problem might have intended: $0.\dot{a}\dot{b} = \frac{b}{11}$

In that case: $\frac{10a + b}{99} = \frac{b}{11} = \frac{9b}{99}$

$10a + b = 9b$
$10a = 8b$
$a = \frac{4b}{5}$

For integer digits: $b = 5$, $a = 4$

**Answer: $a = 4, b = 5$**

Verification: $0.\dot{4}\dot{5} = \frac{45}{99} = \frac{5}{11}$ ✓
