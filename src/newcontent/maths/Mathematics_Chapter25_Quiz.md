### Question 1

For the inequality $y > 3x - 2$, the boundary line should be:

A) Solid and the region above shaded  
B) Dashed and the region above shaded  
C) Solid and the region below shaded  
D) Dashed and the region below shaded  

**Answer: B**

**Explanation:** Since the inequality uses $>$ (strict inequality), the boundary line should be **dashed** (not included). For $y > ...$, the region **above** the line is shaded.

**Key Point:** $>$ or $<$ → dashed line; $y >$ → shade above the line.

### Question 2

The gradient of a line perpendicular to a line with gradient $m = 4$ is:

A) $4$  
B) $-4$  
C) $\frac{1}{4}$  
D) $-\frac{1}{4}$  

**Answer: D**

**Explanation:** For perpendicular lines, $m_2 = -\frac{1}{m_1}$.

$$m_2 = -\frac{1}{4}$$

**Key Point:** Perpendicular gradient is the negative reciprocal.

### Question 3

The midpoint of the line segment joining $(2, 4)$ and $(6, 8)$ is:

A) $(4, 6)$  
B) $(4, 12)$  
C) $(8, 12)$  
D) $(3, 6)$  

**Answer: A**

**Explanation:** Using the midpoint formula:
$$M = \left(\frac{2+6}{2}, \frac{4+8}{2}\right) = \left(\frac{8}{2}, \frac{12}{2}\right) = (4, 6)$$

**Key Point:** Midpoint = average of x-coordinates, average of y-coordinates.

### Question 4

The distance between points $(0, 0)$ and $(3, 4)$ is:

A) $5$  
B) $7$  
C) $1$  
D) $25$  

**Answer: A**

**Explanation:** Using the distance formula:
$$d = \sqrt{(3-0)^2 + (4-0)^2} = \sqrt{9 + 16} = \sqrt{25} = 5$$

**Key Point:** This is the classic 3-4-5 Pythagorean triple.

### Question 5

Which of the following pairs of lines are perpendicular?

A) $y = 2x + 1$ and $y = 2x - 3$  
B) $y = 2x + 1$ and $y = -\frac{1}{2}x + 4$  
C) $y = 2x + 1$ and $y = \frac{1}{2}x + 1$  
D) $y = 2x + 1$ and $y = -2x + 5$  

**Answer: B**

**Explanation:** For perpendicular lines, $m_1 \times m_2 = -1$.

- A: $2 \times 2 = 4 \neq -1$ (parallel)
- B: $2 \times (-\frac{1}{2}) = -1$ ✓ (perpendicular)
- C: $2 \times \frac{1}{2} = 1 \neq -1$
- D: $2 \times (-2) = -4 \neq -1$

**Key Point:** Check the product of gradients equals $-1$.

### Question 6

For the inequality $y \leq x^2 - 4$, the boundary curve is:

A) A dashed parabola  
B) A solid parabola  
C) A dashed straight line  
D) A solid straight line  

**Answer: B**

**Explanation:** Since the inequality uses $\leq$, the boundary is **included** and should be drawn as a **solid** curve. The boundary is the parabola $y = x^2 - 4$.

**Key Point:** $\leq$ or $\geq$ means solid boundary; the boundary is whatever shape the equation gives.

### Question 7

If the midpoint of $AB$ is $(3, 5)$ and $A = (1, 2)$, then $B$ is:

A) $(2, 3)$  
B) $(4, 8)$  
C) $(5, 8)$  
D) $(5, 7)$  

**Answer: C**

**Explanation:** Let $B = (x, y)$.

Using midpoint formula:
$$\left(\frac{1+x}{2}, \frac{2+y}{2}\right) = (3, 5)$$

This gives:
$$\frac{1+x}{2} = 3 \Rightarrow x = 5$$
$$\frac{2+y}{2} = 5 \Rightarrow y = 8$$

**Answer:** $B = (5, 8)$

**Key Point:** Work backwards from the midpoint formula.

### Question 11

The equation of the line perpendicular to $y = \frac{1}{2}x - 3$ passing through $(4, 5)$ is:

A) $y = 2x - 3$  
B) $y = -2x + 13$  
C) $y = 2x - 3$  
D) $y = -2x + 3$  

**Answer: B**

### Question 13

For a system of inequalities, the solution region is:

A) The union of all shaded regions  
B) The intersection of all shaded regions  
C) Any one of the shaded regions  
D) The unshaded region  

**Answer: B**

**Explanation:** For a system of inequalities, the solution is the region where ALL inequalities are satisfied simultaneously. This is the **intersection** (overlap) of all individual regions.

**Key Point:** All conditions must be true for the solution region.

### Question 16

The boundary line for $y < 2x + 1$ should be drawn as a solid line.

**Answer: FALSE**

**Correction:** The boundary line for $y < 2x + 1$ should be drawn as a **dashed line** because the inequality is strict ($<$). Solid lines are used for $\leq$ or $\geq$.

### Question 17

If two lines are perpendicular, the product of their gradients equals $-1$.

**Answer: TRUE**

**Explanation:** This is the fundamental relationship for perpendicular lines: $m_1 \times m_2 = -1$.

### Question 18

The midpoint of $(x_1, y_1)$ and $(x_2, y_2)$ is $\left(\frac{x_2 - x_1}{2}, \frac{y_2 - y_1}{2}\right)$.

**Answer: FALSE**

**Correction:** The midpoint formula uses **addition**, not subtraction:
$$M = \left(\frac{x_1 + x_2}{2}, \frac{y_1 + y_2}{2}\right)$$

### Question 19

The distance between $(1, 3)$ and $(4, 7)$ is 5 units.

**Answer: TRUE**

**Explanation:**
$$d = \sqrt{(4-1)^2 + (7-3)^2} = \sqrt{9 + 16} = \sqrt{25} = 5$$

### Question 20

A line with gradient $\frac{2}{3}$ is perpendicular to a line with gradient $-\frac{3}{2}$.

**Answer: TRUE**

**Explanation:**
$$\frac{2}{3} \times \left(-\frac{3}{2}\right) = -\frac{6}{6} = -1$$

Since the product is $-1$, the lines are perpendicular.

### Question 21

For the inequality $y \leq mx + c$, the shaded region is always below the line.

**Answer: TRUE**

**Explanation:** For $y \leq$ (less than or equal to), the region below the line is shaded. This includes the boundary line because of the $\leq$ symbol.

---

### Question 22

The perpendicular gradient to $m = -5$ is $m = 5$.

**Answer: FALSE**

**Correction:** The perpendicular gradient to $m = -5$ is:
$$m_2 = -\frac{1}{-5} = \frac{1}{5}$$

The perpendicular gradient is $\frac{1}{5}$, not $5$.

---

### Question 23

The distance formula $d = \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2}$ is derived from Pythagoras' theorem.

**Answer: TRUE**

**Explanation:** The horizontal distance $|x_2 - x_1|$ and vertical distance $|y_2 - y_1|$ form the two legs of a right-angled triangle, with the distance $d$ as the hypotenuse.

---

### Question 24

If the origin satisfies an inequality, the shaded region must contain the origin.

**Answer: TRUE**

**Explanation:** When using the test point method, if the test point (such as the origin) satisfies the inequality, the region containing that point should be shaded.

---

### Question 25

A line with gradient $0$ is perpendicular to a line with undefined gradient (vertical line).

**Answer: TRUE**

**Explanation:** A horizontal line has gradient $0$. A vertical line has undefined gradient (infinite). These lines are always perpendicular to each other.

---

### Question 26

The midpoint of a line segment is always closer to one endpoint than the other.

**Answer: FALSE**

**Correction:** The midpoint is **equidistant** from both endpoints. It is exactly halfway between them.

---

### Question 27

For the inequality $y < x^2$, the region below the parabola is shaded.

**Answer: TRUE**

**Explanation:** Since $y < x^2$, we need y-values that are less than $x^2$. This means shading below the parabola (the dashed curve $y = x^2$).

---

### Question 28

The distance from $(a, b)$ to $(c, d)$ is the same as the distance from $(c, d)$ to $(a, b)$.

**Answer: TRUE**

**Explanation:** Distance is symmetric. Since $(a-c)^2 = (c-a)^2$ and $(b-d)^2 = (d-b)^2$, the distance formula gives the same result regardless of order.

---

### Question 29

If $m_1 \times m_2 = 1$, then the lines are perpendicular.

**Answer: FALSE**

**Correction:** If $m_1 \times m_2 = 1$, the lines are NOT perpendicular. For perpendicular lines, the product must equal **$-1$**. A product of $1$ indicates neither parallel nor perpendicular (unless $m_1 = m_2 = 1$ or $m_1 = m_2 = -1$).

---

### Question 30

The centre of a circle is the midpoint of its diameter.

**Answer: TRUE**

**Explanation:** By definition, the diameter passes through the centre of the circle, and the centre divides the diameter into two equal radii. Therefore, the centre is the midpoint of the diameter.

**Question 32** [5 marks]

(a) Find the gradient of the line perpendicular to the line with equation $2x + 3y = 6$. [2 marks]

(b) Find the equation of this perpendicular line if it passes through the point $(1, 4)$. [3 marks]

**(a) Finding the perpendicular gradient [2 marks]**

| Mark | Response |
|------|----------|
| M1 | Correctly rearranges equation to find original gradient |
| A1 | Correct perpendicular gradient |

**Working:**
$$2x + 3y = 6$$
$$3y = -2x + 6$$
$$y = -\frac{2}{3}x + 2$$

Original gradient: $m_1 = -\frac{2}{3}$

Perpendicular gradient:
$$m_2 = -\frac{1}{-\frac{2}{3}} = \frac{3}{2}$$

**Answer:** Perpendicular gradient = $\frac{3}{2}$

**(b) Equation of perpendicular line [3 marks]**

| Mark | Response |
|------|----------|
| M1 | Uses perpendicular gradient found in (a) |

$$y = \frac{3}{2}x + c$$

Substitute $(1, 4)$:
$$4 = \frac{3}{2}(1) + c$$
$$4 = \frac{3}{2} + c$$
$$c = 4 - \frac{3}{2} = \frac{8-3}{2} = \frac{5}{2}$$

**Answer:** $y = \frac{3}{2}x + \frac{5}{2}$ or $2y = 3x + 5$

## 📌 Typical Problem-Solving Patterns

### Pattern 1: "Shade the region satisfying..."
1. Identify all boundary lines/curves
2. Determine solid or dashed
3. Use test point for each inequality
4. Shade the overlapping region

### Pattern 2: "Find the perpendicular line..."
1. Calculate original gradient
2. Find perpendicular gradient: $m_2 = -\frac{1}{m_1}$
3. Use given point to find y-intercept
4. Write final equation

### Pattern 3: "Find the midpoint..."
1. Apply formula: $\left(\frac{x_1+x_2}{2}, \frac{y_1+y_2}{2}\right)$
2. State as coordinate pair

### Pattern 4: "Find the distance/length..."
1. Apply formula: $\sqrt{(x_2-x_1)^2 + (y_2-y_1)^2}$
2. Simplify (leave in exact form unless asked)

### Pattern 5: "Show the triangle is right-angled..."
1. Calculate all side lengths
2. Apply Pythagoras' theorem
3. Verify $a^2 + b^2 = c^2$

## 📌 Common Mistakes to Avoid

| ❌ Mistake | ✅ Correction |
|------------|---------------|
| Wrong line type for inequalities | Check $\leq, \geq$ vs $<, >$ |
| Forgetting negative sign in perpendicular gradient | $m_2 = -\frac{1}{m_1}$ |
| Adding coordinates instead of averaging | Midpoint divides by 2 |
| Forgetting square root in distance | Final step is $\sqrt{}$ |
| Not testing point for inequality region | Always verify with test point |
