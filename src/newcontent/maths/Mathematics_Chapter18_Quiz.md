## 📝 SECTION B: TRUE OR FALSE

**Instructions:** State whether each statement is TRUE or FALSE. If FALSE, provide the correction.

---

**Q1.** The notation $f(x)$ means $f$ multiplied by $x$.

**Answer: FALSE**

**Correction:** $f(x)$ means the output of function $f$ when the input is $x$, read as "f of x".

---

**Q2.** For $fg(x)$, we apply function $f$ first, then function $g$.

**Answer: FALSE**

**Correction:** For $fg(x)$, we apply function $g$ first, then function $f$. The order is right-to-left (inside-out).

---

**Q3.** $fg(x) = gf(x)$ for all functions $f$ and $g$.

**Answer: FALSE**

**Correction:** In general, $fg(x) \neq gf(x)$. The order of composition matters.

Example: If $f(x) = x^2$ and $g(x) = x + 1$:
- $fg(x) = f(x + 1) = (x + 1)^2$
- $gf(x) = g(x^2) = x^2 + 1$

---

**Q4.** The inverse function $f^{-1}(x)$ is the same as $\frac{1}{f(x)}$.

**Answer: FALSE**

**Correction:** $f^{-1}(x)$ is the inverse function that "undoes" $f(x)$, not the reciprocal. $f^{-1}(x) \neq \frac{1}{f(x)}$.

---

**Q5.** If $f(x) = 2x + 6$, then $f^{-1}(x) = \frac{x - 6}{2}$.

**Answer: TRUE**

**Explanation:**
$$y = 2x + 6$$
$$x = 2y + 6$$
$$x - 6 = 2y$$
$$y = \frac{x - 6}{2}$$ ✓

---

**Q6.** The domain of $f(x) = \sqrt{x - 4}$ is $x > 4$.

**Answer: FALSE**

**Correction:** The domain is $x \geq 4$. The square root is defined for zero and positive values.

---

**Q7.** For any function $f$, we have $f^{-1}f(x) = x$.

**Answer: TRUE**

**Explanation:**
This is the defining property of inverse functions. Applying $f$ and then $f^{-1}$ returns the original input.

---

**Q8.** The range of $f(x) = -x^2 + 3$ is $f(x) \leq 3$.

**Answer: TRUE**

**Explanation:**
Since $x^2 \geq 0$, we have $-x^2 \leq 0$, so $-x^2 + 3 \leq 3$.

---

**Q9.** Given $f(x) = 3x - 2$ and $g(x) = x + 4$, then $fg(x) = 3x + 10$.

**Answer: TRUE**

**Explanation:**
$$fg(x) = f(x + 4) = 3(x + 4) - 2 = 3x + 12 - 2 = 3x + 10$$ ✓

---

**Q10.** The function $f(x) = x^2$ has an inverse for all real $x$.

**Answer: FALSE**

**Correction:** $f(x) = x^2$ is not one-to-one over all real numbers (both 2 and -2 give 4), so it doesn't have an inverse over its full domain. However, if we restrict the domain to $x \geq 0$ or $x \leq 0$, an inverse exists.

---

**Q11.** If $f(a) = b$, then $f^{-1}(b) = a$.

**Answer: TRUE**

**Explanation:**
This follows directly from the definition of inverse functions. The inverse function reverses the mapping.

---

**Q12.** The domain of $f^{-1}(x)$ equals the range of $f(x)$.

**Answer: TRUE**

**Explanation:**
Since the inverse function reverses the mapping, the input values for $f^{-1}$ are the output values of $f$, and vice versa.

### Question 1 (2 marks)

Given $f(x) = 4x - 5$, find:
(a) $f(3)$
(b) The value of $x$ when $f(x) = 7$

**Solution:**

**(a) [M1]** Substitute $x = 3$:
$$f(3) = 4(3) - 5 = 12 - 5 = 7$$

**[A1]** Answer: $f(3) = 7$

**(b) [M1]** Set $f(x) = 7$:
$$4x - 5 = 7$$
$$4x = 12$$
$$x = 3$$

**[A1]** Answer: $x = 3$

### Question 2 (2 marks)

Given $g(x) = x^2 - 3x + 2$, find:
(a) $g(4)$
(b) $g(-1)$

**Solution:**

**(a) [M1]** Substitute $x = 4$:
$$g(4) = 4^2 - 3(4) + 2 = 16 - 12 + 2 = 6$$

**[A1]** Answer: $g(4) = 6$

**(b) [M1]** Substitute $x = -1$:
$$g(-1) = (-1)^2 - 3(-1) + 2 = 1 + 3 + 2 = 6$$

**[A1]** Answer: $g(-1) = 6$

### Question 3 (3 marks)

Find the domain of each function:
(a) $f(x) = \frac{4}{x + 3}$
(b) $g(x) = \sqrt{2x - 6}$
(c) $h(x) = \frac{1}{\sqrt{x + 5}}$

**Solution:**

**(a) [B1]** Denominator cannot be zero:
$$x + 3 \neq 0$$
$$x \neq -3$$

**Domain:** All real $x$, $x \neq -3$

**(b) [B1]** Radicand must be non-negative:
$$2x - 6 \geq 0$$
$$2x \geq 6$$
$$x \geq 3$$

**Domain:** $x \geq 3$

**(c) [B1]** Radicand must be positive (not zero, as in denominator):
$$x + 5 > 0$$
$$x > -5$$

**Domain:** $x > -5$

### Question 4 (3 marks)

Given $f(x) = 2x + 1$ and $g(x) = x - 3$:
(a) Find $fg(x)$
(b) Find $gf(x)$
(c) Show that $fg(x) \neq gf(x)$

**Solution:**

**(a) [M1]** $fg(x) = f(x - 3)$
**[A1]** $= 2(x - 3) + 1 = 2x - 6 + 1 = 2x - 5$

**(b) [M1]** $gf(x) = g(2x + 1)$
**[A1]** $= (2x + 1) - 3 = 2x - 2$

**(c) [B1]** $fg(x) = 2x - 5$ and $gf(x) = 2x - 2$

Since $2x - 5 \neq 2x - 2$, we have shown $fg(x) \neq gf(x)$.

**[A1]** $= x^2 + 4$

**(c) [M1]** Using $fg(x) = x^2 + 8x + 16$:

### Question 10 (5 marks)

The function $f$ is defined by $f(x) = \frac{x - 1}{x + 2}$, $x \neq -2$.

(a) Find $f(3)$. **[1 mark]**
(b) Find $f^{-1}(x)$. **[3 marks]**
(c) State the domain of $f^{-1}$. **[1 mark]$

**Solution:**

**(a) [B1]** $f(3) = \frac{3 - 1}{3 + 2} = \frac{2}{5}$

**(b) [M1]** Let $y = \frac{x - 1}{x + 2}$
Swap: $x = \frac{y - 1}{y + 2}$

**[M1]** $x(y + 2) = y - 1$
$$xy + 2x = y - 1$$
$$xy - y = -2x - 1$$
$$y(x - 1) = -2x - 1$$

**[A1]** $y = \frac{-2x - 1}{x - 1} = \frac{2x + 1}{1 - x}$

$f^{-1}(x) = \frac{2x + 1}{1 - x}$

**(c) [B1]** Domain of $f^{-1}$: $x \neq 1$ (denominator cannot equal zero)

### Question 11 (5 marks)

Given $f(x) = 3x + 2$ and $g(x) = x^2 - 1$:

(a) Find expressions for $fg(x)$ and $gf(x)$. **[3 marks]**
(b) Solve $fg(x) = gf(x)$. **[2 marks]$

**Solution:**

**(a) [M1]** $fg(x) = f(x^2 - 1) = 3(x^2 - 1) + 2 = 3x^2 - 3 + 2 = 3x^2 - 1$

**[M1]** $gf(x) = g(3x + 2) = (3x + 2)^2 - 1 = 9x^2 + 12x + 4 - 1 = 9x^2 + 12x + 3$

**[A1]** $fg(x) = 3x^2 - 1$ and $gf(x) = 9x^2 + 12x + 3$

**(b) [M1]** Set $fg(x) = gf(x)$:
$$3x^2 - 1 = 9x^2 + 12x + 3$$
$$0 = 6x^2 + 12x + 4$$
$$0 = 3x^2 + 6x + 2$$

**[M1]** Using quadratic formula: $a = 3$, $b = 6$, $c = 2$
$$x = \frac{-6 \pm \sqrt{36 - 24}}{6} = \frac{-6 \pm \sqrt{12}}{6} = \frac{-6 \pm 2\sqrt{3}}{6} = \frac{-3 \pm \sqrt{3}}{3}$$

**[A1]** $x = \frac{-3 + \sqrt{3}}{3}$ or $x = \frac{-3 - \sqrt{3}}{3}$

### Question 12 (4 marks)

Given $f(x) = x^2 - 4x + 7$:

(a) Express $f(x)$ in the form $(x - a)^2 + b$. **[2 marks]**
(b) Hence find the range of $f(x)$. **[1 mark]**
(c) State the minimum value of $f(x)$ and the value of $x$ at which it occurs. **[1 mark]$

**(c) [B1]** Minimum value is 3, occurring when $x = 2$.

### Question 13 (6 marks)

The function $f$ is defined by $f(x) = \frac{2x + 1}{x - 3}$, $x \neq 3$.

(a) Find $f(5)$. **[1 mark]**
(b) Find $f^{-1}(x)$. **[3 marks]**
(c) Verify that $f^{-1}f(5) = 5$. **[2 marks]$

**Solution:**

**(a) [B1]** $f(5) = \frac{2(5) + 1}{5 - 3} = \frac{11}{2}$

**(b) [M1]** Let $y = \frac{2x + 1}{x - 3}$
Swap: $x = \frac{2y + 1}{y - 3}$

**[M1]** $x(y - 3) = 2y + 1$
$$xy - 3x = 2y + 1$$
$$xy - 2y = 3x + 1$$
$$y(x - 2) = 3x + 1$$

**[A1]** $y = \frac{3x + 1}{x - 2}$

$f^{-1}(x) = \frac{3x + 1}{x - 2}$, $x \neq 2$

**(c) [M1]** $f^{-1}f(5) = f^{-1}\left(\frac{11}{2}\right) = \frac{3\left(\frac{11}{2}\right) + 1}{\frac{11}{2} - 2}$

**[M1]** $= \frac{\frac{33}{2} + 1}{\frac{11}{2} - \frac{4}{2}} = \frac{\frac{35}{2}}{\frac{7}{2}} = \frac{35}{2} \times \frac{2}{7} = 5$ ✓

### Question 14 (5 marks)

Given $f(x) = 2x + 3$, $g(x) = x - 4$:

(a) Find $gf(x)$. **[2 marks]**
(b) Find $(gf)^{-1}(x)$. **[2 marks]**
(c) Show that $(gf)^{-1}(x) \neq f^{-1}g^{-1}(x)$. **[1 mark]$

**Solution:**

**(a) [M1]** $gf(x) = g(2x + 3) = (2x + 3) - 4 = 2x - 1$

**[A1]** $gf(x) = 2x - 1$

**(b) [M1]** Let $y = 2x - 1$
Swap: $x = 2y - 1$
$$x + 1 = 2y$$
$$y = \frac{x + 1}{2}$$

**[A1]** $(gf)^{-1}(x) = \frac{x + 1}{2}$

**(c) [M1]** First find $f^{-1}(x) = \frac{x - 3}{2}$ and $g^{-1}(x) = x + 4$

$f^{-1}g^{-1}(x) = f^{-1}(x + 4) = \frac{(x + 4) - 3}{2} = \frac{x + 1}{2}$

Actually, $(gf)^{-1}(x) = f^{-1}g^{-1}(x) = \frac{x + 1}{2}$ in this case.

**Correction:** The statement is FALSE for this example as they are equal.

**Note:** In general, $(gf)^{-1} = f^{-1}g^{-1}$, not $g^{-1}f^{-1}$. This is a property of inverse of composite functions.

## 📊 SECTION D: EXAM FOCUS & STRATEGY

### Frequently Tested Question Types

| Type | Description | Typical Marks |
|------|-------------|---------------|
| **Evaluate $f(a)$** | Substitute value into function | 1-2 |
| **Find domain/range** | Identify restrictions | 2-3 |
| **Composite function** | Find $fg(x)$ or $gf(x)$ | 2-3 |
| **Inverse function** | Find $f^{-1}(x)$ | 2-3 |
| **Solve $f(x) = k$** | Find input given output | 2-3 |
| **Combined problems** | Multiple function operations | 4-6 |
| **Verify identities** | Show $f^{-1}f(x) = x$ | 2-3 |

### Typical Problem-Solving Patterns

**Pattern 1: Evaluating Functions**
```
1. Write down the function
2. Substitute the given value (use brackets!)
3. Calculate carefully
4. Check for simplification
```

**Pattern 2: Finding Domain**
```
1. Check for division by zero
2. Check for square root of negative
3. Check for both if combined
4. State clearly with correct notation
```

**Pattern 3: Composite Functions**
```
1. Identify which function to apply first
   - fg(x): apply g first, then f
   - gf(x): apply f first, then g
2. Substitute the inner function into the outer
3. Use brackets for substitution
4. Simplify fully
```

**Pattern 4: Inverse Functions**
```
1. Write y = f(x)
2. Swap x and y
3. Solve for y
4. State f⁻¹(x)
5. Check domain if needed
```

### Common Pitfalls to Avoid

1. **Wrong order in composite functions**
   - Remember: $fg(x) = f(g(x))$, apply $g$ first

2. **Forgetting brackets when substituting**
   - $f(2x) \neq f(x) \times 2$
   - Always use brackets: $f(2x) = (2x)^2 + ...$

3. **Confusing inverse with reciprocal**
   - $f^{-1}(x) \neq \frac{1}{f(x)}$
   - Inverse undoes the function

4. **Incomplete domain restrictions**
   - Check both denominator ≠ 0 AND radicand ≥ 0

5. **Not simplifying fully**
   - Expand brackets and collect like terms

6. **Forgetting to swap variables for inverse**
   - Must swap $x$ and $y$ before solving

7. **Wrong domain for inverse**
   - Domain of $f^{-1}$ = Range of $f$
