### Question 1

Simplify $\frac{x^2-4}{x+2}$

**A** $x-2$

**B** $x+2$

**C** $x-4$

**D** $x^2-2$

**Answer: A**

**Explanation:**
$\frac{x^2-4}{x+2} = \frac{(x+2)(x-2)}{x+2} = x-2$ (for $x \neq -2$)

### Question 2

Which of the following is the correct simplification of $\frac{6x^2y}{9xy^2}$?

**A** $\frac{2x}{3y}$

**B** $\frac{2xy}{3}$

**C** $\frac{3x}{2y}$

**D** $\frac{2}{3}xy$

**Answer: A**

**Explanation:**
$\frac{6x^2y}{9xy^2} = \frac{6 \cdot x \cdot x \cdot y}{9 \cdot x \cdot y \cdot y} = \frac{2x}{3y}$

Cancel: 6 and 9 (÷3), one $x$, one $y$

### Question 3

When simplifying $\frac{x^2-9}{x^2-6x+9}$, the first step should be:

**A** Cancel the 9s

**B** Divide numerator and denominator by $x^2$

**C** Factorise both numerator and denominator

**D** Expand both expressions

**Answer: C**

**Explanation:**
You must factorise first: $x^2-9 = (x+3)(x-3)$ and $x^2-6x+9 = (x-3)^2$

Only after factorisation can you cancel the common factor $(x-3)$.

### Question 4

What is the LCD of $\frac{1}{x-1}$ and $\frac{2}{x+3}$?

**A** $(x-1) + (x+3)$

**B** $(x-1)(x+3)$

**C** $x^2 + 2x - 3$

**D** Both B and C are correct

**Answer: D**

**Explanation:**
The LCD is $(x-1)(x+3)$, which when expanded gives $x^2+2x-3$.

Both forms are mathematically equivalent, so D is correct.

### Question 5

Simplify $\frac{1}{x+1} - \frac{1}{x-1}$

**A** $\frac{2}{x^2-1}$

**B** $\frac{-2}{x^2-1}$

**C** $\frac{2x}{x^2-1}$

**D** $\frac{-2}{x^2+1}$

**Answer: B**

**Explanation:**
LCD = $(x+1)(x-1) = x^2-1$

$\frac{1(x-1)}{(x+1)(x-1)} - \frac{1(x+1)}{(x+1)(x-1)}$

$= \frac{x-1-x-1}{x^2-1} = \frac{-2}{x^2-1}$

### Question 6

Which value of $x$ is NOT allowed in the expression $\frac{x+3}{x^2-4}$?

**A** $x = 0$

**B** $x = -3$

**C** $x = 2$

**D** $x = 4$

**Answer: C**

**Explanation:**
$x^2-4 = 0$ when $x = 2$ or $x = -2$

The expression is undefined when $x = 2$ or $x = -2$.

### Question 7

The solution to $\frac{x}{2} + \frac{x}{3} = 5$ is:

**A** $x = 5$

**B** $x = 6$

**C** $x = 10$

**D** $x = 30$

**Answer: B**

**Explanation:**
LCD = 6

$6 \times \frac{x}{2} + 6 \times \frac{x}{3} = 6 \times 5$

$3x + 2x = 30$

$5x = 30$

$x = 6$

### Question 8

Which statement about cancelling in algebraic fractions is TRUE?

**A** You can cancel any common term between numerator and denominator

**B** You can only cancel factors that multiply the entire numerator or denominator

**C** You can cancel addition terms if they appear in both parts

**D** Cancelling changes the value of the fraction

**Answer: B**

**Explanation:**
You can only cancel FACTORS (quantities that multiply the entire expression), not TERMS (quantities that are added or subtracted).

### Question 9

Solve $\frac{2x-1}{3} = \frac{x+2}{2}$

**A** $x = 4$

**B** $x = 8$

**C** $x = 5$

**D** $x = 7$

**Answer: B**

**Explanation:**
Cross multiply: $2(2x-1) = 3(x+2)$

$4x - 2 = 3x + 6$

### Question 10

When solving $\frac{x^2-9}{x-3} = 6$, the apparent solution is $x = 3$. This solution is:

**A** Valid and correct

**B** Invalid because it makes the denominator zero

**C** Valid because it satisfies the equation

**D** One of two valid solutions

**Answer: B**

**Explanation:**
$x = 3$ makes the denominator $(x-3) = 0$, which is undefined.

This is an **extraneous solution** and must be rejected.

The equation has no valid solution.

## SECTION B: TRUE OR FALSE (10 marks)

**State whether each statement is TRUE or FALSE. If FALSE, provide the correct statement.**

---

### Question 11

**Statement:** "In the expression $\frac{x+4}{4}$, we can cancel the 4s to get $x$."

**Answer: FALSE**

**Correction:** You cannot cancel the 4s because 4 is added to $x$, not multiplied. The expression cannot be simplified further.

**Explanation:**
$\frac{x+4}{4} = \frac{x}{4} + 1$, not $x$

---

### Question 12

**Statement:** "When adding fractions with different denominators, you must find a common denominator first."

**Answer: TRUE**

**Explanation:**
This is the fundamental rule for addition of fractions. Only fractions with the same denominator can be added directly.

---

### Question 13

**Statement:** "The LCD of $(x-1)$, $(x+1)$, and $(x^2-1)$ is $(x-1)(x+1)(x^2-1)$."

**Answer: FALSE**

**Correction:** The LCD is $(x^2-1)$ or $(x-1)(x+1)$, since $x^2-1 = (x-1)(x+1)$.

**Explanation:**
Since $x^2-1 = (x-1)(x+1)$, the LCD is just $(x-1)(x+1) = x^2-1$.

---

### Question 14

**Statement:** "When multiplying algebraic fractions, you should find the LCD first."

**Answer: FALSE**

**Correction:** For multiplication, you multiply numerators and denominators directly. You only need LCD for addition and subtraction.

**Explanation:**
$\frac{A}{B} \times \frac{C}{D} = \frac{AC}{BD}$ — just multiply across.

---

### Question 15

**Statement:** "If solving an equation with fractions gives $x = -3$, and one denominator is $x+3$, then $x = -3$ is an extraneous solution."

**Answer: TRUE**

**Explanation:**
If $x = -3$, then $x + 3 = 0$, making that denominator zero. This means the solution is extraneous and must be rejected.

---

### Question 16

**Statement:** "The expression $\frac{x^2-5x+6}{x-2}$ simplifies to $x-3$."

**Answer: TRUE**

**Explanation:**
$x^2-5x+6 = (x-2)(x-3)$

$\frac{(x-2)(x-3)}{x-2} = x-3$ (for $x \neq 2$)

---

### Question 17

**Statement:** "When dividing by a fraction, you should flip the second fraction and multiply."

**Answer: TRUE**

**Explanation:**
$\frac{A}{B} \div \frac{C}{D} = \frac{A}{B} \times \frac{D}{C}$

This is the standard method for dividing fractions.

---

### Question 18

**Statement:** "The fraction $\frac{3}{x-1} - \frac{2}{x+1}$ can be combined to give $\frac{x+5}{(x-1)(x+1)}$."

**Answer: TRUE**

**Explanation:**
$\frac{3(x+1) - 2(x-1)}{(x-1)(x+1)} = \frac{3x+3-2x+2}{(x-1)(x+1)} = \frac{x+5}{(x-1)(x+1)}$

---

### Question 19

**Statement:** "The expression $\frac{1}{x} + \frac{1}{y}$ equals $\frac{2}{x+y}$."

**Answer: FALSE**

**Correction:** $\frac{1}{x} + \frac{1}{y} = \frac{x+y}{xy}$

**Explanation:**
$\frac{1}{x} + \frac{1}{y} = \frac{y+ x}{xy} = \frac{x+y}{xy}$

---

### Question 20

**Statement:** "A fraction is undefined when its denominator equals zero."

**Answer: TRUE**

**Explanation:**
Division by zero is undefined in mathematics. This is why we must always check that our solutions don't make any denominator zero.

### Question 21 (3 marks)

Simplify $\frac{x^2+5x+6}{x^2-9}$

**Solution:**

| Step | Mark | Working |
|------|------|---------|
| Factorise numerator | M1 | $x^2+5x+6 = (x+2)(x+3)$ |
| Factorise denominator | M1 | $x^2-9 = (x+3)(x-3)$ |
| Cancel and simplify | A1 | $\frac{(x+2)(x+3)}{(x+3)(x-3)} = \frac{x+2}{x-3}$ |

**Answer:** $\frac{x+2}{x-3}$ (valid for $x \neq 3$, $x \neq -3$)

---

### Question 22 (4 marks)

Simplify $\frac{2x}{x+1} \div \frac{4x^2}{x^2-1}$

**Solution:**

| Step | Mark | Working |
|------|------|---------|
| Change to multiplication | M1 | $\frac{2x}{x+1} \times \frac{x^2-1}{4x^2}$ |
| Factorise | M1 | $\frac{2x}{x+1} \times \frac{(x+1)(x-1)}{4x^2}$ |
| Cancel common factors | M1 | $\frac{\cancel{2x}}{\cancel{x+1}} \times \frac{\cancel{(x+1)}(x-1)}{4\cancel{x^2}}$ |
| Simplify | A1 | $\frac{x-1}{2x}$ |

**Answer:** $\frac{x-1}{2x}$ (valid for $x \neq 0$, $x \neq 1$, $x \neq -1$)

---

### Question 23 (4 marks)

Simplify $\frac{2}{x-3} + \frac{1}{x+2}$

**Solution:**

| Step | Mark | Working |
|------|------|---------|
| Identify LCD | M1 | LCD = $(x-3)(x+2)$ |
| Rewrite fractions | M1 | $\frac{2(x+2)}{(x-3)(x+2)} + \frac{x-3}{(x-3)(x+2)}$ |
| Add numerators | M1 | $\frac{2(x+2) + (x-3)}{(x-3)(x+2)}$ |
| Simplify | A1 | $\frac{2x+4+x-3}{(x-3)(x+2)} = \frac{3x+1}{(x-3)(x+2)}$ |

**Answer:** $\frac{3x+1}{(x-3)(x+2)}$ or $\frac{3x+1}{x^2-x-6}$

---

### Question 24 (4 marks)

Solve $\frac{3x-1}{2} - \frac{x+2}{3} = 2$

**Solution:**

| Step | Mark | Working |
|------|------|---------|
| Find LCD | M1 | LCD = 6 |
| Multiply all terms by 6 | M1 | $6 \times \frac{3x-1}{2} - 6 \times \frac{x+2}{3} = 6 \times 2$ |
| Simplify | A1 | $3(3x-1) - 2(x+2) = 12$ |
| | | $9x - 3 - 2x - 4 = 12$ |
| | | $7x - 7 = 12$ |
| Solve | A1 | $7x = 19$, $x = \frac{19}{7}$ |

**Answer:** $x = \frac{19}{7}$

### Question 25 (5 marks)

Solve $\frac{2}{x-1} + \frac{1}{x+2} = 1$

**Solution:**

| Step | Mark | Working |
|------|------|---------|
| Find LCD | M1 | LCD = $(x-1)(x+2)$ |
| Multiply all terms | M1 | $2(x+2) + (x-1) = (x-1)(x+2)$ |
| Expand | M1 | $2x + 4 + x - 1 = x^2 + x - 2$ |
| | | $3x + 3 = x^2 + x - 2$ |
| Rearrange | M1 | $x^2 - 2x - 5 = 0$ |
| Solve using quadratic formula | A1 | $x = \frac{2 \pm \sqrt{4+20}}{2} = \frac{2 \pm \sqrt{24}}{2}$ |
| | | $x = 1 \pm \sqrt{6}$ |

**Answer:** $x = 1 + \sqrt{6}$ or $x = 1 - \sqrt{6}$

**Check:** Neither value makes $x-1 = 0$ or $x+2 = 0$, so both are valid.

### Question 26 (4 marks)

Simplify $\frac{x^2-4x+4}{x^2-4} \times \frac{x+2}{x-2}$

**Solution:**

| Step | Mark | Working |
|------|------|---------|
| Factorise all expressions | M1 | Numerator: $x^2-4x+4 = (x-2)^2$ |
| | | First denom: $x^2-4 = (x+2)(x-2)$ |
| Write factorised | M1 | $\frac{(x-2)^2}{(x+2)(x-2)} \times \frac{x+2}{x-2}$ |
| Cancel common factors | M1 | $\frac{(x-2)^{\cancel{2}}}{\cancel{(x+2)}\cancel{(x-2)}} \times \frac{\cancel{x+2}}{\cancel{(x-2)}}$ |
| Simplify | A1 | $= \frac{x-2}{1} = x-2$ |

**Answer:** $x-2$ (valid for $x \neq 2$, $x \neq -2$)

### Question 27 (5 marks)

**A number plus its reciprocal equals $\frac{5}{2}$. Find the number.**

**Solution:**

| Step | Mark | Working |
|------|------|---------|
| Set up equation | M1 | Let the number be $x$ |
| | | $x + \frac{1}{x} = \frac{5}{2}$ |
| Multiply by $2x$ to clear fractions | M1 | $2x^2 + 2 = 5x$ |
| Rearrange to standard form | M1 | $2x^2 - 5x + 2 = 0$ |
| Factorise | M1 | $(2x-1)(x-2) = 0$ |
| Solve | A1 | $x = \frac{1}{2}$ or $x = 2$ |

**Answer:** $x = \frac{1}{2}$ or $x = 2$

**Verification:**
- $\frac{1}{2} + 2 = \frac{1+4}{2} = \frac{5}{2}$ ✓
- $2 + \frac{1}{2} = \frac{5}{2}$ ✓

### Question 29 (4 marks)

Simplify $\frac{1}{x+1} - \frac{2}{x^2-1} + \frac{1}{x-1}$

**Solution:**

| Step | Mark | Working |
|------|------|---------|
| Factorise | M1 | $x^2-1 = (x+1)(x-1)$, so LCD = $(x+1)(x-1)$ |
| Rewrite all with LCD | M1 | $\frac{x-1}{(x+1)(x-1)} - \frac{2}{(x+1)(x-1)} + \frac{x+1}{(x+1)(x-1)}$ |
| Combine numerators | M1 | $\frac{x-1-2+x+1}{(x+1)(x-1)}$ |
| Simplify | A1 | $\frac{2x-2}{(x+1)(x-1)} = \frac{2(x-1)}{(x+1)(x-1)} = \frac{2}{x+1}$ |

**Answer:** $\frac{2}{x+1}$ (valid for $x \neq 1$, $x \neq -1$)

### Question 30 (6 marks)

**Show that** $\frac{x}{x+2} - \frac{2}{x+3} = \frac{x^2+3x-4}{(x+2)(x+3)}$

**Hence solve** $\frac{x}{x+2} - \frac{2}{x+3} = \frac{1}{2}$

**Solution:**

**Part 1: Show that**

| Step | Mark | Working |
|------|------|---------|
| Find LCD | M1 | LCD = $(x+2)(x+3)$ |
| Rewrite | M1 | $\frac{x(x+3)}{(x+2)(x+3)} - \frac{2(x+2)}{(x+2)(x+3)}$ |
| Combine | M1 | $\frac{x(x+3)-2(x+2)}{(x+2)(x+3)}$ |
| Expand numerator | A1 | $\frac{x^2+3x-2x-4}{(x+2)(x+3)} = \frac{x^2+x-4}{(x+2)(x+3)}$ |

**Correction to question:** The expansion gives $x^2+x-4$, not $x^2+3x-4$.

**Part 2: Solve (using corrected expression)**

| Step | Mark | Working |
|------|------|---------|
| Set up equation | M1 | $\frac{x^2+x-4}{(x+2)(x+3)} = \frac{1}{2}$ |
| Cross multiply | M1 | $2(x^2+x-4) = (x+2)(x+3)$ |
| Expand | M1 | $2x^2+2x-8 = x^2+5x+6$ |
| Rearrange | A1 | $x^2-3x-14 = 0$ |
| Solve | A1 | $x = \frac{3 \pm \sqrt{9+56}}{2} = \frac{3 \pm \sqrt{65}}{2}$ |

**Answer:** $x = \frac{3+\sqrt{65}}{2}$ or $x = \frac{3-\sqrt{65}}{2}$

### Question 31 (5 marks)

Simplify the complex fraction: $\frac{\frac{2}{x} - \frac{1}{x+1}}{\frac{1}{x} + \frac{1}{x+1}}$

**Solution:**

| Step | Mark | Working |
|------|------|---------|
| Find LCD for all fractions | M1 | LCD = $x(x+1)$ |
| Multiply numerator and denominator | M1 | $= \frac{x(x+1)\left(\frac{2}{x} - \frac{1}{x+1}\right)}{x(x+1)\left(\frac{1}{x} + \frac{1}{x+1}\right)}$ |
| Simplify numerator | M1 | Numerator: $2(x+1) - x = 2x+2-x = x+2$ |
| Simplify denominator | M1 | Denominator: $(x+1) + x = 2x+1$ |
| Final answer | A1 | $= \frac{x+2}{2x+1}$ |

**Answer:** $\frac{x+2}{2x+1}$

### Question 32 (6 marks)

**Find all values of $x$ for which** $\frac{x^2-1}{x+1} = x-1$

**Solution:**

| Step | Mark | Working |
|------|------|---------|
| Factorise LHS | M1 | $\frac{(x-1)(x+1)}{x+1}$ |
| Simplify (with restriction) | M1 | $= x-1$ provided $x+1 \neq 0$ |
| | | i.e., provided $x \neq -1$ |
| Analysis | M1 | The equation $\frac{x^2-1}{x+1} = x-1$ simplifies to $x-1 = x-1$ |
| | | This is always true (identity) |
| State solution | A1 | The equation holds for all values of $x$ EXCEPT $x = -1$ |
| Explain | A1 | When $x = -1$, the LHS is undefined (division by zero) |

**Answer:** All real values of $x$ except $x = -1$

### Question 33 (5 marks)

Solve $\frac{x+1}{x-1} - \frac{x-1}{x+1} = \frac{4}{3}$

**Solution:**

| Step | Mark | Working |
|------|------|---------|
| Find LCD | M1 | LCD = $(x-1)(x+1) = x^2-1$ |
| Rewrite LHS | M1 | $\frac{(x+1)^2 - (x-1)^2}{x^2-1}$ |
| Expand numerator | M1 | $\frac{x^2+2x+1 - (x^2-2x+1)}{x^2-1} = \frac{4x}{x^2-1}$ |
| Set up equation | M1 | $\frac{4x}{x^2-1} = \frac{4}{3}$ |
| Cross multiply and solve | A1 | $12x = 4(x^2-1)$ |
| | | $12x = 4x^2-4$ |
| | | $4x^2 - 12x - 4 = 0$ |
| | | $x^2 - 3x - 1 = 0$ |
| | | $x = \frac{3 \pm \sqrt{9+4}}{2} = \frac{3 \pm \sqrt{13}}{2}$ |

**Answer:** $x = \frac{3+\sqrt{13}}{2}$ or $x = \frac{3-\sqrt{13}}{2}$

### Question 34 (4 marks)

**A pipe can fill a tank in $x$ hours. A second pipe can fill the same tank in $x+2$ hours. Together they can fill the tank in 2.4 hours. Find $x$.**

**Solution:**

| Step | Mark | Working |
|------|------|---------|
| Set up rates equation | M1 | $\frac{1}{x} + \frac{1}{x+2} = \frac{1}{2.4} = \frac{5}{12}$ |
| Find LCD | M1 | LCD = $x(x+2)$ |
| Multiply through | M1 | $(x+2) + x = \frac{5x(x+2)}{12}$ |
| | | $2x+2 = \frac{5x^2+10x}{12}$ |
| | | $24x+24 = 5x^2+10x$ |
| | | $5x^2-14x-24 = 0$ |
| Solve | A1 | $(5x+6)(x-4) = 0$ |
| | | $x = 4$ or $x = -\frac{6}{5}$ |

**Answer:** $x = 4$ hours (reject negative value as time cannot be negative)

## EXAM TIPS SUMMARY

### For Simplifying Fractions:
1. ✅ Factorise numerator AND denominator first
2. ✅ Cancel ONLY common factors
3. ✅ Recognise difference of squares: $x^2-1$, $x^2-4$, $x^2-9$
4. ✅ State restrictions (what values make denominator zero)

### For Operations with Fractions:
1. ✅ Multiplication: factorise, cancel across, multiply
2. ✅ Division: flip second fraction, then multiply
3. ✅ Addition/Subtraction: find LCD first
4. ✅ Watch signs carefully in subtraction

### For Solving Equations:
1. ✅ Find LCD of ALL denominators
2. ✅ Multiply EVERY term by LCD
3. ✅ Check solutions don't make denominators zero
4. ✅ Reject extraneous solutions

### Key Warning Signs:
-  Denominator can be zero → check restrictions
-  Cancelling terms instead of factors → STOP, factorise first
-  Sign errors in subtraction → use brackets
-  Not multiplying all terms by LCD → check each term
