##  SECTION A: MULTIPLE CHOICE QUESTIONS (MCQs)

**Instructions:** Choose the correct answer from A, B, C, or D.

---

**Q1.** $y$ is directly proportional to $x$. When $x = 4$, $y = 20$. What is the value of $k$?

- A) 4
- B) 5
- C) 16
- D) 80

**Answer: B**

**Explanation:**
$$y = kx$$
$$20 = k \times 4$$
$$k = \frac{20}{4} = 5$$

---

**Q2.** $y$ is inversely proportional to $x$. When $x = 3$, $y = 12$. What is the equation connecting $y$ and $x$?

- A) $y = 4x$
- B) $y = 36x$
- C) $y = \frac{4}{x}$
- D) $y = \frac{36}{x}$

**Answer: D**

**Explanation:**
$$y = \frac{k}{x}$$
$$12 = \frac{k}{3}$$
$$k = 36$$
$$y = \frac{36}{x}$$

---

**Q3.** Simplify $x^3 \times x^5$

- A) $x^2$
- B) $x^8$
- C) $x^{15}$
- D) $2x^8$

**Answer: B**

**Explanation:**
Using $a^m \times a^n = a^{m+n}$:
$$x^3 \times x^5 = x^{3+5} = x^8$$

---

**Q4.** Simplify $\frac{a^9}{a^4}$

- A) $a^5$
- B) $a^{13}$
- C) $a^{36}$
- D) $a^{-5}$

**Answer: A**

**Explanation:**
Using $a^m \div a^n = a^{m-n}$:
$$\frac{a^9}{a^4} = a^{9-4} = a^5$$

---

**Q5.** Evaluate $7^0$

- A) 0
- B) 1
- C) 7
- D) 70

**Answer: B**

**Explanation:**
Any non-zero number to the power zero equals 1.
$$7^0 = 1$$

---

**Q6.** Simplify $(x^4)^3$

- A) $x^7$
- B) $x^1$
- C) $x^{12}$
- D) $x^{64}$

**Answer: C**

**Explanation:**
Using $(a^m)^n = a^{mn}$:
$$(x^4)^3 = x^{4 \times 3} = x^{12}$$

---

**Q7.** What is $4^{-2}$ equal to?

- A) $-8$
- B) $-16$
- C) $\frac{1}{16}$
- D) $\frac{1}{8}$

**Answer: C**

**Explanation:**
Using $a^{-n} = \frac{1}{a^n}$:
$$4^{-2} = \frac{1}{4^2} = \frac{1}{16}$$

---

**Q8.** Evaluate $25^{\frac{1}{2}}$

- A) 5
- B) 12.5
- C) $\frac{1}{25}$
- D) 25

**Answer: A**

**Explanation:**
$$25^{\frac{1}{2}} = \sqrt{25} = 5$$

---

**Q9.** $y$ is directly proportional to $x^2$. When $x = 3$, $y = 27$. Find $y$ when $x = 5$.

- A) 45
- B) 75
- C) 81
- D) 125

**Answer: B**

**Explanation:**
$$y = kx^2$$
$$27 = k \times 3^2 = 9k$$
$$k = 3$$
$$y = 3x^2$$
$$y = 3 \times 5^2 = 3 \times 25 = 75$$

---

**Q10.** Evaluate $8^{\frac{2}{3}}$

- A) 4
- B) $\frac{16}{3}$
- C) 16
- D) $\frac{8}{3}$

**Answer: A**

**Explanation:**
$$8^{\frac{2}{3}} = (\sqrt[3]{8})^2 = 2^2 = 4$$

---

**Q11.** Simplify $3x^2 \times 4x^3$

- A) $7x^5$
- B) $12x^5$
- C) $12x^6$
- D) $7x^6$

**Answer: B**

**Explanation:**
$$3x^2 \times 4x^3 = 3 \times 4 \times x^2 \times x^3 = 12x^{2+3} = 12x^5$$

---

**Q12.** Express $x^{-3}$ with a positive index.

- A) $x^3$
- B) $\frac{1}{x^3}$
- C) $-x^3$
- D) $\frac{1}{x^{-3}}$

**Answer: B**

**Explanation:**
Using $a^{-n} = \frac{1}{a^n}$:
$$x^{-3} = \frac{1}{x^3}$$

---

**Q13.** $y$ is inversely proportional to $x$. When $x = 2$, $y = 10$. Find $x$ when $y = 4$.

- A) 4
- B) 5
- C) 8
- D) 20

**Answer: B**

**Explanation:**
$$y = \frac{k}{x}$$
$$10 = \frac{k}{2} \Rightarrow k = 20$$
$$y = \frac{20}{x}$$
$$4 = \frac{20}{x} \Rightarrow 4x = 20 \Rightarrow x = 5$$

---

**Q14.** Simplify $\frac{15x^6}{5x^2}$

- A) $3x^3$
- B) $3x^4$
- C) $10x^4$
- D) $3x^8$

**Answer: B**

**Explanation:**
$$\frac{15x^6}{5x^2} = \frac{15}{5} \times \frac{x^6}{x^2} = 3x^{6-2} = 3x^4$$

---

**Q15.** Which of these expressions equals $x^0$?

- A) 0
- B) 1
- C) $x$
- D) $\frac{1}{x}$

**Answer: B**

**Explanation:**
For any non-zero $x$:
$$x^0 = 1$$

---

**Q16.** Evaluate $16^{-\frac{1}{2}}$

- A) $-4$
- B) $4$
- C) $\frac{1}{4}$
- D) $-\frac{1}{4}$

**Answer: C**

**Explanation:**
$$16^{-\frac{1}{2}} = \frac{1}{16^{\frac{1}{2}}} = \frac{1}{\sqrt{16}} = \frac{1}{4}$$

##  SECTION B: TRUE OR FALSE

**Instructions:** State whether each statement is TRUE or FALSE. If FALSE, provide the correction.

---

**Q1.** If $y \propto x$, then $y = kx$ where $k$ is a constant.

**Answer: TRUE**

**Explanation:** This is the definition of direct proportion. $y$ equals $k$ times $x$.

---

**Q2.** If $y$ is inversely proportional to $x$, then $y = \frac{k}{x}$ where $k = \frac{y}{x}$.

**Answer: FALSE**

**Correction:** $k = xy$, not $\frac{y}{x}$.

For inverse proportion $y = \frac{k}{x}$, multiplying both sides by $x$ gives $xy = k$.

---

**Q3.** $x^4 \times x^3 = x^7$

**Answer: TRUE**

**Explanation:** Using $a^m \times a^n = a^{m+n}$:
$$x^4 \times x^3 = x^{4+3} = x^7$$

---

**Q4.** $(x^3)^2 = x^5$

**Answer: FALSE**

**Correction:** $(x^3)^2 = x^{3 \times 2} = x^6$

When raising a power to a power, multiply the indices.

---

**Q5.** $\frac{x^8}{x^2} = x^4$

**Answer: FALSE**

**Correction:** $\frac{x^8}{x^2} = x^{8-2} = x^6$

When dividing, subtract the indices, not divide them.

---

**Q6.** $5^0 = 0$

**Answer: FALSE**

**Correction:** $5^0 = 1$

Any non-zero number to the power zero equals 1.

---

**Q7.** $2^3 \times 3^2 = 6^5$

**Answer: FALSE**

**Correction:** Indices laws only apply when bases are the same.

$2^3 \times 3^2 = 8 \times 9 = 72$

---

**Q8.** $a^{-2} = \frac{1}{a^2}$

**Answer: TRUE**

**Explanation:** A negative index means "one over" the positive power.

$$a^{-2} = \frac{1}{a^2}$$

---

**Q9.** $27^{\frac{1}{3}} = 9$

**Answer: FALSE**

**Correction:** $27^{\frac{1}{3}} = \sqrt[3]{27} = 3$

The index $\frac{1}{3}$ means cube root, not division.

---

**Q10.** If $y \propto x^2$ and $x$ doubles, then $y$ quadruples (increases by a factor of 4).

**Answer: TRUE**

**Explanation:**
If $x$ becomes $2x$:
$$y_{new} = k(2x)^2 = k \times 4x^2 = 4kx^2 = 4y_{old}$$

---

**Q11.** $\frac{1}{x^{-3}} = x^3$

**Answer: TRUE**

**Explanation:**
$$\frac{1}{x^{-3}} = \frac{1}{\frac{1}{x^3}} = x^3$$

---

**Q12.** $(2x)^3 = 2x^3$

**Answer: FALSE**

**Correction:** $(2x)^3 = 2^3 \times x^3 = 8x^3$

Everything inside the brackets is cubed.

### Question 1 (2 marks)

$y$ is directly proportional to $x$. When $x = 6$, $y = 24$.

Find the equation connecting $y$ and $x$.

**Solution:**

**[M1]** Write formula and substitute:
$$y = kx$$
$$24 = k \times 6$$

**[A1]** Solve for $k$ and state equation:
$$k = \frac{24}{6} = 4$$
$$y = 4x$$

**Answer: $y = 4x$**

---

### Question 2 (2 marks)

Simplify $(x^2)^5 \times x^3$

**Solution:**

**[M1]** Apply power of power rule:
$$(x^2)^5 = x^{2 \times 5} = x^{10}$$

**[A1]** Multiply by $x^3$:
$$x^{10} \times x^3 = x^{10+3} = x^{13}$$

**Answer: $x^{13}$**

---

### Question 3 (3 marks)

$y$ is inversely proportional to $x$. When $x = 4$, $y = 5$.

(a) Find the equation connecting $y$ and $x$. **[2 marks]**

(b) Find $y$ when $x = 10$. **[1 mark]**

**Solution:**

**(a) [M1]** Write formula and substitute:
$$y = \frac{k}{x}$$
$$5 = \frac{k}{4}$$

**[A1]** Solve for $k$:
$$k = 5 \times 4 = 20$$
$$y = \frac{20}{x}$$

**(b) [B1]** Substitute $x = 10$:
$$y = \frac{20}{10} = 2$$

**Answer: (a) $y = \frac{20}{x}$ (b) $y = 2$**

---

### Question 4 (3 marks)

Simplify $\frac{12a^8b^6}{4a^3b^2}$

**Solution:**

**[M1]** Divide coefficients and apply division rule to each variable:
$$\frac{12}{4} = 3$$

**[M1]** Apply indices rule for each letter:
$$\frac{a^8}{a^3} = a^{8-3} = a^5$$
$$\frac{b^6}{b^2} = b^{6-2} = b^4$$

**[A1]** Combine:
$$\frac{12a^8b^6}{4a^3b^2} = 3a^5b^4$$

**Answer: $3a^5b^4$**

---

### Question 5 (3 marks)

Evaluate $16^{\frac{3}{2}}$

**Solution:**

**[M1]** Recognise fractional index means square root then cube:
$$16^{\frac{3}{2}} = (\sqrt{16})^3$$

**[M1]** Calculate square root:
$$\sqrt{16} = 4$$

**[A1]** Cube the result:
$$4^3 = 64$$

**Answer: $64$**

### Question 6 (4 marks)

$y$ is directly proportional to the square of $x$. When $x = 5$, $y = 50$.

(a) Find the equation connecting $y$ and $x$. **[2 marks]**

(b) Find the value of $x$ when $y = 162$. **[2 marks]**

**Solution:**

**(a) [M1]** Write formula and substitute:
$$y = kx^2$$
$$50 = k \times 5^2$$
$$50 = 25k$$

**[A1]** Solve for $k$:
$$k = 2$$
$$y = 2x^2$$

**(b) [M1]** Substitute $y = 162$:
$$162 = 2x^2$$
$$x^2 = 81$$

**[A1]** Solve:
$$x = \pm 9$$

**Answer: (a) $y = 2x^2$ (b) $x = \pm 9$**

---

### Question 7 (4 marks)

Simplify $\frac{(x^3)^4 \times x^2}{x^5 \times x^3}$

**Solution:**

**[M1]** Simplify $(x^3)^4$:
$$(x^3)^4 = x^{12}$$

**[M1]** Simplify numerator:
$$x^{12} \times x^2 = x^{14}$$

**[M1]** Simplify denominator:
$$x^5 \times x^3 = x^8$$

**[A1]** Divide:
$$\frac{x^{14}}{x^8} = x^{14-8} = x^6$$

**Answer: $x^6$**

---

### Question 8 (4 marks)

(a) Express $x^{-2}y^3$ with positive indices only. **[2 marks]**

(b) Express $\frac{3}{x^4}$ using a negative index. **[2 marks]**

**Solution:**

**(a) [M1]** Recognise $x^{-2} = \frac{1}{x^2}$:
$$x^{-2}y^3 = \frac{y^3}{x^2}$$

**[A1]** 
$$= \frac{y^3}{x^2}$$

**(b) [M1]** Recognise $\frac{1}{x^4} = x^{-4}$:
$$\frac{3}{x^4} = 3 \times \frac{1}{x^4} = 3x^{-4}$$

**[A1]**
$$= 3x^{-4}$$

**Answer: (a) $\frac{y^3}{x^2}$ (b) $3x^{-4}$**

### Question 9 (5 marks)

The time $t$ hours to complete a journey is inversely proportional to the speed $s$ km/h.

When $s = 50$, $t = 4$.

(a) Find the equation connecting $t$ and $s$. **[2 marks]**

(b) Find the time when $s = 80$. **[2 marks]**

(c) What speed is needed to complete the journey in 5 hours? **[1 mark]**

**Solution:**

**(a) [M1]** Write formula and substitute:
$$t = \frac{k}{s}$$
$$4 = \frac{k}{50}$$

**[A1]** Solve for $k$:
$$k = 4 \times 50 = 200$$
$$t = \frac{200}{s}$$

**(b) [M1]** Substitute $s = 80$:
$$t = \frac{200}{80}$$

**[A1]** Calculate:
$$t = 2.5 \text{ hours}$$

**(c) [B1]** Substitute $t = 5$:
$$5 = \frac{200}{s}$$
$$s = \frac{200}{5} = 40 \text{ km/h}$$

**Answer: (a) $t = \frac{200}{s}$ (b) $2.5$ hours (c) $40$ km/h**

### Question 10 (5 marks)

Simplify fully $(2x^3)^3 \times \frac{8x^{10}}{4x^4}$

**Solution:**

**[M1]** Expand $(2x^3)^3$:
$$(2x^3)^3 = 2^3 \times (x^3)^3 = 8x^9$$

**[M1]** Simplify the fraction:
$$\frac{8x^{10}}{4x^4} = \frac{8}{4} \times x^{10-4} = 2x^6$$

**[M1]** Multiply the results:
$$8x^9 \times 2x^6$$

**[M1]** Multiply coefficients and add indices:
$$8 \times 2 = 16$$
$$x^9 \times x^6 = x^{15}$$

**[A1]** Final answer:
$$16x^{15}$$

**Answer: $16x^{15}$**

### Question 11 (6 marks)

$y$ is inversely proportional to the square of $x$. When $x = 2$, $y = 9$.

(a) Find the equation connecting $y$ and $x$. **[2 marks]**

(b) Find $y$ when $x = 3$. **[2 marks]**

(c) Find the positive value of $x$ when $y = 36$. **[2 marks]**

**Solution:**

**(a) [M1]** Write formula and substitute:
$$y = \frac{k}{x^2}$$
$$9 = \frac{k}{2^2}$$
$$9 = \frac{k}{4}$$

**[A1]** Solve for $k$:
$$k = 36$$
$$y = \frac{36}{x^2}$$

**(b) [M1]** Substitute $x = 3$:
$$y = \frac{36}{3^2} = \frac{36}{9}$$

**[A1]**
$$y = 4$$

**(c) [M1]** Substitute $y = 36$:
$$36 = \frac{36}{x^2}$$

**[A1]** Solve:
$$36x^2 = 36$$
$$x^2 = 1$$
$$x = 1 \text{ (positive value)}$$

**Answer: (a) $y = \frac{36}{x^2}$ (b) $y = 4$ (c) $x = 1$**

### Question 12 (6 marks)

(a) Evaluate $27^{\frac{2}{3}}$ **[2 marks]**

(b) Evaluate $16^{-\frac{3}{2}}$ **[2 marks]**

(c) Given that $8^x = \frac{1}{64}$, find the value of $x$. **[2 marks]**

**Solution:**

**(a) [M1]** Cube root then square:
$$27^{\frac{2}{3}} = (\sqrt[3]{27})^2 = 3^2$$

**[A1]**
$$= 9$$

**(b) [M1]** Recognise negative index and evaluate:
$$16^{-\frac{3}{2}} = \frac{1}{16^{\frac{3}{2}}}$$
$$16^{\frac{1}{2}} = 4$$

**[A1]**
$$= \frac{1}{4^3} = \frac{1}{64}$$

**(c) [M1]** Express both sides with base 8:
$$8^x = \frac{1}{64} = \frac{1}{8^2} = 8^{-2}$$

**[A1]**
$$x = -2$$

**Answer: (a) $9$ (b) $\frac{1}{64}$ (c) $x = -2$**

## 📊 SECTION D: EXAM FOCUS & STRATEGY

### Frequently Tested Question Types

| Type | Description | Typical Marks |
|------|-------------|---------------|
| **Direct proportion equation** | Find $k$, write equation | 2-3 |
| **Inverse proportion equation** | Find $k$, write equation | 2-3 |
| **Proportion with powers** | $y \propto x^2$ or $y \propto \frac{1}{x^2}$ | 3-4 |
| **Simplify indices (basic)** | Apply one law | 1-2 |
| **Simplify indices (complex)** | Multiple laws | 2-3 |
| **Evaluate fractional indices** | Calculate value | 2-3 |
| **Negative indices** | Convert to positive | 2 |
| **Word problems with proportion** | Real-world applications | 4-6 |

### Typical Problem-Solving Patterns

**Pattern 1: Proportion Word Problem**
```
1. Read and identify type (direct/inverse)
2. Write formula (y = kx or y = k/x)
3. Substitute given values
4. Find k
5. Write complete equation
6. Use to find unknown
7. Include units in answer
```

**Pattern 2: Simplify Indices**
```
1. Identify which law applies
2. Apply law step by step
3. Work with coefficients separately
4. Combine like terms
5. Express with positive indices if asked
```

**Pattern 3: Evaluate Fractional/Negative Indices**
```
1. Identify if negative → reciprocal first
2. Identify if fractional → root first
3. Work through systematically
4. Give exact value or decimal as required
```

### Command Word Guidance

| Command | What Examiners Expect |
|---------|----------------------|
| **Simplify** | Reduce to simplest single term |
| **Evaluate** | Calculate numerical value |
| **Express** | Write in specified form |
| **Find** | Determine value or equation |
| **Show that** | Full working proving result |
| **Write down** | No working needed |

### How to Gain Full Marks

**For Proportion Questions (3-4 marks):**
1. ✅ State the proportion type clearly
2. ✅ Write the correct formula ($y = kx$ or $y = \frac{k}{x}$)
3. ✅ Show substitution of known values
4. ✅ Find $k$ explicitly
5. ✅ Write the complete equation
6. ✅ Include units in final answer (word problems)

**For Indices Questions (2-3 marks):**
1. ✅ Apply correct law
2. ✅ Show working clearly
3. ✅ Simplify fully
4. ✅ Express with positive indices if requested

**For Complex Indices (3-4 marks):**
1. ✅ Work step by step
2. ✅ Handle coefficients separately
3. ✅ Apply correct law at each step
4. ✅ Give final simplified answer

### Common Pitfalls to Avoid

1. **Confusing direct and inverse proportion**
   - Read the question carefully
   - Look for "directly" vs "inversely" vs "varies inversely"

2. **Forgetting to square when $y \propto x^2$**
   - Substitute $x^2$, not just $x$
   - Example: When $x = 3$, use $x^2 = 9$

3. **Wrong indices law**
   - Multiplying: ADD powers
   - Dividing: SUBTRACT powers
   - Power of power: MULTIPLY powers

4. **Same base requirement**
   - $x^3 \times y^4$ cannot be simplified
   - Indices laws only work with SAME base

5. **Fractional indices confusion**
   - Denominator = ROOT
   - Numerator = POWER
   - Example: $8^{\frac{2}{3}} = (\sqrt[3]{8})^2$
