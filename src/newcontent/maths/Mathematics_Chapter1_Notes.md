# 1. Working with Fractions

## 1.1 Understanding Fractions

A **fraction** represents a part of a whole. It consists of two parts:
- **Numerator** (top number): the number of parts we have
- **Denominator** (bottom number): the total number of equal parts

$$\text{Fraction} = \frac{\text{numerator}}{\text{denominator}}$$

### Types of Fractions

| Type | Description | Example |
|------|-------------|---------|
| **Proper fraction** | Numerator < Denominator | $\frac{3}{4}, \frac{2}{5}$ |
| **Improper fraction** | Numerator ≥ Denominator | $\frac{7}{3}, \frac{5}{2}$ |
| **Mixed number** | Whole number + proper fraction | $2\frac{1}{3}, 3\frac{4}{5}$ |
| **Equivalent fractions** | Same value, different form | $\frac{1}{2} = \frac{2}{4} = \frac{3}{6}$ |

---

## 1.2 Converting Between Improper Fractions and Mixed Numbers

### Improper Fraction → Mixed Number

**Method:** Divide the numerator by the denominator.
- The **quotient** becomes the whole number
- The **remainder** becomes the new numerator
- The denominator stays the same

**Worked Example 1:**
Convert $\frac{17}{5}$ to a mixed number.

**Solution:**
$$17 \div 5 = 3 \text{ remainder } 2$$

$$\frac{17}{5} = 3\frac{2}{5}$$

**Why this works:** We can think of $\frac{17}{5}$ as having 17 fifths. Since 5 fifths make one whole, we can make 3 wholes (using 15 fifths) with 2 fifths left over.

### Mixed Number → Improper Fraction

**Method:** Multiply the whole number by the denominator, add the numerator, then put over the denominator.

**Formula:**
$$a\frac{b}{c} = \frac{a \times c + b}{c}$$

**Worked Example 2:**
Convert $4\frac{3}{7}$ to an improper fraction.

**Solution:**
$$4\frac{3}{7} = \frac{4 \times 7 + 3}{7} = \frac{28 + 3}{7} = \frac{31}{7}$$

>  **Common Mistake:** Students often add all numbers together: $4\frac{3}{7} \neq \frac{4+3}{7} = \frac{7}{7} = 1$
> 
> **Remember:** You must multiply the whole number by the denominator FIRST!

---

## 1.3 Adding and Subtracting Fractions

### Same Denominators

When fractions have the **same denominator**, simply add or subtract the numerators:

$$\frac{a}{c} + \frac{b}{c} = \frac{a + b}{c}$$

**Worked Example 3:**
Calculate $\frac{3}{11} + \frac{5}{11}$

**Solution:**
$$\frac{3}{11} + \frac{5}{11} = \frac{3 + 5}{11} = \frac{8}{11}$$

### Different Denominators

When denominators are **different**, find a **common denominator** first.

**Key Concept: Common Denominator**
A common denominator is a number that both denominators divide into exactly. The **lowest common denominator (LCD)** is the smallest such number.

**Method:**
1. Find the LCD (often the LCM of the denominators)
2. Convert each fraction to an equivalent fraction with the LCD
3. Add or subtract the numerators
4. Simplify if possible

**Worked Example 4:**
Calculate $\frac{2}{3} + \frac{1}{4}$

**Solution:**
- **Step 1:** Find LCD of 3 and 4
  - Multiples of 3: 3, 6, 9, 12, 15...
  - Multiples of 4: 4, 8, 12, 16...
  - LCD = 12

- **Step 2:** Convert each fraction:
  $$\frac{2}{3} = \frac{2 \times 4}{3 \times 4} = \frac{8}{12}$$
  $$\frac{1}{4} = \frac{1 \times 3}{4 \times 3} = \frac{3}{12}$$

- **Step 3:** Add the numerators:
  $$\frac{8}{12} + \frac{3}{12} = \frac{11}{12}$$

>  **Exam Tip:** Always check if your answer can be simplified. In this case, $\frac{11}{12}$ cannot be simplified as 11 is prime and doesn't divide 12.

**Worked Example 5 (with mixed numbers):**
Calculate $3\frac{1}{2} - 1\frac{2}{3}$

**Method 1 - Convert to improper fractions first:**
$$3\frac{1}{2} = \frac{7}{2} \quad \text{and} \quad 1\frac{2}{3} = \frac{5}{3}$$

LCD of 2 and 3 is 6:
$$\frac{7}{2} = \frac{21}{6} \quad \text{and} \quad \frac{5}{3} = \frac{10}{6}$$

$$\frac{21}{6} - \frac{10}{6} = \frac{11}{6} = 1\frac{5}{6}$$

**Method 2 - Work with mixed numbers:**
$$3\frac{1}{2} = 3\frac{3}{6} \quad \text{and} \quad 1\frac{2}{3} = 1\frac{4}{6}$$

Subtract whole numbers: $3 - 1 = 2$
Subtract fractions: $\frac{3}{6} - \frac{4}{6} = -\frac{1}{6}$ (need to borrow!)

$$= 2 - \frac{1}{6} = 1\frac{5}{6}$$

>  **Common Mistake:** Forgetting to "borrow" when the top fraction is smaller than the bottom fraction in subtraction.

## 1.4 Multiplying Fractions

### The Rule

**Multiply the numerators together, multiply the denominators together:**

$$\frac{a}{b} \times \frac{c}{d} = \frac{a \times c}{b \times d}$$

### Simplifying Before Multiplying

**Key Technique:** Cross-cancellation (simplify diagonally before multiplying)

**Worked Example 6:**
Calculate $\frac{4}{9} \times \frac{3}{8}$

**Method - Without simplifying first:**
$$\frac{4}{9} \times \frac{3}{8} = \frac{4 \times 3}{9 \times 8} = \frac{12}{72} = \frac{1}{6}$$

**Method - With cross-cancellation:**
$$\frac{4}{9} \times \frac{3}{8} = \frac{\cancel{4}^{1}}{\cancel{9}^{3}} \times \frac{\cancel{3}^{1}}{\cancel{8}^{2}} = \frac{1 \times 1}{3 \times 2} = \frac{1}{6}$$

**Why cross-cancellation works:**
- 4 and 8 share a common factor of 4: $4 \div 4 = 1$, $8 \div 4 = 2$
- 3 and 9 share a common factor of 3: $3 \div 3 = 1$, $9 \div 3 = 3$

>  **Exam Tip:** Cross-cancellation reduces the numbers you work with and avoids the need to simplify a large fraction at the end. This saves time and reduces errors!

### Multiplying Mixed Numbers

**Method:** Convert to improper fractions first.

**Worked Example 7:**
Calculate $2\frac{1}{3} \times 1\frac{1}{2}$

**Solution:**
$$2\frac{1}{3} = \frac{7}{3} \quad \text{and} \quad 1\frac{1}{2} = \frac{3}{2}$$

$$\frac{7}{3} \times \frac{3}{2} = \frac{\cancel{7}^{7}}{\cancel{3}^{1}} \times \frac{\cancel{3}^{1}}{2} = \frac{7}{2} = 3\frac{1}{2}$$

## 1.5 Dividing Fractions

### The Rule

**Keep the first fraction, change ÷ to ×, flip the second fraction:**

$$\frac{a}{b} \div \frac{c}{d} = \frac{a}{b} \times \frac{d}{c}$$

This is often remembered as **KCF**: **K**eep, **C**hange, **F**lip

### Why This Works

Dividing by a fraction is the same as multiplying by its reciprocal. The reciprocal of $\frac{c}{d}$ is $\frac{d}{c}$.

$$\frac{a}{b} \div \frac{c}{d} = \frac{a}{b} \times \frac{1}{\frac{c}{d}} = \frac{a}{b} \times \frac{d}{c}$$

**Worked Example 8:**
Calculate $\frac{3}{4} \div \frac{2}{5}$

**Solution:**
$$\frac{3}{4} \div \frac{2}{5} = \frac{3}{4} \times \frac{5}{2} = \frac{15}{8} = 1\frac{7}{8}$$

**Worked Example 9 (with mixed numbers):**
Calculate $3\frac{1}{2} \div 1\frac{3}{4}$

**Solution:**
Convert to improper fractions:
$$3\frac{1}{2} = \frac{7}{2} \quad \text{and} \quad 1\frac{3}{4} = \frac{7}{4}$$

Apply KCF:
$$\frac{7}{2} \div \frac{7}{4} = \frac{7}{2} \times \frac{4}{7} = \frac{\cancel{7}^{1}}{\cancel{2}^{1}} \times \frac{\cancel{4}^{2}}{\cancel{7}^{1}} = \frac{1 \times 2}{1 \times 1} = 2$$

>  **Common Mistake:** Only flip the SECOND fraction, not the first one!

# 2. Order of Operations (BIDMAS)

## 2.1 Understanding BIDMAS

When a calculation has multiple operations, we must follow a specific order:

| Order | Letter | Meaning | Examples |
|-------|--------|---------|----------|
| 1st | **B** | Brackets | $(...), [...], \{...\}$ |
| 2nd | **I** | Indices (Powers) | $2^3, 5^2, \sqrt{9}$ |
| 3rd | **D** | Division | $÷, /$ |
| 4th | **M** | Multiplication | $×, \cdot$ |
| 5th | **A** | Addition | $+$ |
| 6th | **S** | Subtraction | $-$ |

>  **Important:** Division and Multiplication have **equal priority** - work from left to right.
> Similarly, Addition and Subtraction have **equal priority** - work from left to right.

**Alternative acronyms:** BODMAS (Orders instead of Indices), PEMDAS (Parentheses, Exponents, Multiplication, Division, Addition, Subtraction)

## 2.2 Worked Examples

**Worked Example 10:**
Calculate $3 + 4 \times 2$

**Solution:**
Multiplication before addition:
$$3 + 4 \times 2 = 3 + 8 = 11$$

>  **Common Mistake:** Doing addition first gives $7 \times 2 = 14$ (INCORRECT!)

**Worked Example 11:**
Calculate $(3 + 4) \times 2$

**Solution:**
Brackets first:
$$(3 + 4) \times 2 = 7 \times 2 = 14$$

**Key Insight:** Brackets change everything! Example 10 gives 11, Example 11 gives 14.

**Worked Example 12:**
Calculate $20 - 12 \div 4 + 3$

**Solution:**
Following BIDMAS:
1. Division first: $12 \div 4 = 3$
2. Then work left to right: $20 - 3 + 3$
3. $20 - 3 = 17$
4. $17 + 3 = 20$

Answer: $20$

**Worked Example 13:**
Calculate $3 + 2^3 \times 4 - 10 \div 2$

**Solution:**
1. **I**ndices: $2^3 = 8$
2. **D**ivision: $10 \div 2 = 5$
3. **M**ultiplication: $8 \times 4 = 32$
4. Work left to right for addition/subtraction:
   - $3 + 32 = 35$
   - $35 - 5 = 30$

Answer: $30$

**Worked Example 14 (Nested Brackets):**
Calculate $2 \times (3 + 4 \times (5 - 2))$

**Solution:**
Work from innermost brackets outward:
1. Innermost brackets: $5 - 2 = 3$
2. Now: $2 \times (3 + 4 \times 3)$
3. Inside brackets, multiplication first: $4 \times 3 = 12$
4. Inside brackets: $3 + 12 = 15$
5. Finally: $2 \times 15 = 30$

Answer: $30$

**Worked Example 15 (With Negatives):**
Calculate $24 \div (-3 - 5)$

**Solution:**
1. Brackets first: $-3 - 5 = -8$
2. Division: $24 \div (-8) = -3$

Answer: $-3$

## 2.3 Fraction of a Number

Finding a fraction of a number means **multiplying** by that fraction.

**Worked Example 16:**
Find $\frac{3}{5}$ of 40

**Solution:**
$$\frac{3}{5} \times 40 = \frac{3 \times 40}{5} = \frac{120}{5} = 24$$

**Alternative method:**
$$\frac{3}{5} \times 40 = 3 \times \frac{40}{5} = 3 \times 8 = 24$$

**Worked Example 17:**
Find $\frac{2}{3}$ of £45

**Solution:**
$$\frac{2}{3} \times 45 = \frac{2 \times 45}{3} = \frac{90}{3} = 30$$

The answer is £30.

# 3. Significant Figures and Decimal Places

## 3.1 Rounding to Decimal Places

### The Rule

To round to **n decimal places**:
1. Look at the digit in position **(n+1)**
2. If this digit is **5 or more**, round up
3. If this digit is **4 or less**, round down (keep the same)

**Worked Example 18:**
Round 3.847 to:
- a) 1 decimal place
- b) 2 decimal places

**Solution:**
a) 1 d.p. - Look at the second decimal digit (4)
   - Since $4 < 5$, round down
   - **Answer: 3.8**

b) 2 d.p. - Look at the third decimal digit (7)
   - Since $7 \geq 5$, round up
   - **Answer: 3.85**

**Worked Example 19:**
Round 7.295 to 2 decimal places

**Solution:**
- Look at the third decimal digit (5)
- Since $5 \geq 5$, round up
- The 9 becomes 10, so we carry the 1
- **Answer: 7.30** (NOT 7.3 - we must show the placeholder zero!)

>  **Common Mistake:** Writing 7.3 instead of 7.30. When asked for 2 d.p., you MUST give 2 decimal places!

## 3.2 Significant Figures

### What are Significant Figures?

**Significant figures** show the precision of a number. They count from the first non-zero digit.

### Rules for Identifying Significant Figures

| Rule | Example | Significant figures |
|------|---------|---------------------|
| All non-zero digits are significant | 435 | 3 s.f. |
| Zeros between non-zero digits are significant | 4005 | 4 s.f. |
| Leading zeros are NOT significant | 0.0045 | 2 s.f. |
| Trailing zeros after decimal point are significant | 3.200 | 4 s.f. |
| Trailing zeros in whole numbers may be ambiguous | 4500 | Could be 2, 3, or 4 s.f. |

### Rounding to Significant Figures

**Worked Example 20:**
Round 4738 to:
- a) 1 significant figure
- b) 2 significant figures
- c) 3 significant figures

**Solution:**
a) 1 s.f.: Look at digit 2 (7). Since $7 \geq 5$, round up
   - **Answer: 5000**

b) 2 s.f.: Look at digit 3 (3). Since $3 < 5$, round down
   - **Answer: 4700**

c) 3 s.f.: Look at digit 4 (8). Since $8 \geq 5$, round up
   - **Answer: 4740**

**Worked Example 21:**
Round 0.00638 to:
- a) 1 significant figure
- b) 2 significant figures

**Solution:**
First, identify the significant figures: 6, 3, 8

a) 1 s.f.: Look at digit 2 (3). Since $3 < 5$, round down
   - **Answer: 0.006**

b) 2 s.f.: Look at digit 3 (8). Since $8 \geq 5$, round up
   - **Answer: 0.0064**

**Worked Example 22:**
Round 0.0407 to 2 significant figures

**Solution:**
Significant figures: 4, 0, 7
- Look at digit 3 (7). Since $7 \geq 5$, round up
- The 0 becomes 1
- **Answer: 0.041**

## 3.3 Accuracy in Calculations

### Decimal Places in Calculations

**Rule:** When doing multi-step calculations, **don't round until the final answer**. Keep extra decimal places during working.

**Worked Example 23:**
Calculate $\frac{3.7 \times 2.4}{5.1}$ giving your answer to 2 decimal places.

**Solution:**
$$\frac{3.7 \times 2.4}{5.1} = \frac{8.88}{5.1} = 1.74117...$$

Round to 2 d.p.: **Answer: 1.74**

>  **Common Mistake:** Rounding intermediate answers leads to accumulated errors!
> - WRONG: $\frac{3.7 \times 2.4}{5.1} = \frac{8.9}{5.1}$ (rounded) $= 1.745...$ ≈ 1.75
> - RIGHT: Keep full precision until the end

# 4. Exam Focus & Strategy

## 🔥 Frequently Tested Question Types

1. **Mixed fraction calculations** - Questions combining all operations
2. **Word problems involving fractions** - Real-life contexts (sharing, recipes)
3. **Multi-step BIDMAS problems** - Testing order of operations
4. **Rounding accuracy questions** - Often combined with calculations
5. **Calculator vs non-calculator** - Different skills required

## 🧩 Typical Problem-Solving Patterns

### Pattern 1: Fraction Word Problems
- Read carefully to identify the operation
- "of" usually means multiply
- "shared between" usually means divide

### Pattern 2: BIDMAS Trap Questions
- Examiner tests if you know order of operations
- Common trap: $3 + 4 \times 2$ vs $(3 + 4) \times 2$
- Always apply BIDMAS carefully

### Pattern 3: Accuracy Questions
- Often combined with other topics
- Remember to show full working
- Don't round until the final answer

## 📝 Command Word Guidance

| Command Word | What to Do |
|--------------|------------|
| **Calculate** | Show working, give numerical answer |
| **Work out** | Same as calculate - show all steps |
| **Simplify** | Give answer in simplest form |
| **Evaluate** | Calculate the value |
| **Write down** | No working needed, just the answer |
| **Show that** | Prove the given result is correct |
| **Give your answer to...** | Round as specified |

##  How to Gain Full Marks

1. **Show all working** - Even if you can do it mentally
2. **Check your answer** - Does it make sense?
3. **Use estimation** - Helps catch silly errors
4. **Give exact answers when required** - Don't round unnecessarily
5. **Include units** when appropriate
6. **Read the question twice** - Understand what's being asked

## ⚙️ Common Mistakes to Avoid

| Mistake | Correct Approach |
|---------|------------------|
| Adding denominators when adding fractions | Find a common denominator first |
| Flipping both fractions when dividing | Only flip the second fraction |
| Working left-to-right ignoring BIDMAS | Follow BIDMAS order strictly |
| Forgetting to simplify final answers | Always check if fraction can be simplified |
| Rounding too early in calculations | Keep full precision until the end |
| Dropping placeholder zeros | 7.30 must be written with the zero for 2 d.p. |

# 5. Key Formulas Summary

| Operation | Formula |
|-----------|---------|
| Adding fractions | $\frac{a}{c} + \frac{b}{c} = \frac{a+b}{c}$ |
| Subtracting fractions | $\frac{a}{c} - \frac{b}{c} = \frac{a-b}{c}$ |
| Different denominators | Find LCD, convert, then add/subtract |
| Multiplying fractions | $\frac{a}{b} \times \frac{c}{d} = \frac{ac}{bd}$ |
| Dividing fractions | $\frac{a}{b} \div \frac{c}{d} = \frac{a}{b} \times \frac{d}{c}$ |
| Mixed to improper | $a\frac{b}{c} = \frac{ac+b}{c}$ |
| Improper to mixed | Divide, quotient is whole, remainder is numerator |
