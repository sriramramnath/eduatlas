### Question 1

Which of the following is a rational number?

| Option | Answer |
|--------|--------|
| A | $\sqrt{5}$ |
| B | $\pi$ |
| C | $\sqrt{16}$ |
| D | $\sqrt{7}$ |

**Answer: C**

**Explanation:**
$\sqrt{16} = 4$, which can be written as $\frac{4}{1}$. This is rational. The other options are all irrational: $\sqrt{5} \approx 2.236...$, $\pi \approx 3.142...$, $\sqrt{7} \approx 2.646...$ are all non-terminating, non-recurring decimals.

### Question 2

Simplify $\sqrt{50}$

| Option | Answer |
|--------|--------|
| A | $2\sqrt{5}$ |
| B | $5\sqrt{2}$ |
| C | $10\sqrt{5}$ |
| D | $25\sqrt{2}$ |

**Answer: B**

**Explanation:**
$$\sqrt{50} = \sqrt{25 \times 2} = \sqrt{25} \times \sqrt{2} = 5\sqrt{2}$$

### Question 3

Express $0.\overline{7}$ as a fraction.

| Option | Answer |
|--------|--------|
| A | $\frac{7}{10}$ |
| B | $\frac{7}{9}$ |
| C | $\frac{77}{100}$ |
| D | $\frac{7}{100}$ |

**Answer: B**

**Explanation:**
Let $x = 0.777...$
$$10x = 7.777...$$
$$10x - x = 7.777... - 0.777...$$
$$9x = 7$$
$$x = \frac{7}{9}$$

### Question 4

Simplify $\sqrt{12} + \sqrt{27}$

| Option | Answer |
|--------|--------|
| A | $\sqrt{39}$ |
| B | $5\sqrt{3}$ |
| C | $3\sqrt{5}$ |
| D | $2\sqrt{3} + 3\sqrt{3}$ |

**Answer: B**

**Explanation:**
$$\sqrt{12} = \sqrt{4 \times 3} = 2\sqrt{3}$$
$$\sqrt{27} = \sqrt{9 \times 3} = 3\sqrt{3}$$
$$\sqrt{12} + \sqrt{27} = 2\sqrt{3} + 3\sqrt{3} = 5\sqrt{3}$$

### Question 5

Rationalise $\frac{4}{\sqrt{2}}$

| Option | Answer |
|--------|--------|
| A | $2\sqrt{2}$ |
| B | $4\sqrt{2}$ |
| C | $\frac{2}{\sqrt{2}}$ |
| D | $\sqrt{2}$ |

**Answer: A**

**Explanation:**
$$\frac{4}{\sqrt{2}} = \frac{4 \times \sqrt{2}}{\sqrt{2} \times \sqrt{2}} = \frac{4\sqrt{2}}{2} = 2\sqrt{2}$$

### Question 6

What is the conjugate of $3 - \sqrt{5}$?

| Option | Answer |
|--------|--------|
| A | $3 + \sqrt{5}$ |
| B | $-3 - \sqrt{5}$ |
| C | $-3 + \sqrt{5}$ |
| D | $\sqrt{5} - 3$ |

**Answer: A**

**Explanation:**
The conjugate is formed by changing the sign between the terms. The conjugate of $3 - \sqrt{5}$ is $3 + \sqrt{5}$.

### Question 7

Simplify $2\sqrt{3} \times 3\sqrt{6}$

| Option | Answer |
|--------|--------|
| A | $5\sqrt{18}$ |
| B | $6\sqrt{9}$ |
| C | $18\sqrt{2}$ |
| D | $6\sqrt{18}$ |

**Answer: C**

**Explanation:**
$$2\sqrt{3} \times 3\sqrt{6} = 6 \times \sqrt{18}$$
$$= 6 \times \sqrt{9 \times 2}$$
$$= 6 \times 3\sqrt{2}$$
$$= 18\sqrt{2}$$

### Question 8

Which of the following results in a rational number?

| Option | Answer |
|--------|--------|
| A | $\sqrt{2} + \sqrt{3}$ |
| B | $\sqrt{2} \times \sqrt{3}$ |
| C | $\sqrt{2} \times \sqrt{2}$ |
| D | $1 + \sqrt{2}$ |

**Answer: C**

**Explanation:**

This equals 2, which is rational. All other options produce irrational numbers.

### Question 9

Rationalise $\frac{2}{3 + \sqrt{5}}$

| Option | Answer |
|--------|--------|
| A | $3 - \sqrt{5}$ |
| B | $\frac{3 - \sqrt{5}}{2}$ |
| C | $2(3 - \sqrt{5})$ |
| D | $\frac{6 - 2\sqrt{5}}{4}$ |

**Answer: B**

**Explanation:**
Multiply by $\frac{3 - \sqrt{5}}{3 - \sqrt{5}}$:

$$\frac{2(3 - \sqrt{5})}{(3 + \sqrt{5})(3 - \sqrt{5})}$$

Denominator: $9 - 5 = 4$

$$\frac{2(3 - \sqrt{5})}{4} = \frac{3 - \sqrt{5}}{2}$$

Simplify $\frac{\sqrt{27}}{\sqrt{3}}$

| Option | Answer |

### Question 11

Expand and simplify $(2 + \sqrt{3})^2$

| Option | Answer |
|--------|--------|

# Section B: True/False Questions

**Instructions:** State whether each statement is TRUE or FALSE. If FALSE, provide the correct statement.

---

### Question 1

**Statement:** All square roots are irrational numbers.

**Answer: FALSE**

**Correction:** Only square roots of non-perfect squares are irrational. For example, $\sqrt{4} = 2$ and $\sqrt{9} = 3$ are rational numbers.

---

### Question 2

**Statement:** $\sqrt{18}$ can be simplified to $3\sqrt{2}$.

**Answer: TRUE**

---

### Question 3

**Statement:** $\sqrt{a} + \sqrt{b} = \sqrt{a + b}$

**Answer: FALSE**

**Correction:** $\sqrt{a} + \sqrt{b}$ cannot be combined into a single square root. For example, $\sqrt{4} + \sqrt{9} = 2 + 3 = 5$, but $\sqrt{4+9} = \sqrt{13} \approx 3.61 \neq 5$.

---

### Question 4

**Statement:** To rationalise $\frac{1}{\sqrt{5}}$, multiply numerator and denominator by $\sqrt{5}$.

**Answer: TRUE**

---

### Question 5

**Statement:** The product of two irrational numbers is always irrational.

**Answer: FALSE**

**Correction:** The product of two irrational numbers can be rational. For example, $\sqrt{2} \times \sqrt{2} = 2$, which is rational.

---

### Question 6

**Statement:** $2\sqrt{3} + 5\sqrt{3} = 7\sqrt{3}$

**Answer: TRUE**

---

### Question 7

**Statement:** $\frac{\sqrt{12}}{\sqrt{3}} = \sqrt{4}$

**Answer: TRUE**

**Explanation:** $\frac{\sqrt{12}}{\sqrt{3}} = \sqrt{\frac{12}{3}} = \sqrt{4} = 2$

---

### Question 8

**Statement:** The conjugate of $1 - 2\sqrt{3}$ is $-1 + 2\sqrt{3}$.

**Answer: FALSE**

**Correction:** The conjugate of $1 - 2\sqrt{3}$ is $1 + 2\sqrt{3}$. The conjugate is formed by changing only the sign between terms, not the signs of individual terms.

---

### Question 9

**Statement:** A rational number plus an irrational number is always irrational.

**Answer: TRUE**

---

### Question 10

**Answer: FALSE**

**Correction:** $\sqrt{75} = \sqrt{25 \times 3} = 5\sqrt{3}$

Actually, this is TRUE. $\sqrt{75} = \sqrt{25 \times 3} = 5\sqrt{3}$. The statement is correct.

**Answer: TRUE**

**Answer: TRUE**

### Question 12

**Statement:** $\frac{3}{2 - \sqrt{2}} = \frac{3(2 + \sqrt{2})}{2}$

**Answer: TRUE**

**Explanation:**
$$\frac{3}{2 - \sqrt{2}} \times \frac{2 + \sqrt{2}}{2 + \sqrt{2}} = \frac{3(2 + \sqrt{2})}{4 - 2} = \frac{3(2 + \sqrt{2})}{2}$$

### Question 1 [3 marks]

**(a) Express $0.\overline{4}$ as a fraction. [2 marks]**

**(b) State whether your answer is rational or irrational. [1 mark]**

**Mark Scheme Solution:**

**(a)** Let $x = 0.444...$ | M1

$10x = 4.444...$

$10x - x = 4.444... - 0.444...$

$9x = 4$

$x = \frac{4}{9}$ | A1

**(b)** $\frac{4}{9}$ is a rational number (can be written as $\frac{p}{q}$ where $p$ and $q$ are integers) | A1

**Answers:**
- (a) $\frac{4}{9}$
- (b) Rational

### Question 2 [4 marks]

**Simplify:**

**(a) $\sqrt{32}$ [2 marks]**

**(b) $\sqrt{32} + \sqrt{18}$ [2 marks]**

**Mark Scheme Solution:**

**(a)**
$$\sqrt{32} = \sqrt{16 \times 2}$$ | M1
$$= 4\sqrt{2}$$ | A1

**(b)**
$$\sqrt{18} = \sqrt{9 \times 2} = 3\sqrt{2}$$ | M1

$$\sqrt{32} + \sqrt{18} = 4\sqrt{2} + 3\sqrt{2} = 7\sqrt{2}$$ | A1

### Question 3 [4 marks]

**Rationalise the denominator of:**

**(a) $\frac{12}{\sqrt{6}}$ [2 marks]**

**(b) $\frac{12}{\sqrt{6}}$ simplify your answer fully. [2 marks]**

**Mark Scheme Solution:**

**(a)**
$$\frac{12}{\sqrt{6}} = \frac{12 \times \sqrt{6}}{\sqrt{6} \times \sqrt{6}}$$ | M1
$$= \frac{12\sqrt{6}}{6}$$ | A1

**(b)**
$$\frac{12\sqrt{6}}{6} = 2\sqrt{6}$$ | M1, A1

Or equivalently, using full method:

$\sqrt{6} = \sqrt{2 \times 3} = \sqrt{6}$ (already simplified) | M1

$2\sqrt{6}$ | A1

**Answer:** $2\sqrt{6}$

### Question 4 [5 marks]

**Expand and simplify $(3 + \sqrt{5})(3 - \sqrt{5})$**

**Show that the answer is a rational number.**

**Mark Scheme Solution:**

| Working | Mark |
|---------|------|
| $(3 + \sqrt{5})(3 - \sqrt{5})$ | |
| $= 3 \times 3 + 3 \times (-\sqrt{5}) + \sqrt{5} \times 3 + \sqrt{5} \times (-\sqrt{5})$ | M1 |

| $= 9 - 5$ (surds cancel) | M1 |
| $= 4$ | A1 |

### Question 5 [4 marks]

**Rationalise the denominator of $\frac{5}{4 - \sqrt{3}}$**

### Question 6 [5 marks]

**Given that $a = \sqrt{2} + 1$ and $b = \sqrt{2} - 1$,**

**(a) Find $a + b$ [1 mark]**

**(b) Find $a \times b$ [2 marks]**

**(c) Hence find $\frac{a}{b}$ in the form $c + d\sqrt{2}$ [2 marks]**

**Mark Scheme Solution:**

**(a)**
$$a + b = (\sqrt{2} + 1) + (\sqrt{2} - 1) = 2\sqrt{2}$$ | A1

**(b)**
$$a \times b = (\sqrt{2} + 1)(\sqrt{2} - 1)$$ | M1
$$= 2 - 1 = 1$$ | A1

**(c)**
$$\frac{a}{b} = \frac{\sqrt{2} + 1}{\sqrt{2} - 1} \times \frac{\sqrt{2} + 1}{\sqrt{2} + 1}$$ | M1

$$= \frac{(\sqrt{2} + 1)^2}{2 - 1}$$

$$= \frac{2 + 2\sqrt{2} + 1}{1}$$

$$= 3 + 2\sqrt{2}$$ | A1

**Answers:**
- (a) $2\sqrt{2}$
- (b) $1$
- (c) $3 + 2\sqrt{2}$

### Question 7 [4 marks]

**The sides of a rectangle are $\sqrt{12}$ cm and $\sqrt{27}$ cm.**

**(a) Find the perimeter of the rectangle. [2 marks]**

**(b) Find the area of the rectangle. [2 marks]**

**Mark Scheme Solution:**

**Simplify the sides first:**
$$\sqrt{12} = 2\sqrt{3} \text{ cm}$$
$$\sqrt{27} = 3\sqrt{3} \text{ cm}$$

**(a) Perimeter:**
$$P = 2(2\sqrt{3} + 3\sqrt{3})$$ | M1
$$= 2(5\sqrt{3}) = 10\sqrt{3} \text{ cm}$$ | A1

**(b) Area:**
$$A = 2\sqrt{3} \times 3\sqrt{3}$$ | M1
$$= 6 \times 3 = 18 \text{ cm}^2$$ | A1

**Answers:**
- (a) $10\sqrt{3}$ cm
- (b) $18 \text{ cm}^2$

### Question 8 [5 marks]

**Simplify $\frac{1}{\sqrt{5} + 2} + \frac{1}{\sqrt{5} - 2}$**

**Mark Scheme Solution:**

| Step | Working | Mark |
|------|---------|------|
| Rationalise first term | $\frac{1}{\sqrt{5} + 2} \times \frac{\sqrt{5} - 2}{\sqrt{5} - 2} = \frac{\sqrt{5} - 2}{5 - 4} = \sqrt{5} - 2$ | M1, A1 |
| Rationalise second term | $\frac{1}{\sqrt{5} - 2} \times \frac{\sqrt{5} + 2}{\sqrt{5} + 2} = \frac{\sqrt{5} + 2}{5 - 4} = \sqrt{5} + 2$ | M1, A1 |
| Add results | $(\sqrt{5} - 2) + (\sqrt{5} + 2) = 2\sqrt{5}$ | A1 |

**Answer:** $2\sqrt{5}$

### Question 9 [6 marks]

**A right-angled triangle has legs of length $2\sqrt{3}$ cm and $\sqrt{6}$ cm.**

**(a) Find the length of the hypotenuse in the form $k\sqrt{n}$ where $k$ and $n$ are integers and $n$ has no square factors. [4 marks]**

**(b) Find the area of the triangle. [2 marks]**

**Mark Scheme Solution:**

**(a) Using Pythagoras:**

$$h^2 = (2\sqrt{3})^2 + (\sqrt{6})^2$$ | M1

$$= 4 \times 3 + 6$$

$$= 12 + 6 = 18$$ | M1

$$h = \sqrt{18}$$ | M1

$$= 3\sqrt{2} \text{ cm}$$ | A1

**(b) Area:**

$$A = \frac{1}{2} \times 2\sqrt{3} \times \sqrt{6}$$ | M1

$$= \sqrt{3} \times \sqrt{6}$$

$$= \sqrt{18} = 3\sqrt{2} \text{ cm}^2$$ | A1

**Answers:**
- (a) $3\sqrt{2}$ cm
- (b) $3\sqrt{2}$ cm²

### Question 10 [6 marks]

**Show that $\frac{\sqrt{3} + 1}{\sqrt{3} - 1} = 2 + \sqrt{3}$**

**Mark Scheme Solution:**

| Step | Working | Mark |
|------|---------|------|
| Multiply by conjugate | $\frac{\sqrt{3} + 1}{\sqrt{3} - 1} \times \frac{\sqrt{3} + 1}{\sqrt{3} + 1}$ | M1 |
| Numerator | $= \frac{(\sqrt{3} + 1)^2}{(\sqrt{3} - 1)(\sqrt{3} + 1)}$ | |
| Expand numerator | $(\sqrt{3} + 1)^2 = 3 + 2\sqrt{3} + 1 = 4 + 2\sqrt{3}$ | M1 |
| Expand denominator | $(\sqrt{3} - 1)(\sqrt{3} + 1) = 3 - 1 = 2$ | M1 |
| Combine | $= \frac{4 + 2\sqrt{3}}{2}$ | M1 |
| Simplify | $= 2 + \sqrt{3}$ | A1 |
| Conclusion | Therefore $\frac{\sqrt{3} + 1}{\sqrt{3} - 1} = 2 + \sqrt{3}$ ✓ | A1 |

# Section D: Exam Focus & Strategy

## Frequently Tested Question Types

### 1. Simplifying Surds
**Frequency:** Very High

**Pattern:** Simplify $\sqrt{n}$ to the form $k\sqrt{m}$

**Key Skills:**
- Recognise perfect square factors
- Apply $\sqrt{ab} = \sqrt{a} \times \sqrt{b}$

---

### 2. Adding and Subtracting Surds
**Frequency:** Very High

**Pattern:** Combine surds after simplification

**Key Skills:**
- Simplify first
- Only combine like surds
- Handle signs carefully

---

### 3. Rationalising Simple Denominators
**Frequency:** High

**Pattern:** $\frac{a}{\sqrt{b}}$ form

**Key Skills:**
- Multiply by $\sqrt{b}$
- Simplify fraction

---

### 4. Rationalising Binomial Denominators
**Frequency:** High

**Pattern:** $\frac{a}{p + q\sqrt{r}}$ form

**Key Skills:**
- Identify conjugate
- Apply difference of squares
- Simplify result

---

### 5. Exact Trigonometric Values
**Frequency:** Medium-High

**Pattern:** Use exact values with surds

**Key Skills:**
- Memorise exact values
- Rationalise trigonometric expressions

---

## Typical Problem-Solving Patterns

### Pattern 1: Simplification Chain
```
1. Identify largest perfect square factor
2. Split the surd
3. Simplify
4. Check for further simplification
```

### Pattern 2: Combining Surds
```
1. Simplify each surd individually
2. Identify like surds
3. Combine coefficients of like surds
4. State final answer
```

### Pattern 3: Rationalisation
```
1. Identify denominator type
2. Choose appropriate multiplier
3. Multiply both numerator AND denominator
4. Simplify using difference of squares (if binomial)
5. Simplify final fraction
```

### Pattern 4: Geometric Application
```
1. Apply Pythagoras theorem
2. Substitute values
3. Simplify surds
4. Leave in exact form
```

## Common Pitfalls and How to Avoid Them

| Pitfall | How to Avoid |
|---------|--------------|
| $\sqrt{a} + \sqrt{b} = \sqrt{a+b}$ | This is WRONG! Never combine different surds |
| Wrong conjugate sign | Conjugate of $a + b\sqrt{c}$ is $a - b\sqrt{c}$ |
| Forgetting to simplify | Always check: $\frac{6\sqrt{3}}{3} = 2\sqrt{3}$ |
| Missing both parts when rationalising | Multiply BOTH numerator and denominator |
| Calculator dependency | Practice exact form; don't use decimal approximations |
