# 1. FACTORISING EXPRESSIONS

## Key Formulas

| Type | Formula | Example |
|------|---------|---------|
| Common Factor | $ab + ac = a(b + c)$ | $6x^2 + 9x = 3x(2x + 3)$ |
| Simple Quadratic | $x^2 + bx + c = (x + p)(x + q)$ | $x^2 + 5x + 6 = (x+2)(x+3)$ |
| General Quadratic | $ax^2 + bx + c$ (AC method) | $2x^2 + 7x + 3 = (2x+1)(x+3)$ |
| Difference of Squares | $a^2 - b^2 = (a+b)(a-b)$ | $x^2 - 25 = (x+5)(x-5)$ |
| Grouping | $ax + ay + bx + by = (a+b)(x+y)$ | Factorise in pairs |

---

## Factorising Checklist

### Step-by-Step Process:
1. **Check for common factor** ŌåÆ Take out HCF first
2. **Count terms remaining:**
   - **2 terms** ŌåÆ Difference of two squares?
   - **3 terms** ŌåÆ Quadratic factorising
   - **4 terms** ŌåÆ Factorise by grouping
3. **Factorise completely** ŌåÆ Check if factors can be further factorised

---

## Quick Rules for Quadratics

### When $a = 1$: $x^2 + bx + c$

| Sign of $c$ | Sign of $b$ | Signs in Brackets |
|-------------|-------------|-------------------|
| $c > 0$ | $b > 0$ | Both **+** : $(x + p)(x + q)$ |
| $c > 0$ | $b < 0$ | Both **ŌłÆ** : $(x - p)(x - q)$ |
| $c < 0$ | Any | One **+**, One **ŌłÆ** |

**Method:** Find $p$ and $q$ where $p \times q = c$ and $p + q = b$

---

### When $a \neq 1$: $ax^2 + bx + c$ (AC Method)

| Step | Action |
|------|--------|
| 1 | Calculate $a \times c$ |
| 2 | Find two numbers: product = $ac$, sum = $b$ |
| 3 | Split middle term using these numbers |
| 4 | Factorise pairs |
| 5 | Extract common bracket |

---

## Common Factorising Patterns

| Expression | Factorised Form |
|------------|-----------------|
| $x^2 - 1$ | $(x+1)(x-1)$ |
| $x^2 - 4$ | $(x+2)(x-2)$ |
| $x^2 - 9$ | $(x+3)(x-3)$ |
| $x^2 - 16$ | $(x+4)(x-4)$ |
| $x^2 - 25$ | $(x+5)(x-5)$ |
| $4x^2 - 9$ | $(2x+3)(2x-3)$ |
| $9x^2 - 16y^2$ | $(3x+4y)(3x-4y)$ |

---

##  Common Mistakes

- ŌØī Stopping at $3(x^2 - 9)$ instead of $3(x+3)(x-3)$
- ŌØī Forgetting the "1" when all terms share a factor
- ŌØī Wrong signs in quadratic factors
- ŌØī Not checking if further factorising is possible

# 2. SIMPLIFYING FRACTIONS

## Golden Rules

> **ONLY cancel FACTORS, never TERMS**

| Can Cancel | Cannot Cancel |
|------------|---------------|
| $\frac{xy}{x} = y$ | $\frac{x+y}{x} \neq y$ |
| $\frac{(x+2)(x-3)}{x+2} = x-3$ | $\frac{x^2+2}{x} \neq x+2$ |

---

## Operations with Algebraic Fractions

### Multiplication
$$\frac{A}{B} \times \frac{C}{D} = \frac{A \times C}{B \times D}$$

**Tip:** Factorise first, cancel, then multiply.

---

### Division
$$\frac{A}{B} \div \frac{C}{D} = \frac{A}{B} \times \frac{D}{C}$$

**Remember:** Keep, Flip, Change

---

### Addition and Subtraction
$$\frac{a}{c} + \frac{b}{c} = \frac{a+b}{c}$$

**Steps:**
1. Find LCD (Lowest Common Denominator)
2. Rewrite each fraction with LCD
3. Add/subtract numerators
4. Simplify if possible

---

## Quick LCD Guide

| Denominators | LCD |
|--------------|-----|
| $x$ and $x+1$ | $x(x+1)$ |
| $x-2$ and $x+3$ | $(x-2)(x+3)$ |
| $x$ and $x^2$ | $x^2$ |
| $x+1$ and $(x+1)^2$ | $(x+1)^2$ |
| $x-1$ and $x^2-1$ | $x^2-1 = (x+1)(x-1)$ |

---

## Simplification Process

1. **Factorise** numerator and denominator
2. **Cancel** common factors
3. **Write** final answer

**Example:**
$$\frac{x^2-9}{x^2+6x+9} = \frac{(x+3)(x-3)}{(x+3)(x+3)} = \frac{x-3}{x+3}$$

---

##  Common Mistakes

- ŌØī Cancelling terms: $\frac{x+3}{x} \neq 3$
- ŌØī Forgetting to multiply ALL terms by LCD
- ŌØī Sign errors when subtracting fractions
- ŌØī Not factorising before finding LCD

# 3. EQUATIONS WITH FRACTIONS

## Solution Method

| Step | Action |
|------|--------|
| 1 | Find LCD of all denominators |
| 2 | Multiply EVERY term by LCD |
| 3 | Solve the resulting equation |
| 4 | Check answer doesn't make denominator = 0 |

---

## Quick LCD for Equations

| Denominators | LCD |
|--------------|-----|
| 2, 3, 6 | 6 |
| $x$, $x+1$ | $x(x+1)$ |
| $x-2$, $x+2$ | $(x-2)(x+2)$ |
| $x$, $2x$ | $2x$ |

---

## Cross-Multiplication

**When:** $\frac{A}{B} = \frac{C}{D}$

**Method:** $A \times D = B \times C$

---

## Types of Solutions

| Result | Meaning |
|--------|---------|
| $x = \text{number}$ | One solution |
| $x = a$ or $x = b$ | Two solutions |
| No real solutions | Discriminant < 0 |
| $x = \text{denominator}$ | **Invalid!** (extraneous solution) |

---

##  Common Mistakes

- ŌØī Not multiplying ALL terms by LCD
- ŌØī Forgetting to expand brackets after multiplying
- ŌØī Accepting solutions that make denominator = 0
- ŌØī Calculation errors in quadratic formula

# 4. SIMULTANEOUS EQUATIONS

## Linear Simultaneous Equations

### Elimination Method

| Step | Action |
|------|--------|
| 1 | Make coefficients of one variable equal |
| 2 | Add or subtract equations to eliminate that variable |
| 3 | Solve for remaining variable |
| 4 | Substitute back to find other variable |

**When to add:** Same signs ŌåÆ Subtract
**When to subtract:** Different signs ŌåÆ Add

---

### Substitution Method

| Step | Action |
|------|--------|
| 1 | Rearrange one equation for $x$ or $y$ |
| 2 | Substitute into other equation |
| 3 | Solve for the variable |
| 4 | Find the other variable |

**Best when:** One variable has coefficient 1

---

## Linear + Quadratic Simultaneous Equations

### Method

| Step | Action |
|------|--------|
| 1 | Rearrange linear equation for $y$ (or $x$) |
| 2 | Substitute into quadratic equation |
| 3 | Solve resulting quadratic |
| 4 | Find corresponding values |
| 5 | Give answers as coordinate pairs |

---

## Number of Solutions

| Scenario | Solutions |
|----------|-----------|
| Line crosses parabola twice | 2 solutions |
| Line touches parabola (tangent) | 1 solution |
| Line misses parabola | 0 solutions |
| Line crosses circle twice | 2 solutions |
| Line tangent to circle | 1 solution |
| Line misses circle | 0 solutions |

---

## Quick Methods Summary

| Type | Best Method |
|------|-------------|
| Same coefficients for one variable | Elimination |
| One variable has coefficient 1 | Substitution |
| Linear + Quadratic | Substitution |
| Fractions in equations | Clear fractions first |

# QUICK FORMULA REFERENCE

## Quadratic Formula
$$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$

## Discriminant
$$\Delta = b^2 - 4ac$$

| $\Delta$ | Solutions |
|----------|-----------|
| $\Delta > 0$ | Two distinct real solutions |
| $\Delta = 0$ | One repeated real solution |
| $\Delta < 0$ | No real solutions |

## Perfect Squares
$$(a+b)^2 = a^2 + 2ab + b^2$$
$$(a-b)^2 = a^2 - 2ab + b^2$$

# KEY EXAM TIPS

## Time-Saving Strategies

1. **Factorise first** - Always check for common factors before expanding
2. **Use elimination** when coefficients can be easily matched
3. **Use substitution** when one coefficient is 1
4. **Check answers** by substitution (especially for simultaneous equations)
5. **Factorise completely** - Look for difference of two squares in remaining factors

## Mark-Getting Strategies

1. **Show ALL working** - Method marks matter
2. **Write clearly** - Each step on new line
3. **Label equations** - (1), (2), etc.
4. **Check constraints** - Ensure denominators ŌēĀ 0
5. **Verify** - Substitute answers back
