### Question 6
Which calculation gives the UPPER bound of $A \div B$?

**A)** $LB_A \div LB_B$  
**B)** $LB_A \div UB_B$  
**C)** $UB_A \div LB_B$  
**D)** $UB_A \div UB_B$

**Answer: C**

**Solution:**
For division, to get the maximum (upper bound):
- Use the maximum numerator (UB of A)
- Use the minimum denominator (LB of B)

Maximum quotient = $\frac{\text{maximum dividend}}{\text{minimum divisor}}$

Therefore: $UB_A \div LB_B$

### Question 7
Find $\sin(45°) + \cos(45°)$ (calculator in DEG mode)

**A)** 0.5  
**B)** 1  
**C)** $\sqrt{2}$ ≈ 1.414  
**D)** 2

**Answer: C**

**Solution:**
$$\sin(45°) = \frac{\sqrt{2}}{2}$$
$$\cos(45°) = \frac{\sqrt{2}}{2}$$
$$\sin(45°) + \cos(45°) = \frac{\sqrt{2}}{2} + \frac{\sqrt{2}}{2} = \sqrt{2} \approx 1.414$$

### Question 11
**Statement:** When a number is rounded to the nearest 10, the rounding unit is 0.1.

**Answer: FALSE**

**Correction:** When a number is rounded to the nearest 10, the rounding unit is **10**, not 0.1.

---

### Question 12
**Statement:** The upper bound of a rounded number is always included in the error interval.

**Answer: FALSE**

**Correction:** The upper bound is **excluded** from the error interval. We write $LB \leq x < UB$, not $LB \leq x \leq UB$.

---

### Question 13
**Statement:** To estimate $\frac{400}{21}$, you could calculate $\frac{400}{20} = 20$.

**Answer: TRUE**

This is a valid estimation technique. Rounding 21 to 20 makes the division easier while keeping a reasonable estimate.

---

### Question 14
**Statement:** When finding the upper bound of $A - B$, use $UB_A - UB_B$.

**Answer: FALSE**

**Correction:** For $A - B$:
- Upper bound = $UB_A - LB_B$ (subtract the smallest possible B)
- Lower bound = $LB_A - UB_B$ (subtract the largest possible B)

---

### Question 15
**Statement:** $\sin^{-1}(0.5) = 30°$ when the calculator is in DEG mode.

**Answer: TRUE**

$\sin(30°) = 0.5$, therefore $\sin^{-1}(0.5) = 30°$ in degree mode.

---

### Question 16
**Statement:** The percentage error when measuring 50 cm to the nearest cm is 0.5%.

**Answer: FALSE**

**Correction:** 
$$\text{Percentage Error} = \frac{0.5}{50} \times 100\% = 1\%$$

The percentage error is **1%**, not 0.5%.

---

### Question 17
**Statement:** When estimating $38.7 \times 0.52$, rounding both to 1 significant figure gives $40 \times 0.5 = 20$.

**Answer: TRUE**

$38.7 \approx 40$ (1 s.f.) and $0.52 \approx 0.5$ (1 s.f.)
$$40 \times 0.5 = 20$$

---

### Question 18
**Statement:** For a number rounded to 2 decimal places, the rounding unit is 0.01.

**Answer: TRUE**

The last significant digit in a 2 d.p. number is in the hundredths place, so the rounding unit is 0.01.

---

### Question 19
**Statement:** To find the minimum value of $A \times B$, multiply the upper bounds together.

**Answer: FALSE**

**Correction:** To find the **minimum** value of $A \times B$, multiply the **lower bounds** together: $LB_A \times LB_B$.

To find the **maximum** value, multiply the upper bounds: $UB_A \times UB_B$.

---

### Question 20
**Statement:** Using the ANS key on a calculator helps avoid rounding errors in multi-step calculations.

**Answer: TRUE**

The ANS key recalls the exact previous answer, avoiding the rounding that occurs when you write down and re-enter intermediate values.

### Question 24 (4 marks)

A scientist measures the time for a chemical reaction as 25 seconds to the nearest second.

a) Calculate the percentage error in this measurement. **[2 marks]**

b) The scientist repeats the experiment and measures 250 seconds (to the nearest second). Without calculating, explain whether the percentage error is larger, smaller, or the same as in part (a). **[2 marks]**

**a) Percentage error [M1, A1]**

| Step | Working | Mark |
|------|---------|------|
| M1 | States half unit = 0.5 s and uses formula | Method |
| A1 | $\frac{0.5}{25} \times 100 = \textbf{2\%}$ | Correct answer |

**Working:**
$$\text{Percentage Error} = \frac{0.5}{25} \times 100\% = 2\%$$

**b) Explanation [M1, A1]**

| | Response | Mark |
|---|----------|------|
| M1 | Recognizes that half unit is same (0.5 s) but measured value is different | Understanding |
| A1 | States percentage error is **smaller** with correct reason | Full explanation |

**Acceptable explanations:**
- "The percentage error is smaller because the same absolute error (0.5 s) is a smaller fraction of 250 s than of 25 s."
- "Smaller, because $\frac{0.5}{250} = 0.2\%$ which is less than 2%."

**Key insight:** As measurements get larger (with same precision), percentage error decreases.

### Question 27 (6 marks)

A metal rod is measured as 12.0 cm to the nearest 0.1 cm.

a) Write down the error interval for the length of the rod. **[2 marks]**

b) The mass of the rod is 45 g to the nearest gram. Calculate the upper and lower bounds for the density (mass ÷ volume) of a cylindrical rod. 

*Note: Volume of a cylinder = πr²h. Assume the rod has radius 0.5 cm (exact).* **[4 marks]**

**a) Error interval [M1, A1]**

| Step | Working | Mark |
|------|---------|------|
| M1 | Identifies half unit = 0.05 cm | Method |
| A1 | $11.95 \leq l < 12.05$ | Correct inequality |

**b) Density calculations [M1, M1, A1, A1]**

**Finding bounds:**
- Mass LB = 44.5 g, UB = 45.5 g
- Length LB = 11.95 cm, UB = 12.05 cm
- Radius = 0.5 cm (exact)

**Volume calculations:**
$$V = \pi r^2 h$$

$V_{min} = \pi \times 0.5^2 \times 11.95 = \pi \times 0.25 \times 11.95 \approx 9.39 \text{ cm}^3$

$V_{max} = \pi \times 0.5^2 \times 12.05 = \pi \times 0.25 \times 12.05 \approx 9.46 \text{ cm}^3$

**Density bounds:**
$$\text{Density} = \frac{\text{Mass}}{\text{Volume}}$$

| | Formula | Working | Mark |
|---|---------|---------|------|
| M1 | States density formula | | |
| M1 | Calculates volume bounds | | |
| Min Density | $\frac{LB_{mass}}{UB_{volume}}$ | $\frac{44.5}{9.46} \approx 4.70 \text{ g/cm}^3$ | A1 |
| Max Density | $\frac{UB_{mass}}{LB_{volume}}$ | $\frac{45.5}{9.39} \approx 4.85 \text{ g/cm}^3$ | A1 |

### Question 28 (8 marks)

**Context:** A delivery company charges based on the volume of parcels.

A parcel has dimensions:
- Length = 30 cm (to nearest cm)
- Width = 20 cm (to nearest cm)  
- Height = 15 cm (to nearest cm)

The company charges £2.50 per 1000 cm³ (cubic centimetres).

a) Calculate the nominal volume of the parcel. **[1 mark]**

b) Find the minimum and maximum possible volumes. **[3 marks]**

c) Calculate the difference between the maximum and minimum possible charges. Give your answer to a suitable degree of accuracy. **[3 marks]**

d) The company wants to quote a single price to the customer. State, with a reason, whether they should use the minimum, maximum, or nominal volume for pricing. **[1 mark]**

**a) Nominal volume [A1]**

$$V = 30 \times 20 \times 15 = 9000 \text{ cm}^3$$

**b) Volume bounds [M1, A1, A1]**

**Finding bounds:**

| Dimension | Lower Bound | Upper Bound |
|-----------|-------------|-------------|
| Length | $30 - 0.5 = 29.5$ cm | $30 + 0.5 = 30.5$ cm |
| Width | $20 - 0.5 = 19.5$ cm | $20 + 0.5 = 20.5$ cm |
| Height | $15 - 0.5 = 14.5$ cm | $15 + 0.5 = 15.5$ cm |

**M1:** Correct method for finding bounds

**Volume calculations:**

$V_{min} = 29.5 \times 19.5 \times 14.5 = 8348.625 \text{ cm}^3$ (A1)

$V_{max} = 30.5 \times 20.5 \times 15.5 = 9691.375 \text{ cm}^3$ (A1)

**Accept rounding to appropriate accuracy:** e.g., 8350 cm³ and 9690 cm³

**c) Charge difference [M1, M1, A1]**

**Charges:**
$$\text{Charge} = \frac{\text{Volume}}{1000} \times £2.50$$

| | Calculation | Result |
|---|-------------|--------|
| Max Charge | $\frac{9691.375}{1000} \times £2.50$ | £24.23... |
| Min Charge | $\frac{8348.625}{1000} \times £2.50$ | £20.87... |

**Difference:**
$$£24.23 - £20.87 = £3.36$$

**M1:** Correct charge formula
**M1:** Correct bounds used
**A1:** Difference = **£3.40** (to 3 s.f. or 2 d.p. for money)

**d) Recommendation [A1 - reasoning required]**

**Recommendation:** Maximum volume (or minimum volume if protecting customer)

**Reason:** The company should use maximum volume to ensure they cover costs for any possible parcel size within the measured range. This is a conservative approach that guarantees profitability.

**Alternative acceptable answer:** Use maximum volume to ensure they charge enough for the largest possible parcel.

## 🎯 EXAM FOCUS & STRATEGY SECTION

### Frequently Tested Question Types

| Type | Description | Typical Marks |
|------|-------------|---------------|
| **Bounds finding** | Direct LB/UB calculation | 2-3 |
| **Combined bounds** | Area, perimeter, speed with bounds | 3-5 |
| **Error intervals** | Writing inequality notation | 2-3 |
| **Estimation** | 1 s.f. approximation | 2-3 |
| **Percentage error** | Uncertainty calculation | 2-3 |
| **Density/Speed bounds** | Multi-step with division | 4-6 |

### Typical Problem-Solving Patterns

#### Pattern A: Simple Bounds (2 marks)
1. Identify rounding unit
2. Calculate half unit
3. Add/subtract from given value

#### Pattern B: Combined Bounds (4-5 marks)
1. Find all individual bounds
2. Apply operation rules (use table)
3. Calculate final bounds
4. Round appropriately

#### Pattern C: Real-World Application (5-6 marks)
1. Extract measurements and their precision
2. Find all relevant bounds
3. Apply correct formulas
4. Interpret results in context

### Command Word Guidance

| Command | Required Action |
|---------|-----------------|
| **Calculate** | Show working, give numerical answer |
| **Estimate** | Use 1 s.f., show rounding clearly |
| **Find** | Similar to calculate, show method |
| **Write down** | No working needed, direct answer |
| **Show that** | Provide clear step-by-step proof |
| **Give your answer to...** | Apply specified rounding |

### How to Gain Full Marks

✅ **Show all working** - Even if you can do it mentally  
✅ **Use correct inequality notation** - $LB \leq x < UB$  
✅ **Round appropriately** - Match precision to context  
✅ **Check reasonableness** - Estimate first  
✅ **Label bounds clearly** - $LB_L$, $UB_W$ etc.  
✅ **State formulas** - Especially for combined operations  

### Common Pitfalls

| ❌ Pitfall | ✅ Solution |
|-----------|-------------|
| Wrong rounding unit | Check "nearest what?" |
| Including upper bound | Remember: $<$ not $\leq$ |
| Wrong division bounds | LB ÷ UB for min, UB ÷ LB for max |
| Over-precise answers | Consider measurement precision |
| Calculator mode wrong | Check DEG for angles |
| Missing working | Always show method clearly |
