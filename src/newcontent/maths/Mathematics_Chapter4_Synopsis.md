# Chapter 4: Numbers 4 - Concise Synopsis
## Compound Percentages & Inverse Percentages

---

## 📌 PART A: COMPOUND PERCENTAGES

### Core Formula

$$\boxed{\text{Final Amount} = P \times \left(1 \pm \frac{r}{100}\right)^n}$$

| Symbol | Meaning |
|--------|---------|
| $P$ | Principal (starting value) |
| $r$ | Percentage rate per period |
| $n$ | Number of periods |
| $+$ | Use for increase |
| $-$ | Use for decrease |

---

### Quick Multiplier Table

| Change | Multiplier |
|--------|------------|
| +10% | 1.10 |
| +20% | 1.20 |
| +25% | 1.25 |
| +50% | 1.50 |
| +100% | 2.00 |
| −10% | 0.90 |
| −20% | 0.80 |
| −25% | 0.75 |
| −50% | 0.50 |
| −100% | 0.00 |

---

### Step-by-Step Method

**To find Final Amount:**
1. Identify $P$, $r$, $n$
2. Write multiplier: $1 \pm \frac{r}{100}$
3. Apply formula: $P \times (\text{multiplier})^n$
4. Calculate and round appropriately

**To find Principal:**
1. Rearrange: $P = \frac{\text{Final}}{(\text{multiplier})^n}$

**To find Rate:**
1. Rearrange: $(\text{multiplier})^n = \frac{\text{Final}}{P}$
2. Take $n^{\text{th}}$ root
3. Subtract 1, multiply by 100

**To find Time ($n$):**
1. Rearrange: $(\text{multiplier})^n = \frac{\text{Final}}{P}$
2. Use trial & improvement or logarithms

---

### Compound vs Simple Interest

| Type | Formula | Growth |
|------|---------|--------|
| Simple | $I = Prt$ | Linear |
| Compound | $A = P(1+r)^n$ | Exponential |

**Remember:** Compound interest always yields MORE over time!

---

### Different Compounding Periods

| Frequency | Formula Adjustment |
|-----------|-------------------|
| Semi-annually | $P\left(1 + \frac{r}{200}\right)^{2n}$ |
| Quarterly | $P\left(1 + \frac{r}{400}\right)^{4n}$ |
| Monthly | $P\left(1 + \frac{r}{1200}\right)^{12n}$ |

## 📌 PART B: INVERSE PERCENTAGES

### Core Formula

$$\boxed{\text{Original Value} = \frac{\text{New Value}}{1 \pm \frac{r}{100}}}$$

- Use $+$ in denominator for **after an increase**
- Use $-$ in denominator for **after a decrease**

---

### Quick Reference Examples

| Given | After | Formula | Calculate |
|-------|-------|---------|-----------|
| £75 | +25% increase | $\frac{75}{1.25}$ | £60 |
| £42 | −30% decrease | $\frac{42}{0.70}$ | £60 |
| £480 | +20% VAT | $\frac{480}{1.20}$ | £400 |
| £153 | −15% loss | $\frac{153}{0.85}$ | £180 |

---

### The "Unitary Method" Shortcut

**Example:** After 40% increase, result is 168

- $140\% \rightarrow 168$
- $1\% \rightarrow \frac{168}{140} = 1.2$
- $100\% \rightarrow 120$

---

### Key Rule for Successive Changes

**Overall Multiplier:** Multiply all individual multipliers

$$\text{Overall} = m_1 \times m_2 \times m_3 \times ...$$

**Example:** +10% then +15%
$$\text{Overall} = 1.10 \times 1.15 = 1.265$$
$$\text{Overall increase} = 26.5\%$$

## 📌 ESSENTIAL REMINDERS

###  Common Mistakes to Avoid

| ❌ Wrong | ✅ Correct |
|----------|-----------|
| $240 - 25\% = 180$ (reverse) | $\frac{240}{1.25} = 192$ |
| $+20\%$ then $-20\%$ = no change | $1.20 \times 0.80 = 0.96$ (−4%) |
| Using simple interest for compound | Use $(1+r)^n$ formula |
| Forgetting to convert % to decimal | Always use $\frac{r}{100}$ |

## 📌 FORMULA TABLE

| Purpose | Formula |
|---------|---------|
| Compound Growth | $A = P\left(1+\frac{r}{100}\right)^n$ |
| Compound Decay | $A = P\left(1-\frac{r}{100}\right)^n$ |
| Reverse (after increase) | $Original = \frac{New}{1+\frac{r}{100}}$ |
| Reverse (after decrease) | $Original = \frac{New}{1-\frac{r}{100}}$ |
| Percentage Change | $\frac{New - Old}{Old} \times 100$ |
| Overall Multiplier | Product of all multipliers |
