# 1. Inequalities on Graphs

## Key Rules

| Inequality | Boundary Line | Shaded Region |
|-----------|---------------|---------------|
| $y > mx + c$ | **Dashed** | Above line |
| $y < mx + c$ | **Dashed** | Below line |
| $y \geq mx + c$ | **Solid** | Above line |
| $y \leq mx + c$ | **Solid** | Below line |

## Quick Method

1. **Draw boundary line** (solid for $\leq, \geq$; dashed for $<, >$)
2. **Test with origin** $(0, 0)$ – if not on line
3. **Shade correct region**

> **📌 Remember:** 
> - $y >$ or $y \geq$ → Shade **above**
> - $y <$ or $y \leq$ → Shade **below**

## Test Point Rule

- If test point **satisfies** inequality → shade that region
- If test point **fails** → shade opposite region
- Use origin $(0, 0)$ as default test point

# 2. Perpendicular Lines

## Gradient Rule

> **📌 Key Formula:**
> $$m_1 \times m_2 = -1$$
> 
> Or: $m_2 = -\frac{1}{m_1}$ (negative reciprocal)

## Quick Examples

| Original Gradient | Perpendicular Gradient |
|-------------------|----------------------|
| $m_1 = 2$ | $m_2 = -\frac{1}{2}$ |
| $m_1 = -3$ | $m_2 = \frac{1}{3}$ |
| $m_1 = \frac{1}{4}$ | $m_2 = -4$ |
| $m_1 = -\frac{2}{5}$ | $m_2 = \frac{5}{2}$ |

## Finding Perpendicular Line Equation

1. Find original gradient $m_1$
2. Calculate perpendicular gradient: $m_2 = -\frac{1}{m_1}$
3. Substitute given point into $y = m_2x + c$
4. Solve for $c$

## Parallel vs Perpendicular

| Type | Gradient Relationship |
|------|----------------------|
| **Parallel** | $m_1 = m_2$ |
| **Perpendicular** | $m_1 \times m_2 = -1$ |

# 3. Midpoints

## Midpoint Formula

> **📌 Key Formula:**
> For points $(x_1, y_1)$ and $(x_2, y_2)$:
> $$M = \left(\frac{x_1 + x_2}{2}, \frac{y_1 + y_2}{2}\right)$$

## Quick Rules

- Midpoint = **average** of x-coordinates, **average** of y-coordinates
- Always divide by 2
- Answer is always between the two points

## Finding Unknown Endpoint

Given midpoint $M$ and endpoint $A$, find endpoint $B$:

$$x_B = 2(x_M) - x_A$$
$$y_B = 2(y_M) - y_A$$

## Quick Example

Midpoint of $(2, 4)$ and $(6, 8)$:
$$M = \left(\frac{2+6}{2}, \frac{4+8}{2}\right) = (4, 6)$$

# 4. Using Pythagoras (Distance Formula)

## Distance Formula

> **📌 Key Formula:**
> For points $(x_1, y_1)$ and $(x_2, y_2)$:
> $$d = \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2}$$

## Pythagoras' Theorem

> **📌 Standard Form:**
> $$a^2 + b^2 = c^2$$
> 
> Where $c$ = hypotenuse (longest side)

## Quick Applications

| Application | Method |
|-------------|--------|
| Length of line segment | Distance formula |
| Radius of circle | Distance from centre to point on circumference |
| Check right-angled triangle | Verify $a^2 + b^2 = c^2$ |
| Find missing coordinate | Set distance equal to given value |

## Quick Example

Distance from $(1, 2)$ to $(4, 6)$:
$$d = \sqrt{(4-1)^2 + (6-2)^2} = \sqrt{9+16} = \sqrt{25} = 5$$

# 7. Quick Reference: Perpendicular Lines

## Negative Reciprocal Pattern

| If $m_1$ is... | Then $m_2$ is... |
|----------------|------------------|
| Positive | Negative |
| Negative | Positive |
| Integer | Fraction |
| Fraction | Integer |

## Verifying Perpendicularity

- Calculate both gradients
- Multiply them
- Result should be $-1$

# 8. Quick Reference: Distance Calculations

## Step-by-Step

1. Find change in $x$: $|x_2 - x_1|$
2. Find change in $y$: $|y_2 - y_1|$
3. Square both changes
4. Add them
5. Take square root

## Common Distance Patterns

| Points | Calculation | Distance |
|--------|-------------|----------|
| $(0,0)$ to $(3,4)$ | $\sqrt{3^2 + 4^2}$ | 5 |
| $(0,0)$ to $(5,12)$ | $\sqrt{5^2 + 12^2}$ | 13 |
| $(0,0)$ to $(1,1)$ | $\sqrt{1^2 + 1^2}$ | $\sqrt{2}$ |
| $(0,0)$ to $(8,6)$ | $\sqrt{8^2 + 6^2}$ | 10 |

# 10. Common Mistakes to Avoid

| ❌ Mistake | ✅ Correction |
|------------|---------------|
| Wrong line type for inequalities | Solid for $\leq, \geq$; dashed for $<, >$ |
| Forgetting negative sign in perpendicular gradient | $m_2 = -\frac{1}{m_1}$ |
| Adding coordinates without dividing by 2 | Midpoint = average of coordinates |
| Forgetting square root in distance formula | Final step: $\sqrt{\text{sum}}$ |
| Confusing parallel and perpendicular | Parallel: same gradient; Perpendicular: negative reciprocal |

# 11. Worked Quick Examples

## Example 1: Inequality Region
**Shade region for $y \geq 2x - 1$**

- Draw $y = 2x - 1$ as **solid line**
- Test $(0, 0)$: $0 \geq -1$ ✓ TRUE
- Shade region **containing origin** (above line)

---

## Example 2: Perpendicular Gradient
**Find perpendicular gradient to $m = -\frac{3}{7}$**

$$m_2 = -\frac{1}{-\frac{3}{7}} = \frac{7}{3}$$

---

## Example 3: Midpoint
**Find midpoint of $(-2, 5)$ and $(4, -3)$**

$$M = \left(\frac{-2+4}{2}, \frac{5+(-3)}{2}\right) = (1, 1)$$

---

## Example 4: Distance
**Find distance from $(2, -1)$ to $(5, 3)$**

$$d = \sqrt{(5-2)^2 + (3-(-1))^2} = \sqrt{9 + 16} = 5$$

---

## Example 5: Perpendicular Line Equation
**Line perpendicular to $y = 4x + 1$ through $(2, 3)$**

- Original gradient: $m_1 = 4$
- Perpendicular gradient: $m_2 = -\frac{1}{4}$
- Use point $(2, 3)$: $3 = -\frac{1}{4}(2) + c$
- $c = 3 + \frac{1}{2} = \frac{7}{2}$
- **Answer:** $y = -\frac{1}{4}x + \frac{7}{2}$
