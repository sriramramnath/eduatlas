## 📌 ALTERNATE SEGMENT THEOREM

### Core Rule
> **Angle between tangent and chord = Angle in alternate segment**

### Key Points
- Tangent must touch circle at the chord's endpoint
- Alternate segment = segment **opposite** to tangent
- Creates "bridge" between external and internal angles

### Quick Formula
$$\angle_{tangent-chord} = \angle_{alternate\ segment}$$

### Visual Pattern
```
    Tangent
        |
    ----●---- (point of contact)
       /|\
      / | \  ← Alternate segment
     /  |  \
    ●   ●   ●
```

## 📌 INTERSECTING CHORDS THEOREM

### Internal Intersection
| Rule | Formula |
|------|---------|
| Product of segments | $AP \times PB = CP \times PD$ |

```
    A
   /|\
  / | \
 /  P  \    ← Intersection inside circle
 D   |   B
  \  |  /
   \ | /
    \|/
     C
```

### External Intersection (Secant-Secant)
| Rule | Formula |
|------|---------|
| External × full length | $PA \times PB = PC \times PD$ |

### Tangent-Secant
| Rule | Formula |
|------|---------|
| Tangent squared | $PT^2 = PA \times PB$ |

## 📌 KEY DIAGRAMS TO RECOGNISE

### Alternate Segment Pattern
- Look for: Tangent line touching circle
- Apply: When finding angles involving tangent and chord

### Intersecting Chords Pattern
- Look for: Two lines crossing inside circle (X-shape)
- Apply: When finding lengths

### Secant Pattern
- Look for: Lines from external point cutting circle twice
- Apply: When finding lengths from external point

## 📌 COMMON EXAM CONFIGURATIONS

### Configuration 1: Tangent + Chord + Angle
**Setup:** Tangent touches circle, chord drawn, angle given
**Solution:** Use alternate segment theorem directly

### Configuration 2: Two Intersecting Chords
**Setup:** Four lengths, three known
**Solution:** $AP \times PB = CP \times PD$

### Configuration 3: External Point + Tangent + Secant
**Setup:** Tangent length and secant segments given
**Solution:** $PT^2 = PA \times PB$

### Configuration 4: Multi-Theorem Problem
**Setup:** Multiple elements requiring several theorems
**Solution:** Apply systematically:
1. Mark tangent-radius right angles (90°)
2. Use angle at centre = 2 × angle at circumference
3. Apply alternate segment for tangent-chord
4. Use cyclic quadrilateral for opposite angles

## 📌 STEP-BY-STEP APPROACH

### For Angle Problems
1. Mark tangent-radius right angles (90°)
2. Look for angle at centre = 2 × angle at circumference
3. Check alternate segment theorem
4. Use cyclic quadrilateral properties
5. Apply triangle angle sum if needed

### For Length Problems
1. Identify intersection type (internal/external)
2. Select correct formula
3. Substitute known values
4. Solve for unknown
5. Check units and reasonableness

## 📌 HIGH-YIELD FACTS

| Fact | Details |
|------|---------|
| Tangent-radius angle | Always 90° |
| Angle in semicircle | Always 90° |
| Cyclic quad opposite angles | Sum to 180° |
| Isosceles from radii | Any triangle with two radii is isosceles |
| Alternate segment | Works both ways (equality) |

## 📌 PROOF REQUIREMENTS

### Alternate Segment Theorem Proof Points
1. Draw diameter through tangent point
2. Use angle in semicircle (90°)
3. Use angles in same segment
4. Use tangent ⊥ radius (90°)

### Key Proof Steps to Memorize
- Construction: Draw diameter and join to chord endpoint
- Use three theorems: Semicircle, same segment, tangent-radius

## 📌 QUICK MEMORY AIDS

### "TAC" for Alternate Segment
- **T**angent at point
- **A**ngle with chord
- **C**orresponds to alternate segment

### "PS" for Intersecting Chords
- **P**roduct of **S**egments
- Same chord segments multiply: $AP \times PB$

### "TSE" for Tangent-Secant
- **T**angent **S**quared **E**quals
- $PT^2 = PA \times PB$
