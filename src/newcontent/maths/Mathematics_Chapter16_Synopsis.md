## PART 1: PROPORTION – QUICK RULES

### Direct Proportion

> **Formula:** $y \propto x \quad \Rightarrow \quad y = kx$

| Key Feature | Value |
|-------------|-------|
| Relationship | $y$ and $x$ increase together |
| Constant | $k = \frac{y}{x}$ |
| Graph | Straight line through origin |
| When $x$ doubles | $y$ doubles |

**Variations:**
| Statement | Equation |
|-----------|----------|
| $y \propto x$ | $y = kx$ |
| $y \propto x^2$ | $y = kx^2$ |
| $y \propto \sqrt{x}$ | $y = k\sqrt{x}$ |
| $y \propto x^3$ | $y = kx^3$ |

---

### Inverse Proportion

> **Formula:** $y \propto \frac{1}{x} \quad \Rightarrow \quad y = \frac{k}{x}$

| Key Feature | Value |
|-------------|-------|
| Relationship | As $x$ increases, $y$ decreases |
| Constant | $k = xy$ |
| Graph | Hyperbola (curved) |
| When $x$ doubles | $y$ halves |

**Variations:**
| Statement | Equation |
|-----------|----------|
| $y \propto \frac{1}{x}$ | $y = \frac{k}{x}$ |
| $y \propto \frac{1}{x^2}$ | $y = \frac{k}{x^2}$ |
| $y \propto \frac{1}{\sqrt{x}}$ | $y = \frac{k}{\sqrt{x}}$ |

---

### Proportion at a Glance

| Type | Formula | Check |
|------|---------|-------|
| **Direct** | $y = kx$ | $\frac{y}{x} = \text{constant}$ |
| **Inverse** | $y = \frac{k}{x}$ | $xy = \text{constant}$ |

---

### Solving Proportion Problems – Quick Steps

```
1. IDENTIFY type (direct or inverse)
2. WRITE formula (y = kx or y = k/x)
3. SUBSTITUTE known values
4. SOLVE for k
5. WRITE complete equation
6. FIND unknown value
```

## PART 2: INDICES LAWS – QUICK RULES

### The Five Essential Laws

> **MEMORISE THESE:**

| Law | Formula | Example |
|-----|---------|---------|
| **Multiply** | $a^m \times a^n = a^{m+n}$ | $x^3 \times x^5 = x^8$ |
| **Divide** | $a^m \div a^n = a^{m-n}$ | $\frac{x^7}{x^2} = x^5$ |
| **Power of Power** | $(a^m)^n = a^{mn}$ | $(x^3)^4 = x^{12}$ |
| **Zero Power** | $a^0 = 1$ | $5^0 = 1$ |
| **Negative** | $a^{-n} = \frac{1}{a^n}$ | $3^{-2} = \frac{1}{9}$ |

---

### Fractional Indices

| Formula | Meaning | Example |
|---------|---------|---------|
| $a^{\frac{1}{2}}$ | $\sqrt{a}$ | $9^{\frac{1}{2}} = 3$ |
| $a^{\frac{1}{3}}$ | $\sqrt[3]{a}$ | $8^{\frac{1}{3}} = 2$ |
| $a^{\frac{m}{n}}$ | $\sqrt[n]{a^m}$ | $8^{\frac{2}{3}} = 4$ |

> **Memory Trick:** Denominator = Root, Numerator = Power

---

### Quick Examples

**Multiplying:**
$$x^3 \times x^4 = x^{3+4} = x^7$$

**Dividing:**
$$\frac{x^8}{x^3} = x^{8-3} = x^5$$

**Power of Power:**
$$(x^2)^5 = x^{2 \times 5} = x^{10}$$

**Negative Index:**
$$x^{-3} = \frac{1}{x^3}$$

**Zero Power:**
$$7x^0 = 7 \times 1 = 7$$

---

### Coefficients with Indices

| Expression | Working | Answer |
|------------|---------|--------|
| $2x^3 \times 5x^4$ | $(2 \times 5) \times x^{3+4}$ | $10x^7$ |
| $(3x^2)^3$ | $3^3 \times (x^2)^3$ | $27x^6$ |
| $\frac{12x^8}{4x^3}$ | $\frac{12}{4} \times x^{8-3}$ | $3x^5$ |

## COMMON MISTAKES 

### Proportion Mistakes

| Mistake | Wrong | Correct |
|---------|-------|---------|
| Wrong proportion type | Inverse: $y = kx$ | $y = \frac{k}{x}$ |
| Forgetting to square | $y \propto x^2$: use $x$ | Use $x^2$ |
| Wrong rearrangement | $y = \frac{k}{x}$ → $k = \frac{y}{x}$ | $k = xy$ |

### Indices Mistakes

| Mistake | Wrong | Correct |
|---------|-------|---------|
| Adding instead of multiplying powers | $(x^3)^2 = x^5$ | $(x^3)^2 = x^6$ |
| Multiplying bases | $2^3 \times 3^2 = 6^5$ | $8 \times 9 = 72$ |
| Zero power | $x^0 = 0$ | $x^0 = 1$ |
| Fractional index | $8^{\frac{1}{3}} = \frac{8}{3}$ | $8^{\frac{1}{3}} = 2$ |
| Same base only | $x^3 \times y^4 = xy^7$ | Cannot simplify! |

| Law | Formula |
|-----|---------|
| Multiply (same base) | $a^m \times a^n = a^{m+n}$ |
| Divide (same base) | $\frac{a^m}{a^n} = a^{m-n}$ |
| Power of power | $(a^m)^n = a^{mn}$ |
| Zero | $a^0 = 1$ |
| Negative | $a^{-n} = \frac{1}{a^n}$ |
| Fractional | $a^{\frac{m}{n}} = \sqrt[n]{a^m}$ |
