## 📊 CUBIC GRAPHS - AT A GLANCE

### Standard Form
$$y = ax^3 + bx^2 + cx + d$$

### Quick Reference Table

| Feature | Detail |
|---------|--------|
| **Degree** | 3 (highest power of x) |
| **y-intercept** | $(0, d)$ - substitute $x = 0$ |
| **x-intercepts** | Solve $ax^3 + bx^2 + cx + d = 0$ |
| **Turning points** | Maximum of 2 |
| **End behavior ($a > 0$)** | Down-left, Up-right |
| **End behavior ($a < 0$)** | Up-left, Down-right |

---

## 🔄 CUBIC TRANSFORMATIONS

| Transformation | Equation Form | Effect |
|----------------|---------------|--------|
| Vertical shift | $y = x^3 + k$ | Up by $k$ |
| Vertical shift | $y = x^3 - k$ | Down by $k$ |
| Horizontal shift | $y = (x - h)^3$ | Right by $h$ |
| Horizontal shift | $y = (x + h)^3$ | Left by $h$ |
| Vertical stretch | $y = ax^3$ | Factor $a$ |
| Reflection | $y = -x^3$ | In x-axis |

### Combined Form
$$y = a(x - h)^3 + k$$

**Order:** Shift → Stretch → Reflect → Shift

## 📐 RECIPROCAL GRAPHS - AT A GLANCE

### Standard Forms
$$y = \frac{a}{x} \quad \text{or} \quad y = \frac{a}{x^2}$$

### Key Features: $y = \frac{a}{x}$

| Feature | Detail |
|---------|--------|
| **Vertical asymptote** | $x = 0$ (y-axis) |
| **Horizontal asymptote** | $y = 0$ (x-axis) |
| **Quadrants ($a > 0$)** | I and III |
| **Quadrants ($a < 0$)** | II and IV |
| **Symmetry** | Rotational about origin |

### Key Features: $y = \frac{a}{x^2}$

| Feature | Detail |
|---------|--------|
| **Vertical asymptote** | $x = 0$ |
| **Horizontal asymptote** | $y = 0$ |
| **Quadrants ($a > 0$)** | I and II only |
| **y-values** | Always positive (for $a > 0$) |
| **Symmetry** | Reflection in y-axis |

---

## 🔄 RECIPROCAL TRANSFORMATIONS

### Translation Form
$$y = \frac{a}{x - h} + k$$

| Asymptote | Equation |
|-----------|----------|
| **Vertical** | $x = h$ |
| **Horizontal** | $y = k$ |

### Quick Rules

| Equation | Transformation |
|----------|----------------|
| $y = \frac{a}{x}$ | Basic shape |
| $y = \frac{a}{x - h}$ | Right by $h$ |
| $y = \frac{a}{x + h}$ | Left by $h$ |
| $y = \frac{a}{x} + k$ | Up by $k$ |
| $y = \frac{a}{x} - k$ | Down by $k$ |
| $y = \frac{-a}{x}$ | Reflection in x-axis |

##  EXAM QUESTION TYPES

### Type 1: Plotting
"Complete the table and draw the graph of..."

**Method:**
1. Calculate y for each x
2. Plot points
3. Draw smooth curve

### Type 2: Reading Values
"Use your graph to find the value of y when $x = 2.5$"

**Method:** Draw vertical line from x-value, read y-value

### Type 3: Solving Equations
"Use your graph to solve $x^3 - 2x = 3$"

**Method:**
1. Draw horizontal line $y = 3$
2. Find intersection points
3. Read x-values

### Type 4: Finding Turning Points
"Write down the coordinates of the turning point"

**Method:** Read from graph to nearest 0.5 small square

### Type 5: Asymptotes
"Write down the equations of the asymptotes"

**Method:** Identify vertical and horizontal asymptotes from equation

##  COMMON ERRORS

| Error | Correction |
|-------|------------|
| Plotting $x = 0$ for reciprocal | **Don't** - undefined |
| Straight line joins | Use **smooth curves** |
| Wrong translation direction | $(x - 3)$ → RIGHT |
| Missing asymptotes | Draw as **dashed lines** |
| Wrong turning point count | Cubics have **max 2** |
| Forgetting negative branches | Both quadrants! |

## 📌 KEY REMEMBERS

### Cubic Graphs
1. ✅ S-shaped curve
2. ✅ Can have 0, 1, or 2 turning points
3. ✅ y-intercept found by setting $x = 0$
4. ✅ $a > 0$ → rises to the right
5. ✅ $a < 0$ → falls to the right

### Reciprocal Graphs
1. ✅ NEVER crosses either axis (for basic form)
2. ✅ Asymptotes are the axes themselves
3. ✅ $y = \frac{1}{x}$ in Quadrants I and III
4. ✅ $y = \frac{1}{x^2}$ in Quadrants I and II
5. ✅ $y = \frac{a}{x-h} + k$ has asymptotes $x = h$, $y = k$

## 🏆 A* TIPS

1. **Speed:** Memorize $x^3$ values: $(-3)^3 = -27$, $(-2)^3 = -8$, $(-1)^3 = -1$
2. **Accuracy:** Use a sharp pencil for curves
3. **Checking:** Points should form smooth pattern - if not, check calculations
4. **Graphical solutions:** Draw auxiliary lines lightly but clearly
5. **Asymptotes:** Always state as equations (e.g., "$x = 2$" not just "$x = 2$ line")
