### Question 2

For a dataset with n = 15 values, what is the position of the upper quartile $Q_3$?

| Option | Answer |
|--------|--------|
| A | 11th value |
| B | 12th value |
| C | 10th value |
| D | 11.25th value |

**Answer:** **B**

**Explanation:**
$$Q_3 \text{ position} = \frac{3(n+1)}{4} = \frac{3 \times 16}{4} = 12$$

The IQR measures the spread of the **middle 50%** of data values, from $Q_1$ to $Q_3$.

### Question 21 (3 marks)

Find the range and interquartile range for the following data:

$$8, 15, 3, 12, 9, 18, 6, 21, 14, 10$$

**Solution:**

**Step 1: Order the data**

| | | | | | | | | | |
|-|-|-|-|-|-|-|-|-|-|
| 3 | 6 | 8 | 9 | 10 | 12 | 14 | 15 | 18 | 21 |

**Step 2: Find range** *(M1)*

$$\text{Range} = 21 - 3 = 18$$ *(A1)*

**Step 3: Find quartiles**

$n = 10$

$$Q_1 \text{ position} = \frac{10+1}{4} = 2.75 \rightarrow \text{between 2nd and 3rd}$$

$$Q_1 = 6 + 0.75(8-6) = 6 + 1.5 = 7.5$$ *(M1)*

$$Q_3 \text{ position} = \frac{3(10+1)}{4} = 8.25 \rightarrow \text{between 8th and 9th}$$

$$Q_3 = 15 + 0.25(18-15) = 15 + 0.75 = 15.75$$

**Step 4: Find IQR** *(M1)*

$$\text{IQR} = 15.75 - 7.5 = 8.25$$ *(A1)*

**Mark Scheme:**
| Mark | Criteria |
|------|----------|
| M1 | Correct method for range or ordering data |
| A1 | Range = 18 |
| M1 | Correct method for finding quartiles |
| A1 | IQR = 8.25 (accept 8 or 8.3 if using method 2) |

**Answers:** Range = 18, IQR = 8.25

### Question 26 (6 marks)

The table shows the times taken by 100 students to complete a puzzle.

| Time (seconds) | Frequency |
|----------------|-----------|
| 20-30 | 12 |
| 30-40 | 28 |
| 40-50 | 35 |
| 50-60 | 18 |
| 60-70 | 7 |

**(a)** Write down the lower boundary of the first class. (1 mark)

**(b)** Calculate an estimate for the lower quartile $Q_1$ using interpolation. (3 marks)

**(c)** Find the number of students who took more than 55 seconds. (2 marks)

**Solution:**

**(a) Lower boundary:**

**Lower boundary = 20 seconds** *(A1)*

**(b) Finding Q₁ using interpolation:**

**Step 1:** Find cumulative frequencies

| Time (s) | Freq | CF | Class Boundaries |
|----------|------|-----|------------------|
| 20-30 | 12 | 12 | 20 ≤ t < 30 |
| 30-40 | 28 | 40 | 30 ≤ t < 40 |
| 40-50 | 35 | 75 | 40 ≤ t < 50 |
| 50-60 | 18 | 93 | 50 ≤ t < 60 |
| 60-70 | 7 | 100 | 60 ≤ t < 70 |

**Step 2:** Find Q₁ position *(M1)*

$$Q_1 \text{ position} = \frac{100}{4} = 25$$

**Step 3:** Identify the class containing Q₁

25 is in the 30-40 class (CF goes from 12 to 40)

**Step 4:** Apply interpolation formula *(M1)*

$$Q_1 = 30 + \left(\frac{25 - 12}{28}\right) \times 10$$

$$Q_1 = 30 + \frac{13}{28} \times 10$$

$$Q_1 = 30 + 4.64 = 34.64 \text{ seconds}$$ *(A1)*

**(c) Students taking more than 55 seconds:**

**Method 1:** Using cumulative frequency

At 55 seconds (interpolating in the 50-60 class):
- CF at 50 = 93
- Need CF at 55: $93 + \frac{5}{10} \times 18 = 93 + 9 = 102$ (but max is 100)

Or simply:
- Students in 60-70 class = 7
- Students in 50-60 class taking more than 55: $\frac{5}{10} \times 18 = 9$ *(M1)*

**Total = 7 + 9 = 16 students** *(A1)*

**Mark Scheme:**
| Mark | Criteria |
|------|----------|
| A1 | Correct lower boundary |
| M1 | Correct Q₁ position identified |
| M1 | Correct interpolation formula used |
| A1 | Q₁ = 34.6 or 34.7 seconds |
| M1 | Correct method to find students above 55 |
| A1 | Answer = 16 students |

# EXAM FOCUS & STRATEGY

## Frequently Tested Question Types

### 1. **Basic Calculations** (2-3 marks)
- Calculate range, IQR from data lists
- Find quartile positions

**Strategy:** Always ORDER data first!

### 2. **Cumulative Frequency** (4-6 marks)
- Construct cumulative frequency table
- Draw the curve
- Read median, quartiles, percentiles

**Strategy:** 
- Start from (lower boundary, 0)
- Plot at upper boundaries
- Draw smooth S-curve

### 3. **Comparison Questions** (2-4 marks)
- Compare two datasets using medians and IQRs
- Make conclusions about performance and consistency

**Strategy:**
- Quote values from both datasets
- Use comparative language
- Interpret what the statistics mean

### 4. **Interpolation** (3-4 marks)
- Estimate quartiles from grouped data

**Strategy:**
- Set up cumulative frequency table
- Identify correct class
- Apply interpolation formula

---

## Typical Problem-Solving Patterns

### Pattern 1: "Find the number of values above/below X"

1. Read cumulative frequency at X
2. Subtract from total (for "above") or use directly (for "below")

### Pattern 2: "Compare these distributions"

1. Compare medians (central tendency)
2. Compare IQRs (consistency)
3. Make a conclusion

### Pattern 3: "Is this value an outlier?"

1. Calculate IQR
2. Find fence values: $Q_1 - 1.5 \times \text{IQR}$ and $Q_3 + 1.5 \times \text{IQR}$
3. Check if value lies outside fences
