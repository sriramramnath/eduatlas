## Algebraic Fractions & Equations with Fractions

---

## ⚡ PART 1: SIMPLIFYING ALGEBRAIC FRACTIONS

### Golden Rule

> ** ONLY cancel FACTORS, never terms. FACTORISE FIRST, then cancel.**

### Key Factorisation Types

| Type | Pattern | Factorisation |
|------|---------|---------------|
| **Difference of Squares** | $a^2 - b^2$ | $(a+b)(a-b)$ |
| **Perfect Square (+)** | $a^2 + 2ab + b^2$ | $(a+b)^2$ |
| **Perfect Square (−)** | $a^2 - 2ab + b^2$ | $(a-b)^2$ |
| **Simple Quadratic** | $x^2 + bx + c$ | $(x+p)(x+q)$ |
| **Common Factor** | $ax + ay$ | $a(x+y)$ |

### Quick Simplification Steps

```
1. Factorise numerator
2. Factorise denominator
3. Cancel common factors
4. State restrictions (denominator ≠ 0)
```

### Common Factorisation Patterns to Recognise

| Expression | Factorises To |
|------------|---------------|
| $x^2 - 1$ | $(x+1)(x-1)$ |
| $x^2 - 4$ | $(x+2)(x-2)$ |
| $x^2 - 9$ | $(x+3)(x-3)$ |
| $x^2 - 16$ | $(x+4)(x-4)$ |
| $4x^2 - 1$ | $(2x+1)(2x-1)$ |
| $4x^2 - 9$ | $(2x+3)(2x-3)$ |
| $x^2 + 2x + 1$ | $(x+1)^2$ |
| $x^2 - 2x + 1$ | $(x-1)^2$ |

## ⚡ PART 2: OPERATIONS WITH ALGEBRAIC FRACTIONS

### Multiplication

$$\frac{A}{B} \times \frac{C}{D} = \frac{AC}{BD}$$

**Method:** Factorise → Cancel across numerators and denominators → Multiply what remains

### Division

$$\frac{A}{B} \div \frac{C}{D} = \frac{A}{B} \times \frac{D}{C}$$

**Method:** Flip second fraction → Multiply as above

### Addition & Subtraction

$$\frac{A}{B} + \frac{C}{D} = \frac{AD + BC}{BD}$$

**Method:**
1. Factorise all denominators
2. Find LCD (Lowest Common Denominator)
3. Rewrite each fraction with LCD
4. Add/subtract numerators
5. Simplify if possible

### Finding the LCD

| Denominators | LCD |
|--------------|-----|
| $(x+1)$ and $(x-2)$ | $(x+1)(x-2)$ |
| $(x+1)$ and $(x+1)(x-2)$ | $(x+1)(x-2)$ |
| $x$ and $(x-1)$ and $(x+1)$ | $x(x-1)(x+1)$ |
| $x$ and $x^2$ | $x^2$ |
| $2$ and $3$ and $x$ | $6x$ |

## ⚡ PART 3: SOLVING EQUATIONS WITH FRACTIONS

### Clear Denominator Method

1. Find LCD of ALL denominators
2. Multiply EVERY term by LCD
3. Solve resulting equation
4. **CHECK** solution doesn't make any denominator zero

### Critical Warning: Extraneous Solutions

> ** ALWAYS check that solutions don't make any denominator = 0. If they do, reject them!**

### Common Scenarios

| Equation Type | After Clearing | Solution Type |
|---------------|----------------|---------------|
| Linear | Linear equation | One solution |
| Quadratic | Quadratic equation | 0, 1, or 2 solutions |
| Special case | Contradiction | No solution |

## 📐 Essential Formulae

### Operations Summary

| Operation | Rule |
|-----------|------|
| Multiply | $\frac{A}{B} \times \frac{C}{D} = \frac{AC}{BD}$ |
| Divide | $\frac{A}{B} \div \frac{C}{D} = \frac{AD}{BC}$ |
| Add (same denom.) | $\frac{A}{C} + \frac{B}{C} = \frac{A+B}{C}$ |
| Add (diff denom.) | $\frac{A}{B} + \frac{C}{D} = \frac{AD+BC}{BD}$ |

### Factorisation Identities

$$a^2 - b^2 = (a-b)(a+b)$$

$$(a+b)^2 = a^2 + 2ab + b^2$$

$$(a-b)^2 = a^2 - 2ab + b^2$$

##  Common Pitfalls

| Mistake | Wrong | Right |
|---------|-------|-------|
| Cancelling terms | $\frac{x+3}{3} = x$ | Cannot cancel; must factorise first |
| Wrong LCD | LCD of $x-1$ and $x^2-1$ is $x-1$ | LCD is $(x-1)(x+1) = x^2-1$ |
| Not multiplying all terms | $x + \frac{1}{x} = 2$ → $x^2 + 1 = 2$ | $x^2 + 1 = 2x$ |
| Sign error | $-(x-2) = -x-2$ | $-(x-2) = -x+2$ |
| Not checking extraneous | Answer $x=2$ for $\frac{x^2-4}{x-2}=6$ | $x=2$ makes denom. = 0, so no solution |

## 🔢 Quick Reference: Domain Restrictions

When simplifying or solving, state that:

$$\text{Denominator} \neq 0$$

| Denominator | Restriction |
|-------------|-------------|
| $x-3$ | $x \neq 3$ |
| $x+2$ | $x \neq -2$ |
| $x^2-4$ | $x \neq 2, -2$ |
| $x(x-1)$ | $x \neq 0, 1$ |
| $(x-1)(x+2)$ | $x \neq 1, -2$ |

##  Speed Tips for the Exam

1. **Recognise patterns:** $x^2-1$, $x^2-4$, $x^2-9$ are always difference of squares
2. **Factorise first:** Always factorise before doing anything else
3. **Cancel across:** In multiplication, cancel factors across the × sign
4. **Check as you go:** Substitute back to verify solutions
5. **State restrictions:** If asked for domain, state what x cannot equal
