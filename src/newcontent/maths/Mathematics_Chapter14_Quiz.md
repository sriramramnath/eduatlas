## 📝 SECTION A: MULTIPLE CHOICE QUESTIONS (MCQs)

**Instructions:** Choose the correct answer from A, B, C, or D.

---

**Q1.** In the formula $A = lw$, what is the subject?

- A) $l$
- B) $w$
- C) $A$
- D) $lw$

**Answer: C**

**Explanation:** The subject of a formula is the variable that stands alone on one side of the equals sign. In $A = lw$, the letter $A$ is on its own on the left side, making it the subject.

---

**Q2.** Evaluate $A = \frac{1}{2}bh$ when $b = 8$ and $h = 5$.

- A) 20
- B) 40
- C) 13
- D) 6.5

**Answer: A**

**Explanation:**
$$A = \frac{1}{2} \times 8 \times 5$$
$$A = \frac{1}{2} \times 40$$
$$A = 20$$

---

**Q3.** Make $r$ the subject of $C = 2\pi r$.

- A) $r = 2\pi C$
- B) $r = \frac{C}{2\pi}$
- C) $r = \frac{2\pi}{C}$
- D) $r = \frac{\pi C}{2}$

**Answer: B**

**Explanation:**
$$C = 2\pi r$$
Divide both sides by $2\pi$:
$$r = \frac{C}{2\pi}$$

---

**Q4.** Evaluate $y = x^2 - 3x$ when $x = -2$.

- A) -10
- B) 10
- C) 2
- D) -2

**Answer: B**

**Explanation:**
$$y = (-2)^2 - 3(-2)$$
$$y = 4 - (-6)$$
$$y = 4 + 6$$
$$y = 10$$

**Note:** Use brackets for negative numbers: $(-2)^2 = 4$, not $-4$.

---

**Q5.** Make $t$ the subject of $v = u + at$.

- A) $t = \frac{v-u}{a}$
- B) $t = \frac{v+u}{a}$
- C) $t = v - u - a$
- D) $t = \frac{a}{v-u}$

**Answer: A**

**Explanation:**
$$v = u + at$$
Subtract $u$: $v - u = at$
Divide by $a$: $t = \frac{v-u}{a}$

---

**Q6.** The formula for the area of a circle is $A = \pi r^2$. Make $r$ the subject.

- A) $r = \frac{A}{\pi}$
- B) $r = \sqrt{\pi A}$
- C) $r = \sqrt{\frac{A}{\pi}}$
- D) $r = \frac{\sqrt{A}}{\pi}$

**Answer: C**

**Explanation:**
$$A = \pi r^2$$
Divide by $\pi$: $\frac{A}{\pi} = r^2$
Square root both sides: $r = \sqrt{\frac{A}{\pi}}$

---

**Q7.** A taxi charges £4 plus £2 per mile. Which formula represents the total cost $C$ for $m$ miles?

- A) $C = 4m + 2$
- B) $C = 2m + 4$
- C) $C = 4 + m + 2$
- D) $C = 2 + 4m$

**Answer: B**

**Explanation:** Fixed charge = £4, plus £2 for each mile = $2m$. Therefore: $C = 2m + 4$ or $C = 4 + 2m$.

---

**Q8.** Evaluate $(-3)^2 + 2 \times (-3)$.

- A) -15
- B) 3
- C) 15
- D) -3

**Answer: B**

**Explanation:**
$$(-3)^2 + 2 \times (-3)$$
$$= 9 + (-6)$$
$$= 9 - 6$$
$$= 3$$

---

**Q9.** Make $h$ the subject of $A = \frac{1}{2}bh$.

- A) $h = \frac{b}{2A}$
- B) $h = \frac{2b}{A}$
- C) $h = \frac{A}{2b}$
- D) $h = \frac{2A}{b}$

**Answer: D**

**Explanation:**
$$A = \frac{1}{2}bh$$
Multiply by 2: $2A = bh$
Divide by $b$: $h = \frac{2A}{b}$

---

**Q10.** What is the value of $v$ in the formula $v = u + at$ when $u = 5$, $a = -2$, and $t = 3$?

- A) 11
- B) -1
- C) 1
- D) -11

**Answer: B**

**Explanation:**
$$v = u + at$$
$$v = 5 + (-2) \times 3$$
$$v = 5 + (-6)$$
$$v = -1$$

---

**Q11.** Make $u$ the subject of $s = \frac{u+v}{2} \times t$.

- A) $u = \frac{2s}{t} - v$
- B) $u = \frac{2s}{t} + v$
- C) $u = \frac{s}{t} - v$
- D) $u = \frac{2s - v}{t}$

**Answer: A**

**Explanation:**
$$s = \frac{u+v}{2} \times t$$
Multiply by 2: $2s = (u+v)t$
Divide by $t$: $\frac{2s}{t} = u + v$
Subtract $v$: $u = \frac{2s}{t} - v$

---

**Q12.** The volume of a sphere is $V = \frac{4}{3}\pi r^3$. Which expression gives $r^3$?

- A) $r^3 = \frac{4V}{3\pi}$
- B) $r^3 = \frac{3V}{4\pi}$
- C) $r^3 = \frac{4\pi V}{3}$
- D) $r^3 = \frac{V}{4\pi}$

**Answer: B**

**Explanation:**
$$V = \frac{4}{3}\pi r^3$$
Multiply by 3: $3V = 4\pi r^3$
Divide by $4\pi$: $\frac{3V}{4\pi} = r^3$

## 📝 SECTION B: TRUE OR FALSE

**Instructions:** State whether each statement is TRUE or FALSE. If FALSE, provide the correction.

---

**Q1.** In the formula $v = u + at$, if we make $a$ the subject, the answer is $a = \frac{v-u}{t}$.

**Answer: TRUE**

**Explanation:** 
$$v = u + at$$
$$v - u = at$$
$$a = \frac{v-u}{t}$$

---

**Q2.** When evaluating $x^2$ where $x = -4$, the answer is $-16$.

**Answer: FALSE**

**Correction:** $(-4)^2 = 16$. The negative number must be in brackets before squaring. Without brackets, $-4^2 = -(4^2) = -16$.

---

**Q3.** The subject of the formula $F = \frac{9}{5}C + 32$ is $F$.

**Answer: TRUE**

**Explanation:** $F$ appears alone on the left side of the equals sign, making it the subject.

---

**Q4.** When rearranging $A = \pi r^2$ to make $r$ the subject, the first step is to multiply by $\pi$.

**Answer: FALSE**

**Correction:** The first step is to **divide** by $\pi$ (the inverse operation). $A = \pi r^2$ → $\frac{A}{\pi} = r^2$.

---

**Q5.** If $A = lw$ and $A = 30$, $l = 6$, then $w = 5$.

**Answer: TRUE**

**Explanation:** 
$$A = lw$$
$$30 = 6 \times w$$
$$w = \frac{30}{6} = 5$$

---

**Q6.** In the formula $\frac{1}{R} = \frac{1}{R_1} + \frac{1}{R_2}$, making $R$ the subject gives $R = R_1 + R_2$.

**Answer: FALSE**

**Correction:** 
$$\frac{1}{R} = \frac{R_2 + R_1}{R_1 R_2}$$
$$R = \frac{R_1 R_2}{R_1 + R_2}$$

---

**Q7.** When substituting negative numbers, you should always use brackets.

**Answer: TRUE**

**Explanation:** Brackets ensure correct application of operations. For example, $(-3)^2 = 9$ but $-3^2 = -9$.

---

**Q8.** To make $x$ the subject of $y = mx + c$, you divide by $m$ first, then subtract $c$.

**Answer: FALSE**

**Correction:** You should subtract $c$ first, then divide by $m$. This follows the reverse BIDMAS rule.
$$y = mx + c$$
$$y - c = mx$$
$$x = \frac{y-c}{m}$$

---

**Q9.** The formula $s = \frac{d}{t}$ rearranged to make $t$ the subject is $t = \frac{s}{d}$.

**Answer: FALSE**

**Correction:** If $s = \frac{d}{t}$, then $st = d$, so $t = \frac{d}{s}$.

---

**Q10.** If a variable appears twice in a formula, you must factorise before making it the subject.

**Answer: TRUE**

**Explanation:** For example, in $A = \pi r^2 + 2\pi rh$, factorise first: $A = \pi r(r + 2h)$, then $r = \frac{A}{\pi(r+2h)}$.

### Question 1 (2 marks)

The formula for the area of a trapezium is $A = \frac{1}{2}(a + b)h$.

Calculate $A$ when $a = 5$, $b = 9$, and $h = 4$.

**Solution:**

**[M1]** Correct substitution shown:
$$A = \frac{1}{2}(5 + 9) \times 4$$

**[A1]** Correct evaluation:
$$A = \frac{1}{2}(14) \times 4$$
$$A = 7 \times 4$$
$$A = 28$$

**Answer: $A = 28$ square units**

---

### Question 2 (3 marks)

(a) Write down a formula for the total cost, $C$ pounds, of buying $n$ books at £8 each plus £3 postage. **[1 mark]**

(b) Calculate the total cost of buying 7 books. **[2 marks]**

**Solution:**

**(a) [M1]**
$$C = 8n + 3$$

**(b) [M1]** Substitute $n = 7$:
$$C = 8(7) + 3$$
$$C = 56 + 3$$

**[A1]**
$$C = 59$$

**Answer: (a) $C = 8n + 3$, (b) £59**

---

### Question 3 (3 marks)

Make $h$ the subject of the formula $V = lwh$.

**Solution:**

**[M1]** Divide both sides by $lw$:
$$\frac{V}{lw} = h$$

**[A1]** Correct rearrangement:
$$h = \frac{V}{lw}$$

**Alternative notation:** $h = \frac{V}{lw}$ or $h = \frac{V}{l \times w}$

**Answer: $h = \frac{V}{lw}$**

### Question 4 (4 marks)

The formula for converting Celsius to Fahrenheit is $F = \frac{9C}{5} + 32$.

(a) Convert $20°C$ to Fahrenheit. **[2 marks]**

(b) Rearrange the formula to make $C$ the subject. **[2 marks]**

**Solution:**

**(a) [M1]** Substitution:
$$F = \frac{9 \times 20}{5} + 32$$
$$F = \frac{180}{5} + 32$$
$$F = 36 + 32$$

**[A1]** 
$$F = 68°F$$

**(b) [M1]** Subtract 32, then multiply by 5, then divide by 9:
$$F = \frac{9C}{5} + 32$$
$$F - 32 = \frac{9C}{5}$$
$$5(F - 32) = 9C$$

**[A1]**
$$C = \frac{5(F - 32)}{9}$$

**Answer: (a) 68°F, (b) $C = \frac{5(F-32)}{9}$**

---

### Question 5 (3 marks)

Make $r$ the subject of $A = \pi r^2 + 2\pi rh$.

**Solution:**

**[M1]** Factorise out $\pi r$:
$$A = \pi r(r + 2h)$$

**[M1]** Divide by $\pi(r + 2h)$:
$$\frac{A}{\pi(r + 2h)} = r$$

**[A1]**
$$r = \frac{A}{\pi(r + 2h)}$$

**Answer: $r = \frac{A}{\pi(r+2h)}$**

### Question 6 (4 marks)

The height $h$ metres of a ball thrown upwards after $t$ seconds is given by:
$$h = 25t - 5t^2$$

(a) Find the height of the ball after 2 seconds. **[2 marks]**

(b) When is the ball at height 20 metres? Give your answer to 1 decimal place. **[2 marks]**

**Solution:**

**(a) [M1]** Substitute $t = 2$:
$$h = 25(2) - 5(2)^2$$
$$h = 50 - 5(4)$$
$$h = 50 - 20$$

**[A1]**
$$h = 30 \text{ metres}$$

**(b) [M1]** Set up equation and solve:
$$20 = 25t - 5t^2$$
$$5t^2 - 25t + 20 = 0$$
Divide by 5: $t^2 - 5t + 4 = 0$
$$(t - 4)(t - 1) = 0$$

**[A1]**
$$t = 1 \text{ second or } t = 4 \text{ seconds}$$

**Answer: (a) 30 m, (b) $t = 1$ s and $t = 4$ s**

---

### Question 7 (5 marks)

The volume of a cone is given by $V = \frac{1}{3}\pi r^2 h$.

(a) Calculate the volume when $r = 3$ cm and $h = 10$ cm. Give your answer in terms of $\pi$. **[2 marks]**

(b) Make $r$ the subject of the formula. **[3 marks]**

**Solution:**

**(a) [M1]** Substitution:
$$V = \frac{1}{3} \times \pi \times 3^2 \times 10$$
$$V = \frac{1}{3} \times \pi \times 9 \times 10$$
$$V = \frac{90\pi}{3}$$

**[A1]**
$$V = 30\pi \text{ cm}^3$$

**(b) [M1]** Multiply by 3:
$$3V = \pi r^2 h$$

**[M1]** Divide by $\pi h$:
$$\frac{3V}{\pi h} = r^2$$

**[A1]** Square root:
$$r = \sqrt{\frac{3V}{\pi h}}$$

**Answer: (a) $30\pi$ cm³, (b) $r = \sqrt{\frac{3V}{\pi h}}$**

### Question 8 (4 marks)

The kinetic energy of a moving object is given by $E = \frac{1}{2}mv^2$ where $E$ is in Joules, $m$ is mass in kg, and $v$ is velocity in m/s.

(a) Calculate the kinetic energy when $m = 2$ kg and $v = 6$ m/s. **[2 marks]**

(b) Rearrange the formula to make $v$ the subject. **[2 marks]**

**Solution:**

**(a) [M1]** Substitution:
$$E = \frac{1}{2} \times 2 \times 6^2$$
$$E = \frac{1}{2} \times 2 \times 36$$
$$E = 36$$

**[A1]**
$$E = 36 \text{ Joules}$$

**(b) [M1]** Rearrange:
$$E = \frac{1}{2}mv^2$$
$$2E = mv^2$$
$$\frac{2E}{m} = v^2$$

**[A1]**
$$v = \sqrt{\frac{2E}{m}}$$

**Answer: (a) 36 J, (b) $v = \sqrt{\frac{2E}{m}}$**

### Question 9 (5 marks)

A company uses the formula $C = 50 + 0.25n$ to calculate the monthly charge for a phone contract, where $C$ is the charge in pounds and $n$ is the number of minutes used.

(a) Calculate the charge when 200 minutes are used. **[2 marks]**

(b) A customer's bill was £87.50. How many minutes did they use? **[3 marks]**

**Solution:**

**(a) [M1]** Substitute $n = 200$:
$$C = 50 + 0.25(200)$$
$$C = 50 + 50$$

**[A1]**
$$C = £100$$

**(b) [M1]** Set up equation:
$$87.50 = 50 + 0.25n$$

**[M1]** Rearrange:
$$87.50 - 50 = 0.25n$$
$$37.50 = 0.25n$$

**[A1]** Solve:
$$n = \frac{37.50}{0.25}$$
$$n = 150 \text{ minutes}$$

**Answer: (a) £100, (b) 150 minutes**

### Question 10 (6 marks)

The surface area of a cylinder is given by $S = 2\pi r^2 + 2\pi rh$.

(a) Factorise the expression $2\pi r^2 + 2\pi rh$. **[1 mark]**

(b) Make $h$ the subject of the formula. **[2 marks]**

(c) Calculate $h$ when $S = 80\pi$ and $r = 4$ cm. **[3 marks]**

**Solution:**

**(a) [B1]**
$$2\pi r^2 + 2\pi rh = 2\pi r(r + h)$$

**(b) [M1]** Start from original formula:
$$S = 2\pi r^2 + 2\pi rh$$
$$S - 2\pi r^2 = 2\pi rh$$

**[A1]**
$$h = \frac{S - 2\pi r^2}{2\pi r}$$

**(c) [M1]** Substitute values:
$$h = \frac{80\pi - 2\pi(4)^2}{2\pi(4)}$$
$$h = \frac{80\pi - 2\pi(16)}{8\pi}$$
$$h = \frac{80\pi - 32\pi}{8\pi}$$

**[M1]** Simplify:
$$h = \frac{48\pi}{8\pi}$$

**[A1]**
$$h = 6 \text{ cm}$$

**Answer: (a) $2\pi r(r+h)$, (b) $h = \frac{S-2\pi r^2}{2\pi r}$, (c) 6 cm**

## 📊 SECTION D: EXAM FOCUS & STRATEGY

### Frequently Tested Question Types

| Type | Description | Typical Marks |
|------|-------------|---------------|
| **Direct Substitution** | Plug values into given formula | 2-3 |
| **Two-step Rearrangement** | Make a variable the subject | 2-3 |
| **Worded Formula** | Create formula from description | 2-3 |
| **Reverse Problem** | Find input given output | 3-4 |
| **Subject Appears Twice** | Factorise before rearranging | 3-4 |
| **Combined Operations** | Rearrange + substitute | 4-5 |

### Typical Problem-Solving Patterns

**Pattern 1: The Setup → Calculate → Check**
```
1. Write formula
2. Substitute values
3. Calculate answer
4. Check reasonableness
```

**Pattern 2: The Rearrange → Use → Apply**
```
1. Identify what's needed
2. Rearrange formula
3. Substitute values
4. Calculate answer
```

### Command Word Guidance

| Command | What Examiners Expect |
|---------|----------------------|
| **Evaluate** | Numerical answer with working |
| **Express** | Algebraic expression (no numbers) |
| **Find** | Single answer with method |
| **Calculate** | Numerical answer with full working |
| **Make ... the subject** | Rearranged formula with new subject |
| **Show that** | Full working proving the result |
| **Hence** | Must use previous answer |

### How to Gain Full Marks

**For Substitution (2-3 marks):**
1. ✅ Write the formula clearly
2. ✅ Show substitution: $A = \frac{1}{2}(6+8)(5)$
3. ✅ Show intermediate steps
4. ✅ Give final answer with units

**For Change of Subject (3-4 marks):**
1. ✅ Write formula clearly
2. ✅ Show each step on new line
3. ✅ Write correct operation being performed
4. ✅ Check answer by back-substitution

**For Worded Problems (3-5 marks):**
1. ✅ Define variables clearly
2. ✅ Write formula correctly
3. ✅ Show all working
4. ✅ State final answer with units
5. ✅ Check answer makes sense
