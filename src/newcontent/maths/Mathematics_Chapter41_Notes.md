# 1. TRIANGLES

## 1.1 Types of Triangles

### Classification by Sides

| Type | Properties | Example |
|------|-----------|---------|
| **Equilateral** | All three sides equal; All angles = 60° | $\triangle ABC$ where $AB = BC = CA$ |
| **Isosceles** | Two sides equal; Two base angles equal | $\triangle ABC$ where $AB = AC$ |
| **Scalene** | All sides different; All angles different | $\triangle ABC$ with $AB \neq BC \neq CA$ |

### Classification by Angles

| Type | Properties |
|------|-----------|
| **Acute-angled** | All three angles < 90° |
| **Right-angled** | One angle = 90° |
| **Obtuse-angled** | One angle > 90° |

---

## 1.2 Fundamental Angle Rules for Triangles

### KEY FORMULA: Angle Sum of a Triangle

$$\text{Sum of interior angles} = 180°$$

For any triangle with angles $a$, $b$, and $c$:
$$a + b + c = 180°$$

> ** COMMON MISTAKE:** Students often forget this rule applies to ALL triangles, regardless of type.

---

### 📐 Worked Example 1: Finding Missing Angles

**Problem:** In triangle $ABC$, angle $A = 47°$ and angle $B = 68°$. Find angle $C$.

**Solution:**
$$A + B + C = 180°$$
$$47° + 68° + C = 180°$$
$$115° + C = 180°$$
$$C = 180° - 115°$$
$$C = 65°$$

> ** EXAM TIP:** Always write down the rule you're using (e.g., "Angles in a triangle sum to 180°") to secure method marks.

---

## 1.3 Special Triangle Properties

### Isosceles Triangle Properties

1. **Two equal sides** (called legs)
2. **Two equal angles** (called base angles)
3. **One line of symmetry** (through the vertex angle)
4. **The angle between equal sides** is called the vertex angle

**Notation:** In an isosceles triangle where $AB = AC$:
- $\angle B = \angle C$ (base angles are equal)
- $\angle A$ is the vertex angle

---

### 📐 Worked Example 2: Isosceles Triangle

**Problem:** In an isosceles triangle $PQR$, $PQ = PR$ and $\angle P = 34°$. Find $\angle Q$ and $\angle R$.

**Solution:**
Since $PQ = PR$, the triangle is isosceles with equal base angles at $Q$ and $R$.

Let $\angle Q = \angle R = x$

$$\angle P + \angle Q + \angle R = 180°$$
$$34° + x + x = 180°$$
$$34° + 2x = 180°$$
$$2x = 180° - 34°$$
$$2x = 146°$$
$$x = 73°$$

Therefore, $\angle Q = \angle R = 73°$

---

### Equilateral Triangle Properties

1. **All sides equal**: $AB = BC = CA$
2. **All angles equal**: $\angle A = \angle B = \angle C = 60°$
3. **Three lines of symmetry**
4. **Rotational symmetry of order 3**

> **🎯 KEY INSIGHT:** Every equilateral triangle is also isosceles, but not every isosceles triangle is equilateral.

---

## 1.4 Exterior Angle Theorem

### KEY FORMULA

$$\text{Exterior angle} = \text{Sum of two interior opposite angles}$$

For triangle $ABC$ with exterior angle at $A$:
$$\text{Exterior angle at } A = \angle B + \angle C$$

**Derivation:**
- Interior angle at $A$ + Exterior angle at $A$ = 180° (straight line)
- Interior angle at $A$ = 180° - $(\angle B + \angle C)$
- Therefore: Exterior angle at $A$ = 180° - Interior angle at $A$ = $\angle B + \angle C$

---

### 📐 Worked Example 3: Exterior Angle

**Problem:** Find the value of $x$ in the triangle shown, where an exterior angle is $118°$ and one interior opposite angle is $42°$.

**Solution:**
Using the exterior angle theorem:
$$\text{Exterior angle} = \text{Sum of interior opposite angles}$$
$$118° = 42° + x$$
$$x = 118° - 42°$$
$$x = 76°$$

---

## 1.5 Angles in Special Configurations

### Angles on a Straight Line

$$\text{Angles on a straight line sum to } 180°$$

### Angles at a Point

$$\text{Angles at a point sum to } 360°$$

### Vertically Opposite Angles

**Vertically opposite angles are equal.**

When two lines intersect, four angles are formed. The angles directly opposite each other are equal.

# 2. QUADRILATERALS

## 2.1 Types of Quadrilaterals

### Hierarchy of Quadrilaterals

```
Quadrilateral
    ├── Trapezium (one pair of parallel sides)
    ├── Parallelogram (two pairs of parallel sides)
    │     ├── Rectangle (parallelogram with right angles)
    │     ├── Rhombus (parallelogram with equal sides)
    │     └── Square (rectangle AND rhombus)
    └── Kite (two pairs of adjacent equal sides)
```

---

## 2.2 Properties of Each Quadrilateral

### KEY TABLE: Quadrilateral Properties

| Property | Square | Rectangle | Rhombus | Parallelogram | Kite | Trapezium |
|----------|--------|-----------|---------|---------------|------|-----------|
| Opposite sides parallel | ✓ | ✓ | ✓ | ✓ | ✗ | One pair only |
| Opposite sides equal | ✓ | ✓ | ✓ | ✓ | ✗ | ✗ |
| All sides equal | ✓ | ✗ | ✓ | ✗ | Two pairs adjacent | ✗ |
| All angles = 90° | ✓ | ✓ | ✗ | ✗ | ✗ | ✗ |
| Diagonals bisect each other | ✓ | ✓ | ✓ | ✓ | ✗ | ✗ |
| Diagonals perpendicular | ✓ | ✗ | ✓ | ✗ | ✓ | ✗ |
| Diagonals equal length | ✓ | ✓ | ✗ | ✗ | ✗ | ✗ |
| Line symmetry | 4 | 2 | 2 | 0 | 1 | 0 or 1 |
| Rotational symmetry | Order 4 | Order 2 | Order 2 | Order 2 | 0 | 0 or 1 |

---

## 2.3 Angle Sum of Quadrilaterals

### KEY FORMULA

$$\text{Sum of interior angles of a quadrilateral} = 360°$$

**Derivation:** A quadrilateral can be divided into 2 triangles by drawing one diagonal. Since each triangle has angles summing to $180°$:
$$\text{Sum} = 2 \times 180° = 360°$$

---

### 📐 Worked Example 4: Parallelogram Angles

**Problem:** In parallelogram $ABCD$, $\angle A = 72°$. Find all other angles.

**Solution:**

**Step 1:** In a parallelogram, opposite angles are equal.
$$\angle C = \angle A = 72°$$

**Step 2:** In a parallelogram, adjacent angles are supplementary.
$$\angle A + \angle B = 180°$$
$$72° + \angle B = 180°$$
$$\angle B = 108°$$

**Step 3:** $\angle D = \angle B = 108°$ (opposite angles equal)

**Answer:** $\angle A = 72°$, $\angle B = 108°$, $\angle C = 72°$, $\angle D = 108°$

---

### 📐 Worked Example 5: Kite Angles

**Problem:** In kite $PQRS$, $PQ = PS$ and $QR = SR$. If $\angle P = 124°$ and $\angle Q = 50°$, find $\angle R$.

**Solution:**

**Step 1:** In a kite, one pair of opposite angles are equal (the angles between unequal sides).
Since $PQ = PS$ and $QR = SR$, the equal angles are at $Q$ and $S$.
$$\angle Q = \angle S = 50°$$

**Step 2:** Using the angle sum of a quadrilateral:
$$\angle P + \angle Q + \angle R + \angle S = 360°$$
$$124° + 50° + \angle R + 50° = 360°$$
$$224° + \angle R = 360°$$
$$\angle R = 136°$$

---

## 2.4 Properties Involving Diagonals

### Parallelogram Diagonals
- Diagonals **bisect each other** (divide into two equal parts)
- Diagonals are **NOT necessarily equal** or perpendicular

### Rectangle Diagonals
- Diagonals **bisect each other**
- Diagonals are **equal in length**

### Rhombus Diagonals
- Diagonals **bisect each other at right angles** (90°)
- Diagonals **bisect the interior angles**

### Square Diagonals
- All properties of rectangle AND rhombus
- Diagonals are equal, perpendicular, and bisect angles

### Kite Diagonals
- One diagonal is the **perpendicular bisector** of the other
- The diagonal connecting unequal angles is bisected

# 3. POLYGONS

## 3.1 Polygon Terminology

| Term | Definition |
|------|-----------|
| **Polygon** | A closed 2D shape with straight sides |
| **Regular polygon** | All sides equal AND all angles equal |
| **Convex polygon** | All interior angles < 180° |
| **Concave polygon** | At least one interior angle > 180° |

### Names of Polygons

| Sides | Name | Sides | Name |
|-------|------|-------|------|
| 3 | Triangle | 8 | Octagon |
| 4 | Quadrilateral | 9 | Nonagon |
| 5 | Pentagon | 10 | Decagon |
| 6 | Hexagon | 12 | Dodecagon |
| 7 | Heptagon | $n$ | $n$-gon |

---

## 3.2 Interior Angle Sum Formula

### KEY FORMULA

$$\text{Sum of interior angles} = (n - 2) \times 180°$$

where $n$ = number of sides

**Derivation:**
- A polygon can be divided into $(n - 2)$ triangles by drawing diagonals from one vertex
- Each triangle has angle sum = $180°$
- Total = $(n - 2) \times 180°$

---

### 📐 Worked Example 6: Finding Angle Sum

**Problem:** Find the sum of interior angles of a heptagon (7 sides).

**Solution:**
$$\text{Sum} = (n - 2) \times 180°$$
$$\text{Sum} = (7 - 2) \times 180°$$
$$\text{Sum} = 5 \times 180°$$
$$\text{Sum} = 900°$$

---

## 3.3 Interior Angle of a Regular Polygon

### KEY FORMULA

$$\text{Each interior angle} = \frac{(n - 2) \times 180°}{n}$$

---

### 📐 Worked Example 7: Regular Polygon Interior Angle

**Problem:** Find the interior angle of a regular octagon.

**Solution:**
$$\text{Interior angle} = \frac{(8 - 2) \times 180°}{8}$$
$$= \frac{6 \times 180°}{8}$$
$$= \frac{1080°}{8}$$
$$= 135°$$

---

## 3.4 Exterior Angle of a Regular Polygon

### KEY FORMULAS

**Sum of exterior angles:**
$$\text{Sum of exterior angles} = 360° \text{ (for ANY polygon)}$$

**Each exterior angle of a regular polygon:**
$$\text{Exterior angle} = \frac{360°}{n}$$

**Relationship between interior and exterior angles:**
$$\text{Interior angle} + \text{Exterior angle} = 180°$$

---

### 📐 Worked Example 8: Finding Number of Sides

**Problem:** The exterior angle of a regular polygon is $24°$. How many sides does it have?

**Solution:**
$$\text{Exterior angle} = \frac{360°}{n}$$
$$24° = \frac{360°}{n}$$
$$n = \frac{360°}{24°}$$
$$n = 15$$

The polygon has 15 sides (a pentadecagon).

---

### 📐 Worked Example 9: Finding Number of Sides from Interior Angle

**Problem:** Each interior angle of a regular polygon is $156°$. Find the number of sides.

**Method 1: Using exterior angle**
$$\text{Exterior angle} = 180° - 156° = 24°$$
$$n = \frac{360°}{24°} = 15$$

**Method 2: Using interior angle formula**
$$156 = \frac{(n-2) \times 180}{n}$$
$$156n = 180(n-2)$$
$$156n = 180n - 360$$
$$360 = 180n - 156n$$
$$360 = 24n$$
$$n = 15$$

---

## 3.5 Problem-Solving with Polygons

### 📐 Worked Example 10: Irregular Polygon

**Problem:** A hexagon has angles $110°, 130°, 95°, 140°, 115°$. Find the sixth angle.

**Solution:**
$$\text{Sum of interior angles} = (6 - 2) \times 180° = 720°$$

Let the sixth angle be $x$:
$$110° + 130° + 95° + 140° + 115° + x = 720°$$
$$590° + x = 720°$$
$$x = 720° - 590°$$
$$x = 130°$$

# 4. CONSTRUCTIONS

## 4.1 Essential Equipment

- **Compass**: For drawing circles and arcs
- **Ruler**: For measuring and drawing straight lines
- **Protractor**: For measuring and drawing angles
- **Sharp pencil**: For accurate marks

---

## 4.2 Basic Constructions

### Construction 1: Bisecting a Line Segment

**Steps:**
1. Open compass to more than half the length of the line
2. With compass point at one end, draw an arc above and below the line
3. With compass point at the other end, draw arcs intersecting the first arcs
4. Draw a line through the intersections - this is the perpendicular bisector

> **🎯 EXAM TIP:** Construction arcs must be shown clearly. Do not erase them!

---

### Construction 2: Bisecting an Angle

**Steps:**
1. With compass point at the vertex, draw an arc cutting both arms
2. From each intersection point, draw arcs that intersect inside the angle
3. Draw a line from the vertex through this intersection
4. This line bisects the angle

---

### Construction 3: Perpendicular from a Point on a Line

**Steps:**
1. With compass at the point, mark equal distances on the line on both sides
2. With compass at each mark, draw arcs above the line that intersect
3. Connect the point to the intersection
4. This creates a perpendicular at the point

---

### Construction 4: Perpendicular from a Point to a Line

**Steps:**
1. With compass at the external point, draw an arc cutting the line twice
2. With compass at each intersection, draw arcs on the opposite side of the line
3. Connect the external point to where these arcs intersect
4. This creates the perpendicular from point to line

---

### Construction 5: Constructing a 60° Angle

**Steps:**
1. Draw a base line with point $P$
2. With compass at $P$, draw an arc cutting the line at $Q$
3. With compass at $Q$ (same radius), draw an arc intersecting the first arc at $R$
4. Draw line $PR$ - angle $QPR = 60°$

**Why this works:** $PQR$ forms an equilateral triangle, so all angles = $60°$

---

### Construction 6: Constructing a 90° Angle

**Method:** Construct a perpendicular bisector or use angle bisection:
1. Construct a $60°$ angle
2. Bisect the $60°$ angle to get $30°$
3. Bisect again to get the remaining angle

Or:
1. Draw a semicircle from a point on a line
2. From the endpoints, draw arcs intersecting above
3. Connect to get $90°$

---

## 4.3 Triangle Constructions

### Constructing a Triangle Given Three Sides (SSS)

**Example:** Construct $\triangle ABC$ with $AB = 7$ cm, $BC = 5$ cm, $AC = 4$ cm

**Steps:**
1. Draw base $AB = 7$ cm
2. With compass radius 5 cm from $B$, draw an arc
3. With compass radius 4 cm from $A$, draw an arc intersecting the first
4. Connect the intersection to $A$ and $B$

> ** NOTE:** Construction is only possible if the sum of any two sides exceeds the third side (triangle inequality theorem).

---

### Constructing a Triangle Given Two Sides and Included Angle (SAS)

**Example:** Construct $\triangle PQR$ with $PQ = 6$ cm, $QR = 4$ cm, $\angle PQR = 55°$

**Steps:**
1. Draw base $PQ = 6$ cm
2. At $Q$, construct angle of $55°$ using protractor or by construction
3. Mark point $R$ at distance 4 cm from $Q$ along the angle arm
4. Connect $P$ to $R$

---

### Constructing a Triangle Given Two Angles and a Side (ASA)

**Example:** Construct $\triangle ABC$ with $AB = 8$ cm, $\angle A = 40°$, $\angle B = 65°$

**Steps:**
1. Draw base $AB = 8$ cm
2. At $A$, construct $40°$ angle
3. At $B$, construct $65°$ angle
4. Where the angle arms meet is point $C$

---

## 4.4 Loci

### Definition
A **locus** (plural: loci) is the set of all points that satisfy a given condition.

### Common Loci

| Locus Description | Shape |
|-------------------|-------|
| Points equidistant from a fixed point | Circle |
| Points equidistant from two fixed points | Perpendicular bisector of the line joining them |
| Points equidistant from two intersecting lines | Angle bisector |
| Points at a fixed distance from a line segment | Parallel lines + semicircles at ends |

---

### 📐 Worked Example 11: Locus Problem

**Problem:** Point $P$ is equidistant from points $A$ and $B$, and also 3 cm from $A$. Describe the position of $P$.

**Solution:**
- "Equidistant from $A$ and $B$" means $P$ lies on the perpendicular bisector of $AB$
- "3 cm from $A$" means $P$ lies on a circle, radius 3 cm, center $A$
- Therefore, $P$ is at the intersection of these two loci

# 5. SIMILAR TRIANGLES

## 5.1 Definition of Similarity

Two shapes are **similar** if they have the same shape but may be different sizes. For triangles:

### KEY CONCEPT: Conditions for Similar Triangles

Two triangles are similar if ANY ONE of these conditions is met:

| Condition | Name | Description |
|-----------|------|-------------|
| AAA | Angle-Angle-Angle | All three angles are equal |
| SSS | Side-Side-Side (proportional) | All three sides are in proportion |
| SAS | Side-Angle-Side | Two sides in proportion and included angle equal |

> ** EXAM TIP:** For AAA, you only need to prove TWO angles equal (the third follows automatically since angles sum to 180°)

---

## 5.2 Scale Factor

### KEY FORMULA

$$\text{Scale factor} = \frac{\text{Length on image}}{\text{Length on object}}$$

For similar shapes with scale factor $k$:
- All linear dimensions multiply by $k$
- Areas multiply by $k^2$
- Volumes multiply by $k^3$ (for 3D shapes)

---

### 📐 Worked Example 12: Finding Scale Factor

**Problem:** Triangle $ABC$ is similar to triangle $DEF$. If $AB = 4$ cm and $DE = 10$ cm, find the scale factor.

**Solution:**
$$\text{Scale factor} = \frac{DE}{AB} = \frac{10}{4} = 2.5$$

---

## 5.3 Finding Missing Lengths in Similar Triangles

### 📐 Worked Example 13: Basic Similar Triangles

**Problem:** Triangles $PQR$ and $XYZ$ are similar. $PQ = 6$ cm, $QR = 9$ cm, $PR = 12$ cm. If $XY = 10$ cm, find $YZ$ and $XZ$.

**Solution:**

**Step 1:** Find the scale factor
$$\text{Scale factor} = \frac{XY}{PQ} = \frac{10}{6} = \frac{5}{3}$$

**Step 2:** Apply to other sides
$$YZ = QR \times \frac{5}{3} = 9 \times \frac{5}{3} = 15 \text{ cm}$$

$$XZ = PR \times \frac{5}{3} = 12 \times \frac{5}{3} = 20 \text{ cm}$$

---

## 5.4 Proving Triangles Similar

### 📐 Worked Example 14: Proving Similarity

**Problem:** In the diagram, $AB$ is parallel to $DE$, and $C$ is a point such that lines $CA$ and $CE$ are straight. Prove that triangles $ABC$ and $DEC$ are similar.

**Solution:**

| Statement | Reason |
|-----------|--------|
| $\angle BAC = \angle DEC$ | Alternate angles (AB || DE) |
| $\angle ABC = \angle EDC$ | Alternate angles (AB || DE) |
| $\angle ACB = \angle DCE$ | Vertically opposite angles |
| Therefore $\triangle ABC \sim \triangle DEC$ | AAA similarity |

---

## 5.5 Area of Similar Shapes

### KEY FORMULA

$$\frac{\text{Area of shape A}}{\text{Area of shape B}} = \left(\frac{\text{Length of A}}{\text{Length of B}}\right)^2 = k^2$$

---

### 📐 Worked Example 15: Area Ratio

**Problem:** Two similar triangles have corresponding sides in the ratio 3:5. The smaller triangle has area 27 cm². Find the area of the larger triangle.

**Solution:**
$$\frac{\text{Area}_{\text{large}}}{\text{Area}_{\text{small}}} = \left(\frac{5}{3}\right)^2 = \frac{25}{9}$$

$$\text{Area}_{\text{large}} = 27 \times \frac{25}{9} = 3 \times 25 = 75 \text{ cm}^2$$

---

### 📐 Worked Example 16: Finding Scale Factor from Area

**Problem:** The areas of two similar shapes are 16 cm² and 81 cm². Find the ratio of their corresponding lengths.

**Solution:**
$$\frac{\text{Area}_1}{\text{Area}_2} = \frac{16}{81}$$

$$\frac{\text{Length}_1}{\text{Length}_2} = \sqrt{\frac{16}{81}} = \frac{4}{9}$$

---

## 5.6 Applications of Similar Triangles

### Problem Type: Shadow/Height Problems

**Problem:** A 2 m vertical stick casts a shadow 3 m long. At the same time, a building casts a shadow 45 m long. Find the height of the building.

**Solution:**
The triangles formed by stick/shadow and building/shadow are similar (same sun angle).

$$\frac{\text{Height of building}}{\text{Height of stick}} = \frac{\text{Shadow of building}}{\text{Shadow of stick}}$$

$$\frac{h}{2} = \frac{45}{3}$$

$$h = 2 \times 15 = 30 \text{ m}$$

---

### Problem Type: Reflection Problems

**Problem:** A person 1.6 m tall stands 4 m from a mirror. They can see the top of a building in the mirror. The mirror is 40 m from the building. Find the height of the building.

**Solution:**
Using similar triangles formed by the reflections:
$$\frac{\text{Building height}}{\text{Person height}} = \frac{\text{Distance to mirror (building)}}{\text{Distance to mirror (person)}}$$
$$\frac{h}{1.6} = \frac{40}{4}$$
$$h = 1.6 \times 10 = 16 \text{ m}$$

## 6.3 Common Mistakes to Avoid

###  TOP 10 MISTAKES

1. **Forgetting to subtract when finding missing angles**
   - Always check: $x = 180° - \text{(sum of known angles)}$

2. **Confusing interior and exterior angles**
   - Interior + Exterior = 180° (they are supplementary)

3. **Incorrect scale factor application**
   - Area ratio = (length ratio)²

4. **Not showing construction arcs**
   - Examiners need to see all construction lines

5. **Mixing up quadrilateral properties**
   - Memorize the properties table

6. **Using wrong angle rule**
   - Always identify the configuration first

7. **Not giving reasons in proofs**
   - Every statement needs a justification

8. **Rounding too early in calculations**
   - Keep exact values until final answer

9. **Forgetting units in answers**
   - Always include ° for angles, cm for lengths

10. **Not checking if answer makes sense**
    - Angles in triangle should sum to 180°
    - Polygon angles should be consistent

---

## 6.4 How to Gain Full Marks

### Method Marks (M)
- Show the formula or rule used
- Show substitution into formula
- Show intermediate steps

### Accuracy Marks (A)
- Get the correct numerical answer
- Include correct units
- Give answer to required accuracy

### Reason Marks (R)
- State the theorem or rule used
- Explain why a rule applies
- Give geometric reasons

---

### Example: Scoring Full Marks

**Question:** Find angle $x$ in the triangle with angles $47°$ and $x$ and exterior angle $118°$. [3 marks]

**Full marks solution:**
$$\text{Exterior angle} = \text{sum of interior opposite angles (1M)}$$
$$118° = 47° + x \quad \text{(1M)}$$
$$x = 118° - 47° = 71° \quad \text{(1A)}$$
