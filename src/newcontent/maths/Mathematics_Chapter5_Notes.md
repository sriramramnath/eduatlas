## 1. Using Calculators

### 1.1 Introduction to Calculator Use

The scientific calculator is an essential tool in IGCSE Mathematics. However, **knowing how to use it correctly is just as important as knowing which buttons to press**. Many students lose marks due to calculator errors that could easily be avoided.

> ** KEY EXAM INSIGHT:** Always check your calculator is in the correct mode (DEG for degrees, not RAD) and that you understand the order of operations your calculator follows.

### 1.2 Order of Operations (BIDMAS/BODMAS)

Calculators follow the mathematical order of operations:

| Order | Operation | Example |
|-------|-----------|---------|
| 1st | **B**rackets | $(3 + 2) \times 4 = 20$ |
| 2nd | **I**ndices (Powers) | $3^2 = 9$ |
| 3rd | **D**ivision | $12 \div 4 = 3$ |
| 4th | **M**ultiplication | $3 \times 4 = 12$ |
| 5th | **A**ddition | $5 + 3 = 8$ |
| 6th | **S**ubtraction | $10 - 4 = 6$ |

#### Worked Example 1:
Calculate: $3 + 4 \times 2 - 6 \div 2$

**Solution:**
$$= 3 + (4 \times 2) - (6 \div 2)$$
$$= 3 + 8 - 3$$
$$= 8$$

> ** TIP:** Division and multiplication have equal priority (work left to right). Addition and subtraction have equal priority (work left to right).

### 1.3 Essential Calculator Functions

#### A. Standard Arithmetic Operations

| Function | Calculator Key | Example |
|----------|---------------|---------|
| Addition | $+$ | $5 + 3 = 8$ |
| Subtraction | $-$ | $10 - 4 = 6$ |
| Multiplication | $\times$ | $7 \times 6 = 42$ |
| Division | $\div$ | $15 \div 3 = 5$ |

#### B. Powers and Roots

| Function | Calculator Key | Example | Result |
|----------|---------------|---------|--------|
| Square | $x^2$ | $7^2$ | $49$ |
| Cube | $x^3$ (or shift function) | $4^3$ | $64$ |
| Any power | $x^y$ or $\wedge$ | $2^5$ | $32$ |
| Square root | $\sqrt{}$ | $\sqrt{144}$ | $12$ |
| Cube root | $\sqrt[3]{}$ | $\sqrt[3]{27}$ | $3$ |
| Any root | shift + $x^y$ | $\sqrt[4]{81}$ | $3$ |

#### Worked Example 2:
Calculate $\sqrt{256} + \sqrt[3]{125}$

**Solution:**
$$\sqrt{256} + \sqrt[3]{125} = 16 + 5 = 21$$

#### C. Fractions

| Function | Calculator Key | Notes |
|----------|---------------|-------|
| Enter fraction | $a \frac{b}{c}$ or dedicated fraction key | Use arrow keys to navigate |
| Mixed numbers | Usually shift + fraction key | Enter whole, then numerator and denominator |
| Improper ↔ Mixed | Often a conversion key | Check your model |

#### Worked Example 3:
Calculate $\frac{3}{4} + \frac{5}{6}$

**Solution:**
Using the fraction key:
1. Press fraction key, enter 3, arrow down, enter 4
2. Press $+$
3. Press fraction key, enter 5, arrow down, enter 6
4. Press $=$

Result: $\frac{19}{12}$ or $1\frac{7}{12}$

**Manual verification:** $\frac{3}{4} + \frac{5}{6} = \frac{9}{12} + \frac{10}{12} = \frac{19}{12}$

#### D. Trigonometric Functions

| Function | Calculator Key | Important |
|----------|---------------|-----------|
| Sine | $\sin$ | Ensure DEG mode for degrees |
| Cosine | $\cos$ | Or RAD for radians (rarely used in IGCSE) |
| Tangent | $\tan$ | |
| Inverse ($\sin^{-1}$, etc.) | shift + function | For finding angles |

> ** COMMON MISTAKE:** Forgetting to check if the calculator is in DEG (degrees) mode. If in RAD mode, all trigonometric answers will be incorrect.

#### Worked Example 4:
Find the value of $\sin(30°) + \cos(60°)$

**Solution:**
1. Check calculator is in DEG mode
2. Press $\sin$, enter 30, press $)$ or $=$
3. Press $+$
4. Press $\cos$, enter 60, press $)$ or $=$
5. Press $=$

$$\sin(30°) + \cos(60°) = 0.5 + 0.5 = 1$$

#### Worked Example 5:
Find angle $x$ where $\sin(x) = 0.5$

**Solution:**
$$x = \sin^{-1}(0.5) = 30°$$

#### E. Scientific Notation

| Function | Calculator Key | Example |
|----------|---------------|---------|
| Enter scientific notation | $\times 10^x$ or EXP | $3.5 \times 10^6$ |
| Standard form display | Often automatic | May show as $3.5 \times 10^6$ or $3.5E6$ |

#### Worked Example 6:
Calculate $(4.2 \times 10^5) \times (3.0 \times 10^{-3})$

**Solution:**
$$= 4.2 \times 3.0 \times 10^{5 + (-3)}$$
$$= 12.6 \times 10^2$$
$$= 1.26 \times 10^3$$

Calculator input: 4.2 EXP 5 × 3.0 EXP (-) 3 =

### 1.4 Memory Functions

| Function | Purpose |
|----------|---------|
| M+ | Add to memory |
| M- | Subtract from memory |
| MR / MRC | Recall from memory |
| MC | Clear memory |
| STO | Store value |
| RCL | Recall stored value |

> ** TIP:** Use memory to store intermediate results in multi-step calculations to avoid rounding errors.

### 1.5 Common Calculator Errors

| Error | Cause | Solution |
|-------|-------|----------|
| **Syntax Error** | Incorrect input format | Check brackets and operators |
| **Math Error** | Division by zero, $\sqrt{-ve}$ | Check calculation logic |
| **Stack Error** | Too many pending operations | Simplify in stages |
| **Incorrect answer** | Wrong mode (RAD vs DEG) | Reset to DEG mode |

### 1.6 Using the ANS Key

The ANS key recalls the previous answer, useful for:
- Multi-step calculations
- Avoiding transcription errors
- Maintaining precision

#### Worked Example 7:
Find $\sqrt{\frac{15 + 7}{4}}$

**Solution:**
Method 1 (Direct input):
$$\sqrt{\frac{15 + 7}{4}} = \sqrt{\frac{22}{4}} = \sqrt{5.5} \approx 2.35$$

Method 2 (Using ANS):
1. Calculate $15 + 7 = 22$
2. Divide by 4: ANS ÷ 4 = 5.5
3. Take square root: $\sqrt{ANS} \approx 2.35$

### 1.7 Rounding and Accuracy

> ** CRITICAL EXAM RULE:** Always give your answer to an appropriate degree of accuracy (usually 3 significant figures unless specified otherwise). Show more decimal places in working.

| Instruction | Meaning |
|-------------|---------|
| "To 2 d.p." | Two decimal places |
| "To 3 s.f." | Three significant figures |
| "Exact value" | Do not round |
| "Appropriate accuracy" | Usually 3 s.f. |

## 2. Estimating Answers

### 2.1 Why Estimate?

Estimation is a crucial skill for:
- **Checking reasonableness** of calculator answers
- **Quick mental calculations**
- **Catching calculator errors**
- **Real-world problem solving**

> ** EXAM INSIGHT:** Estimation questions are common in exams. They test whether you understand the magnitude of numbers and operations.

### 2.2 Rounding for Estimation

The key principle: **Round each number to ONE significant figure** (unless the context suggests otherwise).

#### Worked Example 8:
Estimate $\frac{29.7 \times 51.2}{4.1}$

**Solution:**
Round each number to 1 significant figure:
$$\frac{30 \times 50}{4} = \frac{1500}{4} = 375$$

**Actual calculation:** $\frac{29.7 \times 51.2}{4.1} \approx 370.5$

> **✅ Good estimate:** The estimate (375) is close to the actual value (370.5), confirming our calculation is reasonable.

#### Worked Example 9:
Estimate $\sqrt{82.3} \times 19.8$

**Solution:**
$$\sqrt{80} \times 20 \approx 9 \times 20 = 180$$

**Actual calculation:** $\sqrt{82.3} \times 19.8 \approx 9.07 \times 19.8 \approx 179.6$

### 2.3 Special Cases in Estimation

#### A. Division with Awkward Numbers

When dividing, round to make the division easier:

**Example:** Estimate $\frac{387}{42}$

Round to compatible numbers:
$$\frac{400}{40} = 10$$

#### B. Square Root Estimation

Know perfect squares near common values:

| Number | $\sqrt{n}$ |
|--------|-----------|
| 25 | 5 |
| 36 | 6 |
| 49 | 7 |
| 64 | 8 |
| 81 | 9 |
| 100 | 10 |
| 121 | 11 |
| 144 | 12 |

**Example:** Estimate $\sqrt{70}$

$$64 < 70 < 81$$
$$\sqrt{70} \approx 8.4 \text{ (actual: } 8.37...)$$

#### C. Percentages

For percentage estimation:
- 10% = divide by 10
- 5% = half of 10%
- 1% = divide by 100
- 50% = divide by 2
- 25% = divide by 4

#### Worked Example 10:
Estimate 17% of £38.50

**Solution:**
Round: 17% ≈ 20%, £38.50 ≈ £40

$$20\% \text{ of } £40 = £8$$

**Actual calculation:** $0.17 \times 38.50 = £6.545$

> **Note:** The estimate is rough but gives the correct order of magnitude.

### 2.4 Estimation for Checking Calculations

Always estimate before or after using a calculator to catch errors.

#### Worked Example 11:
A student calculates $23.4 \times 0.89 = 2082.6$

**Check:** 
$$23.4 \times 0.89 \approx 20 \times 1 = 20$$

**Error found:** The student must have entered $23.4 \times 89$ instead of $23.4 \times 0.89$.

### 2.5 Upper and Lower Bounds in Estimation

When estimating, consider whether rounding up or down gives a better estimate:

| Operation | Strategy |
|-----------|----------|
| Multiplication | Round both numbers in same direction (both up or both down) |
| Division | Round in opposite directions for bounds |
| Addition | Round both in same direction |
| Subtraction | Round in opposite directions |

## 3. Upper and Lower Bounds

### 3.1 Understanding Bounds

When a quantity is measured or rounded, there is always some **uncertainty**. The true value lies within a range bounded by the **lower bound** and **upper bound**.

> **KEY DEFINITION:**
> - **Lower Bound**: The smallest possible value a rounded/measured number could have been.
> - **Upper Bound**: The largest possible value a rounded/measured number could have been.

### 3.2 Bounds for Rounded Numbers

When a number is rounded to a given accuracy, the bounds are found by:

$$\text{Lower Bound} = \text{Given value} - \frac{1}{2} \times \text{Rounding unit}$$

$$\text{Upper Bound} = \text{Given value} + \frac{1}{2} \times \text{Rounding unit}$$

#### Worked Example 12:
A length is recorded as 8 cm to the nearest cm. Find the lower and upper bounds.

**Solution:**
- Rounding unit: 1 cm
- Half the rounding unit: 0.5 cm

**Lower Bound:** $8 - 0.5 = 7.5$ cm

**Upper Bound:** $8 + 0.5 = 8.5$ cm

**The actual length $l$ satisfies:** $7.5 \text{ cm} \leq l < 8.5 \text{ cm}$

> ** IMPORTANT:** The upper bound is **exclusive** (not included). We write $l < 8.5$, not $l \leq 8.5$.

### 3.3 Bounds for Different Rounding Types

| Rounding To | Rounding Unit | Example | Lower Bound | Upper Bound |
|-------------|---------------|---------|-------------|-------------|
| Nearest 10 | 10 | 50 | 45 | 55 |
| Nearest 1 | 1 | 8 | 7.5 | 8.5 |
| Nearest 0.1 (1 d.p.) | 0.1 | 3.4 | 3.35 | 3.45 |
| Nearest 0.01 (2 d.p.) | 0.01 | 5.67 | 5.665 | 5.675 |
| Nearest 100 | 100 | 2300 | 2250 | 2350 |

#### Worked Example 13:
A number is given as 12.3 to 1 decimal place. Find the bounds.

**Solution:**
- Rounding unit: 0.1
- Half the rounding unit: 0.05

**Lower Bound:** $12.3 - 0.05 = 12.25$

**Upper Bound:** $12.3 + 0.05 = 12.35$

**Range:** $12.25 \leq n < 12.35$

### 3.4 Bounds for Significant Figures

For significant figures, first determine the place value of the last significant figure.

| Rounding To | Example | Place Value of Last SF | Bounds |
|-------------|---------|----------------------|--------|
| 1 s.f. | 3000 | 1000 | $2500 \leq n < 3500$ |
| 2 s.f. | 470 | 10 | $465 \leq n < 475$ |
| 2 s.f. | 0.047 | 0.001 | $0.0465 \leq n < 0.0475$ |
| 3 s.f. | 12.8 | 0.1 | $12.75 \leq n < 12.85$ |

#### Worked Example 14:
The population of a town is 48,000 to 2 significant figures. Find the possible range.

**Solution:**
- The last significant figure (8) is in the thousands place
- Rounding unit: 1000
- Half the rounding unit: 500

**Lower Bound:** $48,000 - 500 = 47,500$

**Upper Bound:** $48,000 + 500 = 48,500$

**Range:** $47,500 \leq \text{population} < 48,500$

### 3.5 Bounds for Measured Quantities

When a measurement is given, the precision determines the bounds.

| Measured To | Precision | Bounds |
|-------------|-----------|--------|
| Nearest cm | 1 cm | ±0.5 cm |
| Nearest mm | 1 mm | ±0.5 mm |
| Nearest 0.1 cm | 0.1 cm | ±0.05 cm |

#### Worked Example 15:
The length of a rectangle is measured as 12 cm to the nearest cm, and the width is measured as 5.4 cm to the nearest 0.1 cm.

a) Find the lower and upper bounds of each measurement.

b) Calculate the upper and lower bounds of the perimeter.

**Solution:**

**a) Finding bounds:**

Length (12 cm to nearest cm):
- Lower Bound: $12 - 0.5 = 11.5$ cm
- Upper Bound: $12 + 0.5 = 12.5$ cm

Width (5.4 cm to nearest 0.1 cm):
- Lower Bound: $5.4 - 0.05 = 5.35$ cm
- Upper Bound: $5.4 + 0.05 = 5.45$ cm

**b) Perimeter bounds:**

Perimeter $= 2 \times \text{length} + 2 \times \text{width}$

**Lower Bound of Perimeter:**
$$P_{min} = 2(11.5) + 2(5.35) = 23 + 10.7 = 33.7 \text{ cm}$$

**Upper Bound of Perimeter:**
$$P_{max} = 2(12.5) + 2(5.45) = 25 + 10.9 = 35.9 \text{ cm}$$

### 3.6 Bounds with Multiplication and Division

> **CRITICAL RULE:**
> - For **Lower Bound** of a product: multiply the lower bounds
> - For **Upper Bound** of a product: multiply the upper bounds
> - For **Lower Bound** of a quotient: divide lower bound by upper bound
> - For **Upper Bound** of a quotient: divide upper bound by lower bound

| Operation | Lower Bound | Upper Bound |
|-----------|-------------|-------------|
| $A + B$ | $LB_A + LB_B$ | $UB_A + UB_B$ |
| $A - B$ | $LB_A - UB_B$ | $UB_A - LB_B$ |
| $A \times B$ | $LB_A \times LB_B$ | $UB_A \times UB_B$ |
| $A \div B$ | $LB_A \div UB_B$ | $UB_A \div LB_B$ |

#### Worked Example 16:
The length of a rectangle is 8 cm (to nearest cm) and width is 5 cm (to nearest cm).

Find the upper and lower bounds of:
a) The area
b) The length of diagonal (use Pythagoras)

**Solution:**

**Finding individual bounds:**
- Length: LB = 7.5 cm, UB = 8.5 cm
- Width: LB = 4.5 cm, UB = 5.5 cm

**a) Area bounds:**

$$A_{min} = 7.5 \times 4.5 = 33.75 \text{ cm}^2$$

$$A_{max} = 8.5 \times 5.5 = 46.75 \text{ cm}^2$$

**b) Diagonal bounds:**

Using Pythagoras: $d = \sqrt{l^2 + w^2}$

**Lower Bound of diagonal:**
$$d_{min} = \sqrt{7.5^2 + 4.5^2} = \sqrt{56.25 + 20.25} = \sqrt{76.5} \approx 8.75 \text{ cm}$$

**Upper Bound of diagonal:**
$$d_{max} = \sqrt{8.5^2 + 5.5^2} = \sqrt{72.25 + 30.25} = \sqrt{102.5} \approx 10.12 \text{ cm}$$

### 3.7 Bounds with Speed, Density, and Other Formulas

When dealing with derived quantities, consider how bounds combine.

#### Worked Example 17:
A car travels 120 km (to nearest km) in 2.5 hours (to nearest 0.1 hour).

Find the upper and lower bounds of the average speed.

**Solution:**

**Finding bounds:**
- Distance: LB = 119.5 km, UB = 120.5 km
- Time: LB = 2.45 h, UB = 2.55 h

**Speed bounds:**

$$\text{Speed} = \frac{\text{Distance}}{\text{Time}}$$

**Maximum Speed** (using max distance, min time):
$$v_{max} = \frac{120.5}{2.45} \approx 49.2 \text{ km/h}$$

**Minimum Speed** (using min distance, max time):
$$v_{min} = \frac{119.5}{2.55} \approx 46.9 \text{ km/h}$$

### 3.8 Error Intervals

An **error interval** expresses the range of possible values for a measured or rounded quantity.

#### Standard Notation:

For a value $x$ rounded to a given accuracy:
$$LB \leq x < UB$$

#### Worked Example 18:
Write the error interval for $n$ where $n = 15$ (to the nearest whole number).

**Solution:**
$$14.5 \leq n < 15.5$$

### 3.9 Percentage Error

The **percentage error** measures the relative uncertainty in a measurement.

$$\text{Percentage Error} = \frac{\text{Maximum possible error}}{\text{Measured value}} \times 100\%$$

The maximum possible error equals half the rounding unit.

#### Worked Example 19:
A length is measured as 25 cm to the nearest cm. Calculate the percentage error.

**Solution:**
- Maximum possible error: 0.5 cm
- Measured value: 25 cm

$$\text{Percentage Error} = \frac{0.5}{25} \times 100\% = 2\%$$

### 3.10 Appropriate Degree of Accuracy

When working with bounds, your final answer should reflect the accuracy of the original measurements.

> ** GUIDELINE:** Your answer cannot be more accurate than the least accurate measurement used.

#### Worked Example 20:
Find the area of a circle with radius 7 cm (to nearest cm). Give your answer to an appropriate degree of accuracy.

**Solution:**

$$A = \pi r^2 = \pi \times 7^2 = 49\pi \approx 153.938 \text{ cm}^2$$

**Bounds consideration:**
- $r_{min} = 6.5$ cm → $A_{min} = \pi \times 6.5^2 \approx 132.7 \text{ cm}^2$
- $r_{max} = 7.5$ cm → $A_{max} = \pi \times 7.5^2 \approx 176.7 \text{ cm}^2$

**Appropriate answer:** $150 \text{ cm}^2$ (2 significant figures)

The actual area could range from about 133 to 177 cm², so giving an answer like 153.938 cm² would be misleadingly precise.

## 4. Exam Focus & Strategy

### 4.1 Frequently Tested Question Types

| Topic | Question Type | Marks | Frequency |
|-------|--------------|-------|-----------|
| Calculator use | Multi-step calculations | 2-4 | High |
| Estimation | Estimate then compare | 2-3 | Medium |
| Bounds | Find LB and UB | 2-3 | High |
| Bounds with operations | Area, speed, density | 3-5 | High |
| Error intervals | Write inequality | 2-3 | Medium |
| Percentage error | Calculate uncertainty | 2-3 | Medium |

### 4.2 Command Words

| Command Word | What It Means | Example |
|--------------|---------------|---------|
| **Calculate** | Use a method to find the answer | "Calculate the upper bound..." |
| **Estimate** | Find an approximate value | "Estimate the value of..." |
| **Write down** | No working required | "Write down the lower bound..." |
| **Show that** | Provide clear working | "Show that the maximum speed is..." |
| **Explain** | Give a reason | "Explain why your answer is an overestimate..." |
| **Give** | Provide without working (often from calculator) | "Give your answer to 3 s.f." |

### 4.3 Step-by-Step Problem-Solving Patterns

#### Pattern 1: Finding Bounds
1. Identify the rounding unit (nearest what?)
2. Calculate half the rounding unit
3. Subtract for lower bound, add for upper bound
4. Express as inequality if required

#### Pattern 2: Combined Bounds (Addition/Subtraction)
1. Find bounds for each quantity separately
2. For addition: add LBs for minimum, add UBs for maximum
3. For subtraction: LB - UB gives minimum, UB - LB gives maximum

#### Pattern 3: Combined Bounds (Multiplication/Division)
1. Find bounds for each quantity separately
2. For multiplication: LB × LB for minimum, UB × UB for maximum
3. For division: LB ÷ UB for minimum, UB ÷ LB for maximum

### 4.4 How to Gain Full Marks

| Question Type | Full Marks Criteria |
|---------------|---------------------|
| Bounds calculation | Correct bounds shown with method |
| Estimation | Sensible rounding + clear working |
| Multi-step bounds | All intermediate bounds calculated |
| Area/volume with bounds | Correct formula + correct combination of bounds |
| Error interval | Correct inequality notation |
| Reasonableness check | Comparison made + conclusion |

### 4.5 Common Mistakes to Avoid

| ❌ Mistake | ✅ Correct Approach |
|-----------|---------------------|
| Using wrong rounding unit | Check: nearest 1, nearest 0.1, etc. |
| Forgetting upper bound is exclusive | Write $x < UB$, not $x \leq UB$ |
| Wrong bounds for division | Remember: LB ÷ UB and UB ÷ LB |
| Calculator in wrong mode | Always check DEG for angles |
| Over-rounding intermediate answers | Keep more digits in working |
| Wrong formula combination | Use table of rules for operations |
| Not showing working | Always show method clearly |

### 4.6 Key Formulas to Memorize

$$\text{Lower Bound} = \text{Given value} - \frac{\text{Rounding unit}}{2}$$

$$\text{Upper Bound} = \text{Given value} + \frac{\text{Rounding unit}}{2}$$

$$\text{Percentage Error} = \frac{\text{Half rounding unit}}{\text{Measured value}} \times 100\%$$

### 4.7 Time-Saving Tips

1. **For estimation questions:** Don't use a calculator – mental math is faster
2. **For bounds:** Identify rounding unit immediately to avoid confusion
3. **For multi-step:** Set up a table of bounds before combining them
4. **For checking:** Always estimate first to catch obvious errors
