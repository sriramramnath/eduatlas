# PART A: PYTHAGORAS' THEOREM

## A1. Core Formula

$$\boxed{a^2 + b^2 = c^2}$$

**Where:** $c$ = hypotenuse (longest side, opposite right angle)

---

## A2. Quick Rules

| Finding | Formula |
|---------|---------|
| Hypotenuse | $c = \sqrt{a^2 + b^2}$ |
| Shorter side | $a = \sqrt{c^2 - b^2}$ |

---

## A3. Key Applications

| Application | Formula/Method |
|-------------|----------------|
| Rectangle diagonal | $d = \sqrt{l^2 + w^2}$ |
| Square diagonal | $d = s\sqrt{2}$ |
| Cuboid space diagonal | $d = \sqrt{l^2 + w^2 + h^2}$ |
| Distance between points | $d = \sqrt{(x_2-x_1)^2 + (y_2-y_1)^2}$ |
| Height of isosceles triangle | Draw altitude → two right triangles |

---

## A4. Pythagorean Triples (MEMORIZE!)

| Triple | Multiples |
|--------|-----------|
| $(3, 4, 5)$ | $(6, 8, 10)$, $(9, 12, 15)$, $(15, 20, 25)$ |
| $(5, 12, 13)$ | $(10, 24, 26)$ |
| $(8, 15, 17)$ | — |
| $(7, 24, 25)$ | — |

---

## A5. Checking for Right Angle

- **Right-angled:** $a^2 + b^2 = c^2$
- **Acute-angled:** $a^2 + b^2 > c^2$
- **Obtuse-angled:** $a^2 + b^2 < c^2$

---

## A6. Common Mistakes

| Mistake | Correction |
|---------|------------|
| Wrong side as hypotenuse | Hypotenuse is opposite right angle |
| Adding for shorter side | Subtract: $a^2 = c^2 - b^2$ |
| Forgetting √ | $c^2 = 169 \Rightarrow c = 13$ |
| Not checking right angle | Pythagoras ONLY for right triangles |

# PART B: CIRCLE THEOREMS

## B1. Seven Key Theorems

### Theorem 1: Angle at Center vs Circumference

$$\boxed{\angle_{center} = 2 \times \angle_{circumference}}$$

- Same arc → Center angle is double circumference angle

---

### Theorem 2: Angle in a Semicircle

$$\boxed{\angle = 90°}$$

- Angle subtended by diameter = 90°
- If you see a diameter → look for right angles

---

### Theorem 3: Angles in Same Segment

$$\boxed{\angle_1 = \angle_2}$$

- Same arc/chord → Equal angles at circumference
- Look for angles standing on same arc

---

### Theorem 4: Cyclic Quadrilateral

$$\boxed{\angle A + \angle C = 180°}$$
$$\boxed{\angle B + \angle D = 180°}$$

- Opposite angles sum to 180°
- All four vertices must be on circle

---

### Theorem 5: Tangent and Radius

$$\boxed{\angle = 90°}$$

- Tangent ⊥ radius at point of contact
- Two tangents from external point are equal

---

### Theorem 6: Alternate Segment Theorem

$$\boxed{\angle_{tangent-chord} = \angle_{alternate\ segment}}$$

- Angle between tangent and chord = angle in opposite segment

---

### Theorem 7: Chord from Center

$$\boxed{\text{Perpendicular from center bisects chord}}$$

- If $OM \perp AB$ → $AM = MB$
- Creates right triangle: use Pythagoras

---

## B3. Problem-Solving Patterns

### Pattern 1: Finding Angles
1. Identify points on circle
2. Check for: diameter, tangent, cyclic quad
3. Apply theorem
4. Use angle sum if needed (triangle = 180°)

### Pattern 2: Finding Lengths
1. Draw radius to tangent point (→ 90°)
2. Draw perpendicular to chord (→ bisects)
3. Use Pythagoras in right triangle

### Pattern 3: Proofs
1. State facts (radii equal, tangents ⊥ radius)
2. Show triangles congruent (RHS, SSS, etc.)
3. Draw conclusion
