# 1. STATISTICAL INVESTIGATION

## The Statistical Enquiry Cycle

| Stage | Key Actions |
|-------|-------------|
| **Plan** | Formulate hypothesis, decide data needed, choose method |
| **Collect** | Design collection method, gather data systematically |
| **Process** | Organise in tables, calculate statistics, draw diagrams |
| **Discuss** | Interpret results, draw conclusions, evaluate limitations |

---

## Types of Data - Quick Reference

| Type | Definition | Examples |
|------|------------|----------|
| **Primary** | Collected firsthand | Surveys, experiments |
| **Secondary** | Pre-collected by others | Government statistics, internet |
| **Qualitative** | Descriptive (non-numerical) | Eye colour, opinions |
| **Quantitative** | Numerical | Height, test scores |
| **Discrete** | Specific values only | Number of children, goals |
| **Continuous** | Any value in range | Height, weight, time |

---

## Sampling Methods

| Method | Key Feature | Bias Risk |
|--------|-------------|-----------|
| **Simple random** | Equal chance for all | Low |
| **Systematic** | Every nth person | Low |
| **Stratified** | Proportional from subgroups | Low |
| **Quota** | Fill quotas from groups | Medium |
| **Convenience** | Whoever is available | High |

### Stratified Sampling Formula

$$\boxed{\text{Sample from stratum} = \frac{\text{Stratum size}}{\text{Total population}} \times \text{Sample size}}$$

# 2. PRESENTING DATA

## Chart Selection Guide

| Data Type | Best Charts |
|-----------|-------------|
| Qualitative | Bar chart, Pie chart |
| Discrete quantitative | Bar chart, Stem & leaf |
| Continuous grouped | Histogram, Frequency polygon, Cumulative frequency |
| Comparing groups | Composite bar, Back-to-back stem & leaf |

---

## Key Formulas for Diagrams

### Pie Chart Angle
$$\boxed{\text{Angle} = \frac{\text{Frequency}}{\text{Total}} \times 360°}$$

### Frequency Density (Histograms)
$$\boxed{\text{Frequency Density} = \frac{\text{Frequency}}{\text{Class Width}}}$$

> **Remember:** Histogram area = frequency

---

## Cumulative Frequency

| Statistic | Position | Read from curve at y = |
|-----------|----------|------------------------|
| Median (Q₂) | n/2 | n/2 |
| Lower Quartile (Q₁) | n/4 | n/4 |
| Upper Quartile (Q₃) | 3n/4 | 3n/4 |

### Interquartile Range
$$\boxed{\text{IQR} = Q_3 - Q_1}$$

---

## Stem and Leaf Diagram

**Construction:**
1. Stem = leading digit(s)
2. Leaf = final digit
3. Leaves in order
4. Include a **Key**

**Advantages:**
- Retains original values
- Easy to find median (middle value)
- Shows distribution shape

# 3. MISLEADING DATA

## Common Misleading Techniques

| Technique | How it Misleads |
|-----------|-----------------|
| Truncated y-axis | Exaggerates small differences |
| Inconsistent scales | Distorts comparisons |
| 3D effects | Makes some areas appear larger |
| Wrong average | Mean distorted by outliers |
| Small samples | Unreliable conclusions |
| Percentage confusion | Mixing % points with % change |

## Questions to Ask

- **Who** collected the data? (Bias?)
- **How** was the sample selected?
- **What** is missing from the presentation?
- **Which** average was used?

# 4. AVERAGES

## The Three Averages - At a Glance

| Average | Formula/Method | When to Use |
|---------|----------------|-------------|
| **Mean** | $\bar{x} = \frac{\Sigma x}{n}$ or $\frac{\Sigma fx}{\Sigma f}$ | No outliers, quantitative data |
| **Median** | Middle value (position $\frac{n+1}{2}$) | Outliers present, skewed data |
| **Mode** | Most frequent value | Categorical data, "most popular" |

---

## Mean Calculations

### Raw Data
$$\bar{x} = \frac{x_1 + x_2 + ... + x_n}{n}$$

### Frequency Table
$$\bar{x} = \frac{\Sigma fx}{\Sigma f}$$

### Grouped Data (Estimated)
$$\bar{x} = \frac{\Sigma f \times \text{midpoint}}{\Sigma f}$$

---

## Median - Quick Method

| n is odd | n is even |
|----------|-----------|
| Position: $\frac{n+1}{2}$ | Average of positions $\frac{n}{2}$ and $\frac{n}{2}+1$ |
| Example: n=7, median = 4th value | Example: n=8, median = average of 4th and 5th |

### From Grouped Data (Interpolation)
$$\boxed{\text{Median} = L + \left(\frac{\frac{n}{2} - F}{f}\right) \times w}$$

Where: L = lower boundary, F = cum. freq. before median class, f = freq. of median class, w = class width

---

## Mode Rules

- **Unimodal:** One mode
- **Bimodal:** Two modes
- **No mode:** All equal frequency
- **Grouped data:** Give **modal class**, not single value

---

## Range and Spread

| Measure | Formula | Use When |
|---------|---------|----------|
| **Range** | Max − Min | Quick spread measure |
| **IQR** | Q₃ − Q₁ | Outliers present |

> **IQR advantage:** Not affected by extreme values

---

## Choosing the Right Average

| Situation | Best Choice | Reason |
|-----------|-------------|--------|
| Symmetric distribution, no outliers | Mean | Represents "typical" well |
| Skewed distribution, outliers | Median | Not affected by extremes |
| Categorical data | Mode | Only option |
| Stocking products | Mode | Most popular size/item |
| Salary comparison | Median | Mean skewed by high earners |

## Common Errors to Avoid

❌ Gaps in histograms (continuous data = no gaps)
❌ Forgetting frequency in $\frac{\Sigma fx}{\Sigma f}$
❌ Using class boundaries instead of midpoints for grouped mean
❌ Confusing percentage points with percentage change
❌ Not checking pie chart angles sum to 360°
❌ Using mean when outliers present
❌ Not interpreting in context
