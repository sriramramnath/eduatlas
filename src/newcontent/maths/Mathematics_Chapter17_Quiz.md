**Q1.** Solve $x^2 - 7x + 12 = 0$

- A) $x = 3$ or $x = 4$
- B) $x = -3$ or $x = -4$
- C) $x = 3$ or $x = -4$
- D) $x = -3$ or $x = 4$

**Answer: A**

**Explanation:**
Find two numbers that multiply to 12 and add to $-7$.
$$(-3) \times (-4) = 12, \quad (-3) + (-4) = -7$$
$$(x - 3)(x - 4) = 0$$
$$x = 3 \text{ or } x = 4$$

**Q2.** Factorise $x^2 - 9x + 20$

- A) $(x - 4)(x - 5)$
- B) $(x + 4)(x + 5)$
- C) $(x - 4)(x + 5)$
- D) $(x + 4)(x - 5)$

**Answer: A**

**Explanation:**
Find two numbers that multiply to 20 and add to $-9$.
$$(-4) \times (-5) = 20, \quad (-4) + (-5) = -9$$
$$x^2 - 9x + 20 = (x - 4)(x - 5)$$

**Q3.** What are the solutions of $x^2 = 49$?

- A) $x = 7$
- B) $x = -7$
- C) $x = 7$ or $x = -7$
- D) $x = \sqrt{49}$

**Answer: C**

**Explanation:**
$$x^2 = 49$$
$$x = \pm\sqrt{49} = \pm 7$$
Both $x = 7$ and $x = -7$ satisfy the equation.

**Q4.** Solve $2x^2 + 5x + 2 = 0$

- A) $x = -2$ or $x = -\frac{1}{2}$
- B) $x = 2$ or $x = \frac{1}{2}$
- C) $x = -2$ or $x = \frac{1}{2}$
- D) $x = 2$ or $x = -\frac{1}{2}$

**Answer: A**

**Explanation:**
Using AC method: $ac = 4$
Find numbers with product 4 and sum 5: $1$ and $4$
$$2x^2 + x + 4x + 2 = 0$$
$$x(2x + 1) + 2(2x + 1) = 0$$
$$(x + 2)(2x + 1) = 0$$
$$x = -2 \text{ or } x = -\frac{1}{2}$$

**Q6.** What is the turning point of $y = x^2 - 6x + 5$?

**Q7.** Using the quadratic formula, solve $x^2 - 4x - 3 = 0$

- A) $x = 2 \pm \sqrt{7}$
- B) $x = -2 \pm \sqrt{7}$
- C) $x = 4 \pm \sqrt{7}$
- D) $x = 2 \pm \sqrt{3}$

**Answer: A**

**Explanation:**
$a = 1, b = -4, c = -3$
$$\Delta = 16 - 4(1)(-3) = 16 + 12 = 28$$
$$x = \frac{4 \pm \sqrt{28}}{2} = \frac{4 \pm 2\sqrt{7}}{2} = 2 \pm \sqrt{7}$$

**Q8.** The discriminant of $x^2 + 6x + 9 = 0$ is:

- A) 0
- B) 36
- C) 72
- D) -36

**Answer: A**

**Explanation:**
$$\Delta = b^2 - 4ac = 6^2 - 4(1)(9) = 36 - 36 = 0$$
This means the equation has one repeated root.

**Q9.** Solve the inequality $x^2 - 4 < 0$

- A) $-2 < x < 2$
- B) $x < -2$ or $x > 2$
- C) $x < 2$
- D) $x > -2$

**Answer: A**

**Explanation:**
$$x^2 - 4 = (x - 2)(x + 2) < 0$$
Critical values: $x = 2$ and $x = -2$
Since $a = 1 > 0$, the parabola opens upward, so negative between roots.
Solution: $-2 < x < 2$

- D) $x \leq -2$ or $x \geq -3$

**Explanation:**
$$x^2 - 5x + 6 = (x - 2)(x - 3) \geq 0$$
Critical values: $x = 2$ and $x = 3$
Since $a = 1 > 0$, the parabola opens upward, so $\geq 0$ outside roots.
Solution: $x \leq 2$ or $x \geq 3$

**Q11.** How many real solutions does $x^2 + 2x + 5 = 0$ have?

## 📝 SECTION B: TRUE OR FALSE

**Instructions:** State whether each statement is TRUE or FALSE. If FALSE, provide the correction.

---

**Q1.** If $x^2 = 25$, then $x = 5$.

**Answer: FALSE**

**Correction:** $x = \pm 5$

When taking the square root of both sides, we must include both positive and negative solutions.

---

**Q2.** A quadratic equation always has two solutions.

**Answer: FALSE**

**Correction:** A quadratic equation can have 0, 1, or 2 real solutions.

Examples:
- $x^2 - 1 = 0$ has two solutions ($x = \pm 1$)
- $x^2 = 0$ has one solution ($x = 0$)
- $x^2 + 1 = 0$ has no real solutions

---

**Q3.** The discriminant of $x^2 + 3x + 2 = 0$ is positive.

**Answer: TRUE**

**Explanation:**
$$\Delta = 3^2 - 4(1)(2) = 9 - 8 = 1 > 0$$
This confirms two distinct real solutions.

---

**Q4.** For the equation $ax^2 + bx + c = 0$, the sum of the roots equals $-\frac{b}{a}$.

**Answer: TRUE**

**Explanation:**
If the roots are $\alpha$ and $\beta$, then $(x - \alpha)(x - \beta) = x^2 - (\alpha + \beta)x + \alpha\beta$
Comparing with $x^2 + \frac{b}{a}x + \frac{c}{a}$:
$$\alpha + \beta = -\frac{b}{a}$$

---

**Q5.** The turning point of $y = (x - 3)^2 + 4$ is at $(3, 4)$.

**Answer: TRUE**

**Explanation:**
In completed square form $y = a(x - h)^2 + k$, the turning point is $(h, k)$.
Here $h = 3$ and $k = 4$, so turning point is $(3, 4)$.

---

**Q6.** If $x^2 - 9 = 0$, then $x = 3$.

**Answer: FALSE**

**Correction:** $x = 3$ or $x = -3$

$$x^2 - 9 = (x - 3)(x + 3) = 0$$
$$x = 3 \text{ or } x = -3$$

---

**Q7.** The quadratic formula can be used to solve any quadratic equation.

**Answer: TRUE**

**Explanation:**
The quadratic formula $x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$ always gives the solutions (if they exist) for any quadratic equation $ax^2 + bx + c = 0$ with $a \neq 0$.

---

**Q8.** The solution to $x^2 + 4x < 0$ is $-4 < x < 0$.

**Answer: TRUE**

**Explanation:**
$$x^2 + 4x = x(x + 4) < 0$$
Critical values: $x = 0$ and $x = -4$
Since $a = 1 > 0$, the parabola opens upward, so negative between roots.
Solution: $-4 < x < 0$

---

**Q9.** Completing the square on $x^2 + 8x$ gives $(x + 4)^2$.

**Answer: FALSE**

**Correction:** Completing the square on $x^2 + 8x$ gives $(x + 4)^2 - 16$

$$x^2 + 8x = x^2 + 8x + 16 - 16 = (x + 4)^2 - 16$$

---

**Q10.** If $ax^2 + bx + c > 0$ for all values of $x$, then $a > 0$ and $\Delta < 0$.

**Answer: TRUE**

**Explanation:**
For the quadratic to be always positive:
- $a > 0$ (opens upward)
- $\Delta < 0$ (no real roots, so doesn't cross x-axis)

This means the parabola sits entirely above the x-axis.

---

**Q11.** The equation $x^2 + 2x + 1 = 0$ has exactly one solution.

**Answer: TRUE**

**Explanation:**
$$x^2 + 2x + 1 = (x + 1)^2 = 0$$
$$x = -1 \text{ (repeated root)}$$
The discriminant $\Delta = 4 - 4(1)(1) = 0$, confirming one repeated root.

---

**Q12.** Dividing both sides of $x^2 = 3x$ by $x$ gives $x = 3$, which is the complete solution.

**Answer: FALSE**

**Correction:** Dividing by $x$ loses the solution $x = 0$. The correct method is:

$$x^2 - 3x = 0$$
$$x(x - 3) = 0$$
$$x = 0 \text{ or } x = 3$$

### Question 1 (2 marks)

Solve $x^2 + 3x - 10 = 0$

**Solution:**

**[M1]** Find two numbers that multiply to $-10$ and add to $3$:
$$5 \times (-2) = -10, \quad 5 + (-2) = 3$$

**[A1]** Factorise and solve:
$$(x + 5)(x - 2) = 0$$
$$x = -5 \text{ or } x = 2$$

**Answer: $x = -5$ or $x = 2$**

### Question 2 (2 marks)

Solve $x^2 - 36 = 0$

**Solution:**

**[M1]** Recognise difference of two squares:
$$(x - 6)(x + 6) = 0$$

**[A1]** Solve:
$$x = 6 \text{ or } x = -6$$

**Answer: $x = 6$ or $x = -6$**

### Question 3 (3 marks)

Solve $2x^2 - 11x + 12 = 0$

**Solution:**

**[M1]** Using AC method: $ac = 24$
Find numbers with product 24 and sum $-11$: $-3$ and $-8$

**[M1]** Split and factorise:
$$2x^2 - 8x - 3x + 12 = 0$$
$$2x(x - 4) - 3(x - 4) = 0$$
$$(2x - 3)(x - 4) = 0$$

**[A1]** Solve:
$$x = \frac{3}{2} \text{ or } x = 4$$

**Answer: $x = \frac{3}{2}$ or $x = 4$**

### Question 4 (3 marks)

Complete the square for $x^2 - 12x + 7$

**Solution:**

**[M1]** Half the coefficient of $x$: $\frac{-12}{2} = -6$

**[M1]** Add and subtract $(-6)^2 = 36$:
$$x^2 - 12x + 36 - 36 + 7$$
$$(x - 6)^2 - 36 + 7$$

**[A1]** Simplify:
$$(x - 6)^2 - 29$$

**Answer: $(x - 6)^2 - 29$**

### Question 5 (3 marks)

Solve $x^2 + 2x - 7 = 0$ using the quadratic formula. Leave your answer in surd form.

**Solution:**

**[M1]** Identify $a = 1$, $b = 2$, $c = -7$ and calculate discriminant:
$$\Delta = 4 - 4(1)(-7) = 4 + 28 = 32$$

**[M1]** Substitute into formula:
$$x = \frac{-2 \pm \sqrt{32}}{2}$$

Simplify surd: $\sqrt{32} = \sqrt{16 \times 2} = 4\sqrt{2}$

**[A1]** 
$$x = \frac{-2 \pm 4\sqrt{2}}{2} = -1 \pm 2\sqrt{2}$$

**Answer: $x = -1 + 2\sqrt{2}$ or $x = -1 - 2\sqrt{2}$**

### Question 6 (4 marks)

$$2(x^2 + 4x + 4 - 4) - 3$$

**[A1]** Complete the square:
$$2(x + 2)^2 - 8 - 3 = 2(x + 2)^2 - 11$$

**(b) [M1]** Since coefficient of $(x + 2)^2$ is positive (2 > 0), this is a minimum.

Minimum occurs when $(x + 2)^2 = 0$, i.e., $x = -2$.

**[A1]** Minimum value = $-11$

**Answer: (a) $2(x + 2)^2 - 11$ (b) Minimum value is $-11$ at $x = -2$**

### Question 9 (5 marks)

Find the value of $k$ for which the equation $x^2 + kx + k + 3 = 0$ has equal roots.

**Solution:**

**[M1]** For equal roots, discriminant must equal zero:
$$\Delta = b^2 - 4ac = 0$$

**[M1]** Identify $a = 1$, $b = k$, $c = k + 3$:
$$k^2 - 4(1)(k + 3) = 0$$

**[M1]** Expand and simplify:
$$k^2 - 4k - 12 = 0$$

**[M1]** Factorise:
$$(k - 6)(k + 2) = 0$$

**[A1]** Solve:
$$k = 6 \text{ or } k = -2$$

**Answer: $k = 6$ or $k = -2$**

### Question 10 (5 marks)

The area of a rectangle is 60 cm². The length is 7 cm more than the width.

(a) Form a quadratic equation in $x$, where $x$ cm is the width. **[2 marks]**

(b) Solve the equation to find the dimensions of the rectangle. **[3 marks]$

**Solution:**

**(a) [M1]** Let width = $x$ cm, length = $(x + 7)$ cm

**[A1]** Area = length × width:
$$x(x + 7) = 60$$
$$x^2 + 7x - 60 = 0$$

**(b) [M1]** Factorise:
$$x^2 + 7x - 60 = 0$$
Find numbers with product $-60$ and sum $7$: $12$ and $-5$
$$(x + 12)(x - 5) = 0$$

**[M1]** Solve:
$$x = -12 \text{ or } x = 5$$

**[A1]** Reject $x = -12$ (width cannot be negative).

Width = 5 cm, Length = 5 + 7 = 12 cm

**Answer: (a) $x^2 + 7x - 60 = 0$ (b) Width = 5 cm, Length = 12 cm**

### Question 11 (6 marks)

A ball is thrown upwards. Its height $h$ metres after $t$ seconds is given by $h = 30t - 5t^2$.

(a) Find the height of the ball after 2 seconds. **[1 mark]**

(b) After how many seconds is the ball at a height of 40 metres? **[3 marks]**

(c) Find the maximum height reached by the ball and the time at which this occurs. **[2 marks]$

**Solution:**

**(a) [B1]** Substitute $t = 2$:
$$h = 30(2) - 5(2)^2 = 60 - 20 = 40 \text{ metres}$$

**(b) [M1]** Set $h = 40$:
$$40 = 30t - 5t^2$$
$$5t^2 - 30t + 40 = 0$$
$$t^2 - 6t + 8 = 0$$

**[M1]** Factorise:
$$(t - 2)(t - 4) = 0$$

**[A1]** Solve:
$$t = 2 \text{ or } t = 4 \text{ seconds}$$

**(c) [M1]** Complete the square:
$$h = -5t^2 + 30t = -5(t^2 - 6t) = -5(t - 3)^2 + 45$$

**[A1]** Maximum when $t = 3$ seconds, maximum height = 45 metres.

**Answer: (a) 40 m (b) $t = 2$ or $t = 4$ seconds (c) Maximum 45 m at $t = 3$ seconds**

**[A1]**

### Question 14 (6 marks)

Prove that the equation $x^2 + 2x + 5 = 0$ has no real solutions.

**Solution:**

**[M1]** Method 1: Use the discriminant
$$\Delta = b^2 - 4ac$$

**[M1]** Identify $a = 1$, $b = 2$, $c = 5$:
$$\Delta = 2^2 - 4(1)(5)$$

**[M1]** Calculate:
$$\Delta = 4 - 20 = -16$$

**[A1]** Since $\Delta = -16 < 0$, there are no real solutions.

**Alternative completion:**

**[M1]** Method 2: Complete the square
$$x^2 + 2x + 5 = (x + 1)^2 - 1 + 5 = (x + 1)^2 + 4$$

**[A1]** Setting equal to zero:
$$(x + 1)^2 + 4 = 0$$
$$(x + 1)^2 = -4$$

Since $(x + 1)^2 \geq 0$ for all real $x$, and $-4 < 0$, there are no real solutions.

**Answer: Discriminant is $-16 < 0$, hence no real solutions exist.**

## 📊 SECTION D: EXAM FOCUS & STRATEGY

### Frequently Tested Question Types

| Type | Description | Typical Marks |
|------|-------------|---------------|
| **Solve by factorising** | Simple or harder quadratics | 2-3 |
| **Complete the square** | Find form, turning point, min/max | 2-4 |
| **Use quadratic formula** | Surd form or decimal | 3-4 |
| **Discriminant problems** | Find $k$, determine nature of roots | 3-5 |
| **Quadratic inequalities** | Find solution region | 3-4 |
| **Word problems** | Area, projectile, profit | 4-6 |
| **Show that...** | Prove properties | 3-5 |

### Typical Problem-Solving Patterns

**Pattern 1: Solving Quadratics**
```
1. Check if factorisable (look for nice numbers)
2. If not, use completing square or formula
3. Always simplify surds fully
4. Check both solutions (use ±)
```

**Pattern 2: Finding Turning Points**
```
1. Complete the square
2. Write in form a(x + p)² + q
3. Turning point is (-p, q)
4. Check if minimum (a > 0) or maximum (a < 0)
```

**Pattern 3: Inequalities**
```
1. Find critical values (roots)
2. Draw sign diagram
3. Identify correct region
4. Check boundaries (≤ vs <)
```

**Pattern 4: Word Problems**
```
1. Define variable clearly
2. Form equation from context
3. Solve with full working
4. Reject invalid solutions
5. State answer with units
```

### Command Word Guidance

| Command | What Examiners Expect |
|---------|----------------------|
| **Solve** | Find all values of $x$ |
| **Factorise** | Product of linear factors only |
| **Complete the square** | Show working, give final form |
| **Find** | Calculate the value |
| **Show that** | Full algebraic proof |
| **Hence** | Use previous result |
| **Give to 2 d.p.** | Round to specified accuracy |
| **Leave in surd form** | Don't decimalise |

### How to Gain Full Marks

**For Solving Questions (3-4 marks):**
1. ✅ Show clear method (factorising or formula)
2. ✅ State both solutions
3. ✅ Simplify surds fully
4. ✅ Check by substitution (optional but good practice)

**For Completing the Square (3-4 marks):**
1. ✅ Show the add/subtract step
2. ✅ Write in correct form $a(x + p)^2 + q$
3. ✅ Simplify all constants
4. ✅ State turning point if asked

**For Inequalities (3-4 marks):**
1. ✅ Find critical values first
2. ✅ Show sign diagram or reasoning
3. ✅ Use correct inequality symbols
4. ✅ Check if boundaries included

**For Word Problems (4-6 marks):**
1. ✅ Define variables clearly
2. ✅ Show equation formation
3. ✅ Solve with full working
4. ✅ Check validity (reject negative lengths, etc.)
5. ✅ Include units in final answer
6. ✅ Write conclusion in words

### Common Pitfalls to Avoid

1. **Forgetting ± when taking square roots**
   - $x^2 = k \Rightarrow x = \pm\sqrt{k}$

2. **Dividing by variable**
   - $x^2 = 3x$ → factorise as $x(x - 3) = 0$, don't divide by $x$

3. **Sign errors in quadratic formula**
   - Remember: $-b$ in numerator, check signs of $a$, $b$, $c$

4. **Wrong inequality region**
   - Always test a point to verify your answer

5. **Accepting invalid solutions**
   - Negative lengths, widths, times don't make sense

6. **Not simplifying surds**
   - $\sqrt{12} = 2\sqrt{3}$, $\sqrt{32} = 4\sqrt{2}$

7. **Forgetting to complete the square properly**
   - Must subtract what you add: $(x + 3)^2 - 9$ for $x^2 + 6x$
