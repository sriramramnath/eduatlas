### Question 1

Find the gradient of the line passing through points $A(2, 5)$ and $B(6, 17)$.

**A)** 2  
**B)** 3  
**C)** 4  
**D)** 12

**Answer: B**

**Solution:**
$$m = \frac{y_2 - y_1}{x_2 - x_1} = \frac{17 - 5}{6 - 2} = \frac{12}{4} = 3$$

---

### Question 2

What is the gradient of the line with equation $y = -2x + 5$?

**A)** 5  
**B)** 2  
**C)** -2  
**D)** -5

**Answer: C**

**Solution:**
Comparing with $y = mx + c$:
- Gradient $m = -2$
- Y-intercept $c = 5$

---

### Question 3

The gradient of a horizontal line is:

**A)** 1  
**B)** 0  
**C)** -1  
**D)** Undefined

**Answer: B**

**Solution:**
A horizontal line has no vertical change (rise = 0), so:
$$m = \frac{0}{\text{run}} = 0$$

---

### Question 4

Find the y-intercept of the line $3y - 6x = 12$.

**A)** 12  
**B)** -6  
**C)** 4  
**D)** -2

**Answer: C**

**Solution:**
Rearrange to $y = mx + c$:
$$3y = 6x + 12$$
$$y = 2x + 4$$

Y-intercept $c = 4$

---

### Question 5

A line has gradient $m = \frac{2}{3}$. What is the gradient of a line perpendicular to it?

**A)** $\frac{3}{2}$  
**B)** $-\frac{3}{2}$  
**C)** $-\frac{2}{3}$  
**D)** $\frac{2}{3}$

**Answer: B**

**Solution:**
For perpendicular lines, $m_1 \times m_2 = -1$
$$m_2 = -\frac{1}{m_1} = -\frac{1}{\frac{2}{3}} = -\frac{3}{2}$$

### Question 6

Which equation represents a line parallel to $y = 4x - 7$?

**A)** $y = -4x + 3$  
**B)** $y = \frac{1}{4}x + 2$  
**C)** $y = 4x + 1$  
**D)** $y = -\frac{1}{4}x - 5$

**Answer: C**

**Solution:**
Parallel lines have the same gradient.
Original line: $m = 4$
Only option C has gradient $m = 4$.

On a distance-time graph, a horizontal line represents:

A horizontal line means distance is not changing over time → the object is stationary.

### Question 9

A conversion graph shows pounds to euros. If £20 = €24, what is the conversion rate?

**A)** €1.2 per £  
**B)** €1.4 per £  
**C)** €0.83 per £  
**D)** €1 per £

**Answer: A**

**Solution:**

### Question 10

The midpoint of the line segment joining $(1, 3)$ and $(5, 9)$ is:

**A)** $(2, 6)$  
**B)** $(3, 6)$  
**C)** $(3, 5)$  
**D)** $(4, 6)$

**Answer: B**

**Solution:**
$$\text{Midpoint} = \left(\frac{x_1 + x_2}{2}, \frac{y_1 + y_2}{2}\right) = \left(\frac{1 + 5}{2}, \frac{3 + 9}{2}\right) = (3, 6)$$

### Question 11

The gradient of the line $x = 5$ is:

**A)** 0  
**B)** 5

**D)** Undefined

**Answer: D**

**Solution:**

### Question 12

Find the equation of the line passing through $(-1, 4)$ and $(3, -4)$.

**A)** $y = 2x + 6$  
**B)** $y = -2x + 2$  
**C)** $y = -x + 3$  
**D)** $y = x + 5$

**Answer: B**

**Solution:**
$$m = \frac{-4 - 4}{3 - (-1)} = \frac{-8}{4} = -2$$

Using point $(-1, 4)$:
$$y = -2x + c$$
$$4 = -2(-1) + c$$
$$4 = 2 + c$$
$$c = 2$$

Answer: $y = -2x + 2$

### Question 1 [3 marks]

Find the gradient of the line joining the points $A(3, -2)$ and $B(-1, 6)$.

**Solution:**

$$m = \frac{y_2 - y_1}{x_2 - x_1}$$

$$m = \frac{6 - (-2)}{-1 - 3} = \frac{6 + 2}{-1 - 3} = \frac{8}{-4} = -2$$

**Answer: Gradient = $-2$** ✓

| Mark | Reasoning |
|------|-----------|
| M1 | Correct formula used or clear rise/run method |
| M1 | Correct substitution with negative numbers |
| A1 | Correct answer of $-2$ |

### Question 3 [5 marks]

Find the equation of the line that passes through the point $(2, 5)$ and is perpendicular to the line $y = \frac{1}{3}x + 4$.

**Solution:**

**Step 1:** Find the gradient of the perpendicular line.

The given line has gradient $m_1 = \frac{1}{3}$.

For perpendicular lines: $m_2 = -\frac{1}{m_1}$

$$m_2 = -\frac{1}{\frac{1}{3}} = -3$$

**Step 2:** Find the equation using point $(2, 5)$.

$$y = -3x + c$$

Substitute $(2, 5)$:
$$5 = -3(2) + c$$
$$5 = -6 + c$$
$$c = 11$$

**Answer: $y = -3x + 11$** ✓

| Mark | Reasoning |
|------|-----------|
| M1 | Correct identification of given gradient |
| M1 | Correct method for perpendicular gradient |
| A1 | Correct perpendicular gradient $m = -3$ |
| M1 | Substitution of point into equation |
| A1 | Correct final equation |

### Question 4 [6 marks]

The distance-time graph shows a car journey.

| Time (hours) | Distance (km) |
|--------------|---------------|
| 0 | 0 |
| 1 | 60 |
| 2 | 120 |
| 3 | 120 |
| 4 | 180 |

**(a)** Calculate the speed during the first 2 hours. [2 marks]

**(b)** What happens between hours 2 and 3? [1 mark]

**(c)** Calculate the speed between hours 3 and 4. [2 marks]

**(d)** Calculate the average speed for the whole journey. [1 mark]

**Solution:**

**(a)**

**Answer: The car is stationary/stopped** ✓

| Mark | Reasoning |
|------|-----------|
| B1 | Correct interpretation of horizontal section |

**(c)**
$$\text{Speed} = \frac{180 - 120}{4 - 3} = \frac{60}{1} = 60 \text{ km/h}$$

**Answer: 60 km/h** ✓

| Mark | Reasoning |
|------|-----------|
| M1 | Correct use of speed formula |
| A1 | Correct answer with units |

**(d)**
$$\text{Average speed} = \frac{\text{Total distance}}{\text{Total time}} = \frac{180}{4} = 45 \text{ km/h}$$

---

##  Typical Problem-Solving Patterns

### Pattern A: "Find the equation of the line..."
1. Identify what's given (two points? gradient + point? parallel to another line?)
2. If two points → find gradient first
3. Find y-intercept by substitution
4. Write final equation in $y = mx + c$ form

### Pattern B: "Interpret the distance-time graph..."
1. Calculate speed from gradient
2. Identify stationary periods (flat sections)
3. Check units carefully
4. For average speed, use total distance ÷ total time

### Pattern C: "Draw the graph of..."
1. Create table of values
2. Calculate y for each x
3. Plot points precisely with small crosses
4. Draw straight line with ruler

##  Common Mistakes to Avoid

| Mistake | Consequence | How to Avoid |
|---------|-------------|--------------|
| Wrong order in gradient formula | Wrong answer | Write $\frac{\Delta y}{\Delta x}$ - y on top! |
| Forgetting negative signs | Wrong gradient | Check calculations with negatives |
| Not rearranging equation | Wrong gradient/intercept | Always write as $y = mx + c$ |
| Saying vertical line has gradient 0 | Lose mark | Vertical = undefined, Horizontal = 0 |
| Plotting points inaccurately | Line doesn't fit | Use sharp pencil, small crosses |
| Forgetting units | Lose accuracy mark | Always include units in context |
