## 📌 Essential Definitions

| Term | Definition |
|------|------------|
| **Histogram** | Graph for continuous data where **area** represents frequency |
| **Class Interval** | A range grouping continuous data |
| **Class Width** | Upper bound − Lower bound |
| **Frequency Density** | Frequency ÷ Class Width (height of bar) |

---

## 🔢 Key Formulas

| Formula | Use When |
|---------|----------|
| $\text{Class Width} = \text{Upper} - \text{Lower}$ | Finding width of interval |
| $\text{Frequency Density} = \frac{\text{Frequency}}{\text{Class Width}}$ | Drawing histogram |
| $\text{Frequency} = \text{FD} \times \text{Class Width}$ | Reading from histogram |
| $\text{Midpoint} = \frac{\text{Lower} + \text{Upper}}{2}$ | Frequency polygons |

---

## ⚡ Quick Rules

### Drawing Histograms
1. ✓ Calculate class width for each interval
2. ✓ Calculate frequency density = Frequency ÷ Width
3. ✓ x-axis = variable, y-axis = frequency density
4. ✓ **Bars must touch** (continuous data)
5. ✓ Height = frequency density, Width = class width

### Reading Histograms
1. Read height (frequency density) from y-axis
2. Read width from x-axis
3. Multiply: **Frequency = Height × Width**

### Finding Total Frequency
$$\text{Total} = \sum (\text{Height} \times \text{Width}) = \text{Total Area}$$

## 🎯 Histogram vs Bar Chart

| Feature | Histogram | Bar Chart |
|---------|-----------|-----------|
| Data type | **Continuous** | Discrete/Categorical |
| Bars | **Touch** | Have gaps |
| Height = | Frequency density | Frequency |
| Area = | **Frequency** | Not meaningful |

| Notation | Meaning |
|----------|---------|
| $0 \leq x < 10$ | 0 up to (not including) 10 |
| $10 \leq x < 20$ | 10 up to (not including) 20 |

## 📈 Distribution Shapes

| Shape | Description | Mean vs Median vs Mode |
|-------|-------------|------------------------|
| **Symmetric** | Mirror image | Mean ≈ Median ≈ Mode |
| **Positive skew** | Tail to right | Mode < Median < Mean |
| **Negative skew** | Tail to left | Mean < Median < Mode |
| **Bimodal** | Two peaks | Two modes |

```

Step 2: Add "Frequency Density" column (F ÷ W)
Step 3: Draw x-axis (variable) and y-axis (FD)
Step 4: Draw bars - width from class, height from FD
Step 5: Ensure bars touch
```

### From Histogram to Table:
```
Step 1: Read width from x-axis
Step 2: Read height (FD) from y-axis

```

##  Estimating Partial Frequencies

**Within one class:** Use proportion
$$\text{Frequency in range} = \text{Class Frequency} \times \frac{\text{Range width}}{\text{Class width}}$$

**Example:** In class $20 \leq x < 40$ (frequency 30), find frequency for $20 \leq x < 25$:
$$30 \times \frac{25-20}{40-20} = 30 \times \frac{5}{20} = 7.5 \approx 8$$

## 🔢 Quick Calculation Examples

**Example 1:** Find FD if F = 24 and width = 8
$$\text{FD} = \frac{24}{8} = 3$$

**Example 2:** Find F if FD = 4.5 and width = 10
$$\text{F} = 4.5 \times 10 = 45$$

**Example 3:** Class $30 \leq x < 50$, find width
$$\text{Width} = 50 - 30 = 20$$

## 📊 Frequency Polygon

To draw:
1. Find midpoint of each class: $\frac{\text{Lower} + \text{Upper}}{2}$
2. Plot (midpoint, frequency)
3. Join with straight lines
4. Close at zero on both ends
