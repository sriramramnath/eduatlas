## Section A: Multiple Choice Questions (10 marks)

### Question 1
A histogram is drawn showing the distribution of heights. The class $150 \leq h < 160$ has a bar of height 2.5. What is the frequency for this class?

A) 2.5  
B) 25  
C) 250  
D) 10

**Answer: B**

**Explanation:**  
Frequency = Frequency Density × Class Width  
Frequency = 2.5 × (160 - 150) = 2.5 × 10 = **25**

---

### Question 2
Which of the following is TRUE about histograms?

A) Bars have gaps between them  
B) Height of each bar equals the frequency  
C) Area of each bar represents the frequency  
D) They are used for categorical data

**Answer: C**

**Explanation:**  
- A is false: bars must touch (continuous data)  
- B is false: height equals frequency density  
- C is true: Area = Frequency Density × Width = Frequency  
- D is false: histograms are for continuous data

---

### Question 3
A histogram shows data grouped as $0 \leq x < 10$, $10 \leq x < 25$, $25 \leq x < 40$. What are the class widths?

A) 10, 15, 15  
B) 10, 15, 14  
C) 10, 14, 14  
D) 10, 16, 15

**Answer: A**

**Explanation:**  
- Width 1: $10 - 0 = 10$  
- Width 2: $25 - 10 = 15$  
- Width 3: $40 - 25 = 15$

### Question 4
The table shows frequency densities for equal-width classes:

| Class | FD |
|-------|-----|
| $0 \leq x < 5$ | 4 |
| $5 \leq x < 10$ | 6 |
| $10 \leq x < 15$ | 2 |

If the total frequency is 60, what is the class width?

A) 3  
B) 4  
C) 5  
D) 6

**Answer: C**

**Explanation:**  
Let width = $w$  
Frequencies: $4w, 6w, 2w$  
Total = $12w = 60$  
$w = 5$

### Question 5
In a histogram, if a class has frequency 36 and frequency density 6, what is the class width?

A) 6  
B) 30  
C) 42  
D) 216

**Answer: A**

**Explanation:**  
$\text{Width} = \frac{\text{Frequency}}{\text{Frequency Density}} = \frac{36}{6} = 6$

### Question 6
Which graph would be most appropriate to display the heights of 200 students?

A) Bar chart  
B) Pie chart  
C) Histogram  
D) Pictogram

**Answer: C**

**Explanation:**  
Height is continuous data, so a histogram is the appropriate choice. Bar charts are for discrete/categorical data.

### Question 7
A histogram bar for class $20 \leq x < 35$ has width 15 and height 4. What is the frequency?

A) 19  
B) 55  
C) 60  
D) 3.75

**Answer: C**

**Explanation:**  
$\text{Frequency} = \text{Width} \times \text{Height} = 15 \times 4 = 60$

### Question 8
A histogram shows a distribution with most data on the left and a long tail extending to the right. This distribution is:

A) Symmetric  
B) Positively skewed  
C) Negatively skewed  
D) Bimodal

**Answer: B**

**Explanation:**  
A positive (right) skew has most data concentrated on the left with a tail extending right. Mean > Median > Mode.

### Question 9
For the class interval $25 \leq x < 40$, what is the midpoint?

A) 32  
B) 32.5  
C) 33  
D) 65

**Answer: B**

**Explanation:**  
$\text{Midpoint} = \frac{25 + 40}{2} = \frac{65}{2} = 32.5$

### Question 10
A histogram has 4 bars with frequency densities 2, 5, 5, 3. The class widths are 10, 10, 10, 10. What is the total frequency?

A) 15  
B) 25

**Answer: C**

**Explanation:**  
Frequencies: $2 \times 10 = 20$, $5 \times 10 = 50$, $5 \times 10 = 50$, $3 \times 10 = 30$

## Section B: True or False (10 marks)

### Question 11
**Statement:** In a histogram, the height of each bar represents the frequency.

**Answer: FALSE**

**Correction:** In a histogram, the height of each bar represents the **frequency density**, not the frequency. The area of the bar represents the frequency.

---

### Question 12
**Statement:** Histogram bars should touch each other without gaps.

**Answer: TRUE**

**Explanation:** Histograms display continuous data, so there are no gaps between the bars. This is a key difference from bar charts.

---

### Question 13
**Statement:** When class widths are equal, the shape of the histogram is the same as a bar chart using frequencies.

**Answer: TRUE**

**Explanation:** When all class widths are equal, frequency density is proportional to frequency, so the relative heights are the same as if frequency was plotted.

---

### Question 14
**Statement:** The frequency density is calculated by multiplying the frequency by the class width.

**Answer: FALSE**

**Correction:** The frequency density is calculated by **dividing** the frequency by the class width: $\text{FD} = \frac{\text{Frequency}}{\text{Class Width}}$

---

### Question 15
**Statement:** A histogram can be used to display data about favourite colours of students.

**Answer: FALSE**

**Correction:** Favourite colour is **categorical data**, not continuous. A bar chart or pie chart would be appropriate, not a histogram.

---

### Question 16
**Statement:** The total frequency equals the total area under all bars in a histogram.

**Answer: TRUE**

**Explanation:** Each bar's area equals its frequency (Height × Width = FD × Width = Frequency). Therefore, the total area equals the sum of all frequencies.

---

### Question 17
**Statement:** For the class $10 \leq x < 20$, the value 20 is included in this class.

**Answer: FALSE**

**Correction:** The notation $10 \leq x < 20$ means values from 10 up to but **not including** 20. The value 20 would be in the next class.

---

### Question 18
**Statement:** A positively skewed distribution has a longer tail on the right side.

**Answer: TRUE**

**Explanation:** Positive skew (right skew) has most data on the left with the tail extending to the right. The mean is greater than the median.

---

### Question 19
**Statement:** When drawing a frequency polygon, you plot the midpoints of each class interval against the frequencies.

**Answer: TRUE**

**Explanation:** The coordinates plotted are (midpoint, frequency) for each class, then joined with straight lines.

---

### Question 20
**Statement:** If a histogram bar has height 0, the frequency for that class must be 0.

**Answer: TRUE**

**Explanation:** If frequency density = 0, then Frequency = FD × Width = 0 × Width = 0. A bar of zero height indicates no data in that class.

## Section C: Structured Questions

### Question 21 (3 marks)

The table shows the masses of 50 parcels.

| Mass (kg) | Frequency |
|-----------|-----------|
| $0 \leq m < 2$ | 8 |
| $2 \leq m < 4$ | 14 |
| $4 \leq m < 8$ | 18 |
| $8 \leq m < 12$ | 10 |

**(a)** Calculate the frequency density for each class. (2 marks)

**(b)** State which class has the highest frequency density. (1 mark)

**Solution:**

**(a)** Calculate frequency densities:

| Mass (kg) | Frequency | Class Width | Frequency Density |
|-----------|-----------|-------------|-------------------|
| $0 \leq m < 2$ | 8 | $2 - 0 = 2$ | $8 \div 2 = 4$ |
| $2 \leq m < 4$ | 14 | $4 - 2 = 2$ | $14 \div 2 = 7$ |
| $4 \leq m < 8$ | 18 | $8 - 4 = 4$ | $18 \div 4 = 4.5$ |
| $8 \leq m < 12$ | 10 | $12 - 8 = 4$ | $10 \div 4 = 2.5$ |

**Mark scheme:**
- M1: Correct method for frequency density (F ÷ W)
- A1: At least 3 correct values

**(b)** The class with highest frequency density is $\mathbf{2 \leq m < 4}$ with FD = 7.

**Mark scheme:**
- B1: Correct class identified

### Question 22 (4 marks)

A histogram is drawn for the following data:

| Time (minutes) | Frequency |
|----------------|-----------|
| $0 \leq t < 10$ | 15 |
| $10 \leq t < 20$ | $x$ |
| $20 \leq t < 40$ | 24 |
| $40 \leq t < 50$ | 12 |

The bar for the class $10 \leq t < 20$ has height 3 on the histogram.

**(a)** Find the value of $x$. (2 marks)

**(b)** Find the total frequency. (2 marks)

**Solution:**

**(a)** Finding $x$:

For the class $10 \leq t < 20$:
- Class width = $20 - 10 = 10$
- Frequency density = 3

$$x = \text{Frequency} = \text{FD} \times \text{Width} = 3 \times 10 = 30$$

**Mark scheme:**
- M1: Correct use of F = FD × Width
- A1: $x = 30$

**(b)** Total frequency:

$$\text{Total} = 15 + x + 24 + 12 = 15 + 30 + 24 + 12 = 81$$

**Mark scheme:**
- M1: Adding all frequencies
- A1: Total = 81

### Question 23 (5 marks)

The histogram shows the distribution of marks scored by students in a test.

| Marks | Frequency Density |
|-------|-------------------|
| $0 \leq m < 20$ | 1.5 |
| $20 \leq m < 40$ | 2.5 |
| $40 \leq m < 60$ | 4.0 |
| $60 \leq m < 80$ | 1.5 |

**(a)** Calculate the frequency for each class. (3 marks)

**(b)** How many students scored less than 40 marks? (1 mark)

**(c)** What percentage of students scored 60 marks or more? (1 mark)

**Solution:**

**(a)** Calculate frequencies:

| Marks | Width | FD | Frequency |
|-------|-------|-----|-----------|
| $0 \leq m < 20$ | 20 | 1.5 | $20 \times 1.5 = 30$ |
| $20 \leq m < 40$ | 20 | 2.5 | $20 \times 2.5 = 50$ |
| $40 \leq m < 60$ | 20 | 4.0 | $20 \times 4.0 = 80$ |
| $60 \leq m < 80$ | 20 | 1.5 | $20 \times 1.5 = 30$ |

**Mark scheme:**
- M1: Correct method F = FD × Width
- A1: At least 2 correct
- A1: All correct

**(b)** Students scoring less than 40 marks:

$$30 + 50 = 80 \text{ students}$$

**Mark scheme:**
- B1: 80 students

**(c)** Percentage scoring 60 or more:

Total students = $30 + 50 + 80 + 30 = 190$  
Students scoring 60+ = 30

$$\text{Percentage} = \frac{30}{190} \times 100 = 15.8\% \text{ (3 sf)}$$

**Mark scheme:**
- B1: Correct percentage (accept 15.8% or 15.79%)

### Question 24 (6 marks)

The table shows the heights of 100 plants grown in a greenhouse.

| Height (cm) | Frequency |
|-------------|-----------|
| $0 \leq h < 10$ | 12 |
| $10 \leq h < 20$ | 28 |
| $20 \leq h < 30$ | $a$ |
| $30 \leq h < 50$ | $b$ |

The frequency density for the class $30 \leq h < 50$ is 0.8.

**(a)** Find the value of $b$. (2 marks)

**(b)** Find the value of $a$. (2 marks)

**(c)** Describe the shape of the distribution. (2 marks)

**Solution:**

**(a)** Finding $b$:

For class $30 \leq h < 50$:
- Width = $50 - 30 = 20$
- FD = 0.8

$$b = 0.8 \times 20 = 16$$

**Mark scheme:**
- M1: Correct method F = FD × Width
- A1: $b = 16$

**(b)** Finding $a$:

Total frequency = 100

$$12 + 28 + a + 16 = 100$$
$$56 + a = 100$$
$$a = 44$$

**Mark scheme:**
- M1: Using total frequency equation
- A1: $a = 44$

**(c)** Distribution shape:

Frequency densities:
- $0 \leq h < 10$: $12 \div 10 = 1.2$
- $10 \leq h < 20$: $28 \div 10 = 2.8$
- $20 \leq h < 30$: $44 \div 10 = 4.4$
- $30 \leq h < 50$: $16 \div 20 = 0.8$

The distribution is **negatively skewed** (most data on the right, tail to the left).

**Mark scheme:**
- B1: Correct identification (negatively skewed / left skewed)
- B1: Justification (most data at higher values / low FD on right)

### Question 25 (5 marks)

A histogram showing the distribution of journey times has the following bars:

- Bar 1: $0 \leq t < 10$, height = 3
- Bar 2: $10 \leq t < 20$, height = 5
- Bar 3: $20 \leq t < 30$, height = 4
- Bar 4: $30 \leq t < 50$, height = 2

**(a)** Find the total number of journeys. (3 marks)

**(b)** Estimate how many journeys took between 15 and 25 minutes. (2 marks)

**Solution:**

**(a)** Total journeys:

| Class | Width | Height | Frequency |
|-------|-------|--------|-----------|
| $0 \leq t < 10$ | 10 | 3 | 30 |
| $10 \leq t < 20$ | 10 | 5 | 50 |
| $20 \leq t < 30$ | 10 | 4 | 40 |
| $30 \leq t < 50$ | 20 | 2 | 40 |

Total = $30 + 50 + 40 + 40 = 160$ journeys

**Mark scheme:**
- M1: Correct method F = Width × Height
- M1: Adding all frequencies
- A1: Total = 160

**(b)** Journeys between 15 and 25 minutes:

From class $10 \leq t < 20$ (frequency 50):
- Range needed: 15 to 20 (half the class)
- Frequency: $50 \times \frac{5}{10} = 25$

From class $20 \leq t < 30$ (frequency 40):
- Range needed: 20 to 25 (half the class)
- Frequency: $40 \times \frac{5}{10} = 20$

Total = $25 + 20 = 45$ journeys

**Mark scheme:**
- M1: Correct proportional method for at least one partial class
- A1: Answer = 45 (accept 44-46)

### Question 27 (5 marks)

Two classes took the same mathematics test. Their results are shown in histograms.

**Class A:**
- Most students scored around 60-70%
- Distribution is roughly symmetric

**Class B:**
- Most students scored around 40-50%
- Distribution has a long tail to the right

Compare the performance of the two classes.

**Solution:**

**Comparison points:**

1. **Central tendency:** Class A performed better overall, with most students scoring around 60-70%, compared to Class B where most scored around 40-50%.

2. **Distribution shape:** Class A has a symmetric distribution, while Class B is positively skewed (tail to the right).

3. **Spread:** Class B has a wider spread of marks, with some students scoring quite high (shown by the right tail), whereas Class A's marks are more concentrated around the centre.

4. **Consistency:** Class A shows more consistent performance (symmetric distribution), while Class B has more variability (skewed distribution).

**Mark scheme:**
- B1: Correct comparison of central tendency
- B1: Correct identification of skewness
- B1: Comment on spread
- B1: Use of appropriate statistical vocabulary
- B1: Clear, organised response

## Section D: Extended Response Question

### Question 28 (8 marks)

A company records the daily journey times (in minutes) for its delivery drivers over one month. The data is summarised below.

| Time (minutes) | Frequency |
|----------------|-----------|
| $0 \leq t < 15$ | 6 |
| $15 \leq t < 30$ | 18 |
| $30 \leq t < 45$ | 12 |
| $45 \leq t < 60$ | 9 |
| $60 \leq t < 90$ | 15 |

**(a)** Calculate the frequency density for each class. (3 marks)

**(b)** Draw a histogram to represent this data. Use a suitable scale. (3 marks)

**(c)** The manager says "more than half of journeys took less than 45 minutes." Is this statement correct? Show your working. (2 marks)

**Solution:**

**(a)** Frequency densities:

| Time (min) | Frequency | Width | FD |
|------------|-----------|-------|-----|
| $0 \leq t < 15$ | 6 | 15 | $6 \div 15 = 0.4$ |
| $15 \leq t < 30$ | 18 | 15 | $18 \div 15 = 1.2$ |
| $30 \leq t < 45$ | 12 | 15 | $12 \div 15 = 0.8$ |
| $45 \leq t < 60$ | 9 | 15 | $9 \div 15 = 0.6$ |
| $60 \leq t < 90$ | 15 | 30 | $15 \div 30 = 0.5$ |

**Mark scheme:**
- M1: Correct formula FD = F ÷ W
- A1: At least 3 correct
- A1: All correct

**(b)** Histogram:

- x-axis: Time (minutes), scale 0 to 90
- y-axis: Frequency Density, scale 0 to 1.4

Bars drawn with:
- $0 \leq t < 15$: width 15, height 0.4
- $15 \leq t < 30$: width 15, height 1.2
- $30 \leq t < 45$: width 15, height 0.8
- $45 \leq t < 60$: width 15, height 0.6
- $60 \leq t < 90$: width 30, height 0.5

**Mark scheme:**
- B1: Correct axes with labels and scales
- B1: Bars correctly positioned (touching, correct widths)
- B1: Heights correct for all bars

**(c)** Verifying the statement:

Total journeys = $6 + 18 + 12 + 9 + 15 = 60$

Journeys less than 45 minutes = $6 + 18 + 12 = 36$

Proportion = $\frac{36}{60} = 0.6 = 60\%$

Since 60% > 50%, the statement **is correct**.

**Mark scheme:**
- M1: Correct calculation of total and/or journeys under 45 min
- A1: Correct conclusion with justification (60% > 50%)

## Exam Focus & Strategy

### Frequently Tested Question Types

1. **Completing frequency tables** with frequency densities
2. **Calculating frequencies** from histogram bars
3. **Drawing histograms** from grouped data
4. **Estimating partial frequencies** (interpolation)
5. **Comparing distributions** using statistical language
6. **Finding missing values** using total frequency

### Typical Problem-Solving Patterns

| Pattern | Approach |
|---------|----------|
| "Complete the table" | Calculate widths first, then FD = F ÷ W |
| "Find the frequency" | F = FD × Width |
| "How many values..." | Identify bars, calculate partial frequencies if needed |
| "Compare distributions" | Use terms: skew, spread, central tendency |

### Command Word Guidance

| Command | Action Required |
|---------|-----------------|
| **Calculate** | Show working, give exact numerical answer |
| **Draw** | Use ruler, label axes, choose suitable scale |
| **Complete** | Fill in missing values in table |
| **Estimate** | Approximate using proportional reasoning |
| **Compare** | Identify similarities and differences |
| **Describe** | State what the data shows |

### How to Gain Full Marks

1. **Show all calculations** - M marks are for method
2. **Use the correct formula** - Write FD = F ÷ W
3. **Check your answer** - Does total match if given?
4. **Label histogram axes** - Include variable name and units
5. **Use proportional reasoning** for partial frequencies
6. **Use statistical vocabulary** - skew, spread, mode, etc.
7. **Give final answers clearly** - Box or underline
