## 1. EXPANDING BRACKETS – QUICK RULES

### Single Brackets
> **Key Rule:** $a(b + c) = ab + ac$

| Expression | Expansion |
|------------|-----------|
| $3(x + 4)$ | $3x + 12$ |
| $-2(5x - 3)$ | $-10x + 6$ |
| $5x(2x - 3y + 1)$ | $10x^2 - 15xy + 5x$ |

### Double Brackets (FOIL Method)
> **F**irst, **O**uter, **I**nner, **L**ast

$$(a + b)(c + d) = ac + ad + bc + bd$$

| Expression | Expansion |
|------------|-----------|
| $(x + 2)(x + 7)$ | $x^2 + 9x + 14$ |
| $(x - 3)(x + 4)$ | $x^2 + x - 12$ |
| $(2x + 1)(3x - 5)$ | $6x^2 - 7x - 5$ |
| $(x - 4)(x - 6)$ | $x^2 - 10x + 24$ |

### Special Cases – MUST MEMORISE!

| Name | Formula | Example |
|------|---------|---------|
| **Perfect Square (+)** | $(a + b)^2 = a^2 + 2ab + b^2$ | $(x + 5)^2 = x^2 + 10x + 25$ |
| **Perfect Square (-)** | $(a - b)^2 = a^2 - 2ab + b^2$ | $(2x - 3)^2 = 4x^2 - 12x + 9$ |
| **Difference of Two Squares** | $(a + b)(a - b) = a^2 - b^2$ | $(x + 6)(x - 6) = x^2 - 36$ |

>  **Common Error:** $(x + 3)^2 \neq x^2 + 9$ — always has a middle term!

## 2. FACTORISING – QUICK RULES

### Single Brackets (Common Factor)
**Method:** Find HCF of all terms → Put outside brackets

| Expression | Factorisation |
|------------|---------------|
| $6x + 9$ | $3(2x + 3)$ |
| $8x^2 - 12x$ | $4x(2x - 3)$ |
| $x^2 - 9x$ | $x(x - 9)$ |

### Quadratics $x^2 + bx + c$

> **Key Method:** Find two numbers that **MULTIPLY to $c$** and **ADD to $b$**

| Sign of $c$ | Sign of $b$ | Number Signs |
|-------------|-------------|--------------|
| Positive (+) | Positive (+) | Both positive: $(x + p)(x + q)$ |
| Positive (+) | Negative (−) | Both negative: $(x - p)(x - q)$ |
| Negative (−) | Positive (+) | Mixed, larger positive: $(x - p)(x + q)$ |
| Negative (−) | Negative (−) | Mixed, larger negative: $(x + p)(x - q)$ |

**Quick Examples:**

| Quadratic | Numbers | Factorisation |
|-----------|---------|---------------|
| $x^2 + 7x + 12$ | $+3, +4$ | $(x + 3)(x + 4)$ |
| $x^2 - 5x + 6$ | $-2, -3$ | $(x - 2)(x - 3)$ |
| $x^2 + 2x - 15$ | $-3, +5$ | $(x - 3)(x + 5)$ |
| $x^2 - 9x + 20$ | $-4, -5$ | $(x - 4)(x - 5)$ |

### Quadratics $ax^2 + bx + c$ (a ≠ 1)

**AC Method:**
1. Multiply $a \times c$
2. Find factors of $ac$ that add to $b$
3. Split middle term
4. Factorise by grouping

| Quadratic | $a \times c$ | Numbers | Split | Factorised |
|-----------|--------------|---------|-------|------------|
| $2x^2 + 7x + 3$ | 6 | $+6, +1$ | $2x^2 + 6x + x + 3$ | $(2x + 1)(x + 3)$ |
| $3x^2 - 10x - 8$ | $-24$ | $-12, +2$ | $3x^2 - 12x + 2x - 8$ | $(3x + 2)(x - 4)$ |

### Special Factorisations

| Name | Expression | Factorisation |
|------|------------|---------------|
| **Difference of Two Squares** | $a^2 - b^2$ | $(a + b)(a - b)$ |
| $x^2 - 25$ | $x^2 - 5^2$ | $(x + 5)(x - 5)$ |
| $4x^2 - 9$ | $(2x)^2 - 3^2$ | $(2x + 3)(2x - 3)$ |
| **Perfect Square (+)** | $a^2 + 2ab + b^2$ | $(a + b)^2$ |
| **Perfect Square (−)** | $a^2 - 2ab + b^2$ | $(a - b)^2$ |

>  **Always check for common factors FIRST!**
> $2x^2 - 8 = 2(x^2 - 4) = 2(x + 2)(x - 2)$

## 3. SOLVING QUADRATIC EQUATIONS – QUICK RULES

### Method 1: By Factorising

**Steps:**
1. Rearrange to $ax^2 + bx + c = 0$
2. Factorise
3. Set each bracket = 0
4. Solve for $x$

| Equation | Factorised | Solutions |
|----------|------------|-----------|
| $x^2 + 5x + 6 = 0$ | $(x + 2)(x + 3) = 0$ | $x = -2, -3$ |
| $x^2 - 4x - 5 = 0$ | $(x - 5)(x + 1) = 0$ | $x = 5, -1$ |
| $x^2 = 9$ | $(x + 3)(x - 3) = 0$ | $x = 3, -3$ |
| $x^2 = 10x$ | $x(x - 10) = 0$ | $x = 0, 10$ |

>  **Never divide by $x$** – you'll lose the solution $x = 0$!

### Method 2: Quadratic Formula

> **MEMORISE:**
> $$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$

**Steps:**
1. Identify $a$, $b$, $c$ from $ax^2 + bx + c = 0$
2. Calculate discriminant $b^2 - 4ac$
3. Substitute into formula
4. Simplify

| Equation | $a, b, c$ | Discriminant | Solutions |
|----------|-----------|--------------|-----------|
| $x^2 + 6x + 5 = 0$ | $1, 6, 5$ | $16$ | $x = -1, -5$ |
| $x^2 - 3x - 7 = 0$ | $1, -3, -7$ | $37$ | $x = \frac{3 \pm \sqrt{37}}{2}$ |

### The Discriminant

| $b^2 - 4ac$ | Roots |
|-------------|-------|
| $> 0$ | Two distinct real roots |
| $= 0$ | One repeated root |
| $< 0$ | No real roots |

### Method 3: Completing the Square

> **Formula:**
> $$x^2 + bx = \left(x + \frac{b}{2}\right)^2 - \left(\frac{b}{2}\right)^2$$

**Example:** $x^2 + 6x + 5 = 0$
$$(x + 3)^2 - 9 + 5 = 0$$
$$(x + 3)^2 = 4$$
$$x + 3 = \pm 2$$
$$x = -1 \text{ or } -5$$

## 4. WORD PROBLEMS – QUICK PATTERNS

### Problem-Solving Steps

```
1. READ → Identify unknown
2. DEFINE → Let x = unknown
3. EQUATION → Write from problem
4. SOLVE → Factorise or formula
5. CHECK → Reject invalid solutions
6. ANSWER → State with units
```

### Common Problem Types

| Type | Setup | Example |
|------|-------|---------|
| **Area** | Area = length × width | Rectangle: $x(x+3) = 40$ |
| **Consecutive Numbers** | $n, n+1$ | $n(n+1) = 56$ |
| **Pythagoras** | $a^2 + b^2 = c^2$ | $x^2 + (x+7)^2 = 13^2$ |
| **Projectile** | Given formula, set $h = 0$ | $h = 20t - 5t^2$ |

### Key Points for Word Problems
- ✅ Reject negative lengths, times, etc.
- ✅ Check answer makes sense
- ✅ State units in final answer
- ✅ Answer the question asked

## 6. COMMON MISTAKES 

| Mistake | Wrong | Correct |
|---------|-------|---------|
| Forgetting middle term | $(x + 3)^2 = x^2 + 9$ | $x^2 + 6x + 9$ |
| Wrong signs in factorising | $x^2 - 5x + 6 = (x + 2)(x + 3)$ | $(x - 2)(x - 3)$ |
| Dividing by $x$ | $x^2 = 5x$ → $x = 5$ | $x = 0$ or $5$ |
| Missing solutions | Only giving one root | Always state BOTH solutions |
| Not checking context | Giving negative length | Reject invalid solutions |
| Forgetting common factor | $2x^2 - 8 = (x+2)(x-2)$ | $2(x+2)(x-2)$ |
