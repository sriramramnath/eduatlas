**Q1.** Expand $3(x - 4)$

- A) $3x - 4$
- B) $3x - 12$
- C) $3x + 12$
- D) $x - 12$

**Answer: B**

**Explanation:**
$$3(x - 4) = 3 \times x - 3 \times 4 = 3x - 12$$

---

**Q2.** Expand $(x + 3)(x + 5)$

- A) $x^2 + 8x + 15$
- B) $x^2 + 15x + 8$
- C) $x^2 + 2x + 15$
- D) $x^2 + 8x + 8$

**Answer: A**

**Explanation:**
$$(x + 3)(x + 5) = x^2 + 5x + 3x + 15 = x^2 + 8x + 15$$

---

**Q3.** What is $(x + 4)^2$ when expanded?

- A) $x^2 + 16$
- B) $x^2 + 8x + 16$
- C) $x^2 + 4x + 16$
- D) $x^2 + 8x + 8$

**Answer: B**

**Explanation:**
Using the perfect square formula: $(a + b)^2 = a^2 + 2ab + b^2$
$$(x + 4)^2 = x^2 + 2(x)(4) + 4^2 = x^2 + 8x + 16$$

---

**Q4.** Expand $(x - 6)(x + 6)$

- A) $x^2 - 12x - 36$
- B) $x^2 - 36$
- C) $x^2 + 36$
- D) $x^2 - 12x + 36$

**Answer: B**

**Explanation:**
This is the difference of two squares: $(a + b)(a - b) = a^2 - b^2$
$$(x - 6)(x + 6) = x^2 - 6^2 = x^2 - 36$$

**Q5.** Factorise $x^2 + 8x + 12$

**Answer: A**

**Explanation:**
Find numbers that multiply to $12$ and add to $8$: $+2$ and $+6$
$$x^2 + 8x + 12 = (x + 2)(x + 6)$$

**Q9.** Solve $x^2 = 16$

- A) $x = 4$ only
- B) $x = \pm 4$
- C) $x = \pm 8$
- D) $x = 8$ only

**Explanation:**

**Q10.** What are the solutions of $x^2 - 3x - 10 = 0$?

- A) $x = 5$ or $x = -2$
- B) $x = -5$ or $x = 2$
- C) $x = 5$ or $x = 2$
- D) $x = -5$ or $x = -2$

**Answer: A**

**Explanation:**
$$x^2 - 3x - 10 = 0$$
Numbers with product $-10$ and sum $-3$: $-5$ and $+2$
$$(x - 5)(x + 2) = 0$$
$$x = 5 \text{ or } x = -2$$

**Q11.** Using the quadratic formula, solve $x^2 - 5x + 4 = 0$

- A) $x = 1$ or $x = 4$
- B) $x = -1$ or $x = -4$
- C) $x = 2$ or $x = 2$
- D) $x = 1$ or $x = -4$

**Answer: A**

**Explanation:**
Using $x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$ with $a = 1$, $b = -5$, $c = 4$:
$$x = \frac{5 \pm \sqrt{25 - 16}}{2}$$
$$x = \frac{5 \pm 3}{2}$$
$$x = \frac{8}{2} = 4 \quad \text{or} \quad x = \frac{2}{2} = 1$$

**Q12.** Expand $-2x(3x - 5)$

## 📝 SECTION B: TRUE OR FALSE

**Instructions:** State whether each statement is TRUE or FALSE. If FALSE, provide the correction.

---

**Q1.** $(x + 5)^2 = x^2 + 25$

**Answer: FALSE**

**Correction:** $(x + 5)^2 = x^2 + 10x + 25$

The middle term is $2 \times x \times 5 = 10x$, which cannot be omitted.

---

**Q2.** $x^2 - 36 = (x + 6)(x - 6)$

**Answer: TRUE**

**Explanation:** This is the difference of two squares formula: $a^2 - b^2 = (a + b)(a - b)$

---

**Q3.** The solutions of $x^2 - 7x + 10 = 0$ are $x = 5$ and $x = 2$.

**Answer: TRUE**

**Explanation:** 
$$x^2 - 7x + 10 = (x - 5)(x - 2) = 0$$
$$x = 5 \text{ or } x = 2$$

---

**Q4.** To solve $x^2 = 6x$, we can divide both sides by $x$ to get $x = 6$.

**Answer: FALSE**

**Correction:** Never divide by $x$ as you lose the solution $x = 0$.

Correct method:
$$x^2 - 6x = 0$$
$$x(x - 6) = 0$$
$$x = 0 \text{ or } x = 6$$

---

**Q5.** $4x^2 - 9$ factorises to $(2x + 3)(2x - 3)$.

**Answer: TRUE**

**Explanation:** This is difference of two squares: $(2x)^2 - 3^2 = (2x + 3)(2x - 3)$

---

**Q6.** When factorising $x^2 + 3x - 10$, we need two numbers that multiply to give $-10$ and add to give $-3$.

**Answer: FALSE**

**Correction:** We need numbers that multiply to give $-10$ and add to give $+3$ (the coefficient of $x$).

The numbers are $+5$ and $-2$: $x^2 + 3x - 10 = (x + 5)(x - 2)$

---

**Q7.** The quadratic formula is $x = \frac{-b \pm \sqrt{b^2 + 4ac}}{2a}$

**Answer: FALSE**

**Correction:** The correct formula is $x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$

The discriminant is $b^2 - 4ac$, not $b^2 + 4ac$.

---

**Q8.** If $b^2 - 4ac > 0$, a quadratic equation has two distinct real solutions.

**Answer: TRUE**

**Explanation:** The discriminant tells us:
- $b^2 - 4ac > 0$: Two distinct real roots
- $b^2 - 4ac = 0$: One repeated root
- $b^2 - 4ac < 0$: No real roots

---

**Q9.** $(x - 4)(x + 3) = x^2 - x - 12$

**Answer: TRUE**

**Explanation:** 
$$(x - 4)(x + 3) = x^2 + 3x - 4x - 12 = x^2 - x - 12$$

---

**Q10.** Factorising $x^2 - 6x + 9$ gives $(x - 3)^2$.

**Answer: TRUE**

**Explanation:** This is a perfect square trinomial: $x^2 - 6x + 9 = x^2 - 2(x)(3) + 3^2 = (x - 3)^2$

### Question 1 (2 marks)

Expand and simplify $(x + 7)(x - 2)$

**Solution:**

**[M1]** Use FOIL method:
$$(x + 7)(x - 2) = x \times x + x \times (-2) + 7 \times x + 7 \times (-2)$$
$$= x^2 - 2x + 7x - 14$$

**[A1]** Collect like terms:
$$= x^2 + 5x - 14$$

**Answer: $x^2 + 5x - 14$**

---

### Question 2 (2 marks)

Expand and simplify $(2x - 3)^2$

**Solution:**

**[M1]** Write as product or use formula:
$$(2x - 3)^2 = (2x - 3)(2x - 3)$$
$$= 4x^2 - 6x - 6x + 9$$

Or using $(a - b)^2 = a^2 - 2ab + b^2$:
$$= (2x)^2 - 2(2x)(3) + 3^2$$

**[A1]**
$$= 4x^2 - 12x + 9$$

**Answer: $4x^2 - 12x + 9$**

---

### Question 3 (3 marks)

Factorise fully $x^2 + 2x - 15$

**Solution:**

**[M1]** Find two numbers with product $-15$ and sum $+2$:
- Numbers are $+5$ and $-3$

**[M1]** Write the factorisation:
$$x^2 + 2x - 15 = (x + 5)(x - 3)$$

**[A1]** Check: $(x + 5)(x - 3) = x^2 - 3x + 5x - 15 = x^2 + 2x - 15$ ✓

**Answer: $(x + 5)(x - 3)$**

---

### Question 4 (3 marks)

Solve $x^2 - 9x + 20 = 0$

**Solution:**

**[M1]** Factorise:
Find numbers with product $+20$ and sum $-9$: $-4$ and $-5$
$$x^2 - 9x + 20 = (x - 4)(x - 5) = 0$$

**[M1]** Set each bracket to zero:
$$x - 4 = 0 \quad \text{or} \quad x - 5 = 0$$

**[A1]** Solve:
$$x = 4 \quad \text{or} \quad x = 5$$

**Answer: $x = 4$ or $x = 5$**

### Question 5 (3 marks)

Factorise $2x^2 - 18$

**Solution:**

**[M1]** Take out common factor of $2$:
$$2x^2 - 18 = 2(x^2 - 9)$$

**[M1]** Recognise difference of two squares:
$$x^2 - 9 = x^2 - 3^2 = (x + 3)(x - 3)$$

**[A1]**
$$2x^2 - 18 = 2(x + 3)(x - 3)$$

**Answer: $2(x + 3)(x - 3)$**

### Question 6 (4 marks)

(a) Expand and simplify $(x + 4)(x + 4)(x - 2)$ **[2 marks]**

(b) Hence solve $(x + 4)^2(x - 2) = 0$ **[2 marks]**

**Solution:**

**(a) [M1]** First expand $(x + 4)(x + 4)$:
$$(x + 4)^2 = x^2 + 8x + 16$$

**[A1]** Now multiply by $(x - 2)$:
$$(x^2 + 8x + 16)(x - 2)$$
$$= x^3 - 2x^2 + 8x^2 - 16x + 16x - 32$$
$$= x^3 + 6x^2 - 32$$

**Answer: $x^3 + 6x^2 - 32$**

**(b) [M1]** From the equation $(x + 4)^2(x - 2) = 0$, set each factor to zero:
$$x + 4 = 0 \quad \text{or} \quad x - 2 = 0$$

**[A1]**
$$x = -4 \quad \text{or} \quad x = 2$$

**Answer: $x = -4$ or $x = 2$**

### Question 7 (4 marks)

Use the quadratic formula to solve $x^2 - 6x + 2 = 0$. Give your answers to 2 decimal places.

**Solution:**

**[M1]** Identify coefficients:
$$a = 1, \quad b = -6, \quad c = 2$$

**[M1]** Calculate discriminant:
$$b^2 - 4ac = (-6)^2 - 4(1)(2) = 36 - 8 = 28$$

**[M1]** Apply formula:
$$x = \frac{-(-6) \pm \sqrt{28}}{2(1)}$$
$$x = \frac{6 \pm \sqrt{28}}{2}$$
$$x = \frac{6 \pm 5.292}{2}$$

**[A1]** Calculate both solutions:
$$x = \frac{6 + 5.292}{2} = 5.65 \quad \text{(2 d.p.)}$$
$$x = \frac{6 - 5.292}{2} = 0.35 \quad \text{(2 d.p.)}$$

**Answer: $x = 5.65$ or $x = 0.35$ (2 d.p.)**

### Question 8 (4 marks)

(a) Write $x^2 - 10x$ in the form $(x - a)^2 - b$ **[2 marks]**

(b) Hence find the minimum value of $x^2 - 10x + 5$ **[2 marks]**

**Solution:**

**(a) [M1]** Complete the square:
$$x^2 - 10x = (x - 5)^2 - 25$$

**[A1]**
$$x^2 - 10x = (x - 5)^2 - 25$$

**Answer: $a = 5$, $b = 25$**

**(b) [M1]** Add 5 to the completed square form:
$$x^2 - 10x + 5 = (x - 5)^2 - 25 + 5 = (x - 5)^2 - 20$$

**[A1]** Minimum value occurs when $(x - 5)^2 = 0$, giving minimum $= -20$

**Answer: Minimum value is $-20$**

### Question 9 (5 marks)

A rectangular garden has length 4 metres more than its width. The area of the garden is 45 m².

(a) Form an equation for the area of the garden. **[2 marks]**

(b) Solve the equation to find the dimensions of the garden. **[3 marks]**

**Solution:**

**(a) [M1]** Define variables:
Let width $= x$ metres
Length $= (x + 4)$ metres

**[A1]** Area equation:
$$\text{Area} = x(x + 4) = 45$$

**Answer: $x(x + 4) = 45$ or $x^2 + 4x = 45$**

**(b) [M1]** Rearrange to form quadratic equation:
$$x^2 + 4x - 45 = 0$$

**[M1]** Factorise:
Find numbers with product $-45$ and sum $+4$: $+9$ and $-5$
$$(x + 9)(x - 5) = 0$$

**[M1]** Solve:
$$x = -9 \quad \text{or} \quad x = 5$$

**[A1]** Reject negative value (width cannot be negative):
Width $= 5$ m, Length $= 5 + 4 = 9$ m

**Answer: Width = 5 m, Length = 9 m**

### Question 10 (5 marks)

Solve $2x^2 + 5x - 3 = 0$

**Solution:**

**[M1]** Find numbers with product $a \times c = 2 \times (-3) = -6$ and sum $b = 5$:
Numbers are $+6$ and $-1$

**[M1]** Split the middle term:
$$2x^2 + 6x - x - 3 = 0$$

**[M1]** Factorise by grouping:
$$2x(x + 3) - 1(x + 3) = 0$$
$$(2x - 1)(x + 3) = 0$$

**[A1]** Set each bracket to zero:
$$2x - 1 = 0 \Rightarrow x = \frac{1}{2}$$
$$x + 3 = 0 \Rightarrow x = -3$$

**Answer: $x = \frac{1}{2}$ or $x = -3$**

### Question 11 (6 marks)

The height $h$ metres of a ball thrown upwards after $t$ seconds is given by $h = 30t - 5t^2$.

(a) Find the height of the ball after 2 seconds. **[1 mark]**

(b) When is the ball at height 25 metres? **[3 marks]**

(c) After how many seconds does the ball hit the ground? **[2 marks]**

**Solution:**

**(a) [B1]** Substitute $t = 2$:
$$h = 30(2) - 5(2)^2 = 60 - 20 = 40$$

**Answer: 40 metres**

**(b) [M1]** Set $h = 25$:
$$25 = 30t - 5t^2$$

**[M1]** Rearrange:
$$5t^2 - 30t + 25 = 0$$
$$t^2 - 6t + 5 = 0$$

**[M1]** Factorise and solve:
$$(t - 5)(t - 1) = 0$$
$$t = 1 \text{ second} \quad \text{or} \quad t = 5 \text{ seconds}$$

**Answer: $t = 1$ s or $t = 5$ s**

**(c) [M1]** Set $h = 0$:
$$0 = 30t - 5t^2$$
$$5t(6 - t) = 0$$

**[A1]**
$$t = 0 \text{ (start)} \quad \text{or} \quad t = 6 \text{ seconds}$$

**Answer: 6 seconds**

### Question 12 (6 marks)

(a) Expand $(2x + 1)(x - 3)(x + 2)$ **[3 marks]**

(b) Factorise $x^3 - x$ completely. **[3 marks]**

**Solution:**

**(a) [M1]** First expand $(x - 3)(x + 2)$:
$$(x - 3)(x + 2) = x^2 + 2x - 3x - 6 = x^2 - x - 6$$

**[M1]** Multiply by $(2x + 1)$:
$$(2x + 1)(x^2 - x - 6)$$
$$= 2x^3 - 2x^2 - 12x + x^2 - x - 6$$

**[A1]** Simplify:
$$= 2x^3 - x^2 - 13x - 6$$

**Answer: $2x^3 - x^2 - 13x - 6$**

**(b) [M1]** Factorise out common factor $x$:
$$x^3 - x = x(x^2 - 1)$$

**[M1]** Recognise difference of two squares:
$$x^2 - 1 = x^2 - 1^2 = (x + 1)(x - 1)$$

**[A1]** Complete factorisation:
$$x^3 - x = x(x + 1)(x - 1)$$

**Answer: $x(x + 1)(x - 1)$**

## 📊 SECTION D: EXAM FOCUS & STRATEGY

### Frequently Tested Question Types

| Type | Description | Typical Marks |
|------|-------------|---------------|
| **Expand single bracket** | Multiply out and simplify | 1-2 |
| **Expand double brackets** | FOIL, special cases | 2-3 |
| **Factorise simple quadratics** | $x^2 + bx + c$ | 2-3 |
| **Factorise with coefficient** | $ax^2 + bx + c$ | 3-4 |
| **Solve by factorising** | Rearrange, factorise, solve | 3-4 |
| **Quadratic formula** | Non-factorisable cases | 3-4 |
| **Difference of two squares** | Recognise and apply | 2-3 |
| **Word problems** | Area, numbers, projectiles | 4-6 |
| **Complete the square** | Vertex form, minimum | 3-5 |

### Typical Problem-Solving Patterns

**Pattern 1: The Expand → Simplify**
```
1. Write expression
2. Expand each bracket
3. Collect like terms
4. Check answer
```

**Pattern 2: The Factorise → Solve**
```
1. Rearrange to ax² + bx + c = 0
2. Factorise (find numbers)
3. Set each bracket = 0
4. State both solutions
```

**Pattern 3: Word Problem**
```
1. Define variable
2. Write equation
3. Solve equation
4. Check validity (positive, makes sense)
5. State answer with units
```

### Command Word Guidance

| Command | What Examiners Expect |
|---------|----------------------|
| **Expand** | Remove all brackets, simplify |
| **Factorise** | Express as product of factors/brackets |
| **Solve** | Find all values of the unknown |
| **Simplify** | Collect like terms to simplest form |
| **Express** | Algebraic answer (no numerical evaluation) |
| **Hence** | Must use previous result |
| **Show that** | Full working proving the result |
| **Give your answer to...** | Round to specified accuracy |

### How to Gain Full Marks

**For Expanding (2-3 marks):**
1. ✅ Show all four products (FOIL)
2. ✅ Include signs correctly
3. ✅ Collect like terms
4. ✅ Present final answer clearly

**For Factorising (2-3 marks):**
1. ✅ Check for common factors first
2. ✅ Show the numbers you found (product/sum)
3. ✅ Write the factorised form
4. ✅ Check by expanding back

**For Solving Quadratics (3-4 marks):**
1. ✅ Rearrange to standard form
2. ✅ Show factorisation clearly
3. ✅ Set each bracket = 0
4. ✅ State BOTH solutions
5. ✅ Reject invalid solutions in word problems

**For Word Problems (4-6 marks):**
1. ✅ Define variables clearly (Let x = ...)
2. ✅ Write the equation
3. ✅ Show all working
4. ✅ Check validity of solutions
5. ✅ State final answer with units
6. ✅ Check reasonableness
