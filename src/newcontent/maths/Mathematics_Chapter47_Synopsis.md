# PART 1: CIRCLES

## 📐 Key Terms
| Term | Definition |
|------|------------|
| Radius ($r$) | Centre to edge |
| Diameter ($d$) | Through centre, $d = 2r$ |
| Arc | Part of circumference |
| Sector | "Pizza slice" - 2 radii + arc |
| Segment | Chord + arc |
| Tangent | Line touching circle at one point |

---

## 📏 Essential Formulas

### Basic Circle
$$C = 2\pi r = \pi d$$
$$A = \pi r^2$$

### Arc Length
$$\text{Arc} = \frac{\theta}{360} \times 2\pi r$$

### Sector Area
$$\text{Sector} = \frac{\theta}{360} \times \pi r^2$$

### Segment Area
$$\text{Segment} = \text{Sector} - \text{Triangle}$$
$$= \frac{\theta}{360}\pi r^2 - \frac{1}{2}r^2\sin\theta$$

---

## ⚡ Quick Rules - Circles

- **Perimeter of sector** = Arc + $2r$
- **Angle proportion**: All formulas use $\frac{\theta}{360}$
- **Major vs Minor**: Major angle > 180°
- **Finding radius from area**: $r = \sqrt{\frac{A}{\pi}}$
- **Finding radius from circumference**: $r = \frac{C}{2\pi}$

---

##  Circle Theorems Summary

| Theorem | Rule |
|---------|------|
| Centre vs Circumference | Angle at centre = $2 \times$ angle at circumference |
| Semicircle | Angle in semicircle = $90°$ |
| Same Segment | Angles in same segment are equal |
| Cyclic Quadrilateral | Opposite angles sum to $180°$ |
| Tangent-Radius | Tangent $\perp$ Radius at point of contact |
| Tangents from Point | Two tangents from same point are equal |
| Alternate Segment | Angle between tangent and chord = angle in alternate segment |

# PART 2: SOLIDS

## 📦 Volume Formulas

| Shape | Formula |
|-------|---------|
| **Prism** | $V = \text{Area of cross-section} \times \text{length}$ |
| **Cuboid** | $V = lwh$ |
| **Cylinder** | $V = \pi r^2 h$ |
| **Cone** | $V = \frac{1}{3}\pi r^2 h$ |
| **Pyramid** | $V = \frac{1}{3} \times \text{base area} \times h$ |
| **Sphere** | $V = \frac{4}{3}\pi r^3$ |
| **Hemisphere** | $V = \frac{2}{3}\pi r^3$ |

---

## 📐 Surface Area Formulas

| Shape | Curved SA | Total SA |
|-------|-----------|----------|
| **Cylinder** | $2\pi rh$ | $2\pi r(h + r)$ |
| **Cone** | $\pi rl$ | $\pi r(l + r)$ |
| **Sphere** | — | $4\pi r^2$ |
| **Hemisphere** | $2\pi r^2$ | $3\pi r^2$ |

>  $l$ = slant height, $h$ = vertical height

---

## ⚡ Quick Rules - Solids

### Cone: Finding Slant Height
$$l = \sqrt{r^2 + h^2}$$

### Hollow Objects
$$V_{\text{material}} = V_{\text{outer}} - V_{\text{inner}}$$

### Compound Solids
- **Volume**: Add all parts (subtract hollow sections)
- **Surface Area**: Only count **external** faces

### Hemisphere TSA
- **Curved only**: $2\pi r^2$
- **With base**: $3\pi r^2$

---

## 🔢 Key Volume Relationships

| Relationship | Formula |
|--------------|---------|
| Cone : Cylinder | Cone = $\frac{1}{3}$ of cylinder (same $r$, $h$) |
| Pyramid : Prism | Pyramid = $\frac{1}{3}$ of prism (same base, height) |
| Hemisphere : Sphere | Hemisphere = $\frac{1}{2}$ of sphere |

# PART 3: SIMILARITY

## 📊 Scale Factor Rules

| Given | To Find Area | To Find Volume |
|-------|--------------|----------------|
| Linear SF ($k$) | Multiply by $k^2$ | Multiply by $k^3$ |
| Area ratio | — | Multiply by $(k^2)^{3/2}$ |
| Volume ratio | Multiply by $(k^3)^{2/3}$ | — |

---

## 🎯 The Three Rules

### Rule 1: Length
$$\frac{\text{Length}_2}{\text{Length}_1} = k$$

### Rule 2: Area
$$\frac{\text{Area}_2}{\text{Area}_1} = k^2$$

### Rule 3: Volume
$$\frac{\text{Volume}_2}{\text{Volume}_1} = k^3$$

---

## ⚡ Quick Conversion Methods

| From | To Length | To Area | To Volume |
|------|-----------|---------|-----------|
| **Length ($k$)** | $k$ | $k^2$ | $k^3$ |
| **Area ($k^2$)** | $\sqrt{k^2}$ | $k^2$ | $(\sqrt{k^2})^3$ |
| **Volume ($k^3$)** | $\sqrt[3]{k^3}$ | $(\sqrt[3]{k^3})^2$ | $k^3$ |

---

## 📐 Similar Triangles Conditions

Triangles are similar if they satisfy:
- **AA** - Two angles equal
- **SSS** - All sides in proportion
- **SAS** - Two sides in proportion + included angle equal

# 🎯 EXAM QUICK TIPS

## Common Exam Questions

### Type 1: Arc/Sector Calculation
```
Given: radius r, angle θ
1. Arc = (θ/360) × 2πr
2. Sector = (θ/360) × πr²
3. Perimeter of sector = Arc + 2r
```

### Type 2: Compound Solid Volume
```
1. Identify all shapes
2. Calculate each volume
3. Add (for joined) or subtract (for hollow)
```

### Type 3: Surface Area of Compound Solid
```
1. List all external faces
2. Calculate each area
3. Do NOT include joined/hidden faces
```

### Type 4: Similarity Problems
```
1. Find linear scale factor k
2. For area: use k²
3. For volume: use k³
4. Apply to unknown
```

##  Common Mistakes to Avoid

| Topic | Mistake | Correction |
|-------|---------|------------|
| Circles | Using $d = r$ | $d = 2r$ |
| Circles | Using $\frac{\theta}{2\pi}$ | Use $\frac{\theta}{360}$ for degrees |
| Cone | Using $h$ instead of $l$ in SA | $l = \sqrt{r^2 + h^2}$ |
| Hemisphere | TSA = $2\pi r^2$ | TSA = $3\pi r^2$ (includes base) |
| Similarity | Area SF = $k$ | Area SF = $k^2$ |
| Similarity | Volume SF = $k^2$ | Volume SF = $k^3$ |
