### Question 1
Find the next term in the sequence: $5, 9, 13, 17, 21, ...$

**A)** 24  
**B)** 25  
**C)** 26  
**D)** 27

**Answer: B**

**Explanation:**
- Common difference: $d = 9 - 5 = 4$
- Each term increases by 4
- Next term: $21 + 4 = 25$

### Question 2
What is the nth term of the sequence $3, 8, 13, 18, 23, ...$?

**A)** $5n - 2$  
**B)** $5n + 3$  
**C)** $5n - 3$  
**D)** $4n + 3$

**Answer: A**

**Explanation:**
- Common difference: $d = 8 - 3 = 5$
- nth term starts with $5n$
- For $n = 1$: $u_1 = 3$, so $5(1) + c = 3$
- Therefore $c = 3 - 5 = -2$
- nth term: $u_n = 5n - 2$

**Check:** $n = 2$: $5(2) - 2 = 8$ ✓

### Question 6
Is 100 a term in the sequence $u_n = 7n - 3$?

**A)** Yes, it is the 14th term  
**B)** Yes, it is the 15th term

**D)** No, because the sequence is decreasing

### Question 7
The first term of an arithmetic sequence is 8 and the common difference is 5. What is the nth term?

**A)** $5n + 3$  
**B)** $5n + 8$  
**C)** $8n + 5$  
**D)** $5n - 3$

**Answer: A**

**Explanation:**
Using $u_n = a + (n-1)d$:
$$u_n = 8 + (n-1)(5)$$
$$u_n = 8 + 5n - 5$$
$$u_n = 5n + 3$$

### Question 8
Find the sum of the first 10 terms of the arithmetic sequence $2, 5, 8, 11, ...$

**A)** 145  
**B)** 155  
**C)** 165  
**D)** 175

**Answer: B**

**Explanation:**
$a = 2$, $d = 3$, $n = 10$

$$S_n = \frac{n}{2}[2a + (n-1)d]$$
$$S_{10} = \frac{10}{2}[2(2) + 9(3)]$$
$$S_{10} = 5[4 + 27]$$
$$S_{10} = 5 \times 31 = 155$$

What type of sequence is: $1, 4, 9, 16, 25, ...$?

**A)** Arithmetic (linear)  
**B)** Geometric  
**C)** Quadratic  
**D)** Fibonacci

**Answer: C**

**Explanation:**
Check differences:

| Term | 1 | 4 | 9 | 16 | 25 |
|---|---|---|---|---|---|
| 1st diff | | 3 | 5 | 7 | 9 |
| 2nd diff | | | 2 | 2 | 2 |

Constant 2nd difference = **quadratic sequence**.

(This is the sequence of square numbers: $u_n = n^2$)

### Question 10
The sum of the first $n$ terms of an arithmetic sequence is $S_n = n^2 + 2n$. Find the 5th term.

**A)** 10  
**B)** 11  
**C)** 12  
**D)** 13

**Answer: B**

**Explanation:**
$$u_n = S_n - S_{n-1}$$
$$u_5 = S_5 - S_4$$
$$S_5 = 5^2 + 2(5) = 25 + 10 = 35$$
$$S_4 = 4^2 + 2(4) = 16 + 8 = 24$$
$$u_5 = 35 - 24 = 11$$

## SECTION B: TRUE OR FALSE (10 Questions)

*State whether each statement is TRUE or FALSE. If FALSE, give the correct statement.*

---

### Question 11
**Statement:** In an arithmetic sequence, the difference between consecutive terms is always constant.

**TRUE**

This is the definition of an arithmetic sequence.

---

### Question 12
**Statement:** If the nth term of a sequence is $u_n = 3n - 2$, then the first term is 3.

**FALSE**

**Correction:** The first term is $u_1 = 3(1) - 2 = 1$.

---

### Question 13
**Statement:** The sum of the first $n$ natural numbers is $\frac{n(n+1)}{2}$.

**TRUE**

For sequence $1, 2, 3, 4, ..., n$:
- $a = 1$, $d = 1$
- $S_n = \frac{n}{2}[2(1) + (n-1)(1)] = \frac{n}{2}(n+1) = \frac{n(n+1)}{2}$

---

### Question 14
**Statement:** A number can only be a term in a sequence if the position number $n$ is a positive integer.

**TRUE**

The position $n$ must be a positive integer (1, 2, 3, ...) for the number to be in the sequence.

---

### Question 15
**Statement:** In a quadratic sequence, the first difference is always constant.

**FALSE**

**Correction:** In a quadratic sequence, the **second** difference is always constant, not the first difference.

---

### Question 16
**Statement:** The sequence $2, 4, 8, 16, 32, ...$ is an arithmetic sequence.

**FALSE**

**Correction:** This is a **geometric sequence** (each term is multiplied by 2). In an arithmetic sequence, terms would be added, not multiplied.

---

### Question 17
**Statement:** For an arithmetic sequence with common difference $d = -2$, the sequence is decreasing.

**TRUE**

A negative common difference means each term is less than the previous term, so the sequence decreases.

---

### Question 18
**Statement:** The formula $u_n = a + (n-1)d$ and $u_n = dn + (a-d)$ are equivalent.

**TRUE**

Expanding $u_n = a + (n-1)d$:
$$u_n = a + nd - d = nd + (a - d)$$

Both formulas give the same result.

---

### Question 19
**Statement:** The sum of the first $n$ even numbers is $n(n+1)$.

**TRUE**

Even numbers: $2, 4, 6, 8, ...$
- $a = 2$, $d = 2$
- $S_n = \frac{n}{2}[2(2) + (n-1)(2)] = \frac{n}{2}[4 + 2n - 2] = \frac{n}{2}(2n + 2) = n(n+1)$

---

### Question 20
**Statement:** If $S_n = 100$ and $n = 10$, the average of all terms is 10.

**TRUE**

Average = $\frac{S_n}{n} = \frac{100}{10} = 10$

### Question 23 (4 marks)

The first four terms of a sequence are: $5, 9, 15, 23, 33, ...$

**(a)** Show that this is a quadratic sequence. **(2 marks)**

**(b)** Find the nth term. **(4 marks)**

**(c)** Find the 10th term. **(1 mark)**

**(a) Show quadratic [2 marks]**

| Mark | Solution |
|---|---|
| M1 | Correct first differences calculated |
| A1 | Correct second differences shown to be constant |

**Difference table:**

| $n$ | 1 | 2 | 3 | 4 | 5 |
|---|---|---|---|---|---|
| Term | 5 | 9 | 15 | 23 | 33 |
| 1st diff | | 4 | 6 | 8 | 10 |
| 2nd diff | | | 2 | 2 | 2 |

The second difference is constant (= 2), therefore it is a **quadratic sequence**.

---

**(b) Find nth term [4 marks]**

| Mark | Solution |
|---|---|
| M1 | Finding $a = \frac{\text{2nd diff}}{2} = 1$ |
| M1 | Comparing with $n^2$ |
| M1 | Finding the linear part |
| A1 | Correct formula: $u_n = n^2 + n + 3$ |

**Step 1:** Find $a$
$$a = \frac{\text{2nd difference}}{2} = \frac{2}{2} = 1$$

So the formula starts with $n^2$.

**Step 2:** Compare with $n^2$

| $n$ | 1 | 2 | 3 | 4 |
|---|---|---|---|---|
| Term | 5 | 9 | 15 | 23 |
| $n^2$ | 1 | 4 | 9 | 16 |
| Difference | 4 | 5 | 6 | 7 |

The difference increases by 1 each time: it's a linear sequence $n + 3$.

**Step 3:** Complete formula
$$u_n = n^2 + n + 3$$

**Check:** $n = 1: 1 + 1 + 3 = 5$ ✓

### Question 24 (5 marks)

The sum of the first $n$ terms of an arithmetic sequence is given by $S_n = 3n^2 + 2n$.

**(a)** Find the first term. **(1 mark)**

**(b)** Find the second term. **(1 mark)**

**(c)** Find a formula for the nth term. **(3 marks)**

**(d)** Find the sum of the 10th to 20th terms (inclusive). **(3 marks)**

**(a) First term [1 mark]**

| Mark | Solution |
|---|---|
| A1 | Correct answer: 5 |

$$u_1 = S_1 = 3(1)^2 + 2(1) = 3 + 2 = 5$$

---

**(b) Second term [1 mark]**

| Mark | Solution |
|---|---|
| M1 | Using $u_2 = S_2 - S_1$ |
| A1 | Correct answer: 11 |

$$S_2 = 3(2)^2 + 2(2) = 12 + 4 = 16$$
$$u_2 = S_2 - S_1 = 16 - 5 = 11$$

---

**(c) Formula for nth term [3 marks]**

| Mark | Solution |
|---|---|
| M1 | Using $u_n = S_n - S_{n-1}$ |
| M1 | Correct algebraic expansion |
| A1 | Correct formula: $u_n = 6n - 1$ |

$$u_n = S_n - S_{n-1}$$
$$u_n = (3n^2 + 2n) - [3(n-1)^2 + 2(n-1)]$$
$$u_n = 3n^2 + 2n - [3(n^2 - 2n + 1) + 2n - 2]$$
$$u_n = 3n^2 + 2n - [3n^2 - 6n + 3 + 2n - 2]$$
$$u_n = 3n^2 + 2n - 3n^2 + 4n - 1$$
$$u_n = 6n - 1$$

**Verification:** $u_1 = 5$, $u_2 = 11$ ✓

---

**(d) Sum of 10th to 20th terms [3 marks]**

| Mark | Solution |
|---|---|
| M1 | Using $S_{20} - S_9$ |
| M1 | Correct substitution |
| A1 | Correct answer: 897 |

Sum of 10th to 20th terms = $S_{20} - S_9$

$$S_{20} = 3(20)^2 + 2(20) = 1200 + 40 = 1240$$
$$S_9 = 3(9)^2 + 2(9) = 243 + 18 = 261$$
$$\text{Sum} = 1240 - 261 = 979$$

Wait, let me recalculate:
$$S_{20} = 3(400) + 40 = 1200 + 40 = 1240$$
$$S_9 = 3(81) + 18 = 243 + 18 = 261$$
$$S_{20} - S_9 = 1240 - 261 = 979$$

**Answer:** 979

### Question 25 (6 marks)

A company's profits over several years are modeled by an arithmetic sequence. In the first year, the profit was £50,000. In the fifth year, the profit was £78,000.

**(a)** Find the common difference. **(2 marks)**

**(b)** Write an expression for the profit in the nth year. **(2 marks)**

**(c)** In which year will the profit first exceed £120,000? **(3 marks)**

**(d)** Calculate the total profit over the first 15 years. **(3 marks)**

**(a) Common difference [2 marks]**

| Mark | Solution |
|---|---|
| M1 | Using $u_5 = a + 4d$ |
| A1 | Correct answer: £7,000 |

$$u_5 = a + 4d$$
$$78,000 = 50,000 + 4d$$
$$4d = 28,000$$
$$d = 7,000$$

---

**(b) Expression for nth year [2 marks]**

| Mark | Solution |
|---|---|
| M1 | Using $u_n = a + (n-1)d$ |
| A1 | Correct expression: $u_n = 7000n + 43,000$ |

$$u_n = 50,000 + (n-1)(7,000)$$
$$u_n = 50,000 + 7,000n - 7,000$$
$$u_n = 7,000n + 43,000$$

---

**(c) Year when profit exceeds £120,000 [3 marks]**

| Mark | Solution |
|---|---|
| M1 | Setting up inequality or equation |
| M1 | Solving correctly |
| A1 | Correct answer with reasoning: Year 12 |

$$7,000n + 43,000 > 120,000$$
$$7,000n > 77,000$$
$$n > 11$$

Since $n$ must be an integer, $n = 12$.

**Answer:** In the 12th year.

---

**(d) Total profit over 15 years [3 marks]**

| Mark | Solution |
|---|---|
| M1 | Correct sum formula |
| M1 | Correct substitution |
| A1 | Correct answer: £1,395,000 |

$$S_n = \frac{n}{2}[2a + (n-1)d]$$
$$S_{15} = \frac{15}{2}[2(50,000) + 14(7,000)]$$
$$S_{15} = \frac{15}{2}[100,000 + 98,000]$$
$$S_{15} = \frac{15}{2}(198,000)$$
$$S_{15} = 15 \times 99,000$$
$$S_{15} = 1,485,000$$

**Answer:** £1,485,000

### Question 26 (5 marks)

**Prove that the sum of the first $n$ odd numbers is $n^2$.**

| Mark | Solution |
|---|---|
| M1 | Identifying the sequence of odd numbers |
| M1 | Stating $a = 1$, $d = 2$ |
| M1 | Using sum formula |
| M1 | Correct simplification |
| A1 | Correct proof with conclusion |

**Proof:**

The odd numbers form an arithmetic sequence: $1, 3, 5, 7, ...$

- First term: $a = 1$
- Common difference: $d = 2$

Using the sum formula:
$$S_n = \frac{n}{2}[2a + (n-1)d]$$
$$S_n = \frac{n}{2}[2(1) + (n-1)(2)]$$
$$S_n = \frac{n}{2}[2 + 2n - 2]$$
$$S_n = \frac{n}{2}[2n]$$
$$S_n = n^2$$

**Therefore, the sum of the first $n$ odd numbers is $n^2$.**

### Question 28 (5 marks)

Two sequences are defined by:
- Sequence A: $u_n = 3n + 1$
- Sequence B: $v_n = 2n + 5$

Find the value of $n$ for which $u_n = v_n$, and find the sum of the first $n$ terms of Sequence A up to this point.

| Mark | Solution |
|---|---|
| M1 | Setting $u_n = v_n$ |
| M1 | Solving for $n$ |
| A1 | Correct value: $n = 4$ |
| M1 | Using sum formula for Sequence A |
| A1 | Correct sum: 34 |

**Finding where $u_n = v_n$:**

$$3n + 1 = 2n + 5$$
$$n = 4$$

**Sum of first 4 terms of Sequence A:**

Terms: $u_1 = 4$, $u_2 = 7$, $u_3 = 10$, $u_4 = 13$

$$S_4 = \frac{4}{2}(4 + 13)$$
$$S_4 = 2 \times 17$$
$$S_4 = 34$$

**Alternative using formula:**
$$S_n = \frac{n}{2}[2a + (n-1)d]$$
For sequence A: $a = 4$, $d = 3$
$$S_4 = \frac{4}{2}[2(4) + 3(3)] = 2[8 + 9] = 34$$

### 🎯 Typical Problem-Solving Patterns

#### Pattern 1: "Find which term..."
1. Set nth term = given value
2. Solve for $n$
3. Check if $n$ is a positive integer

#### Pattern 2: "Given two terms, find..."
1. Use $u_m = a + (m-1)d$
2. Use $u_n = a + (n-1)d$
3. Solve simultaneous equations

#### Pattern 3: "Sum between two positions"
1. Calculate $S_{\text{end}}$
2. Calculate $S_{\text{start}-1}$
3. Subtract: $S_{\text{end}} - S_{\text{start}-1}$
