# Topic 1: Direct Proportion

## 1.1 Understanding Direct Proportion

### Definition
Two quantities are in **direct proportion** if they increase or decrease at the same rate. As one quantity increases, the other increases in the same ratio.

### Mathematical Representation
When $y$ is directly proportional to $x$, we write:

$$y \propto x$$

This means: $y = kx$ where $k$ is the **constant of proportionality**.

---

## 1.2 Key Characteristics of Direct Proportion

| Feature | Description |
|---------|-------------|
| **Relationship** | $y = kx$ |
| **Graph** | Straight line through the origin |
| **Gradient** | Equal to $k$ (constant of proportionality) |
| **Ratio** | $\frac{y}{x} = k$ (constant for all values) |
| **Origin** | Always passes through $(0, 0)$ |

---

## 1.3 Step-by-Step Method for Direct Proportion Problems

### Method 1: Using the Constant of Proportionality

**Step 1:** Write down the proportionality statement: $y \propto x$

**Step 2:** Convert to an equation: $y = kx$

**Step 3:** Substitute known values to find $k$

**Step 4:** Rewrite the equation with the value of $k$

**Step 5:** Substitute the new value to find the unknown

---

### Worked Example 1

**Question:** $y$ is directly proportional to $x$. When $x = 4$, $y = 12$. Find:
(a) The equation connecting $y$ and $x$
(b) The value of $y$ when $x = 7$
(c) The value of $x$ when $y = 27$

**Solution:**

**Step 1:** Write the relationship
$$y \propto x$$

**Step 2:** Write as an equation
$$y = kx$$

**Step 3:** Find $k$ using $x = 4$ and $y = 12$
$$12 = k \times 4$$
$$k = \frac{12}{4} = 3$$

**Step 4:** The equation is:
$$y = 3x$$

**(a)** The equation is $\boxed{y = 3x}$

**(b)** When $x = 7$:
$$y = 3 \times 7 = 21$$

**(c)** When $y = 27$:
$$27 = 3x$$
$$x = \frac{27}{3} = 9$$

---

### Worked Example 2 (Real-World Context)

**Question:** The cost of fuel is directly proportional to the number of litres purchased. 20 litres cost £28.
(a) Find the cost of 35 litres
(b) How many litres can be bought for £49?

**Solution:**

Let $C$ = cost and $L$ = litres

$$C \propto L \Rightarrow C = kL$$

Find $k$: When $L = 20$, $C = 28$
$$28 = k \times 20$$
$$k = \frac{28}{20} = 1.4$$

So: $C = 1.4L$

**(a)** When $L = 35$:
$$C = 1.4 \times 35 = £49$$

**(b)** When $C = 49$:
$$49 = 1.4L$$
$$L = \frac{49}{1.4} = 35 \text{ litres}$$

---

## 1.4 Graphs of Direct Proportion

### Key Graph Features

1. **Straight line** - always
2. **Passes through origin** $(0, 0)$
3. **Gradient = k** (constant of proportionality)

### Interpreting Graphs

| Graph Type | Meaning |
|------------|---------|
| Steeper line | Larger value of $k$ |
| Less steep line | Smaller value of $k$ |
| Line in Quadrant I only | $k > 0$ (both $x$ and $y$ positive) |
| Line through Quadrants I and III | $k > 0$ |
| Line through Quadrants II and IV | $k < 0$ |

---

### Worked Example 3 (Graph-Based)

**Question:** The graph shows $y$ against $x$ where $y \propto x$. The graph passes through the point $(6, 15)$.
(a) Find the equation of the graph
(b) Calculate the value of $y$ when $x = 10$

**Solution:**

**(a)** Since $y \propto x$: $y = kx$

Using point $(6, 15)$:
$$15 = k \times 6$$
$$k = \frac{15}{6} = 2.5$$

**Equation:** $\boxed{y = 2.5x}$

**(b)** When $x = 10$:
$$y = 2.5 \times 10 = 25$$

---

## 🔴 Common Mistakes with Direct Proportion

| Mistake | Correction |
|---------|------------|
| Writing $y \propto x$ without converting to $y = kx$ | Always convert to an equation before solving |
| Forgetting to find $k$ first | $k$ must be found using known values |
| Not checking units are consistent | Ensure all values use the same units |
| Assuming $y = mx + c$ applies | Direct proportion has $c = 0$ |
| Mixing up which variable to solve for | Read the question carefully |

# Topic 2: Inverse Proportion

## 2.1 Understanding Inverse Proportion

### Definition
Two quantities are in **inverse proportion** if one increases as the other decreases in the same ratio. As one quantity doubles, the other halves.

### Mathematical Representation
When $y$ is inversely proportional to $x$, we write:

$$y \propto \frac{1}{x}$$

This means: $y = \frac{k}{x}$ where $k$ is the constant of proportionality.

---

## 2.2 Key Characteristics of Inverse Proportion

| Feature | Description |
|---------|-------------|
| **Relationship** | $y = \frac{k}{x}$ or $xy = k$ |
| **Graph** | Curve (hyperbola) |
| **Product** | $xy = k$ (constant for all values) |
| **Behavior** | As $x$ increases, $y$ decreases |
| **Asymptotes** | Approaches axes but never touches them |

---

## 2.3 Step-by-Step Method for Inverse Proportion Problems

### Method 1: Using $y = \frac{k}{x}$

**Step 1:** Write the proportionality statement: $y \propto \frac{1}{x}$

**Step 2:** Convert to an equation: $y = \frac{k}{x}$

**Step 3:** Substitute known values to find $k$

**Step 4:** Rewrite the equation with the value of $k$

**Step 5:** Substitute to find unknowns

### Method 2: Using $xy = k$

This is often quicker: find $k$ by multiplying $x$ and $y$ for any pair of values.

---

### Worked Example 4

**Question:** $y$ is inversely proportional to $x$. When $x = 3$, $y = 8$. Find:
(a) The equation connecting $y$ and $x$
(b) The value of $y$ when $x = 4$
(c) The value of $x$ when $y = 2$

**Solution:**

**Step 1:** Write the relationship
$$y \propto \frac{1}{x}$$

**Step 2:** Write as equation
$$y = \frac{k}{x}$$

**Step 3:** Find $k$ using $x = 3$, $y = 8$
$$8 = \frac{k}{3}$$
$$k = 24$$

**Alternative using Method 2:**
$$k = xy = 3 \times 8 = 24$$

**Step 4:** The equation is:
$$y = \frac{24}{x}$$

**(a)** The equation is $\boxed{y = \frac{24}{x}}$ or $xy = 24$

**(b)** When $x = 4$:
$$y = \frac{24}{4} = 6$$

**(c)** When $y = 2$:
$$2 = \frac{24}{x}$$
$$x = \frac{24}{2} = 12$$

---

### Worked Example 5 (Real-World Context)

**Question:** The time taken to complete a journey is inversely proportional to the speed. A journey takes 3 hours at 60 km/h.
(a) How long would it take at 90 km/h?
(b) What speed is needed to complete the journey in 2.5 hours?

**Solution:**

Let $T$ = time and $S$ = speed

$$T \propto \frac{1}{S} \Rightarrow T = \frac{k}{S}$$

Find $k$: When $S = 60$, $T = 3$
$$k = T \times S = 3 \times 60 = 180$$

So: $T = \frac{180}{S}$ or $TS = 180$

**(a)** When $S = 90$:
$$T = \frac{180}{90} = 2 \text{ hours}$$

**(b)** When $T = 2.5$:
$$2.5 \times S = 180$$
$$S = \frac{180}{2.5} = 72 \text{ km/h}$$

---

## 2.4 Graphs of Inverse Proportion

### Key Graph Features

1. **Curved graph** - a hyperbola
2. **Never passes through origin**
3. **Asymptotic to both axes**
4. **Quadrant I only** (when $k > 0$)

### Shape Recognition

| Feature | Description |
|---------|-------------|
| **As $x \to 0$** | $y \to \infty$ (curve approaches y-axis) |
| **As $x \to \infty$** | $y \to 0$ (curve approaches x-axis) |
| **Asymptotes** | $x = 0$ and $y = 0$ |

---

### Worked Example 6 (Graph Interpretation)

**Question:** A graph shows $y$ against $x$ where $y$ is inversely proportional to $x$. The graph passes through $(4, 6)$.
(a) Write down the equation
(b) Sketch the graph for $x > 0$

**Solution:**

**(a)** $y = \frac{k}{x}$ where $k = 4 \times 6 = 24$

**Equation:** $\boxed{y = \frac{24}{x}}$

**(b)** The graph is a hyperbola in Quadrant I:
- Passes through $(4, 6)$
- Approaches the axes but never touches them
- Decreasing curve (as $x$ increases, $y$ decreases)

---

## 🔴 Common Mistakes with Inverse Proportion

| Mistake | Correction |
|---------|------------|
| Using $y = kx$ instead of $y = \frac{k}{x}$ | Recognise the keyword "inversely" |
| Forgetting that $xy = k$ (constant) | This is a useful check |
| Drawing a straight line graph | Inverse proportion gives a curve |
| Not recognising asymptotic behaviour | The curve never crosses the axes |
| Confusing "inversely proportional" with "directly proportional" | Read keywords carefully |

# Topic 3: Fractional Indices

## 3.1 Understanding Fractional Indices

### Definition
Fractional indices represent **roots** of numbers. The denominator of the fraction indicates the type of root.

### Fundamental Rule

$$a^{\frac{1}{n}} = \sqrt[n]{a}$$

This means: $a^{\frac{1}{2}} = \sqrt{a}$, $a^{\frac{1}{3}} = \sqrt[3]{a}$, etc.

---

## 3.2 Key Rules for Fractional Indices

| Rule | Formula | Example |
|------|---------|---------|
| **Square root** | $a^{\frac{1}{2}} = \sqrt{a}$ | $9^{\frac{1}{2}} = 3$ |
| **Cube root** | $a^{\frac{1}{3}} = \sqrt[3]{a}$ | $8^{\frac{1}{3}} = 2$ |
| **nth root** | $a^{\frac{1}{n}} = \sqrt[n]{a}$ | $32^{\frac{1}{5}} = 2$ |
| **Power and root** | $a^{\frac{m}{n}} = (\sqrt[n]{a})^m$ | $8^{\frac{2}{3}} = 4$ |

---

## 3.3 Evaluating $a^{\frac{m}{n}}$

### Step-by-Step Method

**Step 1:** Identify $m$ (numerator) and $n$ (denominator)

**Step 2:** Take the $n$th root first (denominator)

**Step 3:** Raise the result to power $m$ (numerator)

### Formula
$$a^{\frac{m}{n}} = \left(\sqrt[n]{a}\right)^m = \sqrt[n]{a^m}$$

---

### Worked Example 7

**Question:** Evaluate without a calculator:
(a) $16^{\frac{1}{2}}$
(b) $27^{\frac{1}{3}}$
(c) $32^{\frac{1}{5}}$

**Solution:**

**(a)** $16^{\frac{1}{2}} = \sqrt{16} = \boxed{4}$

**(b)** $27^{\frac{1}{3}} = \sqrt[3]{27} = \boxed{3}$
   (Since $3^3 = 27$)

**(c)** $32^{\frac{1}{5}} = \sqrt[5]{32} = \boxed{2}$
   (Since $2^5 = 32$)

---

### Worked Example 8

**Question:** Evaluate without a calculator:
(a) $8^{\frac{2}{3}}$
(b) $16^{\frac{3}{2}}$
(c) $125^{\frac{2}{3}}$

**Solution:**

**(a)** $8^{\frac{2}{3}}$
- Method: Take cube root, then square
- Cube root of 8 = 2 (since $2^3 = 8$)
- Square 2 = 4
- **Answer:** $\boxed{4}$

**(b)** $16^{\frac{3}{2}}$
- Method: Take square root, then cube
- Square root of 16 = 4
- Cube 4 = 64
- **Answer:** $\boxed{64}$

**(c)** $125^{\frac{2}{3}}$
- Method: Take cube root, then square
- Cube root of 125 = 5 (since $5^3 = 125$)
- Square 5 = 25
- **Answer:** $\boxed{25}$

---

## 3.4 Properties of Fractional Indices

### Multiplication Rule
$$a^{\frac{1}{n}} \times a^{\frac{1}{n}} = a^{\frac{1}{n} + \frac{1}{n}} = a^{\frac{2}{n}}$$

### Power of a Power
$$\left(a^{\frac{m}{n}}\right)^p = a^{\frac{mp}{n}}$$

### Worked Example 9

**Question:** Simplify $\left(4^{\frac{1}{2}}\right)^3$

**Solution:**
$$\left(4^{\frac{1}{2}}\right)^3 = 4^{\frac{3}{2}}$$

Evaluate: $\sqrt{4} = 2$, then $2^3 = 8$

**Alternative:** $4^{\frac{1}{2}} = 2$, then $2^3 = 8$

**Answer:** $\boxed{8}$

---

## 🔴 Common Mistakes with Fractional Indices

| Mistake | Correction |
|---------|------------|
| Calculating power before root | Denominator = root first, numerator = power second |
| $a^{\frac{1}{2}} = a \div 2$ | $a^{\frac{1}{2}} = \sqrt{a}$ |
| Forgetting perfect squares/cubes | Learn: 1, 4, 9, 16, 25, 36, 49, 64, 81, 100, 121, 144 |
| $a^{\frac{2}{3}} = (a^2)^{\frac{1}{3}}$ (wrong order) | Can do either order, but $\sqrt[3]{a}$ first is often easier |

# Topic 4: Negative Indices

## 4.1 Understanding Negative Indices

### Definition
A negative index indicates the **reciprocal** of the positive power.

### Fundamental Rule

$$a^{-n} = \frac{1}{a^n}$$

This means we take the reciprocal and change the sign of the index.

---

## 4.2 Key Rules for Negative Indices

| Rule | Formula | Example |
|------|---------|---------|
| **Basic rule** | $a^{-n} = \frac{1}{a^n}$ | $2^{-3} = \frac{1}{8}$ |
| **Reciprocal rule** | $\frac{1}{a^{-n}} = a^n$ | $\frac{1}{3^{-2}} = 9$ |
| **Fraction rule** | $\left(\frac{a}{b}\right)^{-n} = \left(\frac{b}{a}\right)^n$ | $\left(\frac{2}{3}\right)^{-1} = \frac{3}{2}$ |
| **Zero power** | $a^0 = 1$ (for $a \neq 0$) | $5^0 = 1$ |

---

## 4.3 Step-by-Step Method for Negative Indices

**Step 1:** Write as a reciprocal with positive index

**Step 2:** Evaluate the positive power

**Step 3:** Simplify if necessary

---

### Worked Example 10

**Question:** Evaluate:
(a) $2^{-3}$
(b) $5^{-2}$
(c) $10^{-1}$

**Solution:**

**(a)** $2^{-3} = \frac{1}{2^3} = \frac{1}{8}$

**(b)** $5^{-2} = \frac{1}{5^2} = \frac{1}{25}$

**(c)** $10^{-1} = \frac{1}{10^1} = \frac{1}{10} = 0.1$

---

### Worked Example 11 (Fractions with Negative Indices)

**Question:** Evaluate:
(a) $\left(\frac{2}{3}\right)^{-1}$
(b) $\left(\frac{3}{4}\right)^{-2}$
(c) $\left(\frac{1}{5}\right)^{-3}$

**Solution:**

**(a)** $\left(\frac{2}{3}\right)^{-1} = \frac{3}{2} = 1\frac{1}{2}$
   (Reciprocal of the fraction)

**(b)** $\left(\frac{3}{4}\right)^{-2} = \left(\frac{4}{3}\right)^2 = \frac{16}{9} = 1\frac{7}{9}$

**(c)** $\left(\frac{1}{5}\right)^{-3} = 5^3 = 125$

---

## 4.4 Combining Negative and Fractional Indices

### The Complete Framework

When both negative and fractional indices appear:

$$a^{-\frac{m}{n}} = \frac{1}{a^{\frac{m}{n}}} = \frac{1}{(\sqrt[n]{a})^m}$$

---

### Worked Example 12

**Question:** Evaluate without a calculator:
(a) $4^{-\frac{1}{2}}$
(b) $8^{-\frac{2}{3}}$
(c) $16^{-\frac{3}{4}}$

**Solution:**

**(a)** $4^{-\frac{1}{2}} = \frac{1}{4^{\frac{1}{2}}} = \frac{1}{\sqrt{4}} = \frac{1}{2}$

**(b)** $8^{-\frac{2}{3}}$
- Method: $8^{\frac{2}{3}} = (\sqrt[3]{8})^2 = 2^2 = 4$
- Therefore: $8^{-\frac{2}{3}} = \frac{1}{4}$

**(c)** $16^{-\frac{3}{4}}$
- Method: $16^{\frac{3}{4}} = (\sqrt[4]{16})^3 = 2^3 = 8$
- Therefore: $16^{-\frac{3}{4}} = \frac{1}{8}$

---

## 4.5 Scientific Notation and Negative Indices

### Understanding Small Numbers

$$10^{-1} = 0.1$$
$$10^{-2} = 0.01$$
$$10^{-3} = 0.001$$
$$10^{-6} = 0.000001$$

### Worked Example 13

**Question:** Express as decimals:
(a) $3.2 \times 10^{-2}$
(b) $4.7 \times 10^{-4}$

**Solution:**

**(a)** $3.2 \times 10^{-2} = 3.2 \times 0.01 = 0.032$

**(b)** $4.7 \times 10^{-4} = 4.7 \times 0.0001 = 0.00047$

---

## 🔴 Common Mistakes with Negative Indices

| Mistake | Correction |
|---------|------------|
| $a^{-n} = -a^n$ | $a^{-n} = \frac{1}{a^n}$ (reciprocal, not negative) |
| $\left(\frac{2}{3}\right)^{-1} = -\frac{2}{3}$ | $\left(\frac{2}{3}\right)^{-1} = \frac{3}{2}$ |
| Forgetting to flip fractions | $\left(\frac{a}{b}\right)^{-n} = \left(\frac{b}{a}\right)^n$ |
| $0^0$ is undefined | Avoid this case |
| Mixing up $a^{-2}$ and $a^{\frac{1}{2}}$ | Negative = reciprocal; fractional = root |

# Topic 5: Combined Rules and Problem Solving

## 5.1 Laws of Indices Summary

| Law | Rule | Example |
|-----|------|---------|
| **Multiplication** | $a^m \times a^n = a^{m+n}$ | $2^3 \times 2^4 = 2^7$ |
| **Division** | $a^m \div a^n = a^{m-n}$ | $2^5 \div 2^2 = 2^3$ |
| **Power of power** | $(a^m)^n = a^{mn}$ | $(2^3)^2 = 2^6$ |
| **Zero index** | $a^0 = 1$ | $5^0 = 1$ |
| **Negative index** | $a^{-n} = \frac{1}{a^n}$ | $2^{-3} = \frac{1}{8}$ |
| **Fractional index** | $a^{\frac{1}{n}} = \sqrt[n]{a}$ | $8^{\frac{1}{3}} = 2$ |

---

## 5.2 Complex Problems

### Worked Example 14 (Multi-Step)

**Question:** Evaluate: $27^{\frac{2}{3}} \times 9^{-\frac{1}{2}}$

**Solution:**

First, evaluate each part:

$27^{\frac{2}{3}} = (\sqrt[3]{27})^2 = 3^2 = 9$

$9^{-\frac{1}{2}} = \frac{1}{\sqrt{9}} = \frac{1}{3}$

Multiply: $9 \times \frac{1}{3} = 3$

**Answer:** $\boxed{3}$

---

### Worked Example 15 (Simplification)

**Question:** Simplify: $\frac{a^3 \times a^{-2}}{a^{-1}}$

**Solution:**

**Method 1: Using laws of indices**
$$\frac{a^3 \times a^{-2}}{a^{-1}} = \frac{a^{3+(-2)}}{a^{-1}} = \frac{a^1}{a^{-1}} = a^{1-(-1)} = a^2$$

**Method 2: Step by step**
$$a^3 \times a^{-2} = a^{3-2} = a^1 = a$$
$$\frac{a}{a^{-1}} = a \times a^1 = a^2$$

**Answer:** $\boxed{a^2}$

---

### Worked Example 16 (Word Problem - Proportion)

**Question:** The force between two objects is inversely proportional to the square of the distance between them. When the distance is 2 m, the force is 50 N. Find the force when the distance is 5 m.

**Solution:**

Let $F$ = force and $d$ = distance

$$F \propto \frac{1}{d^2}$$

So: $F = \frac{k}{d^2}$

Find $k$: When $d = 2$, $F = 50$
$$50 = \frac{k}{2^2} = \frac{k}{4}$$
$$k = 200$$

Equation: $F = \frac{200}{d^2}$

When $d = 5$:
$$F = \frac{200}{5^2} = \frac{200}{25} = 8 \text{ N}$$

**Answer:** $\boxed{8 \text{ N}}$

---

### Worked Example 17 (Inverse Proportion with Powers)

**Question:** $y$ is inversely proportional to $x^2$. When $x = 3$, $y = 4$. Find $y$ when $x = 6$.

**Solution:**

$$y \propto \frac{1}{x^2} \Rightarrow y = \frac{k}{x^2}$$

Find $k$: When $x = 3$, $y = 4$
$$4 = \frac{k}{3^2} = \frac{k}{9}$$
$$k = 36$$

Equation: $y = \frac{36}{x^2}$

When $x = 6$:
$$y = \frac{36}{6^2} = \frac{36}{36} = 1$$

**Answer:** $\boxed{y = 1}$

## ⭐ Key Exam Insights

1. **Direct proportion questions**: Look for keywords like "directly proportional", "varies directly", or the symbol $\propto$

2. **Inverse proportion questions**: Look for "inversely proportional", "varies inversely", or phrases like "as one increases, the other decreases"

3. **Finding $k$**: Always the first step - substitute known values

4. **Graph work**: 
   - Direct proportion → straight line through origin
   - Inverse proportion → curve (hyperbola)

5. **Indices**: 
   - Negative means reciprocal
   - Fractional denominator means root
   - Work systematically: roots first, powers second

6. **Show your working**: Marks are awarded for method, not just answers
