# 1. Distance-Time Graphs

## Key Points
- **Horizontal axis:** Time
- **Vertical axis:** Distance from starting point
- **Gradient = Speed**

| Line Type | Meaning |
|-----------|---------|
| **Positive slope** | Moving away from start |
| **Horizontal line** | Stationary (speed = 0) |
| **Steeper slope** | Moving faster |
| **Negative slope** | Returning toward start |
| **Curved line** | Speed is changing (accelerating/decelerating) |

## Instantaneous Speed (Curved Graphs)
- Draw a **tangent** at the point
- Calculate **gradient of tangent**

---

# 2. Speed Formulas

## Core Formula
$$\text{Speed} = \frac{\text{Distance}}{\text{Time}}$$

## Rearranged Forms
$$\text{Distance} = \text{Speed} \times \text{Time}$$
$$\text{Time} = \frac{\text{Distance}}{\text{Speed}}$$

## Average Speed
$$\text{Average Speed} = \frac{\text{Total Distance}}{\text{Total Time}}$$

>  **NOT** the average of individual speeds!

---

# 3. Unit Conversions

## Speed Units
| From | To | Multiply by |
|------|-----|-------------|
| km/h | m/s | $\frac{5}{18}$ or divide by 3.6 |
| m/s | km/h | $\frac{18}{5}$ or × 3.6 |

## Quick Examples
- $36 \text{ km/h} = 10 \text{ m/s}$
- $20 \text{ m/s} = 72 \text{ km/h}$
- $54 \text{ km/h} = 15 \text{ m/s}$

# 4. Speed-Time Graphs

## Key Points
- **Horizontal axis:** Time
- **Vertical axis:** Speed
- **Gradient = Acceleration**
- **Area under graph = Distance travelled**

| Line Type | Meaning |
|-----------|---------|
| **Horizontal line** | Constant speed (a = 0) |
| **Positive slope** | Speeding up (acceleration) |
| **Negative slope** | Slowing down (deceleration) |
| **Curved line** | Changing acceleration |

---

# 5. Acceleration

## Definition
**Acceleration = Rate of change of speed**

## Formula
$$a = \frac{v - u}{t}$$

Where:
- $a$ = acceleration (m/s²)
- $v$ = final speed (m/s)
- $u$ = initial speed (m/s)
- $t$ = time (s)

## Rearranged Forms
$$v = u + at$$
$$u = v - at$$
$$t = \frac{v - u}{a}$$

## Key Terms
| Term | Meaning |
|------|---------|
| **Positive acceleration** | Speeding up (v > u) |
| **Deceleration** | Slowing down (v < u) – give as positive value |
| **Uniform acceleration** | Constant acceleration (straight line on graph) |

# 6. Distance from Speed-Time Graph

## Area Formulas

| Shape | Formula |
|-------|---------|
| Rectangle | $\text{length} \times \text{width}$ |
| Triangle | $\frac{1}{2} \times \text{base} \times \text{height}$ |
| Trapezium | $\frac{1}{2}(a + b) \times h$ |

## Multi-Section Graphs
$$\text{Total Distance} = \text{Area}_1 + \text{Area}_2 + \text{Area}_3 + ...$$

# 10. Common Mistakes to Avoid

| ❌ Mistake | ✅ Correction |
|------------|---------------|
| Averaging individual speeds for average speed | Use: $\frac{\text{total distance}}{\text{total time}}$ |
| Forgetting units | Always write units |
| Confusing graph types | D-T gradient = speed; S-T gradient = acceleration |
| Negative deceleration | State as positive with "deceleration" label |
| Wrong area formula | Split into triangles/rectangles/trapeziums |
| Not reading scale correctly | Check if axis starts at zero |

---

# 11. Quick Problem-Solving Steps

## Finding Speed from Distance-Time Graph:
1. Select two points on the line
2. Calculate: $\frac{\text{rise}}{\text{run}} = \frac{\Delta d}{\Delta t}$
3. Include units

## Finding Acceleration from Speed-Time Graph:
1. Select two points on the line
2. Calculate: $\frac{\text{rise}}{\text{run}} = \frac{\Delta v}{\Delta t}$
3. Include units (m/s²)

## Finding Distance from Speed-Time Graph:
1. Identify shapes under the graph
2. Calculate area of each shape
3. Add all areas together
4. Include units (m)

## Finding Average Speed:
1. Calculate total distance
2. Calculate total time
3. Divide: $\frac{\text{total distance}}{\text{total time}}$
