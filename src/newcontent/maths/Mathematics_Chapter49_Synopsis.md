## Essential Formulas

### Right-Angled Triangle Trigonometry

| Ratio | Formula | When to Use |
|-------|---------|-------------|
| **SOH** | $\sin\theta = \frac{O}{H}$ | Have Opposite & Hypotenuse |
| **CAH** | $\cos\theta = \frac{A}{H}$ | Have Adjacent & Hypotenuse |
| **TOA** | $\tan\theta = \frac{O}{A}$ | Have Opposite & Adjacent |

### Pythagoras' Theorem

$$\text{Hypotenuse}^2 = \text{Side}_1^2 + \text{Side}_2^2$$

$$c = \sqrt{a^2 + b^2}$$

### 3D Shape Formulas

| Shape | Formula | Use |
|-------|---------|-----|
| **Cuboid space diagonal** | $d = \sqrt{l^2 + w^2 + h^2}$ | Diagonal from corner to opposite corner |
| **Cuboid face diagonal** | $d = \sqrt{l^2 + w^2}$ | Diagonal across one face |
| **Pyramid slant height** | $l = \sqrt{h^2 + \left(\frac{s}{2}\right)^2}$ | Height of triangular face |
| **Cone slant height** | $l = \sqrt{r^2 + h^2}$ | From base edge to apex |

## Finding Angles - Quick Rules

### Angle Between Line and Plane

1. Drop perpendicular from line to plane
2. Find projection on the plane
3. **Vertical height = Opposite to angle**
4. **Horizontal distance = Adjacent to angle**
5. Use: $\tan\theta = \frac{\text{vertical height}}{\text{horizontal distance}}$

```
    P
    |\
    | \
  h |  \ line
    | θ\
    O---A
      d
```
θ = angle between line and plane

### Angle Between Two Planes

1. Find the line of intersection
2. Draw perpendiculars from each plane to this line
3. Angle between these perpendiculars = angle between planes

## Problem-Solving Steps

### Standard Approach (Triangle Method)

| Step | Action |
|------|--------|
| 1 | Draw and label 3D shape |
| 2 | Identify the triangle containing the unknown |
| 3 | Calculate missing sides using Pythagoras (if needed) |
| 4 | Apply trigonometry (SOH/CAH/TOA) |
| 5 | Calculate and check reasonableness |

### For Multi-Stage Problems

1. **First stage:** Often use Pythagoras to find a diagonal or distance
2. **Second stage:** Use result to form another right-angled triangle
3. **Third stage:** Apply trigonometry for final angle or length

---

## Key Diagram Types

### Cuboid Problems
- Space diagonal triangle: height + base diagonal
- Base diagonal: use corner-to-corner across rectangle

### Pyramid Problems
- Apex-centered triangle: vertical height, half-base, slant height
- Slant edge triangle: vertical height, half-diagonal of base, slant edge

### Cone Problems
- Side view: radius, height, slant height form right triangle

## Common Mistakes 

| Mistake | Correction |
|---------|------------|
| Wrong trig ratio | Label O, A, H clearly before choosing |
| Slant vs vertical height | Slant = along surface; vertical = perpendicular to base |
| Forgetting half base | Apex calculations use $\frac{s}{2}$ not $s$ |
| Calculator in radians | Check for "D" or "DEG" mode |
| Not showing working | Write formula → substitute → calculate |

## Memory Aid

**For cuboid diagonals:** "Three dimensions, three squares, one root"
$$d = \sqrt{l^2 + w^2 + h^2}$$

**For angle with base:** "Up and along, then tan"
$$\tan\theta = \frac{\text{up (height)}}{\text{along (horizontal)}}$$

**SOH-CAH-TOA:** Standard trig ratios apply identically in 3D
