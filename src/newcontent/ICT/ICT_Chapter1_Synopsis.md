### 6. Device Selection Criteria

**Key Questions to Ask:**

1. **Purpose** — What tasks will the device perform?
2. **Performance** — What processor/RAM/storage is needed?
3. **Portability** — Does it need to be mobile?
4. **Budget** — What is the total cost of ownership?
5. **Compatibility** — Does it work with existing systems?
6. **User** — Who will use it and what is their technical ability?

---

### 7. High-Yield Exam Points

**Must-Know Facts:**

- **RAM is volatile** — data lost when power off
- **SSD has no moving parts** — faster, more durable than HDD
- **Capacitive touchscreens support multi-touch** — resistive does not
- **Laser printers use toner** — inkjet uses liquid ink
- **OLED pixels self-emit light** — LCD requires backlight
- **Wi-Fi is for local networking** — Cellular is for mobile internet
- **Desktop PCs are upgradeable** — laptops have limited upgrade options

**Common Exam Scenarios:**

| Scenario | Recommended Device | Reason |
|----------|-------------------|--------|
| Student taking notes in class | Tablet with keyboard | Portable, long battery, versatile |
| Video editing studio | High-end desktop PC | Powerful processor, large storage, upgradeable |
| Field researcher collecting data | Tablet/smartphone | Portable, camera, GPS, connectivity |
| Office worker with desk | Desktop PC | Powerful, ergonomic, cost-effective |
| Travelling salesperson | Laptop | Portable, full functionality, connectivity |
| Child learning to read | Tablet | Touchscreen intuitive, educational apps |

### 8. Quick Definition List

| Term | Definition |
|------|------------|
| **Digital Device** | Electronic device that processes data in binary format |
| **ADC** | Converts analogue signals to digital data |
| **Resolution** | Number of pixels determining display clarity |
| **Volatile Memory** | Memory that loses data without power (RAM) |
| **Non-volatile Memory** | Memory that retains data without power (SSD, HDD) |
| **Peripheral** | External device connected to computer |
| **Form Factor** | Physical size and shape of a device |
| **Ergonomics** | Design for comfortable, efficient use |

### 9. Advantages and Disadvantages Summary

**Desktop Computers:**
- ✅ Powerful, upgradeable, cost-effective
- ❌ Not portable, requires separate peripherals

**Laptops:**
- ✅ Portable, all-in-one design
- ❌ Limited upgrades, battery dependence

**Tablets:**
- ✅ Highly portable, touchscreen intuitive
- ❌ Limited power, virtual keyboard

**Smartphones:**
- ✅ Always connected, pocket-sized
- ❌ Small screen, limited processing power

**Wearables:**
- ✅ Convenient, health monitoring
- ❌ Limited functionality, small display

---

### 10. Exam Tips

1. **Read questions carefully** — identify the specific device or feature being asked about
2. **Always justify recommendations** — link device features to user needs
3. **Consider context** — home, business, education, travel scenarios differ
4. **Mention relevant specifications** — don't just say "fast processor," specify why
5. **Include connectivity** — most answers should mention how devices connect
6. **Balance advantages and disadvantages** — exam questions often ask for both
