### 4. Language Translators

| Type | How It Works | Advantages | Disadvantages |
|------|--------------|------------|---------------|
| **Compiler** | Translates entire program before execution | Fast execution; standalone file | Errors found after compilation |
| **Interpreter** | Translates line by line during execution | Immediate error detection | Slower execution |
| **Assembler** | Converts assembly to machine code | Direct hardware control | Processor-specific |

### 7. Open Source vs Proprietary Software

| Aspect | Open Source | Proprietary |
|--------|-------------|-------------|
| **Cost** | Usually free | Paid/licensed |
| **Source Code** | Publicly available | Closed/protected |
| **Modification** | Allowed | Not allowed |
| **Support** | Community-based | Vendor-provided |
| **Examples** | LibreOffice, Firefox, Linux | Microsoft Office, Windows, Photoshop |

---

### 8. Off-the-Shelf vs Custom Software

| Factor | Off-the-Shelf | Custom |
|--------|---------------|--------|
| **Availability** | Immediate | Development time required |
| **Cost** | Lower initial cost | Higher development cost |
| **Fit** | General needs | Exact requirements |
| **Support** | Vendor support | Internal/contracted |
| **Updates** | Regular vendor updates | Self-maintained |
| **Risk** | Lower | Higher (project failure) |

### 10. High-Yield Exam Points

**Must-Know Facts:**

- **OS manages hardware** — memory, processor, storage, I/O
- **Utility software maintains systems** — antivirus, defrag, backup
- **Compilers create executables** — interpreters do not
- **GUI uses windows, icons, menus, pointers** — WIMP interface
- **Drivers translate OS commands** to device-specific instructions
- **Open source = accessible source code** — can be modified
- **Proprietary = closed source** — cannot be modified

**Classification Quick Guide:**

| Is it... | Answer |
|----------|--------|
| Windows 10 | System software (Operating System) |
| Antivirus | System software (Utility) |
| Microsoft Word | Application software (General-purpose) |
| Photoshop | Application software (Special-purpose) |
| Printer driver | System software (Driver) |
| Google Chrome | Application software (Web browser) |

### 11. Quick Definition List

| Term | Definition |
|------|------------|
| **Software** | Programs instructing computers to perform tasks |
| **Operating System** | System software managing hardware resources |
| **Utility Software** | Programs for system maintenance |
| **GUI** | Graphical User Interface using visual elements |
| **CLI** | Command Line Interface using text commands |
| **Compiler** | Translates source code to executable |
| **Interpreter** | Executes code line by line |
| **API** | Interface allowing software components to communicate |
| **Malware** | Malicious software (viruses, trojans, etc.) |

---

### 12. Exam Tips

1. **Know the difference between system and application software** — this is frequently tested
2. **Always give specific examples** when asked about software types
3. **Explain why features are useful** — don't just list them
4. **Compare open source and proprietary** — advantages AND disadvantages of each
5. **Remember utility software** — often overlooked but important
6. **Understand the role of drivers** — enable hardware communication
