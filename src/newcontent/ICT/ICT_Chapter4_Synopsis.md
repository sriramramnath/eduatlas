## Quick Reference Summary

### 1. Communication Methods Comparison

| Method | Speed | Range | Advantages | Disadvantages |
|--------|-------|-------|------------|---------------|
| **Copper Cable (UTP)** | Up to 10Gbps | 100m max | Cheap, easy install | Interference, distance limited |
| **Fibre Optic** | Terabits/sec | Up to 100km | Fastest, long distance, secure | Expensive, difficult to repair |
| **Wi-Fi** | Up to 9.6Gbps | 30-100m | Wireless, convenient | Interference, security concerns |
| **Bluetooth** | Up to 3Mbps | 10-100m | Low power, simple pairing | Short range, low bandwidth |
| **Mobile (4G/5G)** | Up to 20Gbps | Kilometres | Wide coverage, mobile | Data costs, coverage gaps |
| **Satellite** | High | Global | Remote area coverage | High latency, expensive |

---

### 2. Wired vs Wireless

| Aspect | Wired | Wireless |
|--------|-------|----------|
| **Reliability** | More reliable | Susceptible to interference |
| **Speed** | Generally faster | Improving but typically slower |
| **Security** | More secure | Easier to intercept |
| **Mobility** | Fixed location | Mobile within range |
| **Installation** | More complex | Simpler deployment |
| **Cost** | Lower hardware cost | May require more access points |

---

### 3. Factors Affecting Communication

| Factor | Definition | Impact |
|--------|------------|--------|
| **Bandwidth** | Maximum data transfer rate | Higher = more data capacity |
| **Latency** | Time delay for data transfer | Lower = faster response |
| **Interference** | Signal disruption | Reduces quality/speed |
| **Distance** | Physical separation | Reduces signal strength |
| **Users** | Number sharing connection | More users = less bandwidth each |

### 4. Key Protocols

| Protocol | Full Name | Purpose |
|----------|-----------|---------|
| **TCP/IP** | Transmission Control Protocol/Internet Protocol | Core internet protocol |
| **HTTP** | Hypertext Transfer Protocol | Web page requests |
| **HTTPS** | HTTP Secure | Encrypted web traffic |
| **FTP** | File Transfer Protocol | File transfers |
| **SMTP** | Simple Mail Transfer Protocol | Sending emails |
| **POP3** | Post Office Protocol v3 | Downloading emails |
| **IMAP** | Internet Message Access Protocol | Accessing emails on server |
| **DNS** | Domain Name System | Translates domain names to IP |

### 9. High-Yield Exam Points

**Must-Know Facts:**

- **Fibre optic = fastest** — uses light pulses
- **TCP provides reliability** — resends lost packets
- **IP handles addressing** — routes packets to destinations
- **HTTPS = encrypted HTTP** — secure web browsing
- **SMTP sends email** — POP3/IMAP receives email
- **Bandwidth ≠ speed** — bandwidth is capacity
- **Latency causes lag** — important for gaming/video calls
- **5GHz less congested** — better for interference

**Protocol Port Numbers:**

| Protocol | Port |
|----------|------|
| HTTP | 80 |
| HTTPS | 443 |
| FTP | 20, 21 |
| SMTP | 25, 587 |
| POP3 | 110 |
| IMAP | 143 |

### 10. Quick Definition List

| Term | Definition |
|------|------------|
| **Bandwidth** | Maximum data transfer capacity (bps) |
| **Latency** | Time delay in transmission (ms) |
| **Protocol** | Rules governing data communication |
| **Packet** | Unit of data transmitted over networks |
| **Fibre Optic** | Cable transmitting data via light pulses |
| **Interference** | Disruption affecting wireless signals |
| **Contention Ratio** | Users sharing one connection |
| **DNS** | Domain name to IP address translation |
| **Encryption** | Converting data to secure format |

---

### 11. Exam Tips

1. **Distinguish bandwidth and latency** — capacity vs delay
2. **Know protocol purposes** — what each protocol does
3. **Compare wired vs wireless** — advantages of each
4. **Explain interference factors** — what causes it and how to reduce
5. **Use correct units** — Mbps for bandwidth, ms for latency
6. **Mention security** — HTTPS for secure connections
