## SECTION A: Multiple Choice Questions

### Question 1
What does SaaS stand for?

A) Software as a Service  
B) Storage as a Service  
C) System as a Service  
D) Security as a Service  

**Correct Answer: A) Software as a Service**

**Explanation:** SaaS (Software as a Service) delivers complete software applications over the internet. Users access the software through a web browser without needing to install anything locally. Examples include Google Docs, Microsoft 365, and Salesforce.

---

### Question 2
Which cloud service model provides virtual machines and storage?

A) SaaS  
B) PaaS  
C) IaaS  
D) DaaS  

**Correct Answer: C) IaaS**

**Explanation:** IaaS (Infrastructure as a Service) provides fundamental computing infrastructure including virtual machines, storage, and networking. Users manage operating systems and applications while the provider manages the underlying hardware.

---

### Question 3
What type of cloud deployment is used exclusively by a single organisation?

A) Public cloud  
B) Private cloud  
C) Hybrid cloud  
D) Community cloud  

**Correct Answer: B) Private cloud**

**Explanation:** Private cloud is dedicated infrastructure used exclusively by one organisation. It offers greater control, security, and customisation but at higher cost than public cloud options.

---

### Question 4
Which of these is an example of SaaS?

A) Amazon EC2  
B) Microsoft Azure Virtual Machines  
C) Google Docs  
D) Google App Engine  

**Correct Answer: C) Google Docs**

**Explanation:** Google Docs is a complete software application delivered over the internet—making it SaaS. Amazon EC2 and Azure VMs are IaaS (infrastructure), while Google App Engine is PaaS (platform for development).

---

### Question 5
What is a key advantage of cloud computing's scalability?

A) Resources are fixed and cannot change  
B) Resources can be increased or decreased as needed  
C) Resources are always unlimited  
D) Resources are only available to large companies  

**Correct Answer: B) Resources can be increased or decreased as needed**

**Explanation:** Cloud scalability allows organisations to increase resources during high demand and decrease them when demand is low. This elasticity means you only pay for what you use and can handle peak loads without maintaining excess capacity.

---

### Question 6
What is a major disadvantage of cloud computing?

A) It is always free  
B) It requires internet connectivity to access services  
C) It is always slower than local computing  
D) It cannot be used by businesses  

**Correct Answer: B) It requires internet connectivity to access services**

**Explanation:** Cloud computing depends on internet connectivity. Without a reliable internet connection, users cannot access their data or applications stored in the cloud. This is a fundamental limitation.

---

### Question 7
What does "hybrid cloud" mean?

A) A cloud that only works on mobile devices  
B) Combination of public and private cloud resources  
C) A cloud that uses both electricity and solar power  
D) A cloud used only for gaming  

**Correct Answer: B) Combination of public and private cloud resources**

**Explanation:** Hybrid cloud combines public and private clouds, allowing data and applications to move between them. This provides flexibility—for example, keeping sensitive data in a private cloud while using public cloud for additional capacity.

---

### Question 8
Which is an example of cloud storage?

A) A USB flash drive  
B) An external hard drive  
C) Google Drive  
D) A DVD  

**Correct Answer: C) Google Drive**

**Explanation:** Google Drive is cloud storage—data stored on remote servers accessible via internet. USB drives, external hard drives, and DVDs are all local storage devices that you physically possess.

---

### Question 9
What is vendor lock-in in cloud computing?

A) Being unable to afford cloud services  
B) Difficulty switching between cloud providers  
C) Cloud providers going out of business  
D) Internet connection problems  

**Correct Answer: B) Difficulty switching between cloud providers**

**Explanation:** Vendor lock-in occurs when it becomes difficult or expensive to switch from one cloud provider to another, often because of proprietary technologies, data formats, or integrated services that don't transfer easily.

---

### Question 10
What is the pay-as-you-go model in cloud computing?

A) Paying a fixed monthly fee regardless of usage  
B) Paying only for the resources you actually use  
C) Paying before receiving any service  
D) Not paying anything for services  

**Correct Answer: B) Paying only for the resources you actually use**

**Explanation:** Pay-as-you-go is a pricing model where users pay only for the computing resources they consume, similar to utilities like electricity. This contrasts with traditional computing where you pay for hardware upfront regardless of usage.

---

### Question 11
Which cloud service model requires the most user management?

A) SaaS  
B) PaaS  
C) IaaS  
D) All require equal management  

**Correct Answer: C) IaaS**

**Explanation:** IaaS (Infrastructure as a Service) requires users to manage operating systems, applications, and data. The provider only manages the physical hardware. SaaS requires the least management (just use the application), while PaaS is in between.

---

### Question 12
What security concern is specific to public cloud?

A) No security features exist  
B) Multi-tenant environment where resources are shared  
C) Data is never encrypted  
D) Only one person can access data  

**Correct Answer: B) Multi-tenant environment where resources are shared**

**Explanation:** In public cloud, multiple organisations (tenants) share the same infrastructure. While data is logically separated, the shared environment creates potential security risks that don't exist in dedicated private cloud environments.

## SECTION B: True/False Questions

### Statement 1
**Cloud computing requires users to own physical servers.**

**Answer: FALSE**

**Correction:** Cloud computing eliminates the need for users to own physical servers. The cloud provider owns and maintains the hardware, and users access computing resources over the internet.

---

### Statement 2
**Google Docs is an example of Software as a Service (SaaS).**

**Answer: TRUE**

**Explanation:** Google Docs is a complete application delivered over the internet through a browser. Users don't install software and can access it from any device—classic characteristics of SaaS.

---

### Statement 3
**Private cloud is less secure than public cloud.**

**Answer: FALSE**

**Correction:** Private cloud is generally considered more secure than public cloud because the infrastructure is dedicated to a single organisation. There's no multi-tenant risk, and the organisation has full control over security measures.

---

### Statement 4
**Cloud storage allows access to files from any device with internet.**

**Answer: TRUE**

**Explanation:** One of the main advantages of cloud storage is accessibility. Files stored in the cloud can be accessed from any device with internet connectivity—laptop, tablet, or smartphone—anywhere in the world.

---

### Statement 5
**IaaS provides complete software applications ready to use.**

**Answer: FALSE**

**Correction:** IaaS (Infrastructure as a Service) provides computing infrastructure like virtual machines, storage, and networking—not complete applications. SaaS provides complete applications; IaaS requires users to install and manage their own software.

---

### Statement 6
**Hybrid cloud combines public and private cloud resources.**

**Answer: TRUE**

**Explanation:** Hybrid cloud is a combination of public and private cloud resources, connected by technology that allows data and applications to move between them. This provides flexibility to use the best environment for each workload.

---

### Statement 7
**Cloud computing always works without an internet connection.**

**Answer: FALSE**

**Correction:** Cloud computing fundamentally requires internet connectivity to access services and data stored on remote servers. Without internet, users typically cannot access cloud-based applications or data.

---

### Statement 8
**SaaS applications are automatically updated by the provider.**

**Answer: TRUE**

**Explanation:** One advantage of SaaS is that the provider handles all updates and maintenance. Users always access the latest version without needing to install updates themselves.

---

### Statement 9
**Cloud storage is less reliable than local storage.**

**Answer: FALSE**

**Correction:** Cloud storage is typically more reliable than local storage. Providers use redundant systems across multiple data centres, automatic backups, and professional maintenance—making data loss much less likely than on a single local drive.

---

### Statement 10
**PaaS is designed for software developers to build applications.**

**Answer: TRUE**

**Explanation:** PaaS (Platform as a Service) provides tools and environments specifically for developers to build, test, and deploy applications without managing the underlying infrastructure.

---

### Statement 11
**Public cloud is the most expensive cloud deployment model.**

**Answer: FALSE**

**Correction:** Public cloud is typically the least expensive cloud model because resources are shared among many users, spreading the costs. Private cloud is the most expensive due to dedicated infrastructure.

---

### Statement 12
**Cloud providers may store data in different countries.**

**Answer: TRUE**

**Explanation:** Cloud providers operate data centres globally, and data may be stored in different countries depending on the service and settings. This raises data sovereignty and privacy concerns, especially for sensitive data subject to regulations like GDPR.

### Question 4 (3 marks)
**Explain the three cloud service models: IaaS, PaaS, and SaaS.**

**Mark Scheme Answer:**
**IaaS (Infrastructure as a Service):**
- Provides virtual machines, storage, networking [1 mark]
- User manages operating system and applications [1 mark]
- Example: AWS EC2, Azure Virtual Machines [1 mark]

**PaaS (Platform as a Service):**
- Provides development platform and tools [1 mark]
- User develops and deploys applications [1 mark]
- Example: Heroku, Google App Engine [1 mark]

**SaaS (Software as a Service):**
- Provides complete applications [1 mark]
- User simply uses the software via browser [1 mark]
- Example: Google Docs, Microsoft 365 [1 mark]

*Award up to 3 marks for clear explanation of all three.*

### Question 6 (4 marks)
**A small business is deciding whether to use cloud computing or traditional on-premises servers. Discuss the factors they should consider.**

**Mark Scheme Answer:**
**Cost factors:**
- Cloud: Pay-as-you-go, no upfront hardware cost [1 mark]
- On-premises: High initial investment, ongoing maintenance costs [1 mark]

**Control and security:**
- Cloud: Provider manages security, less control [1 mark]
- On-premises: Full control but requires IT expertise [1 mark]

**Scalability:**
- Cloud: Easy to scale up or down as needed [1 mark]
- On-premises: Limited by hardware capacity purchased [1 mark]

**Accessibility:**
- Cloud: Access from anywhere with internet [1 mark]
- On-premises: Primarily accessible on local network [1 mark]

**Recommendation:** For small business, cloud often better due to lower upfront costs, no need for IT staff, and flexibility. [1 mark]

*Award up to 4 marks for discussion covering multiple factors.*
