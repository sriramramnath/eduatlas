## Quick Reference Summary

### 1. Legal Implications

| Area | Key Points |
|------|------------|
| **GDPR** | EU data protection law; consent, rights, security |
| **Intellectual Property** | Copyright, trademarks, patents protection |
| **Computer Misuse** | Criminalises unauthorised access, hacking |
| **Online Harassment** | Cyberbullying and defamation are illegal |

---

### 2. GDPR Key Rights

```
INDIVIDUAL RIGHTS:
✓ Right to access personal data
✓ Right to correct inaccurate data
✓ Right to erasure ("be forgotten")
✓ Right to data portability
✓ Right to object to processing

ORGANISATION DUTIES:
✓ Obtain consent for data collection
✓ Inform about data use
✓ Protect data security
✓ Report data breaches
```

### 9. High-Yield Exam Points

**Must-Know Facts:**

- **GDPR protects personal data** — consent, rights, security
- **Copyright protects creative works** — music, films, software
- **Computer Misuse Act criminalises hacking** — unauthorised access
- **AI can be biased** — reflects training data
- **Digital divide creates inequality** — economic, geographic, age factors
- **E-waste is environmental problem** — toxic, growing
- **Jobs both created and lost** — transformation, not just destruction

**Key Legislation:**

| Law | Purpose |
|-----|---------|
| GDPR | Data protection and privacy |
| Computer Misuse Act | Criminalise hacking |
| Copyright Law | Protect creative works |
| Equality Act | Prevent discrimination |

### 11. Exam Tips

1. **Consider all perspectives** — legal, ethical, social, environmental
2. **Use specific legislation names** — GDPR, Computer Misuse Act
3. **Balance benefits and costs** — technology trade-offs
4. **Include vulnerable groups** — different impacts on different people
5. **Address solutions** — not just problems
6. **Think globally** — implications vary by country/region
