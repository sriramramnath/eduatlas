## Essential Equations

```
Moment:                M = F × d
Principle of Moments:  Σ clockwise = Σ anticlockwise
Mechanical Advantage:  MA = Load ÷ Effort
```

---

## Moments

### Definition
**Moment** = turning effect of a force

```
Moment (Nm) = Force (N) × perpendicular distance (m)
```

### Direction
- **Clockwise moment** → turns object clockwise
- **Anticlockwise moment** → turns object anticlockwise

### Example
```
Force = 20 N, distance = 0.5 m
Moment = 20 × 0.5 = 10 Nm
```

---

## Principle of Moments

> For an object in equilibrium:
> **Sum of clockwise moments = Sum of anticlockwise moments**

### Equilibrium Conditions
1. Resultant force = 0
2. Resultant moment = 0

### Worked Example

```
30 N at 2 m from pivot = ? N at 3 m from pivot

Clockwise = Anticlockwise
30 × 2 = F × 3
60 = 3F
F = 20 N
```

## Centre of Gravity (CG)

### Definition
Point where the entire weight of an object appears to act.

### Finding CG
- Suspend object, draw vertical line
- Repeat from different point
- Intersection = CG

### Stability

| Factor | Effect |
|--------|--------|
| Lower CG | More stable |
| Wider base | More stable |
| CG over base | Stable |
| CG outside base | Topples |

## Levers

### Parts
- **Pivot (fulcrum)** - fixed turning point
- **Effort** - applied force
- **Load** - force being overcome

### Classes of Levers

| Class | Arrangement | Example |
|-------|-------------|---------|
| 1 | Effort - Pivot - Load | Seesaw, crowbar |
| 2 | Pivot - Load - Effort | Wheelbarrow |
| 3 | Pivot - Effort - Load | Tweezers |

### Mechanical Advantage
```
MA = Load ÷ Effort
```
- MA > 1: lever multiplies force
- MA < 1: lever increases distance/speed

---

## Couples

### Definition
Pair of equal, opposite, parallel forces (not along same line)

```
Moment of couple = F × d
```

### Examples
- Steering wheel
- Using a spanner
- Opening a tap

## Quick Calculation Guide

### Finding an Unknown Force
```
Clockwise moments = Anticlockwise moments
F₁ × d₁ = F₂ × d₂

Example: 50 N at 1.5 m = F at 2 m
50 × 1.5 = F × 2
F = 75 ÷ 2 = 37.5 N
```

### Finding an Unknown Distance
```
F₁ × d₁ = F₂ × d₂

Example: 40 N at 3 m = 60 N at d
40 × 3 = 60 × d
d = 120 ÷ 60 = 2 m
```

## Key Exam Points

1. **Perpendicular distance** - always perpendicular to force
2. **Clockwise vs anticlockwise** - visualize the turning
3. **Include beam weight** - if beam has mass, include its moment
4. **Units** - Nm for moment, N for force, m for distance
5. **Equilibrium** - both force and moment balance needed

---

## Common Problem Types

### Type 1: Simple balance
```
F₁ × d₁ = F₂ × d₂
```

### Type 2: Beam with weight
```
Include moment of beam's weight acting at its centre
```

### Type 3: Multiple forces
```
Sum of clockwise = Sum of anticlockwise
(F₁ × d₁) + (F₂ × d₂) = (F₃ × d₃)
```
