## 5.1 Moments

### Definition

A **moment** (or torque) is the turning effect of a force. It depends on:
- The size of the force
- The perpendicular distance from the pivot to the line of action of the force

```
moment = force × perpendicular distance from pivot

M = F × d

Where:
M = moment (Nm)
F = force (N)
d = perpendicular distance from pivot (m)
```

### Unit of Moment

The unit of moment is the **newton metre (Nm)**.

### Direction of Moments

| Direction | Description |
|-----------|-------------|
| **Clockwise moment** | Tends to turn object clockwise |
| **Anticlockwise moment** | Tends to turn object anticlockwise |

### Worked Examples

**Example 1:** A 10 N force acts 0.5 m from a pivot. Calculate the moment.
```
M = F × d = 10 × 0.5 = 5 Nm
```

**Example 2:** A 20 N weight is placed 0.3 m from the pivot on a seesaw. Calculate the moment.
```
M = F × d = 20 × 0.3 = 6 Nm (clockwise)
```

## 5.2 Principle of Moments

### Statement

**The Principle of Moments:** For an object in equilibrium, the sum of clockwise moments equals the sum of anticlockwise moments.

```
sum of clockwise moments = sum of anticlockwise moments
```

### Conditions for Equilibrium

For an object to be in equilibrium (balanced/not rotating):
1. **Resultant force = 0** (no linear acceleration)
2. **Resultant moment = 0** (no rotation)

### Worked Example

A seesaw is balanced with a 30 N child sitting 2 m from the pivot on one side. Where should a 20 N child sit to balance it?

```
Clockwise moment = Anticlockwise moment
30 × 2 = 20 × d
60 = 20d
d = 3 m

The 20 N child should sit 3 m from the pivot.
```

## 5.3 Centre of Gravity

### Definition

The **centre of gravity** (or centre of mass) of an object is the point where the entire weight of the object appears to act.

### Finding the Centre of Gravity

**Method: Balancing method**
1. Suspend the object from a point
2. Draw a vertical line from the suspension point
3. Repeat from a different suspension point
4. Where the lines intersect is the centre of gravity

### Stability and Centre of Gravity

| Factor | Effect on Stability |
|--------|---------------------|
| **Lower centre of gravity** | More stable |
| **Wider base** | More stable |
| **Centre of gravity above base** | Stable |
| **Centre of gravity outside base** | Object topples |

### Stability Principles


## 5.4 Levers

### What is a Lever?

A lever is a simple machine that uses the principle of moments to multiply force.

### Parts of a Lever

| Part | Description |
|------|-------------|
| **Pivot (fulcrum)** | The fixed point about which the lever turns |
| **Effort** | The force applied to the lever |
| **Load** | The force being overcome |

### Types of Levers

| Type | Arrangement | Example |
|------|-------------|---------|
| **Class 1** | Effort - Pivot - Load | Seesaw, scissors, crowbar |
| **Class 2** | Pivot - Load - Effort | Wheelbarrow, nutcracker |
| **Class 3** | Pivot - Effort - Load | Tweezers, fishing rod |

### Mechanical Advantage

```
mechanical advantage = load ÷ effort

MA = L ÷ E
```

A mechanical advantage greater than 1 means the lever multiplies force.

## 5.5 Applications of Moments

### Seesaw (Balancing)

```
        20 N                    30 N
          │                       │
          ▼                       ▼
    ──────┴─────────────────────┴──────
          │←──── 3 m ────→│←── 2 m ──→│
                    ↑
                  PIVOT

Clockwise moment = 20 × 3 = 60 Nm
Anticlockwise moment = 30 × 2 = 60 Nm
Balanced!
```

### Using a Crowbar

```
    ↑ Effort (small force)
    │
    │←────── long distance ──────→│
    ├─────────────────────────────┘
    ↑                             ↓ Load (large force)
  Pivot                      (short distance)
```

Small effort × long distance = Large load × short distance

### Bridges and Beams

Engineers use the principle of moments to calculate:
- Support forces
- Maximum loads
- Structural safety

## 5.6 Couples

### Definition

A **couple** is a pair of equal and opposite forces that act parallel to each other but not along the same line.

### Moment of a Couple

```
moment of couple = force × perpendicular distance between forces

M = F × d
```

A couple produces rotation without translation (no net force, only turning effect).

### Examples of Couples

| Example | Description |
|---------|-------------|
| **Turning a steering wheel** | Two hands apply equal and opposite forces |
| **Using a spanner** | Two forces on opposite sides of the nut |
| **Opening a tap** | Fingers apply opposing forces |

## 5.7 Experiments with Moments

### Investigating the Principle of Moments

**Apparatus:**
- Metre rule
- Pivot (fulcrum)
- Slotted masses
- Mass hanger

**Method:**
1. Balance the metre rule on the pivot
2. Place known masses at measured distances on one side
3. Find the position for balancing masses on the other side
4. Record results in a table
5. Calculate moments and verify principle

**Expected Result:**
Sum of clockwise moments = Sum of anticlockwise moments

## Exam Tips for Chapter 5

1. **Perpendicular distance** - always use the perpendicular distance from pivot to force line
2. **Direction matters** - identify clockwise and anticlockwise correctly
3. **Units** - moments in Nm, distances in m, forces in N
4. **Equilibrium** - both force and moment conditions must be satisfied
5. **Centre of gravity** - for uniform objects, it's at the geometric centre
6. **Show working** - write out moment calculations clearly

## Common Student Errors

| Error | Correction |
|-------|------------|
| Using wrong distance | Use perpendicular distance from pivot |
| Confusing clockwise/anticlockwise | Visualize which way the force would turn the object |
| Forgetting weight of beam | Include the moment due to the beam's own weight |
| Not converting units | Use metres for distance, not cm |
| Confusing force and moment | Force in N, moment in Nm |
