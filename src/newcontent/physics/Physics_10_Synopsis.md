# Chapter 10: Properties of Waves
## Pearson Edexcel IGCSE Physics - Concise Synopsis

---

## Essential Equations

```
Wave equation:    v = f × λ
Period:           T = 1 ÷ f
Frequency:        f = 1 ÷ T
```

---

## Types of Waves

### Transverse Waves
- Particles vibrate **perpendicular** to direction of travel
- Examples: Light, water waves, waves on string
- Features: **Crests** and **Troughs**

### Longitudinal Waves
- Particles vibrate **parallel** to direction of travel
- Examples: Sound, compression in spring
- Features: **Compressions** and **Rarefactions**

---

## Wave Terms

| Term | Symbol | Definition | Unit |
|------|--------|------------|------|
| **Wavelength** | λ | Distance between corresponding points | m |
| **Amplitude** | A | Maximum displacement from rest | m |
| **Frequency** | f | Waves per second | Hz |
| **Period** | T | Time for one wave | s |
| **Wave speed** | v | Speed of energy transfer | m/s |

---

## Wave Equation

```
v = f × λ

wave speed = frequency × wavelength
```

### Example
```
f = 10 Hz, λ = 0.5 m
v = 10 × 0.5 = 5 m/s
```

---

## Wave Behaviours

| Behaviour | Description |
|-----------|-------------|
| **Reflection** | Waves bounce off surfaces |
| **Refraction** | Waves change direction in new medium |
| **Diffraction** | Waves spread through gaps/obstacles |

## Sound Waves

- **Type:** Longitudinal
- **Needs medium:** Cannot travel through vacuum
- **Speed in air:** ~340 m/s
- **Pitch:** Determined by frequency
- **Loudness:** Determined by amplitude

### Speed in Different Media

| Medium | Speed |
|--------|-------|
| Air | ~340 m/s |
| Water | ~1500 m/s |
| Steel | ~5000 m/s |

## Electromagnetic Spectrum

### Properties
- All are **transverse**
- All travel at **3 × 10⁸ m/s** in vacuum
- Can travel through vacuum

### Order (Long λ → Short λ)

```
Radio → Micro → Infrared → Visible → UV → X-ray → Gamma
```

### Approximate Wavelengths

| Type | Wavelength |
|------|------------|
| Radio | > 1 m |
| Microwaves | 1 mm - 1 m |
| Infrared | 700 nm - 1 mm |
| Visible | 400-700 nm |
| UV | 10-400 nm |
| X-rays | 0.01-10 nm |
| Gamma | < 0.01 nm |

### Visible Light Spectrum

```
Red (700 nm) → Orange → Yellow → Green → Blue → Indigo → Violet (400 nm)
```

## Quick Calculation Guide

### Finding Wave Speed
```
f = 20 Hz, λ = 0.1 m
v = f × λ = 20 × 0.1 = 2 m/s
```

### Finding Frequency
```
v = 340 m/s, λ = 0.17 m
f = v ÷ λ = 340 ÷ 0.17 = 2000 Hz
```

### Finding Wavelength
```
v = 3 × 10⁸ m/s, f = 100 MHz = 10⁸ Hz
λ = v ÷ f = 3×10⁸ ÷ 10⁸ = 3 m
```

### Finding Period
```
f = 50 Hz
T = 1 ÷ f = 1 ÷ 50 = 0.02 s
```

## Key Exam Points

1. **v = fλ** - wave equation, can rearrange
2. **Transverse** - perpendicular vibration
3. **Longitudinal** - parallel vibration
4. **Sound** - longitudinal, needs medium
5. **EM waves** - transverse, all same speed in vacuum
6. **Spectrum order** - Radio, Micro, Infra, Visible, UV, X, Gamma

---

## Common Mistakes

| Mistake | Correction |
|---------|------------|
| Sound is transverse | Sound is longitudinal |
| EM waves need medium | EM waves can travel through vacuum |
| Confusing f and λ | f = waves/s, λ = distance |
| Wrong spectrum order | Learn: R, M, I, V, U, X, G |
