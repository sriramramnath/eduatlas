# Chapter 8: Electrical Resistance
## Pearson Edexcel IGCSE Physics - Concise Synopsis

---

## Essential Equations

```
Ohm's Law:        R = V ÷ I
Alternative:      V = I × R
Current:          I = V ÷ R
Resistivity:      R = ρ × L ÷ A
```

---

## Ohm's Law

> Current is directly proportional to voltage (at constant temperature)

```
V = I × R
```

### Ohmic vs Non-Ohmic Conductors

| Type | Characteristic | Example |
|------|---------------|---------|
| **Ohmic** | Constant resistance, straight V-I graph | Metal resistor |
| **Non-ohmic** | Resistance changes, curved V-I graph | Filament lamp, diode |

---

## V-I Graphs

### Fixed Resistor (Ohmic)
```
V │    ╱
  │   ╱ (straight line)
  │  ╱
  │ ╱
  │╱
  └──────→ I
```

### Filament Lamp
```
V │      ╭─── (curve, less steep at high V)
  │     ╱
  │    ╱
  │   ╱
  │  ╱
  │ ╱
  │╱
  └──────→ I
```
- Resistance increases as temperature increases

### Diode
```
I │         ╱ (conducts one way only)
  │        ╱
  │       ╱
  │______╱
  │
  └────────→ V
       0
```
- Very high resistance in reverse direction
- Conducts when forward voltage > ~0.6 V

## Factors Affecting Resistance

### For a Wire

| Factor | Relationship | Effect |
|--------|--------------|--------|
| **Length (L)** | R ∝ L | Longer = more R |
| **Area (A)** | R ∝ 1/A | Thicker = less R |
| **Material** | R ∝ ρ | Depends on resistivity |
| **Temperature** | Metals: R increases with T | Hotter = more R |

### Combined Formula
```
R = ρ × L ÷ A
```

---

## Resistivity

### Definition
Property of a material measuring opposition to current flow.

### Unit
Ohm-metre (Ωm)

### Typical Values

| Material | Resistivity (Ωm) |
|----------|------------------|
| Copper | 1.7 × 10⁻⁸ |
| Nichrome | 1.1 × 10⁻⁶ |
| Glass | 10¹² |

---

## Temperature Effects

### Metals
- Temperature ↑ → Resistance ↑
- More vibrations = more electron collisions

### Semiconductors (Thermistors)
- Temperature ↑ → Resistance ↓
- More charge carriers released

### Superconductors
- Zero resistance below critical temperature
- Applications: MRI, maglev trains

## Quick Calculation Guide

### Finding Resistance
```
V = 12V, I = 3A
R = V/I = 12/3 = 4Ω
```

### Finding Voltage
```
I = 2A, R = 5Ω
V = I × R = 2 × 5 = 10V
```

### Finding Current
```
V = 9V, R = 3Ω
I = V/R = 9/3 = 3A
```

### Using Resistivity
```
ρ = 1.7×10⁻⁸ Ωm, L = 2m, A = 1mm² = 1×10⁻⁶ m²
R = ρL/A = (1.7×10⁻⁸ × 2) ÷ 10⁻⁶
R = 3.4×10⁻⁸ ÷ 10⁻⁶ = 0.034Ω
```

## Key Exam Points

1. **Ohm's Law**: V = IR (at constant temperature)
2. **V-I graphs**: Know shapes for resistor, lamp, diode
3. **Resistance factors**: Length, area, material, temperature
4. **Resistivity**: Material property, unit Ωm
5. **Temperature**: Metals - R increases with T
6. **Unit conversions**: mm² to m² (× 10⁻⁶)

---

## Common Mistakes

| Mistake | Correction |
|---------|------------|
| Using mm² directly | Convert to m² first |
| Assuming all conductors are ohmic | Only at constant T |
| Forgetting temperature effect | Metals: hotter = higher R |
| Wrong formula | V = IR, not I = VR |
