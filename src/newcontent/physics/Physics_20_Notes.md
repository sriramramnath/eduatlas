## 20.1 Magnetic Fields

### Properties of Magnets

- Magnets have **two poles**: North (N) and South (S)
- Like poles **repel**, opposite poles **attract**
- Magnetic poles always come in pairs (no magnetic monopoles)

### Magnetic Field

A **magnetic field** is a region where magnetic forces act.

### Magnetic Field Patterns

**Bar magnet:**

Field lines go from **North to South**.

**Between two magnets:**
```
    N вҶҗв”Җв”Җв”Җв”ҖвҶ’ S    N вҶҗв”Җв”Җв”Җв”ҖвҶ’ S  (attracting)

```

### Plotting Magnetic Fields

- Use a compass or iron filings
- Field lines show direction a North pole would move
- Closer lines = stronger field

## 20.2 Magnetic Materials

### Ferromagnetic Materials

Materials that can be magnetized:
- **Iron**
- **Nickel**
- **Cobalt**
- **Steel** (alloy of iron and carbon)

### Hard and Soft Magnetic Materials

| Type | Properties | Examples | Uses |
|------|------------|----------|------|
| **Soft** | Easy to magnetize, easy to demagnetize | Iron, soft iron | Electromagnets |
| **Hard** | Hard to magnetize, hard to demagnetize | Steel | Permanent magnets |

### Induced Magnetism

- Unmagnetized magnetic materials can become magnetized when placed in a magnetic field
- This is temporary for soft materials
- This can be permanent for hard materials

## 20.5 Electromagnets

### What is an Electromagnet?

An electromagnet is a coil of wire with a soft iron core that becomes magnetic when current flows.

### Factors Affecting Strength

| Factor | Effect |
|--------|--------|
| **More turns** | Stronger magnet |
| **More current** | Stronger magnet |
| **Soft iron core** | Stronger magnet |

### Advantages Over Permanent Magnets

| Feature | Electromagnet | Permanent Magnet |
|---------|---------------|------------------|
| Can be switched on/off | Yes | No |
| Strength can be varied | Yes | No |
| Pole direction can be changed | Yes | No |

### Uses of Electromagnets

| Use | Application |
|-----|-------------|
| **Scrap yards** | Lifting iron/steel |
| **MRI scanners** | Medical imaging |
| **Relays** | Switches |
| **Electric bells** | Sound production |
| **Circuit breakers** | Safety devices |

## 20.6 The Motor Effect

### Force on a Current-Carrying Wire

A current-carrying wire in a magnetic field experiences a force.

### Fleming's Left-Hand Rule

```
    Thumb = Motion (Force)
    First finger = Field (North to South)
    Second finger = Current
    
    в”Ңв”Җв”Җв”Җв”җ
    в”Ӯ F в”Ӯ Thumb (Force/Motion)
    в”ңв”Җв”Җв”Җв”Ө
    в”Ӯ B в”Ӯ First finger (Field)
    в”ңв”Җв”Җв”Җв”Ө
    в”Ӯ C в”Ӯ Second finger (Current)
    в””в”Җв”Җв”Җв”ҳ
```

### Factors Affecting Force

| Factor | Effect |
|--------|--------|
| **Stronger magnetic field** | Greater force |
| **More current** | Greater force |
| **Longer wire in field** | Greater force |

### Equation

```
F = B Г— I Г— L

Where:
F = force (N)
B = magnetic flux density (T)
I = current (A)
L = length of wire in field (m)
```

```
    Commutator (splits ring)
         в”Ӯ
    N вҶҗв”Җв”Җв”јв”Җв”ҖвҶ’ S

       в•І в”Ӯ в•ұ
        в•Ів”Ӯв•ұ
         в”Ӯ
    Brushes (contact commutator)
```

### How it Works

1. Current flows through coil in magnetic field
2. Force on coil causes rotation (Fleming's left-hand rule)
3. Commutator reverses current every half turn
4. Coil continues rotating in same direction

### Increasing Motor Speed

| Method | How |
|--------|-----|
| More current | Increase voltage |
| Stronger magnetic field | Stronger magnets |
| More turns on coil | Increase coil turns |

## 20.8 Loudspeakers

### How They Work

1. Audio signal (varying current) flows through coil
2. Coil is in magnetic field
3. Force on coil varies with current
4. Coil moves cone
5. Cone vibrates air, producing sound

## Common Student Errors

| Error | Correction |
|-------|------------|
| Confusing left and right hand rules | Left hand = motor effect, Right hand = field direction |
| Thinking copper is magnetic | Only iron, nickel, cobalt are magnetic |
| Forgetting commutator function | Reverses current every half turn |
| Confusing field directions | Field goes N to S, current direction matters |
| Thinking all metals are magnetic | Most metals (aluminum, copper) are not magnetic |
