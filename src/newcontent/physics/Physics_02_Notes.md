## 2.1 What is a Force?

### Definition

A **force** is a push or pull that acts on an object. Forces can:
- Change the shape of an object
- Change the speed of an object
- Change the direction of motion

### Force as a Vector

Force is a **vector quantity** - it has:
- **Magnitude** (size) - measured in newtons (N)
- **Direction** - the direction in which the force acts

### Representing Forces

Forces are represented by **arrows**:
- Length of arrow = magnitude of force
- Direction of arrow = direction of force

## 2.2 Types of Forces

### Contact Forces

| Force | Description | Example |
|-------|-------------|---------|
| **Friction** | Force opposing motion between surfaces | Brakes on a car |
| **Air resistance** | Force opposing motion through air | Parachute slowing fall |
| **Tension** | Pulling force in a string/rope | Rope pulling a box |
| **Normal contact force** | Force perpendicular to a surface | Ground pushing up on your feet |
| **Upthrust** | Upward force on object in fluid | Boat floating on water |

### Non-Contact Forces

| Force | Description | Example |
|-------|-------------|---------|
| **Gravitational force (weight)** | Force due to gravity | Object falling to Earth |
| **Magnetic force** | Force between magnets | Magnet attracting iron |
| **Electrostatic force** | Force between charges | Charged balloon sticking to wall |

## 2.3 Weight and Mass

### Definitions

| Term | Definition | Unit | Type |
|------|------------|------|------|
| **Mass** | Amount of matter in an object | kilograms (kg) | Scalar |
| **Weight** | Force of gravity acting on an object | newtons (N) | Vector |

### The Relationship

```
weight = mass × gravitational field strength

W = m × g

Where:
W = weight (N)
m = mass (kg)
g = gravitational field strength (N/kg)
```

### Value of g

| Location | g (N/kg) |
|----------|----------|
| Earth surface | 10 N/kg (or 9.8 N/kg) |
| Moon surface | 1.6 N/kg |
| Jupiter surface | 25 N/kg |

### Key Differences

| Mass | Weight |
|------|--------|
| Never changes | Changes with location |
| Measured in kg | Measured in N |
| Scalar quantity | Vector quantity |
| Measured with balance | Measured with spring balance/newton meter |

### Common Exam Mistake
Students often confuse mass and weight:
- Mass is constant (same everywhere in universe)
- Weight depends on gravitational field strength
- On the Moon, your mass stays the same but weight is about 1/6 of Earth's

## 2.4 Resultant Forces

### Definition

The **resultant force** is the single force that has the same effect as all the forces acting on an object combined.

### Finding Resultant Forces

**Forces in the same direction:** Add them
```
← 5 N     ← 3 N
Resultant = 5 + 3 = 8 N to the left
```

**Forces in opposite directions:** Subtract them
```
→ 10 N     ← 4 N
Resultant = 10 - 4 = 6 N to the right
```

**Perpendicular forces:** Use Pythagoras
```
    ↑ 4 N
    │
    └──→ 3 N

Resultant = √(3² + 4²) = √25 = 5 N
Direction: tan⁻¹(4/3) = 53° from horizontal
```

### Balanced and Unbalanced Forces

| Situation | Resultant Force | Motion |
|-----------|----------------|--------|
| Balanced forces | Zero | Stationary OR constant velocity |
| Unbalanced forces | Non-zero | Acceleration |

## 2.5 Forces and Elastic Deformation

### Elastic and Inelastic Deformation

| Type | Description | Example |
|------|-------------|---------|
| **Elastic deformation** | Object returns to original shape when force removed | Stretching a spring within limit |
| **Inelastic (plastic) deformation** | Object does not return to original shape | Stretching a spring too far |

### Hooke's Law

For an elastic object, the extension is directly proportional to the force applied (within the limit of proportionality).

```
F = k × e

Where:
F = force applied (N)
k = spring constant (N/m)
e = extension (m)
```

**Extension** = stretched length - original length

### The Limit of Proportionality

```
Force (N)
    │
    │      ╱ (beyond limit - non-linear)
    │     ╱
    │    ╱ (limit of proportionality)
    │   ●
    │  ╱ (Hooke's Law region - linear)
    │ ╱
    │╱
    └──────────────────→ Extension (m)
```

- **Up to limit**: F ∝ e (straight line through origin)
- **Beyond limit**: Spring may be permanently stretched

### Spring Constant (k)

- Measures the **stiffness** of a spring
- Higher k = stiffer spring (harder to stretch)
- Unit: **newtons per metre (N/m)**

### Calculating Spring Constant

```
k = F ÷ e

Example: A 5 N weight extends a spring by 0.1 m
k = 5 ÷ 0.1 = 50 N/m
```

## 2.6 Force-Extension Graphs

### Interpreting the Graph

```
Force (N)
    │
 10 ┤        ╱ (non-linear region)
    │       ╱
  5 ┤      ● (limit of proportionality)
    │     ╱
    │    ╱ (linear region - Hooke's Law)
    │   ╱
  0 ┤──┴──┴──┴──┴──→ Extension (m)
    0   2   4   6
```

| Feature | Interpretation |
|---------|---------------|
| Gradient | Spring constant k (in linear region) |
| Straight line through origin | Hooke's Law obeyed |
| Limit of proportionality | Point where line stops being straight |
| Area under graph | Work done (energy stored) |

### Worked Example

A spring has original length 10 cm. When a 4 N weight is added, it stretches to 14 cm.

(a) Calculate the extension:
```
Extension = 14 cm - 10 cm = 4 cm = 0.04 m
```

(b) Calculate the spring constant:
```
k = F ÷ e = 4 ÷ 0.04 = 100 N/m
```

## 2.7 Energy Stored in a Stretched Spring

### Elastic Potential Energy

The energy stored in a stretched (or compressed) spring is called **elastic potential energy**.

```
E = ½ × F × e

or

E = ½ × k × e²

Where:
E = elastic potential energy (J)
F = force applied (N)
e = extension (m)
k = spring constant (N/m)
```

### Derivation

The energy stored equals the **area under the force-extension graph**:

For a triangle: Area = ½ × base × height = ½ × e × F = ½ × e × (k × e) = ½ke²

### Worked Example

A spring with k = 200 N/m is stretched by 0.05 m. Calculate the energy stored.

```
E = ½ × k × e²
E = ½ × 200 × (0.05)²
E = 100 × 0.0025
E = 0.25 J
```

## 2.8 Friction

### What is Friction?

Friction is a force that opposes motion between two surfaces in contact.

### Types of Friction

| Type | Description | Example |
|------|-------------|---------|
| **Static friction** | Acts on stationary objects | Box that won't move when pushed lightly |
| **Dynamic (kinetic) friction** | Acts on moving objects | Sliding book across table |
| **Rolling friction** | Acts on rolling objects | Car tyres on road |

### Factors Affecting Friction

| Factor | Effect |
|--------|--------|
| **Nature of surfaces** | Rougher surfaces = more friction |
| **Normal force** | Greater force pressing surfaces = more friction |
| **Lubrication** | Reduces friction |

### Useful and Problematic Friction

**Useful friction:**
- Walking (prevents slipping)
- Brakes on vehicles
- Writing with pencil

**Problematic friction:**
- Wear on machine parts
- Energy loss in engines
- Air resistance slowing vehicles

### Reducing Friction

| Method | How it works |
|--------|--------------|
| Lubrication | Oil/grease separates surfaces |
| Ball bearings | Rolling friction < sliding friction |
| Streamlining | Reduces air resistance |
| Smooth surfaces | Less interlocking between surfaces |

## 2.9 Investigating Springs

### Experimental Method

**Aim:** Investigate how force affects spring extension

**Apparatus:**
- Spring
- Metre rule
- Mass hanger and masses
- Clamp stand

**Method:**
1. Measure original length of spring (no load)
2. Add masses in increments (e.g., 1 N at a time)
3. Measure new length each time
4. Calculate extension = new length - original length
5. Record results in table
6. Plot force-extension graph

**Safety:**
- Wear safety goggles (spring may break)
- Place cushion below (masses may fall)

### Expected Results

| Force (N) | Length (cm) | Extension (cm) |
|-----------|-------------|----------------|
| 0 | 10.0 | 0 |
| 1 | 12.0 | 2.0 |
| 2 | 14.0 | 4.0 |
| 3 | 16.0 | 6.0 |
| 4 | 18.0 | 8.0 |

Graph shows straight line through origin (Hooke's Law obeyed).

## Exam Tips for Chapter 2

1. **Always convert to SI units** - metres for extension, kg for mass
2. **Extension vs length** - Extension = stretched length - original length
3. **Weight vs mass** - Weight is a force (N), mass is amount of matter (kg)
4. **Hooke's Law only applies** up to the limit of proportionality
5. **Spring constant** - Higher k means stiffer spring
6. **Resultant forces** - Consider direction (vectors)

---

## Common Student Errors

| Error | Correction |
|-------|------------|
| Using length instead of extension in Hooke's Law | Extension = length - original length |
| Confusing mass and weight | Mass = kg, Weight = N, W = mg |
| Thinking mass changes with location | Mass is constant; weight changes with g |
| Adding forces without considering direction | Forces are vectors - direction matters |
| Using wrong units for spring constant | k is in N/m, not N/cm |
