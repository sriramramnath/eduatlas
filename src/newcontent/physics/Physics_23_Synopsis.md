## Half-Life

### Definition
Time for:
- Half the nuclei to decay, OR
- Activity to halve

### Formula
```
Remaining = Initial × (½)ⁿ
n = time ÷ half-life
```

### Example Half-Lives
| Isotope | Half-Life |
|---------|-----------|
| U-238 | 4.5 billion yrs |
| C-14 | 5730 yrs |
| Co-60 | 5.3 yrs |
| I-131 | 8 days |
| Tc-99m | 6 hrs |

---

## Activity

- **Unit:** Becquerel (Bq)
- **1 Bq** = 1 decay per second
- Activity ∝ number of undecayed nuclei

---

## Random Nature

- **Random** - Cannot predict which nucleus
- **Spontaneous** - Cannot trigger decay
- **Constant probability** - Per unit time

---

## Key Equations

```
Half-life: t½
Number of half-lives: n = t / t½
Remaining: N = N₀ × (½)ⁿ
```
