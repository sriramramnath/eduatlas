## Quick Reference: Scalar vs Vector Quantities

| Scalar (magnitude only) | Vector (magnitude + direction) |
|------------------------|--------------------------------|
| Distance | Displacement |
| Speed | Velocity |
| Mass | Acceleration |
| Energy | Force |
| Time | Momentum |

---

## Essential Equations

```
speed = distance ÷ time          →    v = d/t
velocity = displacement ÷ time   →    v = s/t
acceleration = Δv ÷ Δt           →    a = (v-u)/t
v = u + at
```

## Graph Analysis Summary

### Distance-Time Graphs

| Feature | Interpretation |
|---------|---------------|
| Gradient | **Speed** |
| Steeper gradient | Faster speed |
| Horizontal line | Stationary (speed = 0) |
| Curved line | Changing speed |

### Velocity-Time Graphs

| Feature | Interpretation |
|---------|---------------|
| Gradient | **Acceleration** |
| Area under graph | **Displacement** |
| Horizontal line | Constant velocity (a = 0) |
| Line above time axis | Moving in positive direction |
| Line below time axis | Moving in opposite direction |

## Terminal Velocity

### Sequence of Events (e.g., skydiver)

1. **Start**: Weight > Air resistance → **accelerates**
2. **Speed increases**: Air resistance increases
3. **Eventually**: Weight = Air resistance → **terminal velocity**
4. **Parachute opens**: Large air resistance → rapid deceleration
5. **New terminal velocity**: Lower speed, safe landing

### Factors Affecting Terminal Velocity

- ↑ Mass → ↑ Terminal velocity
- ↑ Surface area → ↓ Terminal velocity
- ↑ Streamlining → ↑ Terminal velocity

## Motion Under Gravity

| Quantity | Value |
|----------|-------|
| g (acceleration due to gravity) | **10 m/s²** (or 9.8 m/s² if specified) |
| Direction | Downwards |
| Free fall (no air resistance) | Constant acceleration |

## Key Exam Points

1. **Distance** = total path length (scalar)
2. **Displacement** = straight-line from start to finish (vector)
3. **Speed** = rate of change of distance (scalar)
4. **Velocity** = rate of change of displacement (vector)
5. **Acceleration** = rate of change of velocity (vector)
6. **Area under v-t graph** = displacement
7. **Gradient of v-t graph** = acceleration
8. **Gradient of d-t graph** = speed

## Common Calculation Types

### Type 1: Finding Speed/Velocity
```
A car travels 150 m in 10 s.
Speed = 150 ÷ 10 = 15 m/s
```

### Type 2: Finding Acceleration
```
A car accelerates from 5 m/s to 25 m/s in 4 s.
a = (25 - 5) ÷ 4 = 20 ÷ 4 = 5 m/s²
```

### Type 3: Using v = u + at
```
A ball is dropped (u = 0) for 3 s.
v = 0 + (10 × 3) = 30 m/s
```

### Type 4: Area Under v-t Graph
```
Triangle: Area = ½ × base × height
Rectangle: Area = base × height
Trapezium: Area = ½ × (a + b) × height
```
