# Chapter 12: Light Waves
## Pearson Edexcel IGCSE Physics - Concise Synopsis

---

## Essential Equations

```
Law of reflection:     i = r
Refractive index:      n = c ÷ v
Critical angle:        sin(c) = 1 ÷ n
```

---

## Reflection

### Law of Reflection
> Angle of incidence = Angle of reflection

### Types
| Type | Surface | Result |
|------|---------|--------|
| **Specular** | Smooth (mirror) | Clear image |
| **Diffuse** | Rough | Scattered light |

### Plane Mirror Images
- Virtual
- Upright
- Same size
- Laterally inverted
- Same distance behind mirror

---

## Refraction

### Definition
Change in direction when light enters a different medium.

### Rules
- **Air → Glass**: Slows down, bends **towards** normal
- **Glass → Air**: Speeds up, bends **away** from normal

### Refractive Index
```
n = c ÷ v
```

| Medium | n (approx) |
|--------|------------|
| Vacuum/Air | 1.0 |
| Water | 1.33 |
| Glass | 1.5 |
| Diamond | 2.42 |

---

## Total Internal Reflection (TIR)

### Conditions
1. Light travels from **denser → less dense**
2. Angle of incidence **> critical angle**

### Critical Angle
```
sin(c) = 1 ÷ n
```

### Applications
- Optical fibres
- Prismatic binoculars
- Diamond sparkle
- Periscopes

---

## Optical Fibres

### Structure
- **Core**: Higher refractive index
- **Cladding**: Lower refractive index
- **Protective coating**

### How They Work
Light undergoes repeated TIR at core-cladding boundary.

### Advantages
- High bandwidth
- Low signal loss
- Lightweight
- No interference
- Secure

## Lenses

### Types

| Type | Shape | Effect |
|------|-------|--------|
| **Converging (convex)** | Thicker in middle | Converges light |
| **Diverging (concave)** | Thinner in middle | Diverges light |

### Converging Lens Ray Rules
1. Parallel → through focal point
2. Through F → parallel
3. Through centre → undeviated

### Image Formation (Converging)

| Object | Image |
|--------|-------|
| Beyond 2F | Real, inverted, smaller |
| At 2F | Real, inverted, same |
| F to 2F | Real, inverted, larger |
| Inside F | Virtual, upright, larger |

### Magnifying Glass
- Object inside F
- Virtual, upright, magnified

---

## Colour

### Dispersion
Prism splits white light into spectrum:
```
Red → Orange → Yellow → Green → Blue → Indigo → Violet
```

- **Red**: Least deviated
- **Violet**: Most deviated

### Primary Colours of Light
- Red, Green, Blue

### Object Colours
- **Red object**: Reflects red, absorbs others
- **White object**: Reflects all
- **Black object**: Absorbs all

## Quick Calculation Guide

### Critical Angle
```
n = 1.5 (glass)
sin(c) = 1/1.5 = 0.667
c = sin⁻¹(0.667) = 41.8°
```

### Refractive Index
```
v = 2 × 10⁸ m/s (in glass)
n = c/v = 3×10⁸ / 2×10⁸ = 1.5
```
